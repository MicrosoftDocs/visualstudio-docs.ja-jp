# <a name="javascript-language-reference"></a>JavaScript Language Reference (JavaScript 言語リファレンス)

JavaScript は、web ページやその他のアプリケーションに埋め込むことができるスクリプト言語です。  
  
Microsoft による JavaScript の実装は、ECMAScript 言語仕様第5版に準拠しています。 これは、ECMA 規格に含まれていない別の機能も提供します。

> [!IMPORTANT]
> Microsoft はコミュニティ全体の取り組みに参加し、 [mdn web ドキュメント](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference)を web のワンストップ (premiere development リファレンス) に参加させました。最初に、Microsoft の JavaScript API リファレンスの500以上のページすべてを[docs.microsoft.com](https://docs.microsoft.com)から mdn に対応するようにし、従来の JS 拡張機能のリファレンスを移植およびサービスの目的で移植しました。
