# <a name="size-property-set-javascript"></a>size プロパティ (Set) (JavaScript)
セット内の要素数を返します。  
  
## <a name="syntax"></a>構文  
  
```JavaScript  
sizeVar = setObj.size  
```  
  
#### <a name="parameters"></a>パラメーター  
 `sizeVar`  
 必須です。 任意の数値です。  
  
 `setObj`  
 必須です。 任意の `Set` オブジェクトを指定します。
