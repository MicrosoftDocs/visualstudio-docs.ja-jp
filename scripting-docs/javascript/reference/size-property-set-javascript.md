# size プロパティ (Set) (JavaScript)
セット内の要素数を返します。  
  
## 構文  
  
```javascript  
sizeVar = setObj.size  
```  
  
#### パラメーター  
 `sizeVar`  
 必須です。  任意の数値です。  
  
 `setObj`  
 必須です。  任意の `Set` オブジェクトを指定します。
