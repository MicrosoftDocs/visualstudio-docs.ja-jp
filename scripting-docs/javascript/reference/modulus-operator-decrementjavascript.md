# <a name="modulus-operator--javascript"></a>剰余演算子 (JavaScript)
数値式の値を別の数値式の値で除算し、残りの部分を生成します。  
  
## <a name="syntax"></a>構文  
  
```  
result = expression1 % expression2  
```  
  
## <a name="arguments"></a>引数  
 `result`  
 任意の変数。  
  
 `expression1`  
 任意の数式。  
  
 `expression2`  
 任意の数式。
