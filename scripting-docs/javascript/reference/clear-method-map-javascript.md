# <a name="clear-method-map-javascript"></a>clear メソッド (Map) (JavaScript)
マップからすべての要素を削除します。  
  
## <a name="syntax"></a>構文  
  
```JavaScript  
mapObj.clear()  
```  
  
#### <a name="parameters"></a>パラメーター  
 `mapObj`  
 必須です。 クリアするマップ。
