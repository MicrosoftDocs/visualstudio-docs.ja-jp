# <a name="tostring-method-map-javascript"></a>toString メソッド (Map) (JavaScript)
マップの文字列表現を返します。  
  
## <a name="syntax"></a>構文  
  
```JavaScript  
mapObj.toString()  
```  
  
#### <a name="parameters"></a>パラメーター  
 `mapObj`  
 必須です。 `Map` オブジェクト。  
  
## <a name="property-valuereturn-value"></a>プロパティ値/戻り値  
 マップの文字列形式。
