# toString メソッド (Map) (JavaScript)
マップの文字列表現を返します。  
  
## 構文  
  
```javascript  
mapObj.toString()  
```  
  
#### パラメーター  
 `mapObj`  
 必須です。  `Map` オブジェクト。  
  
## プロパティ値\/戻り値  
 マップの文字列形式。
