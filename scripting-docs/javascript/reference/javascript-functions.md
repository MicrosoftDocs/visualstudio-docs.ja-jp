|[keys 関数](../../javascript/reference/object-keys-function-javascript.md)|オブジェクトの列挙可能なプロパティおよびメソッドの名前を返します。|`Object`|  
|[log 関数](../../javascript/reference/math-log-function-javascript.md)|指定された数値の自然対数を返します。|`Math`|

|[parse 関数 \(JSON\)](../../javascript/reference/json-parse-function-javascript.md)|JSON のテキストを逆シリアル化して、メモリ内オブジェクトまたは配列を作成します。|`JSON`|  
|[parseFloat 関数](../../javascript/reference/parsefloat-function-javascript.md)|文字列から変換された浮動小数点数を返します。|`Global`|

|[preventExtensions 関数](../../javascript/reference/object-preventextensions-function-javascript.md)|オブジェクトへの新しいプロパティの追加を防止します。|`Object`|  
|[random 関数](../../javascript/reference/math-random-function-javascript.md)|0 ～ 1 の範囲の擬似乱数を返します。|`Math`|  
|[round 関数](../../javascript/reference/math-round-function-javascript.md)|指定された数式の値に最も近い整数を返します。|`Math`|

|[ScriptEngineMajorVersion 関数](../../javascript/reference/scriptenginemajorversion-function-javascript.md)|使用中のスクリプト エンジンのメジャー バージョン番号を返します。|`Global`|  
|[ScriptEngineMinorVersion 関数](../../javascript/reference/scriptengineminorversion-function-javascript.md)|使用中のスクリプト エンジンのマイナー バージョン番号を返します。|`Global`|
