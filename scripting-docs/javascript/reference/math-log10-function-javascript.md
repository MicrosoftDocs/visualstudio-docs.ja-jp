# <a name="mathlog10-function-javascript"></a>Math.log10 関数 (JavaScript)
数値の底 10 の対数を返します。  
  
## <a name="syntax"></a>構文  
  
```  
Math.log10(x)   
```  
  
## <a name="return-value"></a>戻り値  
 `x` が `NaN` の場合、この関数は `NaN` を返します。 `x` がゼロ (0) より小さい場合、この関数は `NaN` を返します。
