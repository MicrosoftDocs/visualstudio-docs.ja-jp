# valueOf メソッド (WeakMap) (JavaScript)
指定されたオブジェクトのプリミティブ値を返します。  
  
## 構文  
  
```javascript  
weakmapObj.valueOf()  
```  
  
#### パラメーター  
 `weakmapObj`  
 必須です。  `WeakMap` オブジェクト。  
  
## プロパティ値\/戻り値  
 `WeakMap` インスタンスを戻します。
