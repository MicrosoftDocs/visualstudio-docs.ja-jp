# <a name="valueof-method-weakmap-javascript"></a>valueOf メソッド (WeakMap) (JavaScript)
指定されたオブジェクトのプリミティブ値を返します。  
  
## <a name="syntax"></a>構文  
  
```JavaScript  
weakmapObj.valueOf()  
```  
  
#### <a name="parameters"></a>パラメーター  
 `weakmapObj`  
 必須です。 `WeakMap` オブジェクト。  
  
## <a name="property-valuereturn-value"></a>プロパティ値/戻り値  
 返します、`WeakMap`インスタンス。
