# <a name="mathlog1p-function-javascript"></a>Math.log1p 関数 (JavaScript)
1 + 数値の自然対数を返します。  
  
## <a name="syntax"></a>構文  
  
```  
Math.log1p(x)   
```  
  
## <a name="return-value"></a>戻り値  
 `x` が `NaN` の場合、この関数は `NaN` を返します。 `x` がゼロ (0) より小さい場合、この関数は `NaN` を返します。
