# <a name="mathexp-function-javascript"></a>Math.exp 関数 (JavaScript)
E (自然対数の底) を累乗したを返します。  
  
## <a name="syntax"></a>構文  
  
```  
  
Math.exp(number)   
```  
  
## <a name="remarks"></a>コメント  
 必要な`number`引数が数値の累乗を表す式*e*です。  
  
 戻り値は数値です。 定数*e*オイラーの番号、2.71828 とほぼ等しくはおよび`number`は、指定された引数。  
  
 **適用されます**: [Math オブジェクト](../../javascript/reference/math-object-javascript.md)
