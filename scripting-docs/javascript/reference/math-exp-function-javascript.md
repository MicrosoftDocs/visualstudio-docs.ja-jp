# Math.exp 関数 (JavaScript)
e \(自然対数の底\) の累乗を返します。  
  
## 構文  
  
```  
  
Math.exp(number)   
```  
  
## 解説  
 必須の `number` 引数は、*e* の累乗を表す数式です。  
  
 戻り値は数値です。  定数 *e* はオイラー数 \(約 2.71828\) です。`number` は引数として渡された値です。  
  
 **対象**: [Math オブジェクト](../../javascript/reference/math-object-javascript.md)
