# JavaScript の将来の予約語
JavaScript では、現在のバージョンで特別な意味を持たない場合でも、将来の予約済みキーワードを識別子として使用しないでください。  
  
 JavaScript の予約語の一覧については、「[JavaScript の予約語](../../javascript/reference/javascript-reserved-words.md)」を参照してください。  
  
## 将来的に使用される予約語  
 将来の予約語を次に示します。  
  
1.  class  
  
2.  enum  
  
3.  export  
  
4.  extends  
  
5.  import  
  
6.  super  
  
## 厳格モードでの将来の予約語  
 厳格モードのみでの将来の予約語を次に示します。  厳格モードの詳細については、「[use strict ディレクティブ](../../javascript/reference/use-strict-directive.md)」を参照してください。  
  
1.  implements  
  
2.  interface  
  
3.  package  
  
4.  private  
  
5.  protected  
  
6.  public  
  
7.  static  
  
8.  yield
