# <a name="javascript-future-reserved-words"></a>JavaScript の将来の予約語
JavaScript では、現在のバージョンで特別な意味を持たない場合でも、将来の予約済みキーワードを識別子として使用しないでください。  
  
 JavaScript の予約語の一覧は、次を参照してください。 [JavaScript の予約語](../../javascript/reference/javascript-reserved-words.md)です。  
  
## <a name="future-reserved-words"></a>将来的に使用される予約語  
 将来の予約語を次に示します。  
  
1.  クラス  
  
2.  enum  
  
3.  export  
  
4.  extends  
  
5.  import  
  
6.  super  
  
## <a name="future-reserved-words-in-strict-mode"></a>厳格モードでの将来の予約語  
 厳格モードのみでの将来の予約語を次に示します。 厳格モードの詳細については、次を参照してください。 [use strict ディレクティブ](../../javascript/reference/use-strict-directive.md)です。  
  
1.  実装  
  
2.  interface  
  
3.  package  
  
4.  private  
  
5.  protected  
  
6.  public  
  
7.  static  
  
8.  yield
