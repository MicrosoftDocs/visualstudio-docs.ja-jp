# <a name="mathsinh-function-javascript"></a>Math.sinh 関数 (JavaScript)
数値の逆ハイパーボリック サインを返します。  
  
## <a name="syntax"></a>構文  
  
```  
  
Math.sinh(x)   
```  
  
## <a name="remarks"></a>コメント  
 戻り値は `(exp(x) - exp(-x))/2` と同じです。 x が `NaN` の場合、結果は `NaN` です。
