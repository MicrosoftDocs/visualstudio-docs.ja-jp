# clear メソッド (WeakMap) (JavaScript)
`WeakMap` オブジェクトからすべての要素を削除します。  
  
## 構文  
  
```javascript  
weakmapObj.clear()  
```  
  
#### パラメーター  
 `weakmapObj`  
 必須です。  クリアする `WeakMap` オブジェクト。
