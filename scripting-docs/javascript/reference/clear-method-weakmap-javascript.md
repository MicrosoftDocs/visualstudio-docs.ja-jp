# <a name="clear-method-weakmap-javascript"></a>clear メソッド (WeakMap) (JavaScript)

          `WeakMap` オブジェクトからすべての要素を削除します。  
  
## <a name="syntax"></a>構文  
  
```JavaScript  
weakmapObj.clear()  
```  
  
#### <a name="parameters"></a>パラメーター  
 `weakmapObj`  
 必須です。 クリアする `WeakMap` オブジェクト。
