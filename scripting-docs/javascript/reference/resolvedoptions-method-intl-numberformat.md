# <a name="resolvedoptions-method-intlnumberformat"></a>resolvedOptions メソッド (Intl.NumberFormat)
`NumberFormat` フォーマッタ オブジェクトのプロパティと値を含むオブジェクトを返します。  
  
## <a name="syntax"></a>構文  
  
```  
numberFormatObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>パラメーター  
 `numberFormatObj`  
 情報の取得元の `NumberFormat` オブジェクト。  
  
## <a name="remarks"></a>コメント  
 返されたオブジェクトのプロパティは、`NumberFormat` オブジェクトの計算されたプロパティに対応します。
