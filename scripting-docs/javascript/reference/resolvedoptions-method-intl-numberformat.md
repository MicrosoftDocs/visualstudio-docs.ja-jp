# resolvedOptions メソッド (Intl.NumberFormat)
`NumberFormat` フォーマッタ オブジェクトのプロパティと値を含むオブジェクトを返します。  
  
## 構文  
  
```  
numberFormatObj.resolvedOptions()  
```  
  
#### パラメーター  
 `numberFormatObj`  
 情報の取得元の `NumberFormat` オブジェクト。  
  
## 解説  
 返されたオブジェクトのプロパティは、`NumberFormat` オブジェクトの計算されたプロパティに対応します。
