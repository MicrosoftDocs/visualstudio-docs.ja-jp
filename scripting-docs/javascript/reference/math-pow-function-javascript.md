# <a name="mathpow-function-javascript"></a>Math.pow 関数 (JavaScript)
指定した電力にかかった基本式の値を返します。  
  
## <a name="syntax"></a>構文  
  
```  
  
Math.pow(  
base, exponent)   
```  
  
## <a name="parameters"></a>パラメーター  
 *base*  
 必須です。 式のベース値。  
  
 *指数*  
 必須です。 式の指数の値。  
  
## <a name="example"></a>例  
 次の例では、baseexponent に等しい数値式は、1000 を返します。  
  
```JavaScript  
Math.pow(10,3);  
```
