# Math.pow 関数 (JavaScript)
exponent 引数を指数とする base 引数の累乗を返します。  
  
## 構文  
  
```  
  
Math.pow(  
base, exponent)   
```  
  
## パラメーター  
 *base*  
 必須です。  累乗の基数とする値を数式で指定します。  
  
 *exponent*  
 必須です。  累乗の指数部とする値を数式で指定します。  
  
## 使用例  
 次の例では、baseexponent に等しい数式が 1000 を返します。  
  
```javascript  
Math.pow(10,3);  
```
