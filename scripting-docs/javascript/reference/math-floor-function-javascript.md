# Math.floor 関数 (JavaScript)
指定された数値引数以下の最大の整数を返します。  
  
## 構文  
  
```  
  
Math.floor(number)   
```  
  
## 解説  
 *number* 引数は必須で、数式を指定します。  
  
 戻り値は、数値引数以下の値となる最大の整数値です。  
  
 **対象**: [Math オブジェクト](../../javascript/reference/math-object-javascript.md)
