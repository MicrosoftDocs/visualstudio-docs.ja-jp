# <a name="mathfloor-function-javascript"></a>Math.floor 関数 (JavaScript)
指定された数値以下の最大の整数を返します。  
  
## <a name="syntax"></a>構文  
  
```  
  
Math.floor(number)   
```  
  
## <a name="remarks"></a>コメント  
 必要な*数*引数は、数値式です。  
  
 戻り値は、その数値の引数と等しいかそれよりも小さいを最大の整数に等しい整数値には。  
  
 **適用されます**: [Math オブジェクト](../../javascript/reference/math-object-javascript.md)
