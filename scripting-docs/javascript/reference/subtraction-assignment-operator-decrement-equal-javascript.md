# <a name="subtraction-assignment-operator---javascript"></a>減算代入演算子 (-=) (JavaScript)
変数の値から式の値を減算し、その結果を変数に代入します。  
  
## <a name="syntax"></a>構文  
  
```  
  
result -= expression  
```  
  
## <a name="parameters"></a>パラメーター  
 *結果*  
 任意の数値変数です。  
  
 *式*  
 任意の数式。  
  
## <a name="remarks"></a>コメント  
 使用して、`-=`演算子は、以下の手順と同じでは正確にします。  
  
```JavaScript  
result = result - expression  
```
