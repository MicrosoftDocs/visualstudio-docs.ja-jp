# <a name="debugdebuggerenabled-property"></a>Debug.debuggerEnabled プロパティ
スクリプトのコンテキストのデバッグが有効になっているかどうかを判断します。 デバッグが有効または無効、デバッガーがアタッチされているかどうか。  
  
## <a name="syntax"></a>構文  
  
```  
  
var dbgEnabled = Debug.debuggerEnabled;  
```
