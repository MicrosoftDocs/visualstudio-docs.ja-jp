# Debug.debuggerEnabled プロパティ
スクリプトのコンテキストでデバッグを有効にするかどうかを指定します。  デバッグは、デバッガーがアタッチされているかどうかによって有効または無効になります。  
  
## 構文  
  
```  
  
var dbgEnabled = Debug.debuggerEnabled;  
```
