# Infinity 定数 (JavaScript)
最大の浮動小数点数を超える数値。  `Number.POSITIVE_INFINITY` の初期値を返します。  Negative Infinity \(\-Infinity\) is smaller than the smallest floating point number.  
  
## 構文  
  
```  
Infinity   
```  
  
## 解説  
 `Infinity` 定数は `Global` オブジェクトのメンバーで、スクリプト エンジンが初期化されたときに使用できるようになります。  
  
## 必要条件  
 `Infinity` プロパティは [!INCLUDE[jsv3text](../../javascript/reference/includes/jsv3text-md.md)] で導入され、[!INCLUDE[jsv9textspecific](../../javascript/reference/includes/jsv9textspecific-md.md)] で読み取り専用になりました。  
  
 **対象**: [Global オブジェクト](../../javascript/reference/global-object-javascript.md)
