# <a name="clear-method-set-javascript"></a>clear メソッド (Set) (JavaScript)
セットからすべての要素を削除します。  
  
## <a name="syntax"></a>構文  
  
```JavaScript  
setObj.clear()  
```  
  
#### <a name="parameters"></a>パラメーター  
 `setObj`  
 必須です。 クリアするセット。
