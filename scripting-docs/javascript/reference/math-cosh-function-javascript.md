# <a name="mathcosh-function-javascript"></a>Math.cosh 関数 (JavaScript)
数値のハイパーボリック コサインを返します。  
  
## <a name="syntax"></a>構文  
  
```  
  
Math.cosh(x)   
```  
  
## <a name="remarks"></a>コメント  
 戻り値は `(exp(x) + exp(-x))/2` と同じです。 `x` が `NaN` の場合、結果は `NaN`です。  
  
 **適用されます**: [Math オブジェクト](../../javascript/reference/math-object-javascript.md)
