# Math.cosh 関数 (JavaScript)
数値のハイパーボリック コサインを返します。  
  
## 構文  
  
```  
  
Math.cosh(x)   
```  
  
## 解説  
 戻り値は `(exp(x) + exp(-x))/2` と同じです。  `x` が `NaN` の場合、結果は `NaN`です。
