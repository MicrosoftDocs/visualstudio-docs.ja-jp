# <a name="valueof-method-map-javascript"></a>valueOf メソッド (Map) (JavaScript)
指定されたオブジェクトのプリミティブ値を返します。  
  
## <a name="syntax"></a>構文  
  
```JavaScript  
mapObj.valueOf()  
```  
  
#### <a name="parameters"></a>パラメーター  
 `mapObj`  
 必須です。 `Map` オブジェクト。  
  
## <a name="property-valuereturn-value"></a>プロパティ値/戻り値  
 マップ インスタンスを戻します。
