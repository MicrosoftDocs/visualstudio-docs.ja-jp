# valueOf メソッド (Map) (JavaScript)
指定されたオブジェクトのプリミティブ値を返します。  
  
## 構文  
  
```javascript  
mapObj.valueOf()  
```  
  
#### パラメーター  
 `mapObj`  
 必須です。  `Map` オブジェクト。  
  
## プロパティ値\/戻り値  
 マップ インスタンスを戻します。
