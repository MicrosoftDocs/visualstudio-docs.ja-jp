# <a name="mathtanh-function-javascript"></a>Math.tanh 関数 (JavaScript)
数値のハイパーボリック タンジェントを返します。  
  
## <a name="syntax"></a>構文  
  
```  
  
Math.tanh(  
x  
)  
  
```  
  
## <a name="remarks"></a>コメント  
 戻り値は `(exp(x) - exp(-x))/(exp(x) + exp(-x))` と同じです。 `x` が `NaN` の場合、結果は `NaN`です。
