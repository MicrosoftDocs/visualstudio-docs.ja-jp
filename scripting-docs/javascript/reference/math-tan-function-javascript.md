# <a name="mathtan-function-javascript"></a>Math.tan 関数 (JavaScript)
タンジェント (正接) を返します。  
  
## <a name="syntax"></a>構文  
  
```  
  
Math.tan(  
radians  
)  
  
```  
  
## <a name="remarks"></a>コメント  
 必要な`radians`引数はラジアンで表した角度を表す数値式です。  
  
 戻り値は数値の引数の正接`radians`です。
