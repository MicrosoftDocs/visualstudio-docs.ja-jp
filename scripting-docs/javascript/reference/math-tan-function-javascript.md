# Math.tan 関数 (JavaScript)
指定された数値のタンジェントの値を返します。  
  
## 構文  
  
```  
  
Math.tan(  
radians  
)  
  
```  
  
## 解説  
 必須の `radians` 引数は、ラジアン単位の角度を含む数式です。  
  
 戻り値は、`radians` の数値引数のタンジェントです。
