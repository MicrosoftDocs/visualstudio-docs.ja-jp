# JavaScript プロパティ
[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] のプロパティの一覧を次の表に示します。  
  
## プロパティ  
  
|プロパティ|説明|[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] オブジェクト|  
|-----------|--------|-----------------------------------------------------------------------|  
|[\_\_proto\_\_ プロパティ](../../javascript/reference/proto-property-object-javascript.md)|オブジェクトのプロトタイプを設定します。|`Object`|  
|[0...n プロパティ](../../javascript/reference/0-dot-dot-dot-n-properties-arguments-javascript.md)|**arguments** オブジェクトから個々の引数の値を返します。|`arguments`|  
|[$1...$9 プロパティ](../../javascript/reference/dollar-1-dot-dot-dot-dollar-9-properties-regexp-javascript.md)|パターン一致で検出された、記憶されている最新の 9 つの部分を返します。|`RegExp`|  
|[arguments プロパティ](../../javascript/reference/arguments-property-function-javascript.md)|現在実行中の関数に渡された各引数を含む配列を返します。|`Function`|  
|[callee プロパティ](../../javascript/reference/callee-property-arguments-javascript.md)|実行されている `Function` オブジェクトを返します。|`arguments`|  
|[caller プロパティ](../../javascript/reference/caller-property-function-javascript.md)|現在の関数を呼び出した関数への参照を返します。|`Function`|  
|[constructor プロパティ](../../javascript/reference/constructor-property-object-javascript.md)|オブジェクトを作成する関数を指定します。|複数|  
|[description プロパティ](../../javascript/reference/description-property-error-javascript.md)|特定のエラーと関連付けられたエラーを説明する文字列を設定するか、または返します。|`Error`|  
|[global プロパティ](../../javascript/reference/global-property-regular-expression-javascript.md)|正規表現で使用する global フラグ \(**g**\) の状態を表すブール値を返します。|`Regular Expression`|  
|[ignoreCase プロパティ](../../javascript/reference/ignorecase-property-regular-expression-javascript.md)|正規表現で使用する ignoreCase フラグ \(**i**\) の状態を表すブール値を返します。|`Regular Expression`|  
|[index プロパティ](../../javascript/reference/index-property-regexp-javascript.md)|検索文字列と一致する最初の文字について、対象文字列内の先頭からの位置を返します。|`RegExp`|  
|[input プロパティ](../../javascript/reference/input-property-dollar-regexp-javascript.md)|検索の対象となった文字列を返します。|`RegExp`|  
|[lastIndex プロパティ](../../javascript/reference/lastindex-property-regexp-javascript.md)|検索文字列と一致する最後の文字について、対象文字列内の先頭からの位置を返します。|`RegExp`|  
|[lastMatch プロパティ](../../javascript/reference/lastmatch-property-dollar-regexp-javascript.md)|正規表現による検索で最後に一致した文字を返します。|`RegExp`|  
|[lastParen プロパティ](../../javascript/reference/lastparen-property-dollar-regexp-javascript.md)|正規表現による検索で、最後にパターン化されたサブマッチがある場合に、それを返します。|`RegExp`|  
|[leftContext プロパティ](../../javascript/reference/leftcontext-property-dollar-grave-regexp-javascript.md)|検索した文字列の先頭から、最後に一致した先頭の前までの文字を返します。|`RegExp`|  
|[length プロパティ \(arguments\)](../../javascript/reference/length-property-arguments-javascript.md)|関数を呼び出すときに実際に渡された引数の数を返します。|`arguments`|  
|[length プロパティ \(Array\)](../../javascript/reference/length-property-array-javascript.md)|配列内で定義されている最後の要素のインデックスより 1 だけ大きい整数値を示します。|`Array`|  
|[length プロパティ \(Function\)](../../javascript/reference/length-property-function-javascript.md)|関数に定義されている引数の数を返します。|`Function`|  
|[length プロパティ \(String\)](../../javascript/reference/length-property-string-javascript.md)|`String` オブジェクトの長さを返します。|`String`|  
|[message プロパティ](../../javascript/reference/message-property-error-javascript.md)|エラー メッセージの文字列を返します。|`Error`|  
|[multiline プロパティ](../../javascript/reference/multiline-property-regular-expression-javascript.md)|正規表現で使用する multiline フラグ \(**m**\) の状態を表すブール値を返します。|`Regular Expression`|  
|[name プロパティ](../../javascript/reference/name-property-error-javascript.md)|エラー名を返します。|`Error`|  
|[number プロパティ](../../javascript/reference/number-property-error-javascript.md)|特定のエラーと関連付けられた数値を設定したり、この数値を取得したりします。|`Error`|  
|[prototype プロパティ](../../javascript/reference/prototype-property-object-javascript.md)|指定されたオブジェクトのクラスのプロトタイプへの参照を返します。|複数|  
|[rightContext プロパティ](../../javascript/reference/rightcontext-property-dollar-regexp-javascript.md)|検索した文字列内で、最後に一致した位置から最後までの文字を返します。|`RegExp`|  
|[source プロパティ](../../javascript/reference/source-property-regular-expression-javascript.md)|正規表現パターンのテキストを返します。|`Regular Expression`|
