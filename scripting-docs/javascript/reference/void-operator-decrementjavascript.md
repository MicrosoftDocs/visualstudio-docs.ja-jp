# void 演算子 (JavaScript)
式が値を返すことができないようにします。  
  
## 構文  
  
```  
void expression   
```  
  
## 解説  
 `expression` 引数は、任意の有効な [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] の式です。  
  
 `void` 演算子は、式を評価して `undefined` を返します。  式の評価が必要であっても、スクリプトの他の部分で結果を参照する必要がない場合に使用すると便利です。
