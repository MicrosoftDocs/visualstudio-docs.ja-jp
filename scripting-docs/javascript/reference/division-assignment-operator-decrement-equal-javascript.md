# <a name="division-assignment-operator---javascript"></a>除算代入演算子 (-) (JavaScript)
変数の値を式で指定された値で除算し、その結果を変数に代入します。  
  
## <a name="syntax"></a>構文  
  
```  
  
result /= expression   
```  
  
## <a name="parameters"></a>パラメーター  
 `result`  
 任意の数値変数です。  
  
 `expression`  
 任意の数式。  
  
## <a name="remarks"></a>コメント  
 使用して、`/=`演算子を指定することと同じ:`result = result / expression`です。
