# 除算代入演算子 (/=) (JavaScript)
変数の値を式で指定された値で除算し、その結果を変数に代入します。  
  
## 構文  
  
```  
  
result /= expression   
```  
  
## パラメーター  
 `result`  
 任意の数値変数を指定します。  
  
 `expression`  
 任意の数式を指定します。  
  
## 解説  
 `/=` 演算子を使用することは、`result = result / expression` を指定することと同じです。
