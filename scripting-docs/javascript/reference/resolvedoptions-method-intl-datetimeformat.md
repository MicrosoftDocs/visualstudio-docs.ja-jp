# resolvedOptions メソッド (Intl.DateTimeFormat)
日時フォーマッタ オブジェクトのプロパティと値を含むオブジェクトを返します。  
  
## 構文  
  
```  
dateTimeFormatObj.resolvedOptions()  
```  
  
#### パラメーター  
 `dateTimeFormatObj`  
 情報の取得元の `DateTimeFormat` オブジェクト。  
  
## 解説  
 返されたオブジェクトのプロパティは、`DateTimeFormat` オブジェクトの計算されたプロパティに対応します。
