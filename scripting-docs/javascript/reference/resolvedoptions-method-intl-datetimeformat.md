# <a name="resolvedoptions-method-intldatetimeformat"></a>resolvedOptions メソッド (Intl.DateTimeFormat)
日時フォーマッタ オブジェクトのプロパティと値を含むオブジェクトを返します。  
  
## <a name="syntax"></a>構文  
  
```  
dateTimeFormatObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>パラメーター  
 `dateTimeFormatObj`  
 情報の取得元の `DateTimeFormat` オブジェクト。  
  
## <a name="remarks"></a>コメント  
 返されたオブジェクトのプロパティは、`DateTimeFormat` オブジェクトの計算されたプロパティに対応します。
