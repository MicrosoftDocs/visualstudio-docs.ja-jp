# <a name="mathasin-function-javascript"></a>Math.asin 関数 (JavaScript)
アークサイン (逆正弦) を返します。  
  
## <a name="syntax"></a>構文  
  
```  
  
Math.asin(number)   
```  
  
## <a name="remarks"></a>コメント  
 必要な`number`引数は、アークサインを必要とする数値式です。  
  
 戻り値のアークサインは、`number`ラジアン単位での引数。  
  
 **適用されます**: [Math オブジェクト](../../javascript/reference/math-object-javascript.md)
