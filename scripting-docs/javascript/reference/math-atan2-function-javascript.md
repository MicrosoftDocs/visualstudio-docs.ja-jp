# Math.atan2 関数 (JavaScript)
X 軸から座標 \(*y*,*x*\) までのラジアン単位の角度を返します。  
  
## 構文  
  
```  
  
Math.atan2(y, x)  
```  
  
## パラメーター  
 *x*  
 必須です。  デカルト座標系での x 座標を表す数式を指定します。  
  
 *y*  
 必須です。  デカルト座標系での y 座標を表す数式を指定します。  
  
## 解説  
 戻り値は、\-πとπの間です。  これは、指定した座標 \(*y*,*x*\) の角度をラジアンで表します。
