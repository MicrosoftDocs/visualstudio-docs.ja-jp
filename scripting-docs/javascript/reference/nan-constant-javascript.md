# NaN 定数 (JavaScript)
thatNot 式を指定する特別な定数は非数値です。  
  
## 構文  
  
```  
NaN   
```  
  
## 解説  
 `NaN` 定数 \(非数値\) は `Global` オブジェクトのメンバーで、スクリプト エンジンを初期化する際に使用可能になります。  
  
## 必要条件  
 `NaN` プロパティは [!INCLUDE[jsv3text](../../javascript/reference/includes/jsv3text-md.md)] で導入され、[!INCLUDE[jsv9textspecific](../../javascript/reference/includes/jsv9textspecific-md.md)] で読み取り専用になりました。  
  
 **Applies To**: [Global オブジェクト](../../javascript/reference/global-object-javascript.md)
