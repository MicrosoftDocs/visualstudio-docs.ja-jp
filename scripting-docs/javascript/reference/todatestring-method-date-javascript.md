# <a name="todatestring-method-date-javascript"></a>toDateString メソッド (Date) (JavaScript)
日付を文字列値として返します。  
  
## <a name="syntax"></a>構文  
  
```  
  
objDate.toDateString( )  
```  
  
## <a name="remarks"></a>コメント  
 必要な*objDate*参照は、`Date`オブジェクト。  
  
 `toDateString`形式を簡単に読み取るメソッドが、便利で、現在のタイム ゾーンでの日付を含む文字列値を返します。
