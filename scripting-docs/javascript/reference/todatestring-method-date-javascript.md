# toDateString メソッド (Date) (JavaScript)
日付を文字列値として返します。  
  
## 構文  
  
```  
  
objDate.toDateString( )  
```  
  
## 解説  
 *objDate* 参照は必須で、`Date` オブジェクトを指定します。  
  
 `toDateString` メソッドは、現在のタイムゾーンの日付を便利で読みやすい形式の文字列値にして返します。
