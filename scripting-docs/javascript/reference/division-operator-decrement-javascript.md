# <a name="division-operator---javascript"></a>除算演算子 (-) (JavaScript)
2 つの式の値を除算します。  
  
## <a name="syntax"></a>構文  
  
```  
  
result = number1 / number2   
```  
  
## <a name="parameters"></a>パラメーター  
 `result`  
 任意の数値変数です。  
  
 `number1`  
 任意の数式。  
  
 `number2`  
 任意の数式。
