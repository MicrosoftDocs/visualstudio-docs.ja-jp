# <a name="mathexpm1-function-javascript"></a>Math.expm1 関数 (JavaScript)
e (自然対数の底) を指数で累乗した値から 1 を減算した結果を返します。  
  
## <a name="syntax"></a>構文  
  
```  
  
Math.expm1(number)   
```  
  
## <a name="remarks"></a>コメント  
 必要な `number` の引数は、e の指数を数値で表したものです。  
  
 戻り値は数値です。 定数 e は 2.71828 とほぼ等しいオイラー数で、`number` は指定された引数です。  
  
 **適用されます**: [Math オブジェクト](../../javascript/reference/math-object-javascript.md)
