# <a name="mathcbrt-function-javascript"></a>Math.cbrt 関数 (JavaScript)
数値の立方根を返します。  
  
## <a name="syntax"></a>構文  
  
```  
  
Math.cbrt(  
number  
)   
```  
  
## <a name="remarks"></a>コメント  
 `number` が `NaN` の場合、結果は `NaN` です。
