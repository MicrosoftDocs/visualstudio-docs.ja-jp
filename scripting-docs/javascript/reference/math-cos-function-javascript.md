# <a name="mathcos-function-javascript"></a>Math.cos 関数 (JavaScript)
コサイン (余弦) を返します。  
  
## <a name="syntax"></a>構文  
  
```  
  
Math.cos(radians)   
```  
  
## <a name="remarks"></a>コメント  
 必要な`radians`引数はラジアンで表した角度を表す数値式です。  
  
 戻り値は数値の引数の余弦`radians`です。  
  
 **適用されます**: [Math オブジェクト](../../javascript/reference/math-object-javascript.md)
