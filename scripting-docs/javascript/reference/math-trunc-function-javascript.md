# <a name="mathtrunc-function-javascript"></a>Math.trunc 関数 (JavaScript)
数値の小数部の桁を除去して、整数部分を返します。  
  
## <a name="syntax"></a>構文  
  
```vb  
Math.trunc(number)  
```  
  
## <a name="remarks"></a>コメント  
 必須の `number` 引数は、小数部の切り捨てられた整数を必要とする数値式です。  
  
 `number` が `NaN` の場合、`NaN` を返します。
