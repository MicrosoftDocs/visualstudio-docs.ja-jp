# valueOf メソッド (Set) (JavaScript)
指定されたオブジェクトのプリミティブ値を返します。  
  
## 構文  
  
```javascript  
setObj.valueOf()  
```  
  
#### パラメーター  
 `setObj`  
 必須です。  `Set` オブジェクト。  
  
## プロパティ値\/戻り値  
 セット インスタンスを戻します。
