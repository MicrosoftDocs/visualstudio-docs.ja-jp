# <a name="valueof-method-set-javascript"></a>valueOf メソッド (Set) (JavaScript)
指定されたオブジェクトのプリミティブ値を返します。  
  
## <a name="syntax"></a>構文  
  
```JavaScript  
setObj.valueOf()  
```  
  
#### <a name="parameters"></a>パラメーター  
 `setObj`  
 必須です。 `Set` オブジェクト。  
  
## <a name="property-valuereturn-value"></a>プロパティ値/戻り値  
 セット インスタンスを戻します。
