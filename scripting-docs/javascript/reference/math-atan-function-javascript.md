# Math.atan 関数 (JavaScript)
指定された数値のアークタンジェントを返します。  
  
## 構文  
  
```  
  
Math.atan(number)   
```  
  
## 解説  
 `number` 引数は必須で、アークタンジェントを必要とする数式を指定します。  
  
 戻り値は、`number` 引数のアークタンジェント \(ラジアン\) です。  
  
 **対象**: [Math オブジェクト](../../javascript/reference/math-object-javascript.md)
