# <a name="mathatan-function-javascript"></a>Math.atan 関数 (JavaScript)
アークタンジェント (逆正接) を返します。  
  
## <a name="syntax"></a>構文  
  
```  
  
Math.atan(number)   
```  
  
## <a name="remarks"></a>コメント  
 必要な`number`引数は、アーク タンジェントの値を必要とする数値式です。  
  
 戻り値は、のアーク タンジェント、`number`ラジアン単位での引数。  
  
 **適用されます**: [Math オブジェクト](../../javascript/reference/math-object-javascript.md)
