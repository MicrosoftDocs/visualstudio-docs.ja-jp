# resolvedOptions メソッド (Intl.Collator)
Collator オブジェクトのプロパティと値を含むオブジェクトを返します。  
  
## 構文  
  
```  
collatorObj.resolvedOptions()  
```  
  
#### パラメーター  
 `collatorObj`  
 情報の取得元の `Collator` オブジェクト。  
  
## 解説  
 返されたオブジェクトのプロパティは、`Collator` オブジェクトの計算されたプロパティに対応します。
