# <a name="resolvedoptions-method-intlcollator"></a>resolvedOptions メソッド (Intl.Collator)
Collator オブジェクトのプロパティと値を含むオブジェクトを返します。  
  
## <a name="syntax"></a>構文  
  
```  
collatorObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>パラメーター  
 `collatorObj`  
 情報の取得元の `Collator` オブジェクト。  
  
## <a name="remarks"></a>コメント  
 返されたオブジェクトのプロパティは、`Collator` オブジェクトの計算されたプロパティに対応します。
