# <a name="global-object-javascript"></a>Global オブジェクト (JavaScript)
グローバル関数と定数を 1 つのオブジェクトに収集するのには、目的の組み込みオブジェクトです。  
  
## <a name="remarks"></a>コメント  
 `Global`オブジェクトは、直接は使用されていませんしを使用して作成することはできません、`new`演算子。 したがって、関数と定数使用できるようにすぐに、スクリプト エンジンが初期化されたときに作成されます。
