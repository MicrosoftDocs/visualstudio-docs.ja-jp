# 左シフト代入演算子 (=) (JavaScript)
値を変数に代入します。  
  
## 構文  
  
```  
  
result = expression  
```  
  
## パラメーター  
 `result`  
 任意の変数を指定します。  
  
 `expression`  
 任意の数式を指定します。  
  
## 解説  
 \= 演算子は他の演算子と同様に動作するため、それを含む式は値を持ちます。  このため、代入演算子で次のような使い方をすることもできます。`j = k = l = 0` この場合、`j`、`k`、`l` は 0 です。
