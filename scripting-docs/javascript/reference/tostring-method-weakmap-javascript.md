# <a name="tostring-method-weakmap-javascript"></a>toString メソッド (WeakMap) (JavaScript)
`WeakMap` オブジェクトの文字列表現を戻します。  
  
## <a name="syntax"></a>構文  
  
```JavaScript  
weakmapObj.toString()  
```  
  
#### <a name="parameters"></a>パラメーター  
 `weakmapObj`  
 必須です。 `WeakMap` オブジェクト。  
  
## <a name="property-valuereturn-value"></a>プロパティ値/戻り値  
 `WeakMap` の文字列形式。
