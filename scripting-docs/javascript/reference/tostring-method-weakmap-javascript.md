# toString メソッド (WeakMap) (JavaScript)
`WeakMap` オブジェクトの文字列表現を戻します。  
  
## 構文  
  
```javascript  
weakmapObj.toString()  
```  
  
#### パラメーター  
 `weakmapObj`  
 必須です。  `WeakMap` オブジェクト。  
  
## プロパティ値\/戻り値  
 `WeakMap` の文字列形式。
