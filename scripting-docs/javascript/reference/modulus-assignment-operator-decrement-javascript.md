# <a name="modulus-assignment-operator--javascript"></a>剰余代入演算子 (JavaScript)
変数の値を式で指定された値で除算し、その剰余を変数に代入します。  
  
## <a name="syntax"></a>構文  
  
```  
  
result %= expression  
```  
  
## <a name="arguments"></a>引数  
 `result`  
 任意の変数。  
  
 `expression`  
 任意の数式。  
  
## <a name="remarks"></a>コメント  
 `%=` 演算子は、次の記述とまったく同じです。  
  
```  
result = result % expression  
```
