# 剰余代入演算子 (JavaScript)
変数の値を式で指定された値で除算し、その剰余を変数に代入します。  
  
## 構文  
  
```  
  
result %= expression  
```  
  
## 引数  
 `result`  
 任意の変数。  
  
 `expression`  
 任意の数式。  
  
## 解説  
 `%=` 演算子は、次の記述とまったく同じです。  
  
```  
result = result % expression  
```
