# size プロパティ (Map) (JavaScript)
マップ内の要素数を返します。  
  
## 構文  
  
```javascript  
sizeVar = mapObj.size  
```  
  
#### パラメーター  
 `sizeVar`  
 必須です。  任意の数値です。  
  
 `mapObj`  
 必須です。  任意の `Map` オブジェクトを指定します。
