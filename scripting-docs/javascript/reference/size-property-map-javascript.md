# <a name="size-property-map-javascript"></a>size プロパティ (Map) (JavaScript)
マップ内の要素数を返します。  
  
## <a name="syntax"></a>構文  
  
```JavaScript  
sizeVar = mapObj.size  
```  
  
#### <a name="parameters"></a>パラメーター  
 `sizeVar`  
 必須です。 任意の数値です。  
  
 `mapObj`  
 必須です。 任意の `Map` オブジェクトを指定します。
