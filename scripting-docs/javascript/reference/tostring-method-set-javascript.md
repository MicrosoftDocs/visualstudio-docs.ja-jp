# toString メソッド (Set) (JavaScript)
セットの文字列表現を返します。  
  
## 構文  
  
```javascript  
setObj.toString()  
```  
  
#### パラメーター  
 `setObj`  
 必須です。  `Set` オブジェクト。  
  
## プロパティ値\/戻り値  
 セットの文字列表現。
