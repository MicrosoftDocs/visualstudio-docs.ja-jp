# JavaScript の予約語
[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] には、識別子として使用できない一連の予約語があります。  予約語は [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 言語の構文の一部であり、言語固有の意味があります。  保護された予約語を使用すると、スクリプトを読み込む際にコンパイル エラーが発生します。  
  
 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] には、将来の予約語の一覧もあります。  これらの予約語は将来の使用のために予約されていますが、現在は [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 言語の一部になっていません。  JavaScript の将来の予約済みキーワードの詳細については、「[JavaScript の将来の予約語](../../javascript/reference/javascript-future-reserved-words.md)」を参照してください。  
  
 識別子を選択する場合には、[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] の組み込みオブジェクト名や関数名として既に使用されている語を使わないように十分に注意してください。たとえば、`String`、`parseInt` などは選択できません。  
  
## 予約済みのキーワード

|-|-|-|-|-|

|catch|do|in|this|while|  
|const|else|instanceof|throw|with|  
|continue|finally|let|try||  
|debugger|for|new|typeof||
