# Math.sin 関数 (JavaScript)
指定された数値のサインの値を返します。  
  
## 構文  
  
```  
  
Math.sin(radians)   
```  
  
## 解説  
 `radians` 引数はラジアン単位の角度を含む数式です。  
  
 戻り値は、数値引数のサインの値です。
