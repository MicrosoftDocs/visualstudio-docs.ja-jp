# <a name="mathsin-function-javascript"></a>Math.sin 関数 (JavaScript)
サイン (正弦) を返します。  
  
## <a name="syntax"></a>構文  
  
```  
  
Math.sin(radians)   
```  
  
## <a name="remarks"></a>コメント  
 `radians`引数はラジアンで表した角度を表す数値式です。  
  
 戻り値は、数値の引数の正弦です。
