# Math.sqrt 関数 (JavaScript)
指定された数値の平方根を返します。  
  
## 構文  
  
```  
  
Math.sqrt(  
number  
)   
```  
  
## 解説  
 `number` 引数は必須で、数式を指定します。  
  
 `number` 引数に負の値を指定すると、戻り値は `NaN` になります。
