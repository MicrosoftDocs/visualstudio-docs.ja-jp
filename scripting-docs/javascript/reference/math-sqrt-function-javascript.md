# <a name="mathsqrt-function-javascript"></a>Math.sqrt 関数 (JavaScript)
平方根を返します。  
  
## <a name="syntax"></a>構文  
  
```  
  
Math.sqrt(  
number  
)   
```  
  
## <a name="remarks"></a>コメント  
 必要な `number` 引数は、数値式です。  
  
 場合`number`は負の場合、戻り値は`NaN`します。
