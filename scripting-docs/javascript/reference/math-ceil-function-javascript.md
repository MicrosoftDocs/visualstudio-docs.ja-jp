# Math.ceil 関数 (JavaScript)
指定された数値引数以上の最小の整数を返します。  
  
## 構文  
  
```  
  
Math.ceil(number)   
```  
  
## 解説  
 `number` 引数は必須で、数式を指定します。  
  
 戻り値は、指定された数値引数以上の値となる最小の整数値です。  
  
 **対象**: [Math オブジェクト](../../javascript/reference/math-object-javascript.md)
