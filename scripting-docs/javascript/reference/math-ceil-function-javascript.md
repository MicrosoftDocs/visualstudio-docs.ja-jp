# <a name="mathceil-function-javascript"></a>Math.ceil 関数 (JavaScript)
大きいか等しい数値引数には、最小の整数を返します。  
  
## <a name="syntax"></a>構文  
  
```  
  
Math.ceil(number)   
```  
  
## <a name="remarks"></a>コメント  
 必要な `number` 引数は、数値式です。  
  
 戻り値は、最小の整数値より大きいか等しい数値に等しい整数値です。  
  
 **適用されます**: [Math オブジェクト](../../javascript/reference/math-object-javascript.md)
