# <a name="multiplication-operator--javascript"></a>乗算演算子 (*) (JavaScript)
2 つの式の値を乗算します。  
  
## <a name="syntax"></a>構文  
  
```  
  
result = number1*number2  
```  
  
## <a name="parameters"></a>パラメーター  
 *結果*  
 任意の変数。  
  
 `number1`  
 任意の式。  
  
 `number2`  
 任意の式。
