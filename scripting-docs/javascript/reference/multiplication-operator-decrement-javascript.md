# 乗算演算子 (*) (JavaScript)
2 つの式の値の積を返します。  
  
## 構文  
  
```  
  
result = number1*number2  
```  
  
## パラメーター  
 *result*  
 任意の変数を指定します。  
  
 `number1`  
 任意の式を指定します。  
  
 `number2`  
 任意の式を指定します。
