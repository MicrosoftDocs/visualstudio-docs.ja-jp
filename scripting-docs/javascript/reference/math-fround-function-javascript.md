# Math.fround 関数 (JavaScript)
最も近い単精度浮動小数点形式の数を返します。  
  
## 構文  
  
```  
  
Math.fround(  
number  
)   
```  
  
## 解説  
 `number` が `NaN` の場合、結果は `NaN`です。
