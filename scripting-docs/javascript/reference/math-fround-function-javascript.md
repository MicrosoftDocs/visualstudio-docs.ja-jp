# <a name="mathfround-function-javascript"></a>Math.fround 関数 (JavaScript)
最も近い単精度浮動小数点形式の数を返します。  
  
## <a name="syntax"></a>構文  
  
```  
  
Math.fround(  
number  
)   
```  
  
## <a name="remarks"></a>コメント  
 `number` が `NaN` の場合、結果は `NaN`です。
