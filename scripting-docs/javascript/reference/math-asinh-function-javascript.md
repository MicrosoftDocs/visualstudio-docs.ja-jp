# <a name="mathasinh-function-javascript"></a>Math.asinh 関数 (JavaScript)
数値の逆ハイパーボリック サインを返します。  
  
## <a name="syntax"></a>構文  
  
```  
  
Math.asinh(x)   
```  
  
## <a name="remarks"></a>コメント  
 `x` が `NaN` の場合、結果は `NaN`です。  
  
 **適用されます**: [Math オブジェクト](../../javascript/reference/math-object-javascript.md)
