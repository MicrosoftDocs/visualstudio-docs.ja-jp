# Math.asinh 関数 (JavaScript)
数値の逆ハイパーボリック サインを返します。  
  
## 構文  
  
```  
  
Math.asinh(x)   
```  
  
## 解説  
 `x` が `NaN` の場合、結果は `NaN`です。  
  
 **適用先**: [Math オブジェクト](../../javascript/reference/math-object-javascript.md)
