# &#39;continue&#39; をループの外に設定できません。
ループの外側で **continue** ステートメントを使用しようとしました。  **continue** ステートメントは、以下のループの内部でのみ使用できます。  
  
-   `do-while` ループ  
  
-   `while` ループ  
  
-   **for** ループ  
  
-   **for\/in** ループ  
  
### このエラーを解決するには  
  
-   **continue** ステートメントは、以下のループの内部でのみ使用してください。  
  
    -   `do-while` ループ  
  
    -   `while` ループ  
  
    -   **for** ループ  
  
    -   **for\/in** ループ
