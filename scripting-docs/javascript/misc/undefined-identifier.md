# 未定義の識別子です。
[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] コンパイラが認識しない識別子を使用しようとしました。  次のアイテムを使用すると、未定義の値が返されます。  
  
-   存在しない変数  
  
-   宣言されていますが、値が代入されていない変数  
  
-   存在しないオブジェクト プロパティ  
  
### このエラーを解決するには  
  
-   **var**  ステートメントを使用して変数を宣言します \(`var` x; など\)。
