# <a name="undefined-identifier"></a>未定義の識別子です。
@No__t_0 コンパイラで認識されない識別子を使用しようとしました。 次を使用する場合は、未定義の値が返されます。  
  
- 存在しない変数です。  
  
- 宣言されていても値が割り当てられていない変数  
  
- 存在しないオブジェクトプロパティです。  
  
### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 変数を `var` x のように**var**ステートメントで宣言します。
