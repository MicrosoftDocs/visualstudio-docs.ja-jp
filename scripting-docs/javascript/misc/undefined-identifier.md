# <a name="undefined-identifier"></a>未定義の識別子です。
コンパイラで認識されない識別子を使用しようとしました [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 。 次を使用する場合は、未定義の値が返されます。  
  
- 存在しない変数です。  
  
- 宣言されていても値が割り当てられていない変数  
  
- 存在しないオブジェクトプロパティです。  
  
### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- **Var**ステートメント (x のように) を使用して変数を宣言 `var` します。
