# 正規表現で構文エラーが発生しました。(JavaScript)
検索文字列の構造は、[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] の正規表現の 1 つ以上の文法規則に違反します。  
  
### このエラーを解決するには  
  
-   正規表現の検索文字列の構造が [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] の正規表現の構文に従っていることを確認します。
