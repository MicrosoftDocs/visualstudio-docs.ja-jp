# <a name="syntax-error-in-regular-expression-javascript"></a>正規表現で構文エラーが発生しました。(JavaScript)
検索文字列の構造は、正規表現の文法規則の1つ以上に違反し [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] ます。  
  
### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 正規表現の検索文字列の構造が正規表現の構文に準拠していることを確認し [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] ます。
