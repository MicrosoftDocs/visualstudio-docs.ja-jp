# <a name="expected-constant"></a>定数が必要です。
条件付きコンパイルのテストステートメントで (条件付きコンパイル以外の) 変数を使用しようとしました。 条件付きコンパイルのテストステートメントは、定数に評価される必要があります。  
  
### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 変数をリテラルに置き換えます。  
  
- 変数を条件付きコンパイル変数に置き換えます。
