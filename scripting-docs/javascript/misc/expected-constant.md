# 定数が必要です。
条件付きコンパイルのテスト ステートメントで、（条件なしコンパイル） 変数を使用しようとしました。  条件付きコンパイルのテスト ステートメントは、定数として評価される必要があります。  
  
### このエラーを解決するには  
  
-   変数をリテラルで置き換えます。  
  
-   変数を条件付きコンパイル変数で置き換えます。
