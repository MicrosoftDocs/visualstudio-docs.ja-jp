# <a name="expected-constant"></a>定数が必要です。
ステートメントが条件付きコンパイルのテストで (条件なしのコンパイル) の変数を使用しようとしました。 条件付きコンパイルのテストのステートメントは、定数に評価する必要があります。  
  
### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   リテラル、変数を置換します。  
  
-   変数は、条件付きコンパイル変数で置き換えます。
