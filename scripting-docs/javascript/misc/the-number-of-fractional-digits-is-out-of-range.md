# 小数の桁数が有効範囲を超えています。
**Number.prototype.toExponential** 関数に無効な引数を渡そうとしました。  **toExponential\(\)** 関数の引数は、0 ～ 20 にする必要があります \(0 と 20 を含む\)。  
  
### このエラーを解決するには  
  
-   **toExponential\(\)** の引数の値は、その範囲内に収めてください。
