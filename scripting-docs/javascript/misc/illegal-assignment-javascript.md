# 無効な代入です。(JavaScript)
読み取り専用の識別子に値を代入しようとしました。  読み取り専用の識別子に値を代入することはできません。  たとえば、ホストによって定義されたオブジェクト、外部 COM オブジェクトなど。  
  
### このエラーを解決するには  
  
-   読み取り専用の識別子に値を代入しないでください。
