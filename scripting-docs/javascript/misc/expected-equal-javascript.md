# &#39;=&#39; が必要です。(JavaScript)
条件付きコンパイル ステートメントで使用する変数を作成しようとしましたが、変数の間の等号 \(\=\) および代入する値を記述しませんでした。  
  
### このエラーを解決するには  
  
-   等号を追加します。  次に例を示します。  
  
    ```javascript  
    @set @myvar1 = 1  
    ```
