# <a name="expected-3939-javascript"></a>予想 &#39; = (& a) #39 です。(JavaScript)
条件付きコンパイル ステートメントで使用する変数を作成しようとしましたが、変数とそれに割り当てる値の間の等号 (=) を配置できませんでした。  
  
### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   等号 (=) を追加します。 例:  
  
    ```JavaScript  
    @set @myvar1 = 1  
    ```
