# <a name="expected--javascript"></a>'=' が必要です。(JavaScript)
条件付きコンパイルステートメントで使用する変数を作成しようとしましたが、変数と代入する値の間に等号がありませんでした。  
  
### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 等号を追加します。 次に例を示します。  
  
    ```JavaScript  
    @set @myvar1 = 1  
    ```
