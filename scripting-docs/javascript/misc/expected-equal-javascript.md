# <a name="expected--javascript"></a>'=' が必要です。(JavaScript)
条件付きコンパイル ステートメントを使用する変数を作成しようとしましたが、変数とそれに割り当てる値の間の等号 (=) を配置しませんでした。  
  
### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   等号 (=) を追加します。 例:  
  
    ```JavaScript  
    @set @myvar1 = 1  
    ```
