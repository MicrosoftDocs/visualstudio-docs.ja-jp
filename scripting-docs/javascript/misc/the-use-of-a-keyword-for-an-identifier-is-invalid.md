# キーワードを識別子に使用することはできません。
1048: キーワードを識別子に使用することはできません  
  
 変数、関数、または他のユーザー定義の識別子の名前としてキーワードを使用することはできません。  
  
 キーワードの詳細については、「[JavaScript の予約語](../../javascript/reference/javascript-reserved-words.md)」および「[JavaScript の将来の予約語](../../javascript/reference/javascript-future-reserved-words.md)」を参照してください。
