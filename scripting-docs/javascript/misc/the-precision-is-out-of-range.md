# <a name="the-precision-is-out-of-range"></a>有効桁数の範囲を超えています。
関数に無効な引数を渡すしようとしています。 **Number.prototype.toPrecision**です。 引数に**toPrecision** 1 ~ 21 (包括) である必要があります。  
  
### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   引数を確認してください`toPrecision`が大きすぎるか小さすぎるではありません。
