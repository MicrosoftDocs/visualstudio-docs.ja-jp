# 有効桁数の範囲を超えています。
**Number.prototype.toPrecision** 関数に無効な引数を渡そうとしました。  **toPrecision** への引数は、1 ～ 21 の間にする必要があります （その値を含む）。  
  
### このエラーを解決するには  
  
-   `toPrecision` への引数の値をその範囲内に収めてください。
