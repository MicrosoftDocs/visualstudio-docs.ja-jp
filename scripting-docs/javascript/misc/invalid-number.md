# <a name="invalid-number"></a>無効な数値です。
無効な数値 (ErrbadNumber 1013)  
  
 JavaScript では無効な数値を使用しようとしました。  
  
 有効な数値については、「 [Number オブジェクト](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Number)」を参照してください。
