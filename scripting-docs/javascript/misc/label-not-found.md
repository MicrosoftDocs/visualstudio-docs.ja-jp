# このラベルが定義されていません。
存在しないラベルを参照しています。  ラベルはコード ブロックにマークを付けるために使用できますが、指定されたスコープ内で一意にする必要があります。  
  
### このエラーを解決するには  
  
-   ラベル名のスペルに誤りがないことを確認します。  
  
-   すべてのラベル参照が、現在のスコープで定義されているラベルに対して行われていることを確認します （事前定義を含む）。
