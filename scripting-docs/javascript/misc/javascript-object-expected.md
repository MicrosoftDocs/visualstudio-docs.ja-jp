# JavaScript オブジェクトが必要です。
[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] オブジェクトを必要とする組み込み関数に [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] オブジェクトではないオブジェクトを渡そうとしました。  さまざまな組み込み関数が、ホストまたはコントロールなどの外部コンポーネントによって定義されたオブジェクトではなく、[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] で定義されたオブジェクトを必要とします。  
  
### このエラーを解決するには  
  
-   パラメーターとして渡しているオブジェクトの型が正しいことを確認します。
