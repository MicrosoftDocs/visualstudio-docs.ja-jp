# <a name="expected-"></a>':' が必要です。
三項条件演算子を使用して式を作成しようとしましたが、2番目と3番目のオペランドの間にコロンが含まれていませんでした。 三項 (3 つのオペランド) 条件演算子では、2番目 (true) と3番目 (false) のオペランドの間にコロンが必要です。  
  
### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 2番目と3番目のオペランドの間にコロンを挿入します。
