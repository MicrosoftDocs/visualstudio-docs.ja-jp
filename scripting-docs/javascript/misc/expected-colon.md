# &#39;:&#39; が必要です。
三項条件演算子を使用する式を作成しようとしましたが、2 番目と 3 番目のオペランド間にコロンがありません。  三項 \(3 つのオペランド\) 条件演算子には、2 番目 \(true\) と 3 番目 \(false\) のオペランドの間にコロンが必要です。  
  
### このエラーを解決するには  
  
-   2 番目と 3 番目のオペランドの間にコロンを挿入します。
