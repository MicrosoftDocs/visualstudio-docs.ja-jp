# <a name="expected-var"></a>'var' が必要です
予想される ' var' (ERRnoVar 1034))  
  
 必要な時に、var キーワードを追加できませんでした。  
  
 Var キーワードの詳細については、[var ステートメント](../../javascript/reference/var-statement-javascript.md)を参照してください。
