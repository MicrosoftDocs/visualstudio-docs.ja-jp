# <a name="expected-while"></a>'while' が必要です
条件をループに含めませんでした `while` `do...while` 。 **Do**ステートメントには、 `while` コードブロックの末尾に対応するテストが必要です。  
  
### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 右中 `while` かっこの後に test ステートメントを含めます。
