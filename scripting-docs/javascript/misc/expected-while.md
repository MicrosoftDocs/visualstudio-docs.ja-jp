# &#39;while&#39; が必要です
`do...while` ループに `while` 条件がありません。  **do** ステートメントには、コード ブロックの最後に対応する `while` のテストが必要です。  
  
### このエラーを解決するには  
  
-   右中かっこの後に `while` のテスト ステートメントを追加します。
