# <a name="expected-"></a>']' が必要です。
配列要素への参照が行われたが、右角かっこが含まれていません。 配列の要素を参照する任意の式には、角かっこの開閉の両方を含める必要があります。  
  
### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 配列の要素を参照する式には、右角かっこを追加します。
