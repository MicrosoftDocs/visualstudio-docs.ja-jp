# <a name="object-member-expected"></a>オブジェクトのメンバが必要です。
オブジェクトのデータメンバーに対して操作を実行しようとしましたが、メンバーが存在しません。  
  
### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- データメンバーの名前を正しく入力したことを確認します。
