# &#39;{&#39; が必要です。
関数本体の開始を示す左中かっこが記述されていません。  関数本体のコードは、1 行であっても中かっこで囲む必要があります。  関数本体での中かっこの使用は、ループでの中かっこの使用より厳密です。  
  
### このエラーを解決するには  
  
-   関数本体の開始を示す左中かっこを追加します。
