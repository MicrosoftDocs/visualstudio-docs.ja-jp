# &#39;(&#39; が必要です。(JavaScript)
式をかっこで囲む際の左かっこがありません。  一部の式は、左かっこと右かっこで囲む必要があります。  次の例でのかっこの使用方法に注意してください。  
  
```javascript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### このエラーを解決するには  
  
-   評価式に左かっこを追加してください。
