# <a name="syntax-error-javascript"></a>構文エラー (JavaScript)
JScript ステートメントが、JScript の 1 つ以上の文法規則に違反しています。
  
### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 示されている行番号のプログラムの構文を再確認します。
  
- かっこや中かっこの使用に間違いがないかを確認します。
