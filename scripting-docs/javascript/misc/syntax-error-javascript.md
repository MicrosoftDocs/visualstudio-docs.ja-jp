# <a name="syntax-error-javascript"></a>構文エラー (JavaScript)
@No__t_0 の文法規則の1つ以上に違反するステートメントを作成しました。  
  
### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 示されている行番号でプログラムの構文を再確認します。  
  
- 誘導かっこまたは中かっこを検索します。
