# Array または arguments オブジェクトが必要です。
引数として配列が指定されませんでした。  このエラーは **Function.prototype.apply** メソッドでのみ発生します。  この関数に 2 番目の引数を指定する場合は、`Array` オブジェクトまたは **Arguments** オブジェクトを指定する必要があります。  
  
### このエラーを解決するには  
  
-   2 番目の引数として `Array` オブジェクトまたは **Arguments** オブジェクトを指定します。
