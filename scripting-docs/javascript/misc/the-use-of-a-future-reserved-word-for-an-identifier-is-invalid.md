# 将来の予約語を識別子に使用することはできません。
1049: 将来の予約語を識別子に使用することはできません。  
  
 変数、関数、またはその他のユーザー定義の識別子の名前として将来の予約語を使用することはできません。  
  
 将来の予約語の詳細については、「[JavaScript の将来の予約語](../../javascript/reference/javascript-future-reserved-words.md)」を参照してください。
