# <a name="number-expected"></a>数字が必要です。

### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   のみを呼び出す、 **Number.prototype.toString**または**Number.prototype.valueOf**型のオブジェクトに対するメソッド**数**です。
