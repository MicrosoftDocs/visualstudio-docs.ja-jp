# &#39;@end&#39; が必要です。
条件付きでコンパイルされたコード ブロックを作成しようとしましたが、最後に **@end** ステートメントが記述されていません。  [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] のステートメントを条件付きでコンパイルするには、**@if\/@end** ブロックで囲む必要があります。  
  
### このエラーを解決するには  
  
-   対応する **@end** ステートメントを追加します。
