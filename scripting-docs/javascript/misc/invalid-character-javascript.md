# <a name="invalid-character-javascript"></a>無効な文字です。(JavaScript)
識別子の構成によって有効と認識されていない文字 (または文字) を使用して、[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]コンパイラ。 有効な文字は、次の規則を使用します。  
  
-   最初の文字は、ASCII 文字 (大文字または小文字) またはアンダー スコア (_) である必要があります。  
  
-   2 文字目以降は、ASCII 文字、数字、またはアンダー スコアにできます。  
  
-   識別子名は予約語をすることはできません。  
  
### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   ない文字は使用しないでくださいの一部、[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]言語の定義。
