# 無効な文字です。(JavaScript)
文字 \(または文字列\) を使用して識別子を作成しましたが、[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] コンパイラが有効な文字として識別できない文字がありました。  次の規則に従う文字が有効です。  
  
-   先頭文字は、ASCII 文字 \(大文字でも小文字でもかまいません\) またはアンダースコア \(\_\) 文字にする必要があります。  
  
-   2 文字目以降の文字には、ASCII 文字、数字、またはアンダースコア \(\_\) を使用できます。  
  
-   識別子名に予約語は使用できません。  
  
### このエラーを解決するには  
  
-   [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 言語定義に含まれていない文字は使用しないようにします。
