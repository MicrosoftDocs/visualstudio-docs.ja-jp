# <a name="expected-"></a>' @No__t_0 ' が必要です
@No__t_0 ステートメントを使用して条件付きコンパイルステートメントで使用する変数を作成しようとしましたが、変数名の前にアットマーク " **@** " を配置しませんでした。  
  
### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 変数名の直前に " **@** " というアットマークを追加します。 (例:  
  
    ```JavaScript  
    @set @myvar = 1  
    ```
