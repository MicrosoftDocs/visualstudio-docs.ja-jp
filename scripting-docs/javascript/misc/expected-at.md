# <a name="expected-3939"></a>予想 &#39; @&#39;
使用して条件付きコンパイル ステートメントで使用する変数を作成しようとしています。、`@set`ステートメントでは、挿入されていないが、アット マーク"**@**"変数名の前にします。  
  
### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   追加、アット マーク"**@**"変数名の直前にします。 例:  
  
    ```JavaScript  
    @set @myvar = 1  
    ```
