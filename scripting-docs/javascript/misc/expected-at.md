# <a name="expected-"></a>予想 '\@'
使用して条件付きコンパイル ステートメントで使用する変数を作成しようとしています。、`@set`ステートメントでは、配置しませんでしたが、アット マーク"**@**"変数名の前にします。  
  
### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   追加するアット"**@**"変数名の直前。 例えば:  
  
    ```JavaScript  
    @set @myvar = 1  
    ```
