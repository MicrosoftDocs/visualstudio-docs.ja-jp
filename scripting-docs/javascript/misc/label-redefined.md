# ラベルが再定義されました
作成された新しいラベルに既存のラベル名が指定されました。  ラベルはコード ブロックにマークを付けるために使用できますが、指定されたスコープ内で一意にする必要があります。  
  
### このエラーを解決するには  
  
-   プログラム内で使用するすべてのラベルは、それぞれのスコープ内で必ず一意になるようにしてください。
