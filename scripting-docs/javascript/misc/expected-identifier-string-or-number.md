# 識別子、文字列、または数値が必要です
誤ったリテラル構文を使用して、オブジェクト リテラルを宣言しました。  オブジェクト リテラルのプロパティは、識別子、文字列、または数値にする必要があります。  オブジェクト リテラル \(または "オブジェクト初期化子"\) は、"プロパティ:値" のペアをコンマで区切ったリストにして全体をかっこで囲みます。  次に例を示します。  
  
```javascript  
var point = {x:1.2, y:-3.4};  
```  
  
### このエラーを解決するには  
  
-   正しいリテラル構文を使用してください。
