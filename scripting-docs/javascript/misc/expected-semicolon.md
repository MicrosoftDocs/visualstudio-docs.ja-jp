# &#39;;&#39; が必要です。
行に複数のステートメントを配置しようとしましたが、セミコロンで区切っていませんでした。  セミコロンはステートメントを終了するために使用されます。  単一行に複数のステートメントを配置することはできますが、それぞれをセミコロンで区切る必要があります。  
  
### このエラーを解決するには  
  
-   各ステートメントの末尾をセミコロンでマークします。  
  
-   関数呼び出しでかっこを適切に使用していることを確認します。  
  
-   **for** ループのヘッダー内のセミコロンを忘れていないことを確認します。
