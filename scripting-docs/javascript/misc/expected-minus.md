# &#39;/&#39; が必要です。
正規表現リテラルを作成しようとしましたが、いずれかのスラッシュ （\/） がありません。  文字列リテラルが引用符で囲まれた文字列として記述されるように、正規表現リテラルはスラッシュ （\/）文字のペアで囲まれた文字列として表現されます。  
  
### このエラーを解決するには  
  
-   正規表現の最後を示すために終端のスラッシュを挿入します。
