# <a name="expected--"></a>予想 '-'
正規表現をリテラルには、作成しようとしましたが、スラッシュ (/) の 1 つ含まれていません。 文字列リテラルが二重引用符内の文字として書き込まれると同様、正規表現リテラルは、スラッシュ (/) 文字のペア内の文字として表現されます。  
  
### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 正規表現の終わりをマークする終了スラッシュを挿入します。
