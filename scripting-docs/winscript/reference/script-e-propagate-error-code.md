# <a name="scriptepropagate-error-code"></a>SCRIPT_E_PROPAGATE エラー コード
スクリプト エラーは、別のスレッドで可能性のある呼び出し元に反映中です。 ホストは、呼び出し元に、エラー コードを渡す必要があります。  
  
## <a name="syntax"></a>構文  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
