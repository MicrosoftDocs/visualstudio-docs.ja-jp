# <a name="script_e_propagate-error-code"></a>SCRIPT_E_PROPAGATE エラー コード
スクリプトエラーが呼び出し元に伝達されています。これは別のスレッドに存在する可能性があります。 ホストは、エラーコードを呼び出し元に渡す必要があります。  
  
## <a name="syntax"></a>構文  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
