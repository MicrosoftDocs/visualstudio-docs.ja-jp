# SCRIPT_E_PROPAGATE エラー コード
スクリプト エラーは別のスレッドにある呼び出し元に広められています。  ホストは、呼び出し元にエラー コードを渡す必要があります。  
  
## 構文  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
