# <a name="ienumjsstackframesnext-method"></a>IEnumJsStackFrames::Next メソッド
指定した数のフレームを取得します。  
  
## <a name="syntax"></a>構文  
  
```cpp
HRESULT Next(  
   ULONG cFrameCount,  
   JS_NATIVE_FRAME *pFrames,  
   ULONG *pcFetched  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `cFrameCount`  
 [入力] 取得するフレームの数。  
  
 `pFrames`  
 [出力] フレームを格納する配列。  
  
 `pcFetched`  
 [出力] 返されたフレーム数。  
  
## <a name="return-value"></a>戻り値  
  
## <a name="requirements"></a>必要条件  
 **ヘッダー:** jscript9diag.h です
