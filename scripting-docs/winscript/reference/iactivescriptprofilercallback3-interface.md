# IActiveScriptProfilerCallback3 インターフェイス
プロファイリングされた実装が Web ワーカー スレッドのコンテキストで発生していることをプロファイラーに通知するメソッドを提供します。  
  
## 構文  
  
```  
interface IActiveScriptProfilerCallback3 : IActiveScriptProfilerCallback2  
```  
  
## メソッド  
  
|メソッド|説明|  
|----------|--------|  
|[IActiveScriptProfilerCallback3::SetWebWorkerId メソッド](../../winscript/reference/iactivescriptprofilercallback3-setwebworkerid-method.md)|ワーカー スレッドの ID についてプロファイラーにこのプロファイル セッションで使用する通知します。|
