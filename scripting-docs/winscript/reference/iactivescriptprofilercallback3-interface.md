# <a name="iactivescriptprofilercallback3-interface"></a>IActiveScriptProfilerCallback3 インターフェイス
プロファイリングされた実行が web ワーカーのコンテキストで発生していることをプロファイラーに通知するメソッドを提供します。  
  
## <a name="syntax"></a>構文  
  
```cpp
interface IActiveScriptProfilerCallback3 : IActiveScriptProfilerCallback2  
```  
  
## <a name="methods"></a>メソッド  
  
|メソッド|説明|  
|------------|-----------------|  
|[IActiveScriptProfilerCallback3::SetWebWorkerId メソッド](../../winscript/reference/iactivescriptprofilercallback3-setwebworkerid-method.md)|このプロファイルセッションで使用するワーカー ID をプロファイラーに通知します。|
