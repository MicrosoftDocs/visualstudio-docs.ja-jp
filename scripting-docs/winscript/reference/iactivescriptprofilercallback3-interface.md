# <a name="iactivescriptprofilercallback3-interface"></a>IActiveScriptProfilerCallback3 インターフェイス
Web ワーカーのコンテキストで、プロファイリング対象の実行が行われていることをプロファイラーに通知するメソッドを提供します。  
  
## <a name="syntax"></a>構文  
  
```  
interface IActiveScriptProfilerCallback3 : IActiveScriptProfilerCallback2  
```  
  
## <a name="methods"></a>メソッド  
  
|メソッド|説明|  
|------------|-----------------|  
|[IActiveScriptProfilerCallback3::SetWebWorkerId メソッド](../../winscript/reference/iactivescriptprofilercallback3-setwebworkerid-method.md)|このプロファイルのセッションに使用するワーカーの ID をプロファイラーに通知します。|
