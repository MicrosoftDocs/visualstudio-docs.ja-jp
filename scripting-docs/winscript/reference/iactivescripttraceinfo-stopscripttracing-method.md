# IActiveScriptTraceInfo::StopScriptTracing メソッド
停止のスクリプトのトレース。  
  
## 構文  
  
```  
HRESULT StopScriptTracing(void);   
}  
  
```  
  
#### パラメーター  
 このメソッドにはパラメーターがありません。  
  
## 戻り値  
 このメソッドの使用可能な戻り値は次のとおりです。:  
  
1.  S\_OK: 成功。  
  
2.  E\_NOTIMPL: 実行されません。
