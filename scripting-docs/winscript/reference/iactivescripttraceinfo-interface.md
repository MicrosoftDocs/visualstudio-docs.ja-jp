|メソッド|説明|  
|------------|-----------------|  
|[IActiveScriptTraceInfo::StartScriptTracing メソッド](../../winscript/reference/iactivescripttraceinfo-startscripttracing-method.md)|スクリプト トレースを開始します。|  
|[IActiveScriptTraceInfo::StopScriptTracing メソッド](../../winscript/reference/iactivescripttraceinfo-stopscripttracing-method.md)|トレースを停止します。|
