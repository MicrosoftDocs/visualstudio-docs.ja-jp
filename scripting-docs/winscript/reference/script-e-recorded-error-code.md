# <a name="script_e_recorded-error-code"></a>SCRIPT_E_RECORDED エラー コード
スクリプトエンジンとホストの間でエラーが発生しました。 ホストは、エラーコードを呼び出し元に渡す必要があります。  
  
## <a name="syntax"></a>構文  
  
```cpp  
SCRIPT_E_RECORDED   0x86664004L  
```
