# SCRIPT_E_RECORDED エラー コード
エラーは、スクリプト エンジンとホストの間に渡されました。  呼び出し元にエラー コードを渡すホストがあります。  
  
## 構文  
  
```cpp  
SCRIPT_E_RECORDED   0x86664004L  
```
