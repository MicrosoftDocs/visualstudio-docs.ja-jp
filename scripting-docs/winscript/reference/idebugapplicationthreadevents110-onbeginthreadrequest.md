# <a name="idebugapplicationthreadevents110-onbeginthreadrequest"></a>IDebugApplicationThreadEvents110 ::OnBeginThreadRequest
PDM のスレッド切り替えを使用してスレッドへの呼び出しが開始されました。  
  
> [!IMPORTANT]
> [IDebugApplicationThreadEvents110 インターフェイス](../../winscript/reference/idebugapplicationthreadevents110-interface.md)は、PDM version 11.0 以降で実装されています。 activdbg100.h にあります。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT OnBeginThreadRequest( void );  
```  
  
#### <a name="parameters"></a>パラメーター  
 このメソッドにはパラメーターがありません。
