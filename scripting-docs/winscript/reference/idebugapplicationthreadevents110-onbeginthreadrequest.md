# <a name="idebugapplicationthreadevents110-onbeginthreadrequest"></a>IDebugApplicationThreadEvents110 ::OnBeginThreadRequest
呼び出しスレッドは、PDM のスレッドを使用して、切り替えが開始されました。  
  
> [!IMPORTANT]
>  [IDebugApplicationThreadEvents110 インターフェイス](../../winscript/reference/idebugapplicationthreadevents110-interface.md)は、PDM v11.0 以降によって実装されている値を超えています。 activdbg100.h にあります。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT OnBeginThreadRequest( void );  
```  
  
#### <a name="parameters"></a>パラメーター  
 このメソッドにパラメーターがありません。
