# IEnumDebugApplicationNodes インターフェイス
アプリケーションに関連付けられているノードの子ノードを列挙します。  
  
 `IUnknown` から継承するメソッドに加え、`IEnumDebugApplicationNodes` インターフェイスは次のメソッドを公開します。  
  
## Vtable 順序のメソッド  
  
|メソッド|説明|  
|----------|--------|  
|[IEnumDebugApplicationNodes::Next](../../winscript/reference/ienumdebugapplicationnodes-next.md)|列挙体シーケンスのセグメントの指定した数を取得します。|  
|[IEnumDebugApplicationNodes::Skip](../../winscript/reference/ienumdebugapplicationnodes-skip.md)|列挙体シーケンス内の指定した数のセグメントをスキップします。|  
|[IEnumDebugApplicationNodes::Reset](../../winscript/reference/ienumdebugapplicationnodes-reset.md)|列挙体シーケンスを先頭にリセットします。|  
|[IEnumDebugApplicationNodes::Clone](../../winscript/reference/ienumdebugapplicationnodes-clone.md)|現在の列挙状態と同じ状態を格納する列挙子を作成します。|
