# IDebugDocument インターフェイス
すべてのデバッグ ドキュメントの基本インターフェイス。  
  
## 解説  
 `IDebugDocument` のインターフェイスは [IDebugDocumentInfo インターフェイス](../../winscript/reference/idebugdocumentinfo-interface.md)から継承されたメソッドのみを公開します。  これは追加のメソッドを公開しません。
