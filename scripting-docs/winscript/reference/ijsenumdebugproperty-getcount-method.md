# <a name="ijsenumdebugpropertygetcount-method"></a>IJsEnumDebugProperty::GetCount メソッド
オブジェクトのプロパティの数を取得します。  
  
## <a name="syntax"></a>構文  
  
```cpp
HRESULT GetCount(  
   ULONG *pCount  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `pCount`  
 [出力] プロパティの数。  
  
## <a name="return-value"></a>戻り値  
  
## <a name="requirements"></a>必要条件  
 **ヘッダー:** jscript9diag.h です
