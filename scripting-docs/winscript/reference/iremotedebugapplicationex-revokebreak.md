# IRemoteDebugApplicationEx:RevokeBreak
中断コマンドをキャンセルします。  
  
## 構文  
  
```  
HRESULT RevokeBreak( );  
```  
  
#### パラメーター  
 なし。  
  
## 戻り値  
 このメソッドは `HRESULT` を返します。  指定できる値は、に含まれていますが、次の表に、これらはありません。  
  
|値|説明|  
|-------|--------|  
|`S_OK`|メソッドが成功しました。|
