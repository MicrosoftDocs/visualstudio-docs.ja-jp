# IVariantChangeType インターフェイス
呼び出し元が 1 段階の型から別の型に変数の値を変換することができます。  
  
## メソッド  
 `IUnknown` から継承するメソッドに加え、`IVariantChangeType` インターフェイスは次のメソッドを公開します。  
  
|メソッド|説明|  
|----------|--------|  
|[IVariantChangeType::ChangeType](../../winscript/reference/ivariantchangetype-changetype.md)|変数の値を受け取り、指定された型の新しいバリアントを作成します。|
