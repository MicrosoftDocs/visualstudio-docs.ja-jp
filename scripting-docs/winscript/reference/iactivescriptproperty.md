# IActiveScriptProperty
`IActiveScriptProperty` のインターフェイスがコンポーネントのプロパティを取得および設定するために使用されます。  
  
## Vtable 順序のメソッド  
  
|メソッド|説明|  
|----------|--------|  
|[GetProperty](../../winscript/reference/iactivescriptproperty-getproperty.md)|パラメーターで指定されるプロパティを取得します。|  
|[SetProperty](../../winscript/reference/iactivescriptproperty-setproperty.md)|パラメーターで指定されるプロパティを設定します。|
