# IDebugExtendedProperty インターフェイス
`IDebugProperty` のインターフェイスを拡張します。  
  
## Vtable 順序のメソッド  
 `IDebugProperty`から継承されたメソッドに加えてこのインターフェイスは、次にメソッドを公開します。  
  
|メソッド|説明|  
|----------|--------|  
|[IDebugExtendedProperty::GetExtendedPropertyInfo](../../winscript/reference/idebugextendedproperty-getextendedpropertyinfo.md)|`ExtendedDebugPropertyInfo` を取得します `IDebugExtendedProperty`この`.`を記述する|  
|[IDebugExtendedProperty::EnumExtendedMembers](../../winscript/reference/idebugextendedproperty-enumextendedmembers.md)|拡張プロパティのメンバーを列挙します。|  
  
## 要件  
 ヘッダー: dbgprop.h
