# IJsDebugFrame::GetDebugProperty メソッド
このスタック フレームのプロパティ ブラウザーを返します。  
  
## 構文  
  
```  
HRESULT GetDebugProperty(  
   IJsDebugProperty **ppDebugProperty  
);  
```  
  
#### パラメーター  
 `ppDebugProperty`  
 \[out\] Object representing the property browser.  
  
## 戻り値  
  
## 必要条件  
 **Header:** jscript9diag.h
