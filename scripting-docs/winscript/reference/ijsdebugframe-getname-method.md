# <a name="ijsdebugframegetname-method"></a>IJsDebugFrame::GetName メソッド
スタック フレームのユーザー フレンドリ名を取得します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT GetName(  
   BSTR *pName  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `pName`  
 [出力] スタック フレームの名前。  
  
## <a name="return-value"></a>戻り値  
  
## <a name="requirements"></a>要件  
 **ヘッダー:** jscript9diag.h です
