# IJsDebugFrame::GetName メソッド
スタック フレームのユーザー フレンドリ名を取得します。  
  
## 構文  
  
```  
HRESULT GetName(  
   BSTR *pName  
);  
```  
  
#### パラメーター  
 `pName`  
 \[出力\] スタック フレームの名前。  
  
## 戻り値  
  
## 必要条件  
 **ヘッダー:** jscript9diag.h
