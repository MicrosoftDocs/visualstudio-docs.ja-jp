# IEnumDebugStackFrames インターフェイス
スレッドに対応するスタック フレームを列挙します。  
  
## メソッド  
 `IUnknown` から継承するメソッドに加え、`IEnumDebugStackFrames` インターフェイスは次のメソッドを公開します。  
  
|メソッド|説明|  
|----------|--------|  
|[IEnumDebugStackFrames::Next](../../winscript/reference/ienumdebugstackframes-next.md)|列挙体シーケンス内の指定した数のセグメントを取得します。|  
|[IEnumDebugStackFrames::Skip](../../winscript/reference/ienumdebugstackframes-skip.md)|列挙体シーケンス内の指定した数のセグメントをスキップします。|  
|[IEnumDebugStackFrames::Reset](../../winscript/reference/ienumdebugstackframes-reset.md)|列挙体シーケンスを先頭にリセットします。|  
|[IEnumDebugStackFrames::Clone](../../winscript/reference/ienumdebugstackframes-clone.md)|現在の列挙状態と同じ状態を格納する列挙子を作成します。|
