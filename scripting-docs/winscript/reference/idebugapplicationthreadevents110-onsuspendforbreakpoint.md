# IDebugApplicationThreadEvents110::OnSuspendForBreakPoint
スレッドがブレークポイントに対して十分に中断した判断し、通常の実行をかどうかを再開していません。  
  
> [!IMPORTANT]
>  [IDebugApplicationThreadEvents110 インターフェイス](../../winscript/reference/idebugapplicationthreadevents110-interface.md) は PDM v11.0 を超えるによって実装されます。  activdbg100.h.である。  
  
## 構文  
  
```cpp  
HRESULT OnSuspendForBreakPoint( void );  
```  
  
#### パラメーター  
 このメソッドにはパラメーターがありません。
