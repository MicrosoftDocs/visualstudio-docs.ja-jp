# IDebugAsyncOperationCallBack インターフェイス
`IDebugAsyncOperation` のインターフェイスの評価の進行状況に関する状態のイベントを提供します。  
  
## メソッド  
 `IUnknown` から継承するメソッドに加え、`IDebugAsyncOperationCallBack` インターフェイスは次のメソッドを公開します。  
  
|メソッド|説明|  
|----------|--------|  
|[IDebugAsyncOperationCallBack::onComplete](../../winscript/reference/idebugasyncoperationcallback-oncomplete.md)|結果は非同期のデバッグ操作から使用可能であることが示されます。|
