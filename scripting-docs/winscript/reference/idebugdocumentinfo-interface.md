# <a name="idebugdocumentinfo-interface"></a>IDebugDocumentInfo インターフェイス
ドキュメントの情報を提供します。インスタンス化することも、しないこともできます。  
  
 継承されたメソッドだけでなく`IUnknown`、`IDebugDocumentInfo`インターフェイスは、次のメソッドを公開します。  
  
## <a name="methods-in-vtable-order"></a>Vtable 順序のメソッド  
  
|メソッド|説明|  
|------------|-----------------|  
|[IDebugDocumentInfo::GetName](../../winscript/reference/idebugdocumentinfo-getname.md)|指定されたドキュメント名を返します。|  
|[IDebugDocumentInfo::GetDocumentClassId](../../winscript/reference/idebugdocumentinfo-getdocumentclassid.md)|ドキュメントの種類を識別する CLSID を返します。|
