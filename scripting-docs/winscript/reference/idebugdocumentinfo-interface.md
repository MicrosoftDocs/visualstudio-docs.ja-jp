# IDebugDocumentInfo インターフェイス
いる可能性がある、またははインスタンス化されないドキュメントについて説明します。  
  
 `IUnknown` から継承するメソッドに加え、`IDebugDocumentInfo` インターフェイスは次のメソッドを公開します。  
  
## Vtable 順序のメソッド  
  
|メソッド|説明|  
|----------|--------|  
|[IDebugDocumentInfo::GetName](../../winscript/reference/idebugdocumentinfo-getname.md)|指定したドキュメントの名前を返します。|  
|[IDebugDocumentInfo::GetDocumentClassId](../../winscript/reference/idebugdocumentinfo-getdocumentclassid.md)|ドキュメントの種類を識別する CLSID を返します。|
