# SCRIPT_E_REPORTED エラー コード
スクリプト エンジンは [IActiveScriptSite::OnScriptError](../../winscript/reference/iactivescriptsite-onscripterror.md)によってホストにハンドルされない例外が発生しました。  ホストはこのエラーを無視できます。  
  
## 構文  
  
```cpp  
SCRIPT_E_REPORTED   0x80020101L  
  
```
