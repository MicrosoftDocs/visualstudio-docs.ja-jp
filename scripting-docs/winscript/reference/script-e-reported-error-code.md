# <a name="script_e_reported-error-code"></a>SCRIPT_E_REPORTED エラー コード
スクリプトエンジンは、 [IActiveScriptSite:: OnScriptError](../../winscript/reference/iactivescriptsite-onscripterror.md)を使用してホストにハンドルされない例外を報告しました。 ホストはこのエラーを無視できます。  
  
## <a name="syntax"></a>構文  
  
```cpp  
SCRIPT_E_REPORTED   0x80020101L  
```
