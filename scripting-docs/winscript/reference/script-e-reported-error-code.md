# <a name="scriptereported-error-code"></a>SCRIPT_E_REPORTED エラー コード
スクリプト エンジンが未処理の例外を使用してホストに報告して[IActiveScriptSite::OnScriptError](../../winscript/reference/iactivescriptsite-onscripterror.md)します。 ホストは、このエラーを無視できます。  
  
## <a name="syntax"></a>構文  
  
```cpp  
SCRIPT_E_REPORTED   0x80020101L  
```
