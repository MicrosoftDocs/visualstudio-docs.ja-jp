# IJsDebugBreakPoint::Enable メソッド
Enables the breakpoint.  
  
## 構文  
  
```  
HRESULT Enable(void);  
```  
  
## 戻り値  
  
## 解説  
 Returns E\_UNEXPECTED if called on a deleted breakpoint.
