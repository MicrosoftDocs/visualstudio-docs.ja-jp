# <a name="ijsdebugbreakpointenable-method"></a>IJsDebugBreakPoint::Enable メソッド
ブレークポイントを有効にします。  
  
## <a name="syntax"></a>構文  
  
```cpp
HRESULT Enable(void);  
```  
  
## <a name="return-value"></a>戻り値  
  
## <a name="remarks"></a>Remarks  
 削除されたブレークポイントで呼び出された場合は E_UNEXPECTED を返します。  
  
## <a name="requirements"></a>［要件］  
 **ヘッダー:** jscript9diag.h
