# IEnumDebugCodeContexts インターフェイス
ドキュメントのコンテキストに対応するコード コンテキストを列挙します。  
  
 `IUnknown` から継承するメソッドに加え、`IEnumDebugCodeContexts` インターフェイスは次のメソッドを公開します。  
  
## Vtable 順序のメソッド  
  
|メソッド|説明|  
|----------|--------|  
|[IEnumDebugCodeContexts::Next](../../winscript/reference/ienumdebugcodecontexts-next.md)|列挙体シーケンス内の指定した数のセグメントを取得します。|  
|[IEnumDebugCodeContexts::Skip](../../winscript/reference/ienumdebugcodecontexts-skip.md)|列挙体シーケンス内の指定した数のセグメントをスキップします。|  
|[IEnumDebugCodeContexts::Reset](../../winscript/reference/ienumdebugcodecontexts-reset.md)|列挙体シーケンスを先頭にリセットします。|  
|[IEnumDebugCodeContexts::Clone](../../winscript/reference/ienumdebugcodecontexts-clone.md)|現在の列挙状態と同じ状態を格納する列挙子を作成します。|
