# <a name="idebugpropertysetvalueasstring"></a>IDebugProperty::SetValueAsString
指定された文字列からプロパティの値を設定します。  
  
## <a name="syntax"></a>構文  
  
```cpp
HRESULT SetValueAsString (  
   LPCOLESTR pszValue,  
   UINTnRadix,  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `pszValue`  
 から設定する値。  
  
 `nRadix`  
 から数値情報の解釈に使用される基数。  
  
## <a name="return-value"></a>戻り値  
 は、有効な `HRESULT` (通常は `S_OK`) を返します。
