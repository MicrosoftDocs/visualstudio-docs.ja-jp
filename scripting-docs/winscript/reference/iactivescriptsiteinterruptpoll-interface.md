# IActiveScriptSiteInterruptPoll インターフェイス
`IActiveScriptSiteInterruptPoll` のインターフェイスは、ホストがスクリプトが終了することを指定できるようにします。  
  
## メソッド  
 `IUnknown` から継承するメソッドに加え、`IActiveScriptSiteInterruptPoll` インターフェイスは次のメソッドを公開します。  
  
|メソッド|説明|  
|----------|--------|  
|[IActiveScriptSiteInterruptPoll::QueryContinue](../../winscript/reference/iactivescriptsiteinterruptpoll-querycontinue.md)|ホストがスクリプトが終了することを指定できます。|
