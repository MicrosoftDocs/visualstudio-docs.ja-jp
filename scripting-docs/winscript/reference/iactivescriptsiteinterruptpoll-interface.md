# <a name="iactivescriptsiteinterruptpoll-interface"></a>IActiveScriptSiteInterruptPoll インターフェイス
`IActiveScriptSiteInterruptPoll`インターフェイスにより、ホストは、スクリプトが終了することを指定します。  
  
## <a name="methods"></a>メソッド  
 継承されたメソッドだけでなく`IUnknown`、`IActiveScriptSiteInterruptPoll`インターフェイスは、次のメソッドを公開します。  
  
|メソッド|説明|  
|------------|-----------------|  
|[IActiveScriptSiteInterruptPoll::QueryContinue](../../winscript/reference/iactivescriptsiteinterruptpoll-querycontinue.md)|により、ホストは、スクリプトが終了することを指定します。|
