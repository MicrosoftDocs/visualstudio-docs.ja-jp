# IDebugExpressionCallBack インターフェイス
`IDebugExpression` の評価の進行状況に関する状態のイベントを提供します。  
  
## メソッド  
 `IUnknown` から継承するメソッドに加え、`IDebugExpressionCallBack` インターフェイスは次のメソッドを公開します。  
  
|メソッド|説明|  
|----------|--------|  
|[IDebugExpressionCallBack::onComplete](../../winscript/reference/idebugexpressioncallback-oncomplete.md)|式の評価が完了したことを示します。|
