# <a name="idebugexpressioncallback-interface"></a>IDebugExpressionCallBack インターフェイス
進行状況に関連する状態イベントを提供する`IDebugExpression`評価します。  
  
## <a name="methods"></a>メソッド  
 継承されたメソッドだけでなく`IUnknown`、`IDebugExpressionCallBack`インターフェイスは、次のメソッドを公開します。  
  
|メソッド|説明|  
|------------|-----------------|  
|[IDebugExpressionCallBack::onComplete](../../winscript/reference/idebugexpressioncallback-oncomplete.md)|式の評価が完了したことを示します。|
