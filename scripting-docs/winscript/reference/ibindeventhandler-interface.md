# <a name="ibindeventhandler-interface"></a>IBindEventHandler インターフェイス
指定したイベント ハンドラーを処理するオブジェクトを提供する、呼び出し元を有効にします。  
  
## <a name="methods"></a>メソッド  
 継承されたメソッドだけでなく`IUnknown`、`IBindEventHandler`インターフェイスは、次のメソッドを公開します。  
  
|メソッド|説明|  
|------------|-----------------|  
|[IBindEventHandler::BindHandler](../../winscript/reference/ibindeventhandler-bindhandler.md)|イベントをオブジェクトにバインドします。|
