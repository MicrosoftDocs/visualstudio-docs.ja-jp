# IBindEventHandler インターフェイス
呼び出し元が指定できるハンドル イベント ハンドラー オブジェクトを提供できるようにします。  
  
## メソッド  
 `IUnknown` から継承するメソッドに加え、`IBindEventHandler` インターフェイスは次のメソッドを公開します。  
  
|メソッド|説明|  
|----------|--------|  
|[IBindEventHandler::BindHandler](../../winscript/reference/ibindeventhandler-bindhandler.md)|オブジェクトにイベントをバインドします。|
