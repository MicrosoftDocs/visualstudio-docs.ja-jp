# IDebugCookie インターフェイス
デバッグのクッキーが `IMachineDebugManagerCookie` のインターフェイスで使用するために、設定できます。  詳細については、「[IMachineDebugManagerCookie インターフェイス](../../winscript/reference/imachinedebugmanagercookie-interface.md)」を参照してください。  このインターフェイスは、プロセスのデバッグ Manager \(PDM\) によって実装され、スクリプト デバッガーによって実装されます。  
  
## メソッド  
 `IUnknown` から継承するメソッドに加え、`IDebugCookie` インターフェイスは次のメソッドを公開します。  
  
|メソッド|説明|  
|----------|--------|  
|[IDebugCookie::SetDebugCookie](../../winscript/reference/idebugcookie-setdebugcookie.md)|デバッグのアプリケーションのクッキーを設定します。|
