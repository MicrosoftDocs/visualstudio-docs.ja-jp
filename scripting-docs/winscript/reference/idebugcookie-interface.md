# <a name="idebugcookie-interface"></a>IDebugCookie インターフェイス
使用するため、設定するデバッグ cookie を許可、`IMachineDebugManagerCookie`インターフェイスです。 詳細については、次を参照してください。 [IMachineDebugManagerCookie インターフェイス](../../winscript/reference/imachinedebugmanagercookie-interface.md)です。 このインターフェイスはプロセスをデバッグ マネージャー (PDM) によって実装されており、スクリプト デバッガーによって消費されています。  
  
## <a name="methods"></a>メソッド  
 継承されたメソッドだけでなく`IUnknown`、`IDebugCookie`インターフェイスは、次のメソッドを公開します。  
  
|メソッド|説明|  
|------------|-----------------|  
|[IDebugCookie::SetDebugCookie](../../winscript/reference/idebugcookie-setdebugcookie.md)|デバッグ アプリケーションの cookie を設定します。|
