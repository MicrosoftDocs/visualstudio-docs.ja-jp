# PROFILER_EXTERNAL_OBJECT_ADDRESS 型
C\+\+.などのオブジェクトの外部オブジェクトのアドレスは、JavaScript ヒープの外部にあるオブジェクトを C\+\+ 代入します。  [PROFILER\_HEAP\_OBJECT 構造体](../../winscript/reference/profiler-heap-object-structure.md) と [PROFILER\_HEAP\_OBJECT\_RELATIONSHIP 構造体](../../winscript/reference/profiler-heap-object-relationship-structure.md)で使用します。  
  
## 構文  
  
```  
typedef void* PROFILER_EXTERNAL_OBJECT_ADDRESS;  
```
