# JsGetStringLength 関数
文字列値の最大長を取得します。  
  
## 構文  
  
```  
STDAPI_(JsErrorCode) JsGetStringLength(  
   _In_ JsValueRef stringValue,  
   _Out_ int *length  
);  
```  
  
#### パラメーター  
 `stringValue`  
 長さを取得する文字列値。  
  
 `length`  
 文字列の長さ。  
  
## 戻り値  
 操作が成功した場合はコード `JsNoError`、操作が失敗した場合はエラー コード。  
  
## 解説  
 アクティブ スクリプトのコンテキストが必要です。  
  
## 必要条件  
 **ヘッダー:** jsrt.h
