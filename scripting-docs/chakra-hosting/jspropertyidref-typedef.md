# <a name="jspropertyidref-typedef"></a>JsPropertyIdRef Typedef
プロパティの識別子。  
  
## <a name="syntax"></a>構文  
  
```  
typedef JsRef JsPropertyIdRef;  
```  
  
## <a name="remarks"></a>コメント  
 文字列を使用するのではなく、JavaScript オブジェクトのプロパティを参照する場合に、プロパティの識別子を使用します。  
  
## <a name="requirements"></a>要件  
 **ヘッダー:** jsrt.h
