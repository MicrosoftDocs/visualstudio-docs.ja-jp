# JsSetContextData 関数
JsrtContext の内部データを設定します。  
  
## 構文  
  
```  
STDAPI_(JsErrorCode) JsSetContextData(  
  _In_ JsContextRef context,  
  _In_ void *data  
);  
  
```  
  
#### パラメーター  
 `context`  
 データを設定するコンテキスト。  
  
 `data`  
 設定するデータへのポインター。  
  
## 戻り値  
 操作が成功した場合はコード `JsNoError`、操作が失敗した場合はエラー コード。  
  
## 解説  
 アクティブ スクリプトのコンテキストが必要です。  
  
## 必要条件  
 **ヘッダー:** jsrt.h
