# <a name="jsvalueref-typedef"></a>JsValueRef Typedef
JavaScript 値への参照。  
  
## <a name="syntax"></a>構文  
  
```  
typedef JsRef JsValueRef;  
```  
  
## <a name="remarks"></a>コメント  
 JavaScript 値は、未定義、Null、ブール値、文字列、数値、またはオブジェクトのいずれかの型の値です。  
  
## <a name="requirements"></a>要件  
 **ヘッダー:** jsrt.h
