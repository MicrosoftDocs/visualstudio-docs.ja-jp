# JsValueRef Typedef
JavaScript 値への参照。  
  
## 構文  
  
```  
typedef JsRef JsValueRef;  
```  
  
## 解説  
 JavaScript 値は、未定義、Null、ブール値、文字列、数値、またはオブジェクトのいずれかの型の値です。  
  
## 必要条件  
 **ヘッダー:** jsrt.h
