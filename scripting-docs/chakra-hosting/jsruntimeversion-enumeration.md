# <a name="jsruntimeversion-enumeration"></a>JsRuntimeVersion 列挙型
ランタイムのバージョン。  
  
## <a name="syntax"></a>構文  
  
```  
enum JsRuntimeVersion;  
```  
  
## <a name="members"></a>メンバー  
  
### <a name="values"></a>値  
  
|名前|説明|  
|----------|-----------------|  
|`JsRuntimeVersion10`|IE10 バージョンのランタイムを作成します。|  
|`JsRuntimeVersion11`|IE11 バージョンのランタイムを作成します。|  
|`JsRuntimeVersionEdge`|実行時にコンピューターに存在する最新バージョンのランタイムを作成します。|  
  
## <a name="requirements"></a>要件  
 **ヘッダー:** jsrt.h
