# JsRuntimeVersion 列挙型
ランタイムのバージョン。  
  
## 構文  
  
```  
enum JsRuntimeVersion;  
```  
  
## メンバー  
  
### 値  
  
|名前|説明|  
|--------|--------|  
|`JsRuntimeVersion10`|IE10 バージョンのランタイムを作成します。|  
|`JsRuntimeVersion11`|IE11 バージョンのランタイムを作成します。|  
|`JsRuntimeVersionEdge`|実行時にコンピューターに存在する最新バージョンのランタイムを作成します。|  
  
## 必要条件  
 **ヘッダー:** jsrt.h
