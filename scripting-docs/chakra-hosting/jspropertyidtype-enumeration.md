# JsPropertyIdType 列挙型
JavaScript のプロパティの列挙型を入力します。  
  
## 構文  
  
```vb  
enum JsPropertyIdType;  
  
```  
  
## メンバー  
  
### 値  
  
|名前|説明|  
|--------|--------|  
|JsPropertyIdTypeString|JavaScript 文字列のプロパティの列挙型を入力します。|  
|JsPropertyIdTypeSymbol|JavaScript シンボルのプロパティの列挙型を入力します。|  
  
## 必要条件  
 jsrt.h
