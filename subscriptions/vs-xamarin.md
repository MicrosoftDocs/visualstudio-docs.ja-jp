# <a name="activating-the-xamarin-university-training-benefit-in-visual-studio-subscriptions"></a>Visual Studio サブスクリプションの Xamarin University トレーニング特典をアクティブ化する

エキスパートの指導で、iOS や Android のネイティブ アプリを C# で作成する初心者向けビデオ (厳選されたクラス ビデオと教材)  

1.  Xamarin University トレーニング特典を使うには、特典タイルの下部にある **[コードの取得]** リンクをクリックします。   クーポン コードが正常に取得された通知を受け取ります。  **[アクティブ化]** をクリックして続行します。

![Xamarin University トレーニング特典タイル](_img\vs-xamarin\vs-xamarin-tile.png)

2.  [登録] ページで、連絡先情報を入力し、チェック ボックスをオンにして使用条件およびプライバシーの声明に同意し、**[特典の有効化]** をクリックします。 

![Xamarin University トレーニング特典の登録](_img\vs-xamarin\vs-xamarin-registration-resized.png)


3.  必要な作業は以上です。  これで Xamarin University トレーニング特典が有効になったので、Microsoft Visual Studio 用の Xamarin University モバイル トレーニング ページにリダイレクトされます。  

![Xamarin University トレーニング特典ホーム ページ](_img\vs-xamarin\vs-xamarin-home-resized.png)
