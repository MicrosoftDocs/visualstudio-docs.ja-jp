# <a name="buying-visual-studio-cloud-subscriptions---renewals"></a>Visual Studio Cloud サブスクリプションの購入: 更新

クラウドのサブスクリプションの更新は、自動的には行われ、Azure の課金内容に更新料金が表示されます。  以下に予想される結果の例を示します。
## <a name="monthly-subscriptions"></a>月間サブスクリプション
Microsoft は、最初の月を比例配分します。  

**例:** 15 日に購入した場合、最初に課金される料金は約 50% と予想されます。  それ以降の月は、毎月 1 日に完全な料金が課金されます。

## <a name="annual-subscriptions"></a>年間サブスクリプション
年間サブスクリプションは、購入日から 1 年後の次の月の 1 日に更新されます。  最初の更新後には、更新日の一貫性が維持されます。

**例:** 2018 年 1 月 15 日に購入したサブスクリプションの更新日は、2019 年 2 月 1 日になります。

## <a name="faq"></a>FAQ:
### <a name="q--what-if-i-dont-want-to-auto-renew"></a>Q: 自動更新をしたくない場合はどうすればよいですか。
A: 更新料金を回避するには、自動更新の日付より前に Visual Studio クラウド サブスクリプションをキャンセルする必要があります。  サブスクリプションのキャンセルを決めた場合でも、有効期間に残りがある場合は引き続き特典にアクセスできます。
