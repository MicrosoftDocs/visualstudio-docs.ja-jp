# <a name="visual-studio-dev-essentials"></a>Visual Studio Dev Essentials

Visual Studio Dev Essentials では、アプリを開発して任意のプラットフォームに展開するのに必要なものがすべて揃います。 最新のツール、強力なクラウド、トレーニング、サポートなど、これまでにない総合的な開発者向けプログラムが無料で提供されます。

Visual Studio Dev Essentials メンバーシップに含まれるすべてのものについては、[Dev Essentials](https://visualstudio.microsoft.com/dev-essentials/) を参照してください。

## <a name="opt-in-to-visual-studio-dev-essentials"></a>Visual Studio Dev Essentials に参加する

登録する準備ができたら、[[Join or access now]\(今すぐ参加またはアクセス\)](https://my.visualstudio.com/Benefits?wt.mc_id=o~msft~vscom~devessentials-hero~mt689&campaign=o~msft~vscom~devessentials-hero~mt689) ボタンをクリックするだけです。

これで完成です。

## <a name="opt-out-of-visual-studio-dev-essentials"></a>Visual Studio Dev Essentials を退会する

Visual Studio Dev Essentials プログラムの参加を解除したい場合は、次の手順に従ってください。

1. [https://my.visualstudio.com/subscriptions](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs) に移動します。
2. 自分のサブスクリプションにサインインします。  (方法については、[サインインに関するページ](signing-in.md)を参照してください)。
3. サインインできたら、現在のサブスクリプション一覧が表示されます。
4. Visual Studio Dev Essentials に関連付けられている行の右端の リンク 列の **プログラムへの参加解除** をクリックします。
5. ポップアップ ウィンドウが表示されます。 **[確認]** を選択します。

    > [!NOTE]
    > Visual Studio Dev Essentials プログラムに再度参加するのは簡単です。  単純に手順 1 ～ 5 に従い、リンク 列の上の **Visual Studio Dev Essentials に参加する** ボタンをクリックします。 次いで **[確認]** をクリックします。
