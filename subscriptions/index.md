# <a name="visual-studio-subscriptions"></a>Visual Studio サブスクリプション
##  <a name="developer-tools-cloud-services-software-support-and-training"></a>開発者ツール、クラウド サービス、ソフトウェア、サポート、トレーニング

Visual Studio サブスクリプションは、Windows、iOS、Android、Linux など、お好みのプラットフォームとデバイス上で優れたアプリケーションを作成、展開、管理するための最も包括的なリソース セットです。

# <a name="welcome-to-subscriptions"></a>サブスクリプションへようこそ!

お客様がサブスクライバーである場合でも、会社のサブスクリプションを管理している場合でも、Visual Studio サブスクリプションを最大限に活用するのに役立つ情報が見つかります。
