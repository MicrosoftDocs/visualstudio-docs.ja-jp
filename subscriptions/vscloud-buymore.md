# <a name="buying-visual-studio-cloud-subscriptions---buying-additional-subscriptions"></a>Visual Studio クラウド サブスクリプションの購入: サブスクリプションの追加購入

Visual Studio クラウド サブスクリプションを追加購入するには、元の Azure サブスクリプションの購入に使用したアカウントで、Visual Studio の[サブスクリプションの管理ポータル](https://manage.visualstudio.com/)にサインインします。

1.  左上隅で、正しい Azure サブスクリプションを選択していることを確認します。
2.  左側のナビゲーションで、購入するサブスクリプションの下の **[Buy More]**\(追加購入\) を選択します。
3.  サブスクリプションを希望の数量に増やし、**[続行]** をクリックします。
4.  注文を確認し、**[確認]** をクリックします。
5.  サブスクリプションの管理ポータルに戻ると、使用可能なサブスクリプションの数量が増えているはずです。

サブスクリプションの割り当てと管理のヘルプについては、[ドキュメント](/visualstudio/subscriptions/)を確認してください。

## <a name="faq"></a>FAQ:
### <a name="q--how-long-do-i-have-to-wait-before-i-can-assign-subscriptions"></a>Q: サブスクリプションの割り当てに、どのくらいの期間待機する必要がありますか?
A: サブスクリプションは、サブスクリプションの管理ポータルからすぐに割り当てることができます。
