# <a name="locating-your-subscription-id-for-visual-studio-subscriptions"></a>Visual Studio サブスクリプションのサブスクリプション ID を検索する

## <a name="what-is-a-subscription-id"></a>サブスクリプション ID とは何ですか。
"サブスクリプション ID" は、サブスクライバーのサブスクリプションまたはプログラムに関連付けられた 32 桁の GUID です。  状況によっては、特定のサブスクリプションの潜在的な問題を絞り込むために、サポート チームが、サブスクライバーにサブスクライバー ID をたずねることがあります。 

## <a name="how-to-find-your-subscription-id"></a>サブスクリプション ID を見つける方法
1. [https://my.visualstudio.com/subscriptions](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs) に移動します。
2. 自分のサブスクリプションにサインインします。  (方法については、[サインインに関するページ](/visualstudio/subscriptions/signing-in)を参照してください)。
3. サインインした後は、サインインに使用した電子メール アドレスに関連付けられているサブスクリプションの一覧が表示されます。  次のように表示されます。
    - サブスクリプション/プログラムの種類
    - サブスクリプション ID 
    - 種類 (製品版、NFR など)
    - 有効期限日   (有効期限)
    - 更新、サポート情報の取得のためのリンク。Visual Studio Dev Essentials の場合はプログラムの終了のためのリンク。
