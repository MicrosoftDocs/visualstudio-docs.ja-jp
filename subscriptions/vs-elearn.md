# <a name="activating-the-microsoft-e-learning-benefit-in-visual-studio-subscriptions"></a>Visual Studio サブスクリプションの Microsoft E-learning 特典をアクティブ化する

テクノロジを学び、スキルをアップし、マイクロソフト認定試験の準備をすることができます。  サブスクリプションによっては、1 つまたは 2 つのコース コレクションにアクセスできます。  

1.  Microsoft E-Learning の特典を使い始めるには、Microsoft E-Learning タイルの **[コードの取得]** をクリックします。 

    ![Microsoft E-learning 特典タイル](_img\vs-elearn\vs-elearn-tile.png)

2.  コードをクリップボードにコピーします。  特典をアクティブ化するのにコードが必要です。  **[アクティブ化]** をクリックして続行します。 

    ![Microsoft E-learning 特典のコード取得](_img\vs-elearn\vs-elearn-get-code.png)


3.  [Microsoft Imagine Academy](https://imagineacademy.microsoft.com/AccessCodeRedemption/enrollmentcode?channelid=6) サイトで、**[サインイン]** をクリックします。  作成した個人アカウントを使うか、あるいは勤務先または学校の IT 部門から提供された職場/学校アカウントを使うかの選択を求められる場合があります。 

    ![Microsoft E-learning 特典サインイン](_img\vs-elearn\vs-elearn-imagine-resized.png)


4.  前にコピーした登録コードとメール アドレスを入力します。  **[送信]** をクリックして続けます。  

    ![Microsoft E-learning 特典のご利用コード](_img\vs-elearn\vs-elearn-enter-code-resized.png)


5.  **[同意]** チェック ボックスをオンにして、使用条件、プライバシー、Cookie に関する声明を確認します。  **[続行]**をクリックします。  
6.  登録の確認が表示されます。  **[続行]**をクリックします。  
7.  プロファイルの連絡先情報を入力します。  チェック ボックスをオンにして、Imagine Academy、Microsoft 製品、その他の詳細を参照することもできます。  **[保存]**をクリックします。  (**[保存]** ボタンを表示するにはスクロール ダウンが必要な場合があります。)

    ![Microsoft E-learning 特典プロファイル](_img\vs-elearn\vs-elearn-full-profile.png)

8.  Microsoft E-learning ポータルへようこそ。 学習を始める準備ができました。

    ![Microsoft E-learning 特典ポータル](_img\vs-elearn\vs-elearn-portal.png)
