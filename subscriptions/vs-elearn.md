#  <a name="the-microsoft-e-learning-benefit-in-visual-studio-subscriptions"></a>Visual Studio サブスクリプションの Microsoft E-learning 特典

## <a name="overview"></a>概要
テクノロジを学び、スキルをアップし、マイクロソフト認定試験の準備をすることができます。  アクセスできるコース コレクションの数は、Visual Studio サブスクリプション レベルによって異なります。  すべてのサブスクリプションにこの特典が含まれているわけではありません。

|-------------------------------------------------------------------------------|-----------------------|--------------------------------------------------------------------|
| Visual Studio Enterprise Standard                                             | コース コレクション 2 つ  |  [はい]                                                               |
| Visual Studio Enterprise - 年間プラン                                               | コース コレクション 2 つ  |  [はい]                                                               |
| Visual Studio Enterprise - 月間プラン                                              | 使用できません         |                                                                    |
| Visual Studio Professional Standard                                           | コース コレクション 1 つ   |  [はい]                                                               |
| Visual Studio Professional - 年間プラン                                             | コース コレクション 1 つ   |  [はい]                                                               | 
| Visual Studio Professional - 月間プラン                                            | 使用できません         |                                                                    |
| Visual Studio Test Pro                                                        | コース コレクション 1 つ   |  [はい]                                                               |
| MSDN Platforms                                                                | コース コレクション 1 つ   |  [はい]                                                               |
| Visual Studio Dev Essentials                                                  | 使用できません         |                                                                    |
| Visual Studio Enterprise - NFR<sup>1</sup>                                               | コース コレクション 2 つ  |  [はい]                                                               |
| Visual Studio Enterprise - FTE                                                | コース コレクション 2 つ  |  [はい]                                                               |

どのサブスクリプション使用しているかわからない場合は次の手順を実行してください。  [https://my.visualstudio.com/subscriptions](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs) に接続し、電子メール アドレスに割り当てられているすべてのサブスクリプションを表示します。 すべてのサブスクリプションが表示されない場合は、1 つ以上のサブスクリプションが別のメール アドレスに割り当てられている可能性があります。  それらのサブスクリプションを表示するには、そのメール アドレスを使用してサインインする必要があります。 

## <a name="activation-steps"></a>アクティブ化の手順
1.  [https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs) にサインインします。

2.  [特典] ページの [プロフェッショナルな開発] セクションで、Microsoft E-learning のタイルを見つけ、特典タイルの下部にある **[コードを取得]** リンクをクリックします。

    ![Microsoft E-learning 特典タイル](_img\vs-elearn\vs-elearn-tile.png)

2.  コードをクリップボードにコピーします。  特典をアクティブ化するのにコードが必要です。  **[アクティブ化]** をクリックして続行します。 

    ![Microsoft E-learning 特典のコード取得](_img\vs-elearn\vs-elearn-get-code.png)


3.  [Microsoft Imagine Academy](https://imagineacademy.microsoft.com/AccessCodeRedemption/enrollmentcode?channelid=6) サイトで、**[サインイン]** をクリックします。  作成した個人アカウントを使うか、あるいは勤務先または学校の IT 部門から提供された職場/学校アカウントを使うかの選択を求められる場合があります。 

    ![Microsoft E-learning 特典サインイン](_img\vs-elearn\vs-elearn-imagine-resized.png)


4.  前にコピーした登録コードとメール アドレスを入力します。  **[送信]** をクリックして続けます。  

    ![Microsoft E-learning 特典のご利用コード](_img\vs-elearn\vs-elearn-enter-code-resized.png)


5.  **[同意]** チェック ボックスをオンにして、使用条件、プライバシー、Cookie に関する声明を確認します。  **[続行]**をクリックします。  
6.  登録の確認が表示されます。  **[続行]**をクリックします。  
7.  プロファイルの連絡先情報を入力します。  チェック ボックスをオンにして、Imagine Academy、Microsoft 製品、その他の詳細を参照することもできます。  **[保存]**をクリックします。  (**[保存]** ボタンを表示するにはスクロール ダウンが必要な場合があります。)

    ![Microsoft E-learning 特典プロファイル](_img\vs-elearn\vs-elearn-full-profile.png)

8.  Microsoft E-learning ポータルへようこそ。 学習を始める準備ができました。

    ![Microsoft E-learning 特典ポータル](_img\vs-elearn\vs-elearn-portal.png)
