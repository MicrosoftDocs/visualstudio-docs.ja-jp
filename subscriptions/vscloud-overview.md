# <a name="buying-visual-studio-cloud-subscriptions---overview"></a>Visual Studio Cloud サブスクリプションの購入: 概要

クラウド サブスクリプションによって、Visual Studio、VSTS、および必要なサブスクライバー特典をレンタルできます。長期の契約は必要ありません。 課金は Microsoft Azure サブスクリプションを通して処理され、クレジットカード払いと Enterprise Agreement 契約*による購入がサポートされます。さらに、[クラウド ソリューション プロバイダー](https://www.microsoft.com/solution-providers/search) パートナーを通しての購入もサポートされます。 まだ Azure サブスクリプションをお持ちでなくても、心配ありません。最初に購入を行う際に、無料でサブスクリプションをセットアップして支払い手続きに使用できます。

クラウド [サブスクリプションの特典と価格](https://www.visualstudio.com/vs/pricing/)の比較を参照してください。

## <a name="before-you-start"></a>開始する前に
購入手続きをスムーズに進めるために、購入前に次の手順を実行してください。•   購入料金の請求のために、[Azure サブスクリプション](https://azure.microsoft.com/pricing/purchase-options/)が必要です。  Visual Studio Marketplace で購入する前、または初めての購入時に、[サインアップ](https://portal.azure.com/)することができます。  

•   Azure サブスクリプションにサインアップして、Visual Studio Marketplace から購入するには、[個人の Microsoft アカウント](https://www.microsoft.com/account)、所属する組織用に購入し、組織が ID を Azure Active Directory (Azure AD) で管理する場合は[職場または学校のアカウント](https://azure.microsoft.com/en-us/documentation/articles/sign-up-organization/) が必要になります。
