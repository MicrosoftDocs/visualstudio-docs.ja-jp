#   <a name="how-to-buy-a-subscription"></a>サブスクリプションの購入方法
Visual Studio サブスクリプションの購入には 3 つの方法があります。  
- クラウド サブスクリプションは、Visual Studio Marketplace から購入できます。
- 標準サブスクリプションは、Microsoft ストアから購入できます。 
- 組織は、販売店を通じてボリューム ライセンス サブスクリプションを購入できます。  

## <a name="buying-cloud-subscriptions-through-the-visual-studio-marketplace"></a>Visual Studio Marketplace からのクラウド サブスクリプションの購入
Visual Studio Professional と Visual Studio Enterprise の[サブスクリプション](https://www.visualstudio.com/subscriptions/)は、[Visual Studio Marketplace](https://marketplace.visualstudio.com) から購入できます。 これらのサブスクリプションのすべての特典について理解し、IDE の[機能を比較](https://www.visualstudio.com/vs/compare/)し、[サブスクリプション料金](https://www.visualstudio.com/vs/pricing/)を見つけます。

### <a name="before-you-start"></a>開始する前に

*   購入に対する請求のため、[Azure サブスクリプション](https://azure.microsoft.com/en-us/pricing/purchase-options/)が必要になります。 Visual Studio Marketplace で初めて購入する前、または購入時に、[サインアップ](https://account.windowsazure.com/Subscriptions)することができます。
Visual Studio Marketplace での購入に使用できる Azure サブスクリプションについては、[こちら](https://docs.microsoft.com/en-us/vsts/marketplace/marketplace-billing-qa)を参照してください。 

*   Azure サブスクリプションにサインアップして、Visual Studio Marketplace から購入するには、"個人" の [Microsoft アカウント](https://www.microsoft.com/account)、所属する組織用に購入し、組織が ID を Azure Active Directory (Azure AD) で管理する場合は ["職場または学校のアカウント"](https://azure.microsoft.com/en-us/documentation/articles/sign-up-organization/) が必要になります。

### <a name="buy-subscriptions"></a>サブスクリプションの購入


1.  [Visual Studio Marketplace > サブスクリプション](https://marketplace.visualstudio.com/subscriptions)にサインインします。

2.  購入するサブスクリプションを選択します。次に例を示します。

    <img alt="Buy a Visual Studio subscription" src="_img/buy-vs-subscriptions/buy-vs-sub-start.png" style="border: 1px solid #CCCCCC" />

3.  請求に使用する Azure サブスクリプションを選択します。
まだサブスクリプションがない場合は、新しいサブスクリプションのためにサインアップします。

    <img alt="Select Azure subscription for biling" src="_img/buy-vs-subscriptions/buy-vs-sub-Azure-sub.png" style="border: 1px solid #CCCCCC" />

4.  購入するサブスクリプションの数を選択します。

    <img alt="Select number of subscriptions" src="_img/buy-vs-subscriptions/buy-vs-sub-users.png" style="border: 1px solid #CCCCCC" />

    サブスクリプションを自分に割り当てない場合は、**[自分にこのサブスクリプションを割り当てる]** をオフにします。

5.  確認して購入を完了します。 ここでサブスクリプションの特典を確認するには、Visual Studio サブスクリプション ポータル [(https://my.visualstudio.com)](https://my.visualstudio.com) に移動するか、**[自分のサブスクリプションを表示する]** をクリックします。

    <img alt="Finish purchase successfully" src="_img/buy-vs-subscriptions/buy-vs-sub-success.png" style="border: 1px solid #CCCCCC" />

    <img alt="Go to Visual Studio subscriptions portal" src="_img/buy-vs-subscriptions/view-subscription-benefits-subscriptions-portal.png" style="border: 1px solid #CCCCCC" />

    複数のサブスクリプションを購入した場合は、[Visual Studio サブスクリプション管理ポータル]( https://manage.visualstudio.com/cloud)で、または **[ユーザーへの割り当て]** をクリックして、他のユーザーにサブスクリプションを割り当てることができます。

    <img alt="Assign subscriptions to others" src="_img/buy-vs-subscriptions/buy-vs-sub-success-many.png" style="border: 1px solid #CCCCCC" />

    <img alt="View subscriptions" src="_img/buy-vs-subscriptions/assign-subscriptions.png" style="border: 1px solid #CCCCCC" />

## <a name="buying-standard-subscriptions-through-the-microsoft-store"></a>Microsoft ストアから標準サブスクリプションを購入する
Visual Studio の標準サブスクリプションは、Microsoft ストアを通して入手できます。  サブスクリプションを購入または更新するには、次の手順を実行します。

1. [Microsoft ストア](https://www.microsoft.com/store) に接続します。
2. **[ソフトウェア]** タブをクリックします。
3. **[Professional software]\(プロクラスのソフトウェア\)** を選択します。

    <img alt="Software Menu" src="_img/buy-vs-subscriptions/professional-software.png" style="border: 1px solid #CCCCCC" />

4. **ショップ カテゴリ**の一覧で、**[Visual Studio]** を選択します。

    <img alt="Categories List" src="_img/buy-vs-subscriptions/shop-categories.png" style="border: 1px solid #CCCCCC" />

5. **Visual Studio** の一覧で、購入するサブスクリプションを選択します  (Visual Studio Professional 2017 は IDE のスタンドアロン ライセンスで、サブスクリプションは含まれていません)。

    <img alt="Visual Studio Products" src="_img/buy-vs-subscriptions/shop-visual-studio.png" style="border: 1px solid #CCCCCC" />

6. 製品の説明ページでは、サブスクリプションの更新または新規購入を選択できます。  目的のサブスクリプションの種類を選択して、カートに追加します。 

    <img alt="Visual Studio Enterprise options" src="_img/buy-vs-subscriptions/enterprise-options.png" style="border: 1px solid #CCCCCC" />

7. カート内で、**[購入手続き]** ボタンをクリックして、**確認と注文**のページに移動します   (続行するには、Microsoft アカウントにサインインする必要があります)。 

8. 支払いの種類を選択し、**[選択]** をクリックします。

9. ファイルに支払い方法がある場合は、それを選択します。  クレジット カードなどの支払い方法を追加するには、支払い方法の詳細を入力し、**[保存]** をクリックします。

10. **[注文]** をクリックして購入を完了します。  

### <a name="activating-subscriptions-purchased-through-the-microsoft-store"></a>Microsoft ストアから購入した標準サブスクリプションをアクティブ化する

Microsoft ストアで Visual Studio サブスクリプションを購入した場合は、支払いの確認ページまたは確認メールからライセンス認証手順を始めることができます。 注: メールが届いていない場合は、メール アドレスを確認し、迷惑メール フォルダーを確認してください。

サブスクリプションをアクティブ化するには、次の手順を実行します。 

1. **[Click here to sign in]\(サインインするにはここをクリックしてください\)** をクリックします。 [https://my.visualstudio.com/subscriptions/activate](https://my.visualstudio.com/subscriptions/activate) にリダイレクトされます。

2. Visual Studio サブスクリプションへのアクセスに使用するメール アドレスを入力して、**[続行]** をクリックします。

3. メール アドレスによっては、**Microsoft アカウント (MSA)** サインイン ページまたは**職場/学校**アカウントのサインイン ページにリダイレクトされる場合があります。 

4. パスワードを入力するか、サインインに必要な情報を入力します。
5. サインインに成功すると、**[Activate Your Visual Studio Subscription]\(Visual Studio サブスクリプションをアクティブ化する\)** ウィンドウが表示されます。
6. 支払い確認ページまたは確認メールで入手できる**プロダクト キー**情報を入力し、**[アクティブ化]** をクリックします。

7. [https://my.visualstudio.com/subscriptions](https://my.visualstudio.com/subscriptions) に自分のサブスクリプションが表示されるはずです。


## <a name="buying-standard-subscriptions-through-volume-license-programs"></a>ボリューム ライセンス プログラムから標準サブスクリプションを購入する


複数のサブスクリプションの特典を必要とする組織は、さまざまなボリューム ライセンス (VL) プログラムを通じてそれらを購入できます。  ボリューム ライセンスは、Microsoft の認定パートナーを通じて販売されています。  
[ボリューム ライセンス](https://www.microsoft.com/Licensing/how-to-buy/how-to-buy.aspx)を通じた購入方法をご確認ください。 [Marketplace と請求の Q&A](/vsts/marketplace/marketplace-billing-qa) で、Visual Studio Marketplace と Azure の請求に関する複数の一般的な質問に対する回答をご覧ください。
