# <a name="buying-visual-studio-cloud-subscriptions---cancelling-subscriptions"></a>Visual Studio クラウド サブスクリプションの購入: サブスクリプションのキャンセル

Visual Studio クラウド サブスクリプションをキャンセルするには、元の Azure サブスクリプションの購入に使用したアカウントで、Visual Studio の[サブスクリプションの管理ポータル](https://manage.visualstudio.com/)にサインインします。
1.  左上隅で、正しい Azure サブスクリプションを選択していることを確認します。
2.  左側のナビゲーションで、キャンセルするサブスクリプションの下の **[Reduce]**\(削減\) を選択します。
3.  アカウントに関連付けられているサブスクリプションの数量を調整するウィンドウが表示されます。
4.  希望の数量に減らすか、ゼロ (0) でサブスクリプションをキャンセルして、**[続行]** をクリックします。
5.  注文を確認し、**[確認]** をクリックすると完了です。
## <a name="faq"></a>FAQ:
### <a name="q--can-i-still-use-a-subscription-after-i-cancel"></a>Q: キャンセル後にもサブスクリプションは使用できますか?
A: はい、サブスクリプションの有効期間に残りがある場合、引き続き特典にアクセスできます。
