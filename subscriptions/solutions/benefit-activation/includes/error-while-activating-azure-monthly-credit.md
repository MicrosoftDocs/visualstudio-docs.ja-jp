## <a name="im-getting-an-error-while-trying-to-activate-my-azure-monthly-credit"></a>Azure の月単位クレジットをアクティブにしようとするとエラーが表示されます。

Visual Studio サブスクリプションに関連付けられている Azure サブスクリプションあたり、Azure Dev/Test 個別クレジット特典が 1 つ与えられます。 Azure の月単位クレジットを以前アクティブにしている場合、エラーが表示されることがあります。 クレジットをアクティブにしたかどうかわからない場合、[Azure portal](https://portal.azure.com/) にログインしてご確認ください。 

## <a name="verify-your-email"></a>電子メールを確認する 

月単位クレジットをアクティブにすると、Visual Studio サブスクリプションをアクティブにするときに使用した電子メールがリンクされます。  

Visual Studio サブスクリプションに[代替アカウント](https://docs.microsoft.com/visualstudio/subscriptions/vs-alternate-identity)を関連付けている可能性があります。 Visual Studio サブスクリプションに代替アカウントを追加すると、サブスクリプションが割り当てられているものとは異なる ID を利用して、Azure DevOps や Azure などのサブスクリプションの特典にアクセスできます。  

アカウント設定に使用した電子メールで [Azure portal](https://portal.azure.com/) にサインインする必要があります。 たとえば、代替アカウントで月単位クレジットをアクティブにした場合、その電子メールで Azure サブスクリプションに常にサインインする必要があります。 ポータルの[サブスクリプション ページ](https://my.visualstudio.com/subscriptions)の一番下で代替アカウントを追加しているかどうかを確認できます。
