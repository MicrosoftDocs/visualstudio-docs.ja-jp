## <a name="im-unable-to-activate-my-azure-dev--test-pay-as-you-go-subscription"></a>Azure 開発テスト用の従量課金制サブスクリプションをアクティブ化できません。

Visual Studio サブスクライバーとして、Azure でのサブスクリプション ソフトウェアの開発テストとクラウドの使用権を、特別割引価格 (時間単位) で利用できます。 

## <a name="set-up-a-subscription"></a>サブスクリプションを設定する

1. [サブスクリプション ポータル](https://my.visualstudio.com/benefits)にサインインして、Azure 開発テスト用の従量課金制の特典タイルの下部にあるアクティブ化リンクをクリックします。
1. Microsoft Azure のサインアップ ページでは、開発テスト用の従量課金制プラン (PAYG) を設定したり、Visual Studio Enterprise のプランにアップグレードしたりできます。 アップグレードしない場合は、[Continue with the current offer (Pay-As-You-Go Dev/Test)]\(現在のプランで続行する (開発テスト用の従量課金制プラン)\) をクリックします。 
1. 支払い方法を追加し、サブスクリプション契約に同意すれば準備完了です。 これ以降は、[portal.azure.com](https://portal.azure.com/) からサブスクリプションにアクセスできます。 

## <a name="more-information-about-azure-devtest-pay-as-you-go-subscriptions"></a>詳細については、Azure Dev/Test 用の従量課金制のサブスクリプションに関する記事をご覧ください。
 
詳細については、[Azure Dev/Test 用の従量課金制のサブスクリプション](https://docs.microsoft.com/visualstudio/subscriptions/vs-azure-payg)に関する記事をご覧ください。
