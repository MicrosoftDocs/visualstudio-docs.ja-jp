## <a name="how-to-request-a-visual-studio-subscription-from-your-workschool"></a>職場や学校に Visual Studio サブスクリプションを要求する方法
職場や学校を含め、多くの組織ではさまざまなライセンス プログラムを通じて Visual Studio サブスクリプションを購入します。  これらのサブスクリプションは、サブスクリプションの割り当て、基本設定の確立、特定の機能の有効化または無効化を行うことができるサブスクリプション管理者によって管理されます。  

**サブスクリプション管理者に連絡してサブスクリプションを要求する** 組織で管理されているサブスクリプションを使用するには、最初に管理者によってそれがユーザーに割り当てられる必要があります。サブスクリプションを要求するには、管理者に連絡する必要があります。  大規模な組織では、管理者を把握するのが難しい場合があります。  多くの組織に複数の管理者がいます。  

管理者がわからない場合は、サブスクリプションが割り当てられている同僚に連絡すると役立つことがあります。  既存のサブスクライバーは https://my.visualstudio.com/subscriptions にアクセスできます。  右上に **[Contact my Admin]\(管理者に問い合わせる\)** ボタンが表示されます。これを使用して、管理者にメッセージを送信することができます。  しかし、このボタンを使用しても管理者の名前や電子メールは公開されないため、同僚は、要求について連絡するよう管理者に依頼する必要があります。  

**[Contact my Admin]\(管理者に問い合わせる\) 機能を使用する** この機能の詳細については、[管理者への問い合わせ](https://docs.microsoft.com/visualstudio/subscriptions/contact-my-admin)に関する記事を参照してください。
