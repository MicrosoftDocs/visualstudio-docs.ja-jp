## <a name="resolve-issues-signing-in-to-visual-studio-subscriptions-administration-portal"></a>Visual Studio サブスクリプション管理ポータルへのサインインに関する問題を解決する
職場または学校がボリューム ライセンスまたは Visual Studio Marketplace を通じてサブスクリプションを購入した場合は、管理者によって管理されます。  次の 2 種類があります。• 管理者は、サブスクリプションの割り当て、編集、削除を行うことができます。
• スーパー管理者は、管理者と同じ機能を実行できるだけでなく、契約の基本設定を行い、他の管理者やスーパー管理者を指名することもできます。  

**サインインに関する一般的な問題** 管理者がサブスクリプション管理ポータル (https://manage.visualstudio.com ) にアクセスできない場合は、ポータルへのサインインに使用しているメール アドレスが正しくない場合があります。  問題を解決するには、スーパー管理者にお問い合わせください。  誰がスーパー管理者かわからない場合は、契約での管理者がその情報を提供できます。  それは **[契約の詳細]** に一覧表示され、ポータルの左側上部にある 2 番目のボタンを使用してアクセスできます。

**[Contact my Admin]\(管理者に問い合わせる\) を使用する** サブスクリプションも割り当てられているユーザーの場合は、Visual Studio サブスクリプション ポータルの **[Contact my Admin]\(管理者に問い合わせる\)** 機能にアクセスして、管理者やスーパー管理者にメッセージを送信できます。  (これらのメッセージを誰が受け取るかは、スーパー管理者が決定します)。

スーパー管理者に連絡できない場合は、調達組織への連絡が必要になることがあります。  Visual Studio サブスクリプション契約の主要連絡先または通知連絡先は、自動的にスーパー管理者として登録され、ユーザーを支援できます。  1 人が組織を離れただけでスーパー管理者がいなくなることがないように、複数のスーパー管理者を登録しておくことをお勧めします。

**サインインの後でサブスクリプションが表示されない**
ポータルにログインできても、管理対象のサブスクリプションが見つからない場合は、複数の契約で管理者になっていて、現在間違った契約を表示しているおそれがあります。  契約を切り替えるには、左上隅にあるドロップダウン リストで、目的の契約を選択します。
