## <a name="why-cant-i-access-my-subscription"></a>サブスクリプションにアクセスできないのはなぜですか?

サブスクリプションにアクセスできない場合は、次のような理由がいくつか考えられます。

**サブスクリプションの有効期限が切れている可能性があります。**  Microsoft Store からサブスクリプションを購入した場合は、新しいサブスクリプションのコストに比べて割引価格で更新することができます。  サブスクリプションを更新するのに役立つクイック リンクをいくつか以下に示します。
- [Visual Studio Enterprise](https://www.microsoft.com/p/visual-studio-enterprise-subscription/dg7gmgf0dst4?activetab=pivot%3aoverviewtab)
- [Visual Studio Professional](https://www.microsoft.com/p/visual-studio-professional-subscription/dg7gmgf0dst3?activetab=pivot%3aoverviewtab)
- [Visual Studio Test Professional](https://www.microsoft.com/p/visual-studio-test-professional-subscription/dg7gmgf0dst6?activetab=pivot%3aoverviewtab)

**サブスクリプションが管理者によって削除または再割り当てされた可能性があります。**  サブスクリプションが職場または学校から提供されている場合、管理者によって管理されていることがあります。  以前はサブスクリプションにアクセスできていた場合、管理者によって削除されたか、別のユーザーに再割り当てされた可能性があります。  詳細については、[管理者にお問い合わせください](https://docs.microsoft.com/visualstudio/subscriptions/contact-my-admin)。

**管理者によってサブスクリプションが割り当てられていない可能性があります。**  職場または学校で組織のサブスクリプションが管理されている場合、サブスクリプションが割り当てられていない可能性があります。  割り当てられている場合は、サブスクリプションの割り当てを確認する電子メール メッセージが表示されるはずです。  サブスクリプションにサインインする際にその通知を受け取ったのと同じメール アドレスを使用していることを確認してください。 詳細については、[管理者にお問い合わせください](https://docs.microsoft.com/visualstudio/subscriptions/contact-my-admin)。 

**間違った資格情報でサインインしている可能性があります。**  サブスクリプションがあるのは確実であるにもかかわらず、サインインできないか、サインインできてもサブスクリプションが表示されない場合は、サブスクリプションに関連付けられているものとは異なるメール アドレスでサインインしている可能性があります。  複数のサブスクリプションがある場合は、単に間違ったものを表示している可能性もあります。  サブスクリプションを切り替えるには、特典ページの上部にあるサブスクリプション ピッカーを使用します。  サインイン メール アドレスに関連付けられているすべてのサブスクリプションを表示して、使用するものを選択する場合は、[サブスクリプション タブ](https://my.visualstudio.com/subscriptions)を参照してください。
