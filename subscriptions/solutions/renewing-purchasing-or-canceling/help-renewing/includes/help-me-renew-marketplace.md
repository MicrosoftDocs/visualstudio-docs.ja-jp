## <a name="renewing-subscriptions-through-visual-studio-marketplace"></a>Visual Studio Marketplace からサブスクリプションを更新する 

支払い方法が引き続き有効である限り、Marketplace からの Visual Studio サブスクリプションは自動更新されます。 詳細については、[Visual Studio サブスクリプション Azure の請求](https://docs.microsoft.com/visualstudio/subscriptions/vscloud-billing-faq)に関する記事をご覧ください。 

サブスクリプションを更新しない場合は、[Visual Studio Marketplace](https://marketplace.visualstudio.com/subscriptions) で有料ユーザーの数を減らします。 新しい月額料金とユーザー数の削減は、翌月の 1 日に有効になります。
