## <a name="im-unable-to-download-or-activate-visual-studio-for-mac"></a>Visual Studio for Mac をダウンロードまたはアクティブにできない

Visual Studio のサブスクライバーは、Visual Studio IDE にアクセスできます。 Visual Studio for Mac は、サブスクリプション ポータルの[ダウンロード ページ](https://my.visualstudio.com/Downloads)から検索できる、個別のダウンロードです。  

## <a name="find-and-download-visual-studio-for-mac"></a>Visual Studio for Mac を検索してダウンロードする 
1. ダウンロード可能な Visual Studio for Mac は、[サブスクリプション ポータル](https://my.visualstudio.com/benefits)にサインインし、[フィルター処理されたビュー](https://my.visualstudio.com/Downloads?q=Visual%20Studio%20for%20mac&pgroup=)で表示できます。 最新の製品バージョンのみを提供しています。 
2. ダウンロードと設定の手順に従ってください。 
1. 自分のサブスクリプションが割り当てられているメールを使用すると、インストール後すぐに IDE にサインインできます。プロダクト キーは必要ありません。

## <a name="no-download-available"></a>ダウンロード可能なものがない 
使用しているサブスクリプションによって、ダウンロード可能なソフトウェアは異なります。 探しているダウンロードが表示されない場合は、利用できない可能性があります。 使用できるサブスクリプションを確認するには、ポータルの[サブスクリプション ページ](https://my.visualstudio.com/subscriptions)にログインします。 すべての種類のサブスクリプションのソフトウェア タイトルの[広範な一覧をダウンロード](https://download.microsoft.com/download/1/5/4/15454442-CF17-47B9-A65D-DF84EF88511B/Visual_Studio_by_Subscription_Level.xlsx) することができます。
