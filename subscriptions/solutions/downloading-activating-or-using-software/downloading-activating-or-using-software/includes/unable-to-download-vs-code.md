## <a name="im-unable-to-download-or-activate-visual-studio-code"></a>Visual Studio Code をダウンロードまたはアクティブにできない

Visual Studio Code は無料のコード エディターであり、使用するために有料の Visual Studio サブスクリプションは必要ありません。 Visual Studio の Web サイトから、[直接これをダウンロード](https://code.visualstudio.com/download)できます。
