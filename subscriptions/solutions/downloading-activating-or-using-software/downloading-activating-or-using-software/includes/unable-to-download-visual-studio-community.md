## <a name="im-unable-to-download-or-activate-visual-studio-community"></a>Visual Studio Community をダウンロードまたはアクティブにできない
Visual Studio Community は、IDE の無料の軽量バージョンであり、使用するために有料の Visual Studio のサブスクリプションは必要ありません。 Visual Studio の Web サイトから、[直接これをダウンロード](https://visualstudio.microsoft.com/vs/community/)
