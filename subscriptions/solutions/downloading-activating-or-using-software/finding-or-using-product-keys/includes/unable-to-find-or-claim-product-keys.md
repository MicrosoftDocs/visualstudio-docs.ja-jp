## <a name="im-unable-to-find-or-claim-a-product-key"></a>プロダクト キーを見つけることも要求することもできません。

使用できるプロダクト キーは、ご利用の Visual Studio サブスクリプションによって異なります。  

## <a name="view-a-list-of-available-product-keys"></a>利用可能なプロダクト キーの一覧を表示する 

[[プロダクト キー]](https://my.visualstudio.com/productkeys)  ページには、使用できるすべてのキーがまとめて一覧表示されます。 キーを要求したタイミングと理由を特定するためのメモ欄を編集できます。 キーには製品によって異なる種類があります。 キーの中には、要求する必要があるものとすぐに表示できるものがあります。 

キーや製品が表示されない場合、ご利用のサブスクリプション レベルでその製品が使用できない可能性があります。 すべてのサブスクリプションの種類で使用可能なソフトウェアのタイトルが示されている[広範な一覧をダウンロードする](https://download.microsoft.com/download/1/5/4/15454442-CF17-47B9-A65D-DF84EF88511B/Visual_Studio_by_Subscription_Level.xlsx)ことができます。 自分に割り当てられている種類を確認するには、[サブスクリプション ポータル](https://my.visualstudio.com/subscriptions)でご自分のサブスクリプションを表示します。  

## <a name="more-information-on-product-key-types"></a>プロダクト キーの種類に関する詳細情報

詳細については、[プロダクト キーの種類とプロダクト キーの要求](https://docs.microsoft.com/visualstudio/subscriptions/find-keys)に関する記事をご覧ください。
