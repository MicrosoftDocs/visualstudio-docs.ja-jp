## <a name="what-if-you-cant-find-the-ide-feature-you-want-to-use"></a>使用したい IDE 機能が見つからない場合はどうすればよいでしょうか。 

使用できる IDE の特定のバージョンと機能は、サブスクリプションのレベルによって異なります。 Visual Studio は、[サブスクリプション ポータル](https://my.visualstudio.com/benefits)の特典ページにツール カテゴリのタイルとして表示されます。 

## <a name="compare-supported-visual-studio-features"></a>サポートされている Visual Studio 機能の比較 

サブスクリプションに含まれている Visual Studio のバージョンを確認した後に、[サポートされている機能を比較](https://visualstudio.microsoft.com/vs/compare/)して、使用している IDE でその機能が使用可能かどうかを確認します。 いずれかの機能が含まれていないバージョンの IDE を使用している場合は、正しい IDE をダウンロードする必要があります。 

## <a name="upgrade-your-subscription-level"></a>サブスクリプション レベルをアップグレードする  

この機能が別のサブスクリプション レベルに固有のものである場合は、サブスクリプション レベルを変更する必要があります。  

職場または学校から Visual Studio サブスクリプションが与えられる場合、おそらく、サブスクリプション管理者からそれを割り当てられています。 [サブスクリプション ポータル](https://my.visualstudio.com/benefits)で、ナビゲーションの右隅にある [contact my admin]\(管理者に問い合わせる\) ボタンを使用して、組織の管理者に連絡してください。  

サブスクリプションを直接購入した場合、または管理者がいない場合は、必要なレベルの新しいサブスクリプションを購入する必要があります。  

## <a name="how-to-contact-your-admin"></a>管理者に連絡する方法 

多くの組織の規模と複雑さを考えると、管理者が誰かがわからない場合があります。しかし、サブスクリプション ポータルには、管理者への連絡を支援する機能があります。 

1. [サブスクリプション ポータル](https://my.visualstudio.com/benefits)にサインインします。  
1. 右上にある [Contact my Admin]\(管理者に問い合わせる\) をクリックします。 
1. 管理者へのメッセージを入力します。 
1. 管理者に送信されたメッセージのコピーを受け取る場合は、[Send me a copy]\(コピーを送る\) チェックボックスをオンにします。 最後に、[送信] をクリックします。
