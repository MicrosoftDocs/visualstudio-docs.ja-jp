## <a name="windows-developer-technical-support"></a>Windows 開発者向けテクニカル サポート  

Visual Studio サブスクリプション サポートでは、Visual Studio サブスクリプションの販売、サブスクリプション、アカウント、課金に関するサポートが提供されます。 Windows テクニカルまたは開発者アカウントに関する問題がある場合は、[Windows 開発者向けサポート](https://developer.microsoft.com/windows/support/?utm_source=developermscom)にお問い合わせいただく必要があります。 以下の他のリソースを先に試してみることをお勧めします。 

## <a name="windows-technical-documentation"></a>Windows テクニカル ドキュメント 
「[Windows テクニカル ドキュメント](https://docs.microsoft.com/windows/)」に含まれているチュートリアル、トラブルシューティング ガイド、役に立つリソース、その他の各種トピックを確認します。

## <a name="windows-developer-account"></a>Windows 開発者アカウント
お使いの Visual Studio サブスクリプションには Windows 開発者アカウントが含まれていることがあります。 アカウントをアクティブにするには、[サブスクリプション ポータルの特典ページ](https://my.visualstudio.com/Benefits)にアクセスし、特典タイルをクリックします。 

詳細については、[Windows 開発者アカウントの特典](https://docs.microsoft.com/visualstudio/subscriptions/vs-windows-dev)に関する記事を参照してください。 

[Visual Studio サブスクリプションの各特典を比較](https://visualstudio.microsoft.com/vs/benefits/#azure?cat=visual-studio-enterprise-subscription)して、ご利用のサブスクリプションに含まれる内容を確認します。
