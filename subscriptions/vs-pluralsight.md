# <a name="activating-the-pluralsight-training-benefit-in-visual-studio-subscriptions"></a>Visual Studio サブスクリプションの Pluralsight トレーニング特典をアクティブ化する

Pluralsight と Code School の全カタログで、スキルを向上させましょう。  エキスパートが作成したコース、ラーニング パス、適応スキルの測定方法などを利用しましょう。  特典は、お持ちのサブスクリプションの種類によって異なります。  

| サブスクリプション                                                     | 特典                              |
|------------------------------------------------------------------|--------------------------------------|
| Visual Studio Enterprise の標準および年間クラウド サブスクリプション   | 完全な年間サブスクリプション             |
| Visual Studio Professional の標準および年間クラウド サブスクリプション | 選択した 40 コースへの 12 か月間のアクセス |
| Visual Studio Test Professional                                  | 選択した 40 コースへの 12 か月間のアクセス |
| MSDN Platforms                                                   | 選択した 40 コースへの 12 か月間のアクセス |
| Visual Studio Dev Essentials                                     | 完全なカタログへの 3 か月間のアクセス  |  

  

1. サブスクリプションをアクティブ化するには、Pluralsight 特典タイルの **[コードを取得]** リンクをクリックします。

![Pluralsight 特典タイル](_img\vs-pluralsight\vs-pluralsight-tile.png)

2. クーポン コードが正常に取得された通知を受け取ります。  **[アクティブ化]** をクリックします。   

![Pluralsight 特典ランディング ページ](_img\vs-pluralsight\vs-pluralsight-landing-crop.png)

3.  アクティブ化ページで次のようにします。
- フォームで要求された連絡先情報を入力します 
- チェック ボックスをオンにして、使用条件とプライバシー ポリシーに同意します
- **[I agree, activate benefit]\(特典のアクティブ化に同意します\)** をクリックします。 

![Pluralsight 特典アクティブ化](_img\vs-pluralsight\vs-pluralsight-registration-crop.png)

4.  それで完了です!  勉強したいときはいつでも、[Pluralsight](https://app.pluralsight.com/id?) に接続してサインインしてください。  

![Pluralsight 特典サインイン](_img\vs-pluralsight\vs-pluralsight-sign-in.png)
