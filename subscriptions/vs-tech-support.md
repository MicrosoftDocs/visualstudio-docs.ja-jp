# <a name="activating-the-technical-support-benefit-in-visual-studio-subscriptions"></a>Visual Studio サブスクリプションのテクニカル サポート特典をアクティブ化する

非運用環境で発生した障害対応の問題を解決します。  (注: サポート リクエストを送信するには、その前に特典を有効にする必要があります)。Visual Studio Enterprise の Standard および年間のクラウド サブスクリプションは、4 つのインシデントを受け取り、Visual Studio Professional Standard および年間のクラウド サブスクリプションが 2 つのインシデントを受け取ります。  

1.  テクニカル サポート特典の使用を開始するには、テクニカル サポート タイルで **[有効化]** をクリックします。 

![テクニカル サポート特典タイル](_img\vs-tech-support\vs-tech-support-tile.png)

2.  [テクニカル サポート ページ](https://www.visualstudio.com/my/technical-support-vse)にテクニカル サポート特典を有効にするための指示が表示されます。  契約 ID とアクセス ID が必要です。これはテクニカル サポート チームが設定してくれます。    
3.  契約 ID およびアクセス ID まだセットアップしていない場合、お住まいの国の[マイクロソフト テクニカル サポートの番号](http://support.microsoft.com/gp/customer-service-phone-numbers)に連絡し、Visual Studio サブスクリプションのテクニカル サポート連絡先を設定する必要があることを伝えてください。  わずか数分で完了します。  契約 ID とアクセスの ID が提供されるので、将来参照できるように記録してください。  それらの情報が含まれる電子メールも送信されます。  
 
4.  ヨーロッパ、中東およびアフリカでサポート アカウントを有効にするには、お住まいの国を[このリスト](http://support.microsoft.com/activatesupport)から選択してください。   その他のすべての地域では、手順 7. を省略してください。
 
5.  「**Activating a Support Contract**」(サポート連絡先の有効化) ページの手順に従います。プロセスは、国によって異なる場合があります。  一般に、この手順では、電子登録フォームをダウンロードして記入し、マイクロソフトに電子メールで送信します。  処理されると、アカウントが準備され、アカウントの詳細が記載された電子メールを受信します。  
 
6.  自分のアカウントが設定されたら、手順 8 に進み、オンライン サポート インシデントを送信する方法を参照してください。  
7.  ヨーロッパ、中東およびアフリカを除くすべての地域では、お住まいの国の[テクニカル サポート チーム](http://support.microsoft.com/gp/customer-service-phone-numbers)に連絡してアカウントをセットアップすることができます。  お住まいの国の適切な電話番号を見つけるには、お住まいの国の名前に対応する文字をクリックします。  その文字で始まる国の一覧が表示され、これらの場所のサポート言語と電話番号が表示されます。  お住まいの国が一覧にない場合は、選択した言語のサポートを提供する最も近い国を選択します。  
 
8.  電話すると、名前、電話番号、電子メール アドレスなどの基本的な連絡先情報を提供するように求められます。  お客様のサブスクリプション ID も提供する必要があります。  これを見つけるには、[サブスクリプション ポータル](https://my.visualstudio.com/subscriptions)で、[[サブスクリプション]](https://my.visualstudio.com) タブをクリックします。  自分のサブスクリプションのサブスクリプション ID の一覧が表示されます。

![テクニカル サポート特典のサブスクリプション ID を確認する](_img\vs-tech-support\vs-tech-support-subID-cropped.png)

9.  電話の後に、作成された契約 ID とアクセス ID を含む確認の電子メールを受け取ります。

10. 契約 ID およびアクセス ID をセットアップした後に、サポートが必要なときには、アカウントを設定するために電話した番号と同じテクニカル サポート番号に連絡するか、または[オンライン サポート リクエストを送信する](http://support.microsoft.com/oas/)ことができます。  電話によるサポートを希望する場合でも、オンラインでリクエストを送信し、Microsoft サポート担当者にコールバックさせることができます。

11. [インシデントをオンライン送信](http://support.microsoft.com/oas/)する場合は、ヘルプが必要な製品を指定します。  各種の一覧から選択するか、名前で検索できます。  たとえば、Visual Studio Enterprise 2017 のインシデントを送信するには、**[開発者ツール]** グループで **[Visual Studio]** をクリックし、グループ。 
 
![テクニカル サポート特典のインシデント製品を選択する](_img\vs-tech-support\vs-tech-support-select-product.png)
12. Visual Studio 製品の一覧で **[Visual Studio Enterprise 2017]** をクリックします。 
 
13. **[インシデントの作成**] ページで、ドロップダウン リストから該当する問題の種類を選択します。  たとえば、**[統合開発環境 (IDE)]** を選択します。

![テクニカル サポート特典のインシデントを作成する](_img\vs-tech-support\vs-tech-support-create-incident.png)

14. 質問/問題に最も適合する**カテゴリ**を選択します。  この例では、**[ツールの構築]** を選択します。
15. **[要求の開始]** をクリックします。 
 
16. 次のページで求、支払いオプションを選択するように求められます。  
- **[MSDN または Visual Studio サブスクリプションを使用]** を選択します。 
- **[別の契約の使用]** をクリックします。
- アクセス ID と契約 ID を入力します。
- **[追加]**をクリックします。

17. サポート契約を追加すると、プロフェッショナル サポート契約の一覧に表示されます。  それを選択し、**[続行]** をクリックします。
 
![テクニカル サポート特典の支払い](_img\vs-tech-support\vs-tech-support-payment.png)

18. 連絡先情報と、希望する連絡方法を提供し、**[続行]** をクリックします。  
 
19. サポート インシデントのタイトルを入力し、問題を説明します。  

20. 簡単な一連の質問に答えて、問題の重大度を決定します。  

21. サービス条件に同意するためのチェックボックスをクリックします。

22. **[送信]** をクリックしてお客様の問題を送信します。  
 
23. インシデントが正常に送信されたことの確認メッセージが表示されます。  エージェントがインシデントに割り当てられ、エージェントは選択された連絡方法を使用して問題を解決するために連絡します。  重大でない問題の応答時間は、8 時間です。 エージェントは、サービス リクエスト番号を提供し、お客様の問題を解決するのために協力します。
