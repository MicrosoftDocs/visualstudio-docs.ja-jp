# <a name="learn-about-subscription-management"></a>サブスクリプションの管理について

ボリューム ライセンス契約、マイクロソフト製品/サービス契約 (MPSA)、または Visual Studio Cloud サブスクリプションを介してサブスクリプションを取得したお客様はサブスクリプションの管理機能を使用できます。

さまざまな購入チャネルがあるので、適切なアクセス許可とユーザー設定を利用できるようにするために、サブスクリプションの購入方法を知ることはサブスクリプション管理者にとって重要です。 契約の種類ごとに管理者を設定する方法については、ポータルへのアクセスのセクションでご確認ください。

Visual Studio サブスクリプション チームが、"Visual Studio [サブスクリプション管理ポータル](https://visualstudio.microsoft.com/subscriptions-administration/)" を作成しました。  このポータルの目的は、サブスクリプション管理者が組織内のサブスクリプションを割り当てて管理するために使用できるツールと機能をすばやく理解できるようにすることです。

サブスクリプションの管理ポータルでは、次のトピックについて学習できます。
- [サブスクリプションの管理ポータル](https://manage.visualstudio.com)でサブスクリプションを管理する方法。
- サブスクリプションの管理ポータルにまだ移行していない組織が、ボリューム ライセンス契約を使用してサブスクリプションを管理する方法。
- 新しいポータルへ移行する方法。
- サブスクリプション管理者対象のその他のヒントとテクニック。
