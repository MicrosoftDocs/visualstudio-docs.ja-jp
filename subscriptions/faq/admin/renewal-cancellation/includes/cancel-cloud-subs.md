## <a name="how-do-i-cancel-monthly-and-annual-subscriptions"></a>月間および年間サブスクリプションをキャンセルするにはどうすればよいですか?
[Visual Studio Marketplace](https://marketplace.visualstudio.com) から購入した月間および年間サブスクリプションをキャンセルするには、[管理ポータル](https://manage.visualstudio.com)にサインインして、契約のサブスクリプションの数をゼロに設定する必要があります。

サブスクリプションの数を減らすには、次の手順に従います。
1.  https://manage.visualstudio.com にサインインします
2.  複数の契約の管理者である場合は、ドロップダウン リストから目的の契約を選択します。
3.  左上にある **[概要]** アイコンを選択して、サブスクリプションに関する情報を表示します。
4.  キャンセルするサブスクリプションのエントリを見つけて、 **[数量の変更]** を選択します。 これにより、購入数量を更新できる Visual Studio Marketplace が表示されます。 
5.  数量をゼロ (0) に設定します。 サブスクリプションは、予定請求日まではそのまま残りますが、請求日となったときに更新は行われません。
