## <a name="how-do-i-renew-visual-studio-subscriptions"></a>Visual Studio サブスクリプションを更新するにはどうすればよいですか? 

Visual Studio サブスクリプションはいくつかの方法で取得でき、それらの取得方法によって、既存のサブスクリプションの更新プロセスが異なります。

### <a name="how-to-renew-retail-subscriptions"></a>リテール サブスクリプションを更新する方法 

現在、リテール サブスクリプションをお持ちの場合は、次の 2 つの方法で更新できます。 

-  [サブスクリプション ポータル](https://my.visualstudio.com/benefits) から  [[サブスクリプション] ページ](https://my.visualstudio.com/subscriptions)に接続し、[更新] リンクをクリックします。 
-  [Microsoft Store](https://www.microsoft.com/store)  にアクセスして "Visual Studio サブスクリプション" を検索し、使用可能な更新機会を確認してください。 クイック リンクを次に示します。 


    [Visual Studio Enterprise](https://www.microsoft.com/p/visual-studio-enterprise-subscription/dg7gmgf0dst4?activetab=pivot%3aoverviewtab) 

    [Visual Studio Professional](https://www.microsoft.com/p/visual-studio-professional-subscription/dg7gmgf0dst3?activetab=pivot%3aoverviewtab)

    [Visual Studio Test Professional](https://www.microsoft.com/p/visual-studio-test-professional-subscription/dg7gmgf0dst6?activetab=pivot%3aoverviewtab) 


### <a name="how-to-renew-cloud-subscriptions"></a>クラウド サブスクリプションを更新する方法
クラウド サブスクリプションは、[Visual Studio Marketplace](https://marketplace.visualstudio.com/) から購入するものです。  月間および年間の両方のクラウド サブスクリプションは、お客様がキャンセルするまで更新されるため、それらを更新するために必要な操作はありません。

### <a name="how-to-renew-subscriptions-purchased-through-volume-licensing"></a>ボリューム ライセンスで購入したサブスクリプションを更新する方法
お客様のサブスクリプションが職場から提供されている場合は、会社のボリューム ライセンス契約を通じて購入および更新されることがあります。  [サブスクリプション ポータル](https://my.visualstudio.com/benefits)の右上にある **[Contact my Admin]\(管理者に問い合わせる\)** ボタンを使用して、管理者に連絡することができます。
