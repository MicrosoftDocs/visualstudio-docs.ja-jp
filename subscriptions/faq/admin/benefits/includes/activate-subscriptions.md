## <a name="how-do-my-end-users-activate-their-visual-studio-subscription"></a>エンド ユーザーが Visual Studio サブスクリプションをアクティブ化するにはどうすればよいですか?

サブスクリプションはアクティブになっており、[Visual Studio 管理者ポータル](https://manage.visualstudio.com/subscribers)内から割り当てられるとすぐに使用できます。 お客様のサブスクライバーは、すぐに [Visual Studio サブスクライバー ポータル](https://my.visualstudio.com/)または VS IDE にサインインすることができ、サブスクリプションが割り当てられていることを知らせるウェルカム メールを受け取ります。
