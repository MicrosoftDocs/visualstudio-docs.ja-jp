## <a name="how-do-my-developers-download-visual-studio"></a>開発者向けの Visual Studio はどのようにしてダウンロードしますか?
開発者に割り当てられたサブスクリプション レベルによって、アクセスできる Visual Studio IDE のレベルが特定されます。 たとえば、Visual Studio Professional サブスクリプションは Visual Studio Enterprise にはアクセスできません。 適切なレベルが割り当てられていることを確認してください。 

### <a name="for-the-latest-release"></a>最新リリースの場合
1.  サブスクライバーが <https://my.visualstudio.com/benefits> でサブスクリプション ポータルにサインインするようにします。 
0.  [ツール] セクションで、Visual Studio IDE のタイルを見つけて、特典タイルの下部にある **[ダウンロード]** リンクをクリックします。 
0.  Visual Studio で IDE をダウンロードするための [結果のダウンロード] ページにリダイレクトされます。 
0.  ダウンロードと設定の手順に従ってください。
0.  サブスクリプションが割り当てられている電子メールを使用すると、インストール後すぐに IDE にサインインできるようになります。プロダクト キーは必要ありません。 オフライン環境で使用する[プロダクト キーを要求する方法](https://docs.microsoft.com/visualstudio/subscriptions/find-keys)について説明します。

### <a name="find-older-versions"></a>旧バージョンを検索する
サブスクライバーは、サブスクリプション ポータル (<https://my.visualstudio.com>) の[ダウンロード ページ](https://my.visualstudio.com/downloads)で、旧バージョンを検索してダウンロードすることもできます。
