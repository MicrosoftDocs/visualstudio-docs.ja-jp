# <a name="accessibility"></a>アクセシビリティ

macOS には、Visual Studio for Mac の使用に役立つアクセシビリティ ツールや機能が多数組み込まれています。  これらに加え、Visual Studio for Mac には、さまざまな身体機能のユーザーが操作しやすくなる以下の機能も用意されています。

* エディターのテキストの拡大
* 埋め込みテキストの拡大
* コード エディターの配色テーマのカスタマイズ
* キーボード ショートカットのカスタマイズ
* キーボード ナビゲーション

> [!NOTE]
> このガイドでは、Visual Studio for Mac について説明します。 Windows 上の Visual Studio については、「[Visual Studio のユーザー補助機能](/visualstudio/ide/reference/accessibility-features-of-visual-studio)」を参照してください。

* [キーボード ナビゲーションを使用する](accessibility-keyboard.md)
