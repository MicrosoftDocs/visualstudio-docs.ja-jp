# <a name="aspnet-core-web-app-deployment"></a>ASP.NET Core Web アプリのデプロイ

Web アプリの開発において発行 (デプロイ) は重要な手順です。 他のユーザーがアプリケーションを使用できるように、またはテスト目的として、リモート環境に発行できます。 Visual Studio for Mac では、アプリケーションを発行するために 2 つのオプションを提供します。 "Azure App Service に発行する" を使用すると、自分の Web アプリを Azure App Service に発行できます。 [Publish to Folder]\(フォルダーに発行する\) を使用すると、自分のアプリケーションをフォルダーに発行できます。

## <a name="publish-to-azure-app-service"></a>Azure App Service に発行する

Visual Studio for Mac を使用して、Web アプリを Azure App Service に直接発行できます。 ASP.NET Core Web アプリを Azure App Service に発行するには、「[Publish to Azure App Service](publish-app-svc.md)」(Azure App Service に発行する) の記事を参照してください。

## <a name="publish-to-folder"></a>フォルダーに発行する

ASP.NET Core Web アプリをフォルダーに発行することもできます。 Web アプリをフォルダーに発行する方法の詳細については、「[フォルダーに発行する](publish-folder.md)」を参照してください。
