# <a name="troubleshooting-why-can-i-not-see-all-available-templates"></a>トラブルシューティング:使用可能なすべてのテンプレートが表示されない

[新しいプロジェクト] ダイアログでは、インストールされているコンポーネントのテンプレートのみが表示されます。 次のコンポーネントのいずれかが表示されない場合は、以下の説明に従って、適切なコンポーネントがインストールされていることを確認します。


|表示されないテンプレート  |インストールするコンポーネント  |
|---------|---------|
|Android     |Android        |
|iOS     |iOS         |
|tvOS     |iOS         |
|watchOS     |iOS         |
|マルチプラットフォーム     |Android および iOS         |
|Mac     |macOS (Cocoa)         |
|.NET Core     |.NET Core         |

VisualStudioForMacInstaller.dmg をもう一度実行して、コンポーネントをインストールします。 なくなってしまっている場合は、[Visual Studio for Mac](https://aka.ms/vsmac) の Web ページからダウンロードできます。
