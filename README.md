# <a name="visual-studio-2017-documentation"></a>Visual Studio 2017 ドキュメント

ようこそ このリポジトリには、開発作業中である Visual Studio 2017 のテクニカル ドキュメント用のソース ファイルが含まれます。 トピックは、[Visual Studio ドキュメント サイト](https://docs.microsoft.com/visualstudio)で公開されています。

このリポジトリは、2017 年 6 月 23 日に、 https://github.com/Microsoft/vsdocs から移動されました。 以前の URL 宛てのトラフィックはここにリダイレクトされます。

Visual Basic と Visual C# のドキュメントは別のリポジトリ [http://github.com/dotnet/core-docs](http://github.com/dotnet/core-docs) にあり、Visual C++ のドキュメントは [http://github.com/MicrosoftDocs/cpp-docs](http://github.com/MicrosoftDocs/cpp-docs) のリポジトリにあります。

## <a name="contributing-to-the-documentation"></a>ドキュメントへの貢献

このドキュメントへの貢献については、[貢献に関するガイド](CONTRIBUTING.md)をご覧ください。
Visual Studio のドキュメントの改善にご協力ください。このリポジトリのすべての記事では、GitHub 形式のマークダウンが使われています。

Visual Studio の複数の機能領域については、このリポジトリに専用のフォルダーがあります。デバッグについての **debugger** や、Visual Studio の対話型開発環境 (IDE) についてのトピックの **ide** などです。 各フォルダーの **/media** サブフォルダーには、トピックのアート ファイルが含まれます。 詳しくは、[貢献に関するガイド](CONTRIBUTING.md)をご覧ください。
