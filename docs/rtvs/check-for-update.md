# <a name="rtvs-current-version"></a>RTVS の現在のバージョン

Tools for Visual Studio (RTVS) の最新のバージョンは、**1.0.30321.1407** です (推奨)。

これは RTVS の 1.0 リリース バージョンであり、[aka.ms/rtvs-current](https://aka.ms/rtvs-current) から入手できます。

Visual Studio の **[ヘルプ] > [バージョン情報]** で **[R Tools for Visual Studio]** エントリまで下にスクロールして、バージョンを確認できます。

![Visual Studio で RTVS のバージョンを調べる (VS2017)](~/rtvs/media/current-version.png)

Visual Studio と Windows の要件については、「[Installation](installation.md)」(インストール) をご覧ください。

Microsoft R Open および Microsoft R Server の更新については、[Microsoft R 製品](http://aka.ms/rtvs-msft-r)のページをご覧ください。
