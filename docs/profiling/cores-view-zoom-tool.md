# コア ビューのズーム ツール
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

ズーム コントロールは、タイムラインの表示を拡大縮小できるスライダーであり、これを使用すると興味のある特定の領域を集中的に見ることができます。  このコントロールは、現在のビューの中央が拡大縮小の中心点になるため、拡大する前にビューを調整してください。  
  
## その他のズーム方法  
  
-   拡大: タイムライン ビュー内の領域をドラッグして選択します。  マウス ボタンを離すと、タイムライン ビューの選択した範囲が拡大されます。  
  
-   拡大および縮小: タイムライン上でマウスを動かし、Ctrl キーを押しながらマウス ホイールを動かします。
