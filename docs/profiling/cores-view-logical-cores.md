# コア ビューの論理コア
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

各チャネル名は、プロファイリングされたシステムにある論理コアの 1 つを表します。  名前は 0 から論理コアの総数までカウントされて付けられます。  たとえば、4 つのコアのシステムは、論理コア 0 から論理コア 3 までカウントされます。
