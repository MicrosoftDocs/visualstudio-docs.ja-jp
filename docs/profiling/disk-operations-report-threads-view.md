# ディスク操作レポート (スレッド ビュー)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

ディスク操作のレポートはディスク チャネルでディスク I\/O 操作を示します。  
  
 現在表示されている時間ウィンドウでプロファイリングを行っているプロセスによって実行される各ディスクへのアクセスは、この情報は報告されます:  
  
-   ディスク アクセスを実行したプロセスの名前と PID  
  
-   ディスクにアクセスするスレッドの ID。  
  
-   アクセスされたファイル名  
  
-   ファイルごとの読み取り数  
  
-   読み取るバイト数  
  
-   読み取り待機時間 \(ミリ秒\)  
  
-   書き込み数  
  
-   書き込むバイト数  
  
-   ミリ秒の書き込み待機時間 \(
