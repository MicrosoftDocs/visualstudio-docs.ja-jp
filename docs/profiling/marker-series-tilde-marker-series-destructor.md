# marker_series::~marker_series デストラクター
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

破棄の marker\_series はすべて、割り当てられているリソースを解放します。  
  
## 構文  
  
```  
~marker_series();  
```  
  
## 必要条件  
 **ヘッダー:** cvmarkersobj.h  
  
 **名前空間:** Concurrency::diagnostic
