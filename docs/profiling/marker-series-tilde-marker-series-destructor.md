# <a name="marker_seriesmarker_series-destructor"></a>marker_series::~marker_series デストラクター
marker_series オブジェクトを破棄し、すべての割り当て済みリソースを開放します。

## <a name="syntax"></a>構文

```cpp
~marker_series();
```

## <a name="requirements"></a>必要条件
 **ヘッダー:** *cvmarkersobj.h*

 **名前空間:** Concurrency::diagnostic
