# 方法: プロファイリング ツールのレポート ビューの列をカスタマイズする
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Visual Studio Team System Development Edition のパフォーマンス ツールは統合開発環境 \(IDE\) に完全に統合されており、シームレスな状態ユーザー エクスペリエンスを提供する。  次の一覧は、列の追加、削除、または再配置によってカスタマイズできるビューを示しています。  
  
-   [モジュール ビュー](../profiling/modules-view.md)  
  
-   [関数ビュー](../profiling/functions-view.md)  
  
-   [行ビュー](../profiling/lines-view.md)  
  
-   [命令ポインター \(IP\) ビュー](../profiling/instruction-pointers-ips-view.md)  
  
-   [\[呼び出し元\/呼び出し先\] ビュー](../Topic/Caller-Callee%20View.md)  
  
-   [コール ツリー ビュー](../profiling/call-tree-view.md)  
  
-   [割り当てビュー](../profiling/dotnet-memory-allocations-view.md)  
  
-   [オブジェクトの有効期間ビュー](../profiling/object-lifetime-view.md)  
  
-   [マーク ビュー](../profiling/marks-view.md)  
  
-   [プロセス ビュー](../profiling/process-view.md)  
  
### レポート ビューをカスタマイズするには  
  
1.  レポート ビューを開きます。  
  
2.  レポート ビュー上で右クリックし、**\[列の追加と削除\]** をクリックします。  
  
     **\[列の追加と削除\]** ダイアログ ボックスが表示されます。  
  
3.  追加する列を選択します。  
  
4.  矢印ボタンをクリックして列を並べ替えます。  
  
5.  \[OK\] をクリックします。
