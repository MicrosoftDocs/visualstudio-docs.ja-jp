# スレッド別の概要レポート
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

この棒グラフは、それぞれがスレッドを非時間の割合が現在表示されている時間範囲内に各アクティビティ カテゴリに費やすことを示します。「スレッドが実行中であることを実行します;」を意味します。そのほかすべてのカテゴリは、そのスレッドが何かを待機していることを意味します。
