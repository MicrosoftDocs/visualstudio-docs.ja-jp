# タイムライン キャレット
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

実行中のスレッド セグメントのタイムライン上の点を選択すると、タイムラインのキャレットはその上に表示されます。  現在のスタック タブ上に表示される呼び出し履歴が、セグメントでクリックした場所の時間に最も近い呼び出し履歴になります。  キャレットが **\[現在\]** タブの下に表示されるサンプリングされた瞬間に呼び出し履歴を関連付けるために使用されます。  キャレットは、ユーザーが選択した場所に最も近い呼び出し履歴にある呼び出し履歴の正確な位置を示します。
