# <a name="no-disk-activity-to-show-threads-view"></a>表示するディスク アクティビティがありません (スレッド ビュー)
ディスク I/O チャネルの現在の時間範囲のディスク アクティビティを表示できません。 ズームアウトまたはスクロールすると、ディスク アクティビティが表示されることがあります。 また、非表示のディスク チャネルがないことを確認します。
