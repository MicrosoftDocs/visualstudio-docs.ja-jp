# 選択範囲のコピー
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

レポート タブの呼び出し履歴全体をコピーするには、**\[コピー\]** をクリックします。  その後で、このアクションをサポートする任意のプログラムで呼び出し履歴を貼り付けることができます。
