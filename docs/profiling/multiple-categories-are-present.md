# 複数のカテゴリが存在します
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

指定した時間範囲について、複数の種類のアクティビティを含むセクションが選択されています。  2 色表示されており、これらの色は上下に重なっています。  これらの色は、指定された時間範囲で最も頻繁に発生する上位 2 つのアクティビティだけを表しています。  これらの 2 つのアクティビティの頻度は、他方との相関関係を示すように、縦棒の 3 分の 1 または 2 分の 1 として表示されます。  詳細については、スレッド ビューのこのセクションを拡大してください。
