# <a name="export-button-concurrency-visualizer"></a>[エクスポート] ボタン (コンカレンシー ビジュアライザー)
**[エクスポート]** ボタンでは、独自のレコードのためや、Microsoft Excel など、別のツールで使用するために、.*csv* ファイルとして呼び出し履歴をエクスポートできます。
