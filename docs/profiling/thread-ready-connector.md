# スレッド準備完了コネクタ
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

ブロックしているセグメントをクリックして呼び出し履歴とブロック解除スタックを表示すると、スレッド準備完了コネクタも表示されることがあります。  現在のプロセス内の別のスレッドでブロック解除イベントが発生した場合、そのスレッドとブロックされたスレッドが実行再開できるようにした実行セグメントを、そのスレッド準備完了コネクタが視覚的に示します。
