# <a name="unblocking-stack"></a>ブロック解除スタック
現在選択されているスレッドの要素が、現在のプロセスの別のスレッドによってブロック解除された後に実行を開始するブロック セグメントを表している場合、ブロック解除を行ったスレッドの呼び出し履歴がこのタブに表示されます。
