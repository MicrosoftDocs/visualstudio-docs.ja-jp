# 表示されているタイムライン プロファイル
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

スレッド ブロッキング ビューの表示されているタイムライン プロファイル レポートには、統計情報とリンクを示します。  拡大、縮小、水平方向のスクロール、チャネルの非表示、またはチャネルの表示を行うと、アクティブな凡例の数値が、現在の表示に基づいて変更されます。  凡例内の項目に関するレポートを表示するには、その項目をクリックします。
