# 概要ビュー - プロファイラーの .NET メモリ データ
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

概要ビューには、最も多くのメモリを割り当てた .NET 関数と型、およびプロファイリング実行で作成回数が最も多い型に関する情報が一覧表示されます。  通知リンクやレポートの一覧の説明など、詳細については、「[概要ビュー](../profiling/summary-view.md)」を参照してください。  
  
## タイムライン グラフ  
 概要ビューのタイムライン グラフには、プロファイリングが行われた期間のプロファイリングされたアプリケーション別にプロセッサ \(CPU\) 使用率が表示されます。  タイムライン グラフを使用すると、選択した期間に対してフィルター処理した結果を表示できます。  詳細については、「[方法: 概要ビューのタイムラインからレポート ビューをフィルター処理する](../Topic/How%20to:%20Filter%20Report%20Views%20from%20the%20Summary%20Timeline.md)」を参照してください。  
  
## 最も多くのメモリを割り当てている関数  
 プロファイリング実行で最も多くのメモリのバイト数を割り当てた関数が一覧表示されます。  
  
|列|説明|  
|-------|--------|  
|**名前**|関数の名前。|  
|**\[バイト %\]**|プロファイリング実行で割り当てられたすべてのバイト数に対する、この関数、またはこの関数によって呼び出された子関数によって割り当てられたバイト数の割合。|  
  
## 最も多くのメモリを割り当てられた型  
 プロファイリング実行で最も多くのメモリのバイト数が割り当てられた型が一覧表示されます。  
  
|列|説明|  
|-------|--------|  
|**名前**|型の名前。|  
|**\[バイト %\]**|プロファイリング実行で割り当てられたすべてのバイト数に対する、この型に対して割り当てられたバイト数の割合。|  
  
## 最も多くのインスタンスを伴う型  
 プロファイリング実行で作成回数が最も多い型が一覧表示されます。持っています  
  
|列|説明|  
|-------|--------|  
|**名前**|型の名前。|  
|**\[インスタンス %\]**|プロファイリング実行で作成された .NET オブジェクトの合計数に対する、この型のインスタンスであったオブジェクトの割合。|
