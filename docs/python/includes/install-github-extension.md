1. GitHub リポジトリに接続するには、Visual Studio インストーラーを実行し、**[変更]** を選択し、**[個別のコンポーネント]** タブを選択します。下方にスクロールして **[コード ツール]** セクションを表示し、**[Visual Studio 向け GitHub 拡張]** を選択し、**[変更]** を選択します。
