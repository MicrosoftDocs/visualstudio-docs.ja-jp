## <a name="prerequisites"></a>必須コンポーネント

- Python のワークロードがインストールされた Visual Studio 2017。 手順については、「Visual Studio での Python の使用」の「[手順 0](../tutorial-working-with-python-in-visual-studio-step-00-installation.md)」をご覧ください。

Python Tools for Visual Studio がインストールされている Visual Studio の以前のバージョンを使用することもできます。 「[Visual Studio での Python サポートのインストール](../installing-python-support-in-visual-studio.md)」をご覧ください。 「[機能一覧](../overview-of-python-tools-for-visual-studio.md#features-matrix)」で説明されているように、一部の機能は以前のバージョンで使用できない場合があります。
