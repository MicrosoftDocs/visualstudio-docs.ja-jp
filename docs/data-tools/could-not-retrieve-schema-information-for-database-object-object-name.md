# Could not retrieve schema information for database object &lt;object name&gt;
このメッセージは、通常、**サーバー エクスプローラー**またはデータベース エクスプローラーでオブジェクトをクリップボードにコピーし、そのオブジェクトをデータベースから削除した後、デザイナーに貼り付けた場合に表示されます。データベース オブジェクトはもう存在しないので、このメッセージが表示されます。  
  
### このエラーを解決するには  
  
-   **\[OK\]** をクリックしてダイアログ ボックスを閉じます。
