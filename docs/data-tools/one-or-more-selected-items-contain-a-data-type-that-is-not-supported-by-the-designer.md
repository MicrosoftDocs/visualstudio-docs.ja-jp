# <a name="one-or-more-selected-items-contain-a-data-type-that-is-not-supported-by-the-designer"></a>1 つ以上の選択された項目に、デザイナーでサポートされていないデータ型が含まれています

1 つまたは複数の項目のドラッグ**サーバー エクスプ ローラー**または**データベース エクスプ ローラー**上に、 **O/R デザイナー**でサポートされていないデータ型が含まれる、 **O/R デザイナー**、たとえば、 [CLR ユーザー定義型](/dotnet/framework/data/adonet/sql/clr-user-defined-types)します。

## <a name="to-correct-this-error"></a>このエラーを解決するには

1. 目的のテーブルをベースにした、サポートされていないデータ型を含まないビューを作成します。

2. ビューからドラッグ**サーバー エクスプ ローラー**または**データベース エクスプ ローラー**デザイナーにします。
