# アプリケーションでのデータ編集
データ モデルにデータを読み込んだ後、通常は、一部のデータを追加、編集、または削除してから、データ ソースや別のプロセス、またはアプリケーションにデータを送信します。  
  
 次のトピックでは、データの追加、編集、および削除と、アプリケーションのさまざまな段階でデータを操作する方法について詳しく説明します。
