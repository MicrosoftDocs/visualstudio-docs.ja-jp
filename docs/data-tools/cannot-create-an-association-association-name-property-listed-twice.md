# <a name="cannot-create-an-association-ltassociation-namegt---property-listed-twice"></a>関連付けを作成できません&lt;association 名&gt;-プロパティが 2 回一覧表示

関連付けを作成できません\<association 名 >。 同じプロパティが複数リストされています:\<プロパティ名 >。

関連付けを定義して、選択した**関連付けのプロパティ**で、 **[関連付けエディター** ] ダイアログ ボックス。 プロパティは、関連付けのクラスごとに 1 回のみリストできます。

メッセージのプロパティには、親または子のいずれかのクラスの 1 つ以上の時間が表示されます。**関連付けのプロパティ**です。

## <a name="to-resolve-this-condition"></a>この状況の解決方法

- メッセージを確認し、メッセージで指定されているプロパティに注目します。

- をクリックして**OK**メッセージ ボックスを閉じます。

- 検査、**関連付けのプロパティ**と重複するエントリを削除します。

- **[OK]** をクリックします。
