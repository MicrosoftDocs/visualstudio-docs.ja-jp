# <a name="messages-menu-commands"></a>[メッセージ] メニューのコマンド
**メッセージ**メニューは、メッセージ ビューを制御するためのコマンドを提供します。  
  
 **ログ記録の開始/停止**  
 開始またはメッセージの表示を停止します。  
  
 **ログ オプション**  
 開く、**メッセージ** ダイアログ ボックス、メッセージの検索条件を指定できます。  
  
 **ログの消去**  
 アクティブなメッセージ ビューの内容を消去します。
