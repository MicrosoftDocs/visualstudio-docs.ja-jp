# [ウィンドウ] メニューのコマンド
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

\[ウィンドウ\] メニューには、Spy\+\+ ウィンドウの管理用のコマンドが用意されています。  
  
 **\[重ねて表示\]**  
 すべてのウィンドウを再配置し、重ねて表示します。  
  
 **\[上下に並べて表示\]**  
 すべてのウィンドウを再配置し、上下に並べて表示します。  
  
 **\[左右に並べて表示\]**  
 すべてのウィンドウを再配置し、左右に並べて表示します。  
  
 **\[アイコンの整列\]**  
 ウィンドウが最小化されているときに、ビュー アイコンを再整列させます。  
  
 **\[閉じる\] または \[すべて閉じる\]**  
 Spy\+\+ のウィンドウを 1 つまたはすべて閉じます。  
  
 **\[最新の情報に更新\]**  
 現在選択されているビューを更新します。  
  
 **\[ウィンドウ\]**  
 特定のウィンドウを選択できます。
