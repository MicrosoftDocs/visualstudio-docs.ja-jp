# <a name="window-menu-commands"></a>[ウィンドウ] メニューのコマンド
**[ウィンドウ]** メニューには、Spy++ ウィンドウを管理するためのコマンドが用意されています。

 **カスケード** すべてのウィンドウをカスケード パターンで再配置します。

 **上下に並べて表示** すべてのウィンドウを水平タイル パターンで再配置します。

 **左右に並べて表示** すべてのウィンドウを垂直タイル パターンで再配置します。

 **アイコンの整列** ウィンドウが最小化されている場合は、表示アイコンを再配置します。

 **[閉じる] または [すべて閉じる]** Spy++ の 1 つまたはすべてのウィンドウを閉じます。

 **更新** 現在選択されているビューを更新します。

 **ウィンドウ** 特定のウィンドウを選択できます。
