# <a name="spy-views"></a>Spy++ ビュー
Spy++ では、コンピューターで windows、メッセージ、プロセス、および利用可能なスレッドのツリー ビューが表示されます。

## <a name="in-this-section"></a>このセクションの内容
 [Windows ビュー](../debugger/windows-view.md)コントロール、使用可能なウィンドウのツリー ビューが表示されます。

 [メッセージ ビューを](../debugger/messages-view.md)ウィンドウ、プロセス、またはスレッドに関連付けられているメッセージ ストリームを表示します。

 [プロセス ビュー](../debugger/processes-view.md)アクティブなプロセスのツリー ビューが表示されます。

 [スレッド ビュー](../debugger/threads-view.md)アクティブなスレッドのツリー ビューが表示されます。
