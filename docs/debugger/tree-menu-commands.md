# [ツリー] メニューのコマンド
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

\[ツリー\] メニューには、ビューの展開と折りたたみを行うためのコマンドが用意されています。  
  
 **\[1 つ下の階層を表示\]**  
 システム ビューで現在選択されている項目の 1 つ下の階層を表示します。  
  
 **\[ブランチを表示\]**  
 現在選択されている項目の下のすべての階層を展開します。  
  
 **\[すべて展開\]**  
 システム ビュー内のすべての階層を展開し、表示します。  
  
 **\[折りたたむ\]**  
 現在選択されている項目の下のすべての階層を折りたたみます。
