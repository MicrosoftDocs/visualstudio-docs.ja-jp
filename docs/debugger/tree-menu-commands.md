# <a name="tree-menu-commands"></a>[ツリー] メニューのコマンド
**[ツリー]** メニューでは、ビューを展開または折りたたむためのコマンドが提供されます。

 **1 レベル展開** システム ビューで現在選択されている項目を、その 1 レベル下のものを表示して展開します。

 **ブランチを表示** 現在選択されている項目の下にあるすべてのレベルを展開します。

 **すべて展開** システム ビュー内のすべてのレベルを展開して表示します。

 **折りたたむ** 現在選択されている項目の下にあるすべてのレベルを折りたたみます。
