# <a name="how-to-display-thread-properties"></a>方法: スレッドのプロパティを表示する
表示されているスレッドの詳細を検索する[スレッド ビュー](../debugger/threads-view.md)を使用して、[スレッド プロパティ ダイアログ ボックス](../debugger/thread-properties-dialog-box.md)です。  
  
### <a name="to-open-a-thread-properties-dialog-box"></a>スレッド プロパティ ダイアログ ボックスを開く  
  
1.  表示されるツリーからスレッドを選択して[スレッド ビュー](../debugger/threads-view.md)です。  
  
2.  **ビュー** ] メニューの [選択**プロパティ**です。  
  
 プロパティのダイアログ ボックスはモーダルではないため、表示ウィンドウで別のアイテムをクリックして、ダイアログ ボックスの選択項目に関する情報が表示されます。
