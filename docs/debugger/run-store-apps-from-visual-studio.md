# <a name="run-uwp-apps-from-visual-studio"></a>UWP アプリを Visual Studio から実行します。
UWP アプリは、デバイスの広範な範囲で実行し、Visual Studio が提供するさまざまな方法を実行し、それらのアプリと対話します。 「F5」の拡張機能では、開発時にアプリのデバッグ、プロファイル、およびテストが簡単に行えるようになっています。
