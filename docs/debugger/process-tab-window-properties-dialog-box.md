# [プロセス] タブ ([ウィンドウ プロパティ] ダイアログ ボックス)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

\[プロセス\] タブを使用すると、選択したウィンドウのプロセスを特定できます。  [&#91;ウィンドウ プロパティ&#93; ダイアログ ボックス](../debugger/window-properties-dialog-box.md)を表示するには、[ウィンドウ ビュー](../debugger/windows-view.md) ウィンドウにフォーカスを移動します。  ツリーで任意のウィンドウ ノードを選択し、\[表示\] メニューの \[プロパティ\] をクリックします。  
  
 \[プロセス\] タブには、次の項目があります。  
  
|エントリ|Description|  
|----------|-----------------|  
|**\[プロセス ID\]**|このウィンドウを作成したスレッドを所有するプロセスの ID です。  このプロセスのプロパティを表示するには、この値を選択します。|  
|**\[スレッド ID\]**|このウィンドウを作成したスレッドの ID です。  このスレッドのプロパティを表示するには、この値を選択します。|
