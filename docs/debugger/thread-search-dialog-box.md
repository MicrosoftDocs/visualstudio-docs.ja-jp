# [スレッド検索] ダイアログ ボックス
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

このダイアログ ボックスを使用すると、[スレッド ビュー](../debugger/threads-view.md)の特定のスレッドのノードを検索および選択できます。  このダイアログ ボックスを表示するには、スレッド ビュー ウィンドウにフォーカスを移動します。  次に、\[検索\] メニューの \[スレッド検索\] をクリックします。  
  
 次の項目があります。  
  
 **\[スレッド\]**  
 検索対象のスレッドの ID です。  
  
 **Module**  
 検索対象のモジュールの文字列です。  
  
 **\[上へ検索\] または \[下へ検索\]**  
 検索の最初の方向です。
