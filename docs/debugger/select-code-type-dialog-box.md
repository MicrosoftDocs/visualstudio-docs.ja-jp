# <a name="select-code-type-dialog-box"></a>[コードの種類の選択] ダイアログ ボックス
このダイアログ ボックスを開く、**プロセスにアタッチする**クリックしてダイアログ ボックスで、**選択**ボタンをクリックします。  
  
 **デバッグするコードの種類を自動的に判断します。**  
 実行中のコードの種類に基づいて、適切なデバッガーが選択されます。  
  
 **これらのコードの種類をデバッグするには。**  
 表示される一覧から、デバッグするコードの種類を選択します。
