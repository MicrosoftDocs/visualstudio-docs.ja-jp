# <a name="debug-azure-services-in-visual-studio"></a>Visual Studio での Azure サービスをデバッグします。

Visual Studio を使用して、さまざまなシナリオで Azure サービスをデバッグすることができます。

- Azure App Service でホストされている運用アプリをデバッグするを参照してください。[スナップショット デバッガーを使用して ASP.NET アプリのライブ デバッグ](../debugger/debug-live-azure-applications.md)です。

- Application Insights を使用して運用アプリをデバッグし、Azure App Service または Service Fabric でホストを参照してください[アプリ用 .NET での例外でのスナップショットをデバッグ](/azure/application-insights/app-insights-snapshot-debugger)です。

- リモート デバッグ (Azure App Service または Azure の仮想マシン) IIS で ASP.NET を参照してください。 [Azure 上でリモート デバッグ ASP.NET](remote-debugging-azure.md)です。

- Azure Service Fabric の ASP.NET リモート デバッグを参照してください[リモート Service Fabric アプリケーションのデバッグ。](/azure/service-fabric/service-fabric-debugging-your-application#debug-a-remote-service-fabric-application)
