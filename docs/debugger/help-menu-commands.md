# [ヘルプ] メニューのコマンド
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

\[ヘルプ\] メニューには、ヘルプ ファイルを開くためのコマンドが用意されています。  
  
 **\[インデックス\]**  
 Spy\+\+ ヘルプの目次画面を開きます。  
  
 **\[ヘルプの使い方\]**  
 ヘルプの使い方を示したヘルプ ファイルを開きます。  
  
 **\[バージョン情報\]**  
 Spy\+\+ のソフトウェア バージョンと著作権情報を表示します。
