# <a name="help-menu-commands"></a>[ヘルプ] メニューのコマンド
**[ヘルプ]** メニューには、ヘルプ ファイルを開くためのコマンドが用意されています。

 **[インデックス]** : Spy + + ヘルプのコンテンツ画面が開きます。

 **[ヘルプの使用]** : ヘルプの使用方法を説明するヘルプ ファイルが開きます。

 **[Spy++ のバージョン情報]** : Spy++ のソフトウェアのバージョンと著作権に関する情報が表示されます。
