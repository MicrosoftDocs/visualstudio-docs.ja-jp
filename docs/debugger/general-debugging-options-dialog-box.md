## <a name="options-supported-in-older-versions-of-visual-studio"></a>Visual Studio の以前のバージョンでサポートされているオプション

**ネイティブ互換モードを使用して、** デバッガーで Visual Studio 2010 のネイティブ デバッガーを使用して、新しいネイティブ デバッガーではなくこのオプションを選択するとします。
