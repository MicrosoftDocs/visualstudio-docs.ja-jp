# How to: Display Process Properties
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

[プロセス ビュー](../debugger/processes-view.md)に一覧表示されているプロセスの詳細を確認するには、[&#91;プロセス プロパティ&#93; ダイアログ ボックス](../debugger/process-properties-dialog-box.md)を使用します。  
  
### \[プロセス プロパティ\] ダイアログ ボックスを開くには  
  
1.  プロセス ビューに表示されているツリーからプロセスを選択します。  
  
2.  \[表示\] メニューの \[プロパティ\] をクリックします。  
  
 プロパティ ダイアログ ボックスはモーダルではないため、ビュー ウィンドウの別の項目をクリックできます。その場合、選択した項目の情報がダイアログ ボックスに表示されます。
