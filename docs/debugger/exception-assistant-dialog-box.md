# 例外処理アシスタント ダイアログ ボックス
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

**例外処理アシスタント** ダイアログ ボックスは、ランタイム例外がスローされると表示されます。  **例外処理アシスタント** ダイアログ ボックスには、例外の種類、追加情報、およびトラブルシューティングのヒントへのリンクが表示されます。また、このダイアログ ボックスを使用して追加のオンライン ヘルプを検索したり、例外の詳細を表示するなど、特定のアクションを実行したりできます。  
  
## タスク一覧  
 発生した例外のトラブルシューティングに関するトピックを表示するには、**\[トラブルシューティングのヒント\]** ペインに表示されるヒント メッセージのいずれかをクリックします。  
  
 例外に関連するアクションを実行するには、アクション ペインに表示されたアクションのいずれかをクリックします。  
  
 例外処理アシスタントを有効または無効にする方法については、「[\[全般\] \(\[オプション\] ダイアログ ボックス \- \[デバッグ\]\)](../Topic/General,%20Debugging,%20Options%20Dialog%20Box.md)」を参照してください。  
  
## ユーザー インターフェイス要素一覧  
 **例外の種類**  
 スローされた例外の種類が表示されます。  
  
 **追加情報**  
 例外に関する追加情報が表示されます。  
  
 **\[トラブルシューティングのヒント\]**  
 例外の原因を発見する上で役立つトラブルシューティングのヒントへのリンクが表示されます。  
  
 **アクション**  
 例外オブジェクトに関する詳細の表示など、実行できるアクションの一覧が表示されます。  
  
 **\[オンライン ヘルプで検索\]**  
 追加のオンライン ヘルプを検索できます。
