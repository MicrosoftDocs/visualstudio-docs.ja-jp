# <a name="how-to-display-window-properties"></a>方法: ウィンドウのプロパティを表示する
[[ウィンドウ ビュー]](../debugger/windows-view.md) に表示されているウィンドウの詳細については、[[ウィンドウ プロパティ] ダイアログ ボックス](../debugger/window-properties-dialog-box.md)を使用してください。

### <a name="to-open-the-window-properties-dialog-box"></a>[ウィンドウ プロパティ] ダイアログ ボックスを開く

1. 項目を選択して、 **[表示]** から **[プロパティ]** を選択します。

   [プロパティ] ダイアログ ボックスはモーダルではありません。 [表示] ウィンドウで項目ごとにクリックすると、ダイアログ ボックスが自動的に更新され、選択した各項目に関する情報が表示されます。
