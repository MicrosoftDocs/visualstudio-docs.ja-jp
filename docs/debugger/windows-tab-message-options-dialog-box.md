# <a name="windows-tab-message-options-dialog-box"></a>[ウィンドウ] タブ ([メッセージ オプション] ダイアログ ボックス)
使用して、 **Windows**メッセージの一覧のスコープを設定するタブ[メッセージ ビュー](../debugger/messages-view.md)です。 表示する、[メッセージ オプション ダイアログ ボックス](../debugger/message-options-dialog-box.md)、選択**ログ メッセージ**から、**スパイ**メニュー。  
  
 次の設定が [利用可能な**Windows** ] タブ。  
  
 **ファインダー ツール**  
 ドラッグ アンド ドロップに使用するツール ウィンドウを選択します。  
  
 **非表示に spy++**  
 Spy++ のメイン ウィンドウし、のまま、**オプション**他のアプリケーションの上に表示されるダイアログ ボックス。  
  
 **親ウィンドウ**  
 選択したウィンドウとその直接の親ウィンドウのメッセージを表示します。  
  
 **子ウィンドウ**  
 選択したウィンドウと入れ子になった子ウィンドウを含む、そのすべての子ウィンドウのメッセージを表示します。  
  
 **同じスレッドのウィンドウ**  
 選択したウィンドウと同じスレッドが所有する他のすべてのウィンドウのメッセージを表示します。  
  
 **同じプロセスの Windows**  
 選択したウィンドウと同じプロセスが所有する他のすべてのウィンドウのメッセージを表示します。  
  
 **システムのすべてのウィンドウ**  
 すべてのウィンドウのメッセージを表示します。  
  
 **設定を既定として保存します。**  
 新しいメッセージ ストリームの windows の設定を保存します。 これらの設定は、spy++ を終了するときにも保存されます。
