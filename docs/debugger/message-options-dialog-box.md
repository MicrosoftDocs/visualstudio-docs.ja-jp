# <a name="message-options-dialog-box"></a>[メッセージ オプション] ダイアログ ボックス
このダイアログ ボックスを使用して、[[メッセージ ビュー]](../debugger/messages-view.md) に一覧表示されるメッセージを選択します。 このダイアログ ボックスを表示するには、 **[Spy]** メニューから **[ログ メッセージ]** を選択します。

## <a name="in-this-section"></a>このセクションの内容
 [[ウィンドウ] タブ ([メッセージ オプション] ダイアログ ボックス)](../debugger/windows-tab-message-options-dialog-box.md) 一覧表示するメッセージの種類を選択するために使用されます。 ファインダー ツールが含まれています。

 [[メッセージ] タブ ([メッセージ オプション] ダイアログ ボックス)](../debugger/messages-tab-message-options-dialog-box.md) 表示するメッセージの種類を選択するために使用されます。

 [[出力] タブ ([メッセージ オプション] ダイアログ ボックス)](../debugger/output-tab-message-options-dialog-box.md) 表示するメッセージ データを指定するために使用されます。
