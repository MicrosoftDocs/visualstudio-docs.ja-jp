# [ウィンドウ プロパティ] ダイアログ ボックス
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

このダイアログ ボックスを使用すると、特定のウィンドウの詳細を確認できます。  このダイアログ ボックスを表示するには、[ウィンドウ ビュー](../debugger/windows-view.md) ウィンドウにフォーカスを移動します。  ツリーで任意のウィンドウ ノードを選択し、\[表示\] メニューの \[プロパティ\] をクリックします。  
  
## このセクションの内容  
 [&#91;全般&#93; タブ \(&#91;ウィンドウ プロパティ&#93; ダイアログ ボックス\)](../debugger/general-tab-window-properties-dialog-box.md)  
 選択したウィンドウに関する情報が表示されます。  
  
 [&#91;スタイル&#93; タブ \(&#91;ウィンドウ プロパティ&#93; ダイアログ ボックス\)](../debugger/styles-tab-window-properties-dialog-box.md)  
 選択したウィンドウに適用されるスタイルが表示されます。  
  
 [&#91;ウィンドウ&#93; タブ \(&#91;ウィンドウ プロパティ&#93; ダイアログ ボックス\)](../debugger/windows-tab-window-properties-dialog-box.md)  
 選択したウィンドウに関係するウィンドウの情報が表示されます。  
  
 [&#91;クラス&#93; タブ \(&#91;ウィンドウ プロパティ&#93; ダイアログ ボックス\)](../debugger/class-tab-window-properties-dialog-box.md)  
 選択したウィンドウのクラスに関する情報が表示されます。  
  
 [&#91;プロセス&#93; タブ \(&#91;ウィンドウ プロパティ&#93; ダイアログ ボックス\)](../debugger/process-tab-window-properties-dialog-box.md)  
 選択したウィンドウのプロセスを識別します。
