# <a name="find-window-dialog-box"></a>[ウィンドウ検索] ダイアログ ボックス
ウィンドウを選択し、そのプロパティまたはメッセージを表示するには、このダイアログ ボックスを使用します。 このダイアログ ボックスを表示するには、**ウィンドウ検索**から、**スパイ**メニュー。  
  
 次の設定を使用できます。  
  
 **ファインダー ツール**  
 ドラッグ アンド ドロップに使用するツール ウィンドウを選択します。  
  
 **非表示に spy++**  
 Spy++ のメイン ウィンドウし、のまま、**ウィンドウ検索**他のアプリケーションの上に表示されるダイアログ ボックス。  
  
 **ハンドル**  
 検索するウィンドウの一意の ID。  
  
 **[キャプション]**  
 ウィンドウのキャプションのテキスト。  
  
 **クラス**  
 ウィンドウのクラスです。  
  
 **スタイル**  
 ウィンドウのスタイルです。  
  
 **Rect**  
 ウィンドウの外接する四角形。  
  
 **プロパティまたはメッセージを表示します。**  
 [Ok] を表示するかを判断、[ウィンドウのプロパティ ダイアログ ボックス](../debugger/window-properties-dialog-box.md)を開いたり、[メッセージ ビュー](../debugger/messages-view.md)です。
