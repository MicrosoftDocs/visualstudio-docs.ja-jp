# <a name="spy-menu-commands"></a>[スパイ] メニューのコマンド
**[Spy]** メニューには、Spy++ のウィンドウとビューを操作するためのコマンドが用意されています。

 **ウィンドウ** 新しいウィンドウ ビューが開きます。

 **プロセス** 新しいプロセス ビューが開きます。

 **スレッド** 新しいスレッド ビューが開きます。

 **Log Messages\(ログ メッセージ\)** 新しいメッセージ ビューが開き、このビューに対して [[メッセージ オプション] ダイアログ ボックス](../debugger/message-options-dialog-box.md)が開きます。

 **ウィンドウ検索** マウスを使用してウィンドウを選択するための **[FindWindow]** ダイアログ ボックスが表示されます。

 **終了** Spy++ ユーティリティが閉じます。
