# [スパイ] メニューのコマンド
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

\[スパイ\] メニューには、Spy\+\+ のウィンドウとビューを操作するためのコマンドが用意されています。  
  
 **\[ウィンドウ\]**  
 新しいウィンドウ ビューを開きます。  
  
 **\[プロセス\]**  
 新しいプロセス ビューを開きます。  
  
 **\[スレッド\]**  
 新しいスレッド ビューを開きます。  
  
 **\[メッセージのログ出力\]**  
 新しいメッセージ ビューを開き、そのビューの [&#91;メッセージ オプション&#93; ダイアログ ボックス](../debugger/message-options-dialog-box.md)を開きます。  
  
 **\[ウィンドウ検索\]**  
 \[ウィンドウ検索\] ダイアログ ボックスを表示し、マウスを使用してウィンドウを選択できます。  
  
 **\[終了\]**  
 Spy\+\+ ユーティリティを閉じます。
