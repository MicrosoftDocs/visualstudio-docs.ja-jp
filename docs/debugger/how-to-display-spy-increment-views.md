# How to: Display Spy++ Views
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Spy\+\+ を起動すると、[ウィンドウ ビュー](../debugger/windows-view.md)が開きます。これは、コンピューターで使用できるすべてのウィンドウとコントロールのツリー ビューです。  このルート ウィンドウのキャプションは、"ウィンドウ 1" になります。この他にも、[メッセージ ビュー](../debugger/messages-view.md)、[プロセス ビュー](../debugger/processes-view.md)、[スレッド ビュー](../debugger/threads-view.md)という 3 つの Spy\+\+ ビューを利用できます。  目的のビューは、**\[スパイ\]** メニューから表示できます。  詳細については、「[Spy\+\+ ビュー](../debugger/spy-increment-views.md)」を参照してください。
