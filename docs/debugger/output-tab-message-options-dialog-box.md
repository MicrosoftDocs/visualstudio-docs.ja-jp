# <a name="output-tab-message-options-dialog-box"></a>[出力] タブ ([メッセージ オプション] ダイアログ ボックス)
使用して、**出力**タブの一覧に各メッセージからデータを指定する[メッセージ ビュー](../debugger/messages-view.md)します。 表示する、[メッセージ オプション ダイアログ ボックス](../debugger/message-options-dialog-box.md)、選択**ログ メッセージ**から、**スパイ**メニュー。

 次の設定は [使用可能な**出力**] タブ。

 **行番号**行番号を表示します。

 **メッセージ レベルの入れ子**レベルごとに 1 つのピリオドを持つ入れ子になったメッセージのプレフィックスします。

 **生のメッセージ パラメーター** 16 進数を表示**wParam**と**lParam**値。

 **デコードされたメッセージ パラメーター**の特定のメッセージのデコード結果を表示、 **wParam**と**lParam**値。

 **生の戻り値**16 進数を表示**lResult**値を返します。

 **戻り値のデコード**の特定のメッセージのデコード結果を表示、 **lResult**値を返します。

 **メッセージのポストされた時間**(ポストされたメッセージのみ) の Windows システムが起動されてからの経過時間。

 **マウスの位置をメッセージ**(ポストされたメッセージのみ) のメッセージが投稿されたときのマウスの画面座標。

 **最大行数**現在選択されているメッセージ ビューで保持される行の数を制限します。

 **ファイルへのログ メッセージも**メッセージ ログの出力ファイルを指定します。 この出力ファイルは、メッセージ ログ ウィンドウを同時に書き込まれます。

 **既定値として設定を保存**新しいメッセージ ストリームの windows の設定を保存します。 Spy++ を終了すると、これらの設定が保存されます。
