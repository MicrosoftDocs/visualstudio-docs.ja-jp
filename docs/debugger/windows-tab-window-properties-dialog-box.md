# [ウィンドウ] タブ ([ウィンドウ プロパティ] ダイアログ ボックス)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

\[ウィンドウ\] タブを使用すると、選択したウィンドウに関係するウィンドウの情報を表示できます。  [&#91;ウィンドウ プロパティ&#93; ダイアログ ボックス](../debugger/window-properties-dialog-box.md)を表示するには、[ウィンドウ ビュー](../debugger/windows-view.md) ウィンドウにフォーカスを移動します。  ツリーで任意のウィンドウ ノードを選択し、\[表示\] メニューの \[プロパティ\] をクリックします。  
  
 \[ウィンドウ\] タブには、次の項目があります。  
  
|エントリ|Description|  
|----------|-----------------|  
|**\[次のウィンドウ\]**|ウィンドウ ツリー ビューに表示されている同じシーケンス \(Z オーダー\) 内の次の兄弟ウィンドウのハンドルです \(次のウィンドウが存在しない場合は "なし"\)。  次のウィンドウのプロパティを表示するには、このエントリを選択します。|  
|**\[前のウィンドウ\]**|ウィンドウ ツリー ビューに表示されている同じシーケンス \(Z オーダー\) 内の前の兄弟ウィンドウのハンドルです \(前のウィンドウが存在しない場合は "なし"\)。  前のウィンドウのプロパティを表示するには、このエントリを選択します。|  
|**\[親\]**|このウィンドウの親ウィンドウのハンドルです \(親が存在しない場合は "なし"\)。  親ウィンドウのプロパティを表示するには、このエントリを選択します。|  
|**\[1 つ目の子ウィンドウ\]**|ウィンドウ ツリー ビューに表示されているシーケンス \(Z オーダー\) 内にある、このウィンドウの 1 つ目の子ウィンドウのハンドルです \(子ウィンドウが存在しない場合は "なし"\)。  1 つ目の子ウィンドウのプロパティを表示するには、この値を選択します。|  
|**\[オーナー ウィンドウ\]**|このウィンドウのオーナー ウィンドウのハンドルです。  通常は、アプリケーションのメイン ウィンドウがシステム モーダル ダイアログ ウィンドウなどを所有します \(オーナーが存在しない場合は "なし"\)。  オーナー ウィンドウのプロパティを表示するには、このエントリを選択します。|
