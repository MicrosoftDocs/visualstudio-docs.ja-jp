# <a name="idiasymbolgetuavslot"></a>IDiaSymbol::get_uavSlot
再びスロットを取得します。  
  
## <a name="syntax"></a>構文  
  
```C++  
HRESULT get_uavSlot(   
   DWORD* pRetVal);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `pRetVal`  
 [out]ポインター、`DWORD`再びスロットを保持します。  
  
## <a name="return-value"></a>戻り値  
 成功した場合を返します`S_OK`、それ以外を返します`S_FALSE`またはエラー コード。
