# <a name="idiaenumsegmentsget_count"></a>IDiaEnumSegments::get_Count
セグメント数を取得します。

## <a name="syntax"></a>構文

```C++
HRESULT get_Count ( 
   LONG* pRetVal
);
```

#### <a name="parameters"></a>パラメーター
 pRetVal
- [out, retval] セグメント数を返します。

## <a name="return-value"></a>戻り値
 正常に終了した場合は、`S_OK` を返します。それ以外の場合は、エラー コードを返します。
