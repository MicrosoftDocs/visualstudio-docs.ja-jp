# <a name="idiaenumsourcefilesget_count"></a>IDiaEnumSourceFiles::get_Count
ソース ファイルの数を取得します。

## <a name="syntax"></a>構文

```C++
HRESULT get_Count ( 
   LONG* pRetVal
);
```

#### <a name="parameters"></a>パラメーター
 pRetVal

[出力] ソース ファイルの数を返します。

## <a name="return-value"></a>戻り値
 成功した場合は、`S_OK` を返します。それ以外の場合は、エラー コードを返します。
