# <a name="idiaenumdebugstreamdatareset"></a>IDiaEnumDebugStreamData::Reset
列挙のシーケンスの先頭にリセットします。

## <a name="syntax"></a>構文

```C++
HRESULT Reset ();
```

## <a name="return-value"></a>戻り値
 `S_OK` を返します。
