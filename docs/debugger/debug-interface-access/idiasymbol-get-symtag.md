# <a name="idiasymbolgetsymtag"></a>IDiaSymbol::get_symTag
シンボル型の分類子を取得します。  
  
## <a name="syntax"></a>構文  
  
```C++  
HRESULT get_symTag (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `pRetVal`  
 [out]値を返します、 [SymTagEnum 列挙型](../../debugger/debug-interface-access/symtagenum.md)シンボル型の分類子を指定する列挙です。  
  
## <a name="return-value"></a>戻り値  
 成功した場合を返します`S_OK`、それ以外を返します`S_FALSE`またはエラー コード。  
  
> [!NOTE]
>  戻り値の`S_FALSE`プロパティは、シンボルの使用可能なことを意味します。  
  
## <a name="example"></a>例  
  
```C++  
IDiaSymbol* pType;  
DWORD       tag = 0;  
pType->get_symTag( &tag );  
```
