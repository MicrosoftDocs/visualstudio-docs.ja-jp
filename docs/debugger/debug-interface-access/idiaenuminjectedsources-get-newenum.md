```C++  
HRESULT get__NewEnum (   
   IUnknown** pRetVal  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 pRetVal  
 [out, retval]返します、`IUnknown`を表すインターフェイスを<xref:System.Runtime.InteropServices.ComTypes.IEnumVARIANT>この列挙子のバージョン。
