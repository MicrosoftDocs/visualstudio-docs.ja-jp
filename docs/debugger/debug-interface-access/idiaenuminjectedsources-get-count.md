# <a name="idiaenuminjectedsourcesget_count"></a>IDiaEnumInjectedSources::get_Count
挿入されたソースの数を取得します。

## <a name="syntax"></a>構文

```C++
HRESULT get_Count ( 
   LONG* pRetVal
);
```

#### <a name="parameters"></a>パラメーター
 pRetVal

[out] 挿入されたソースの数を返します。

## <a name="return-value"></a>戻り値
 正常に終了した場合は、`S_OK` を返します。それ以外の場合は、エラー コードを返します。
