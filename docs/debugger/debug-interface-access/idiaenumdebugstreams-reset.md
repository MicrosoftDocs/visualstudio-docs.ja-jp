# <a name="idiaenumdebugstreamsreset"></a>IDiaEnumDebugStreams::Reset
列挙のシーケンスを先頭にリセットします。  
  
## <a name="syntax"></a>構文  
  
```C++  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>戻り値  
 `S_OK` を返します。
