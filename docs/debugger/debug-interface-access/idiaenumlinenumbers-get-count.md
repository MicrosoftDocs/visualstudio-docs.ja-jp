# <a name="idiaenumlinenumbersgetcount"></a>IDiaEnumLineNumbers::get_Count
行番号の数を取得します。  
  
## <a name="syntax"></a>構文  
  
```C++  
HRESULT get_Count (   
   LONG* pRetVal  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 pRetVal  
 [out]行番号の数を返します。  
  
## <a name="return-value"></a>戻り値  
 成功した場合を返します`S_OK`、それ以外のエラー コードを返します。
