# <a name="idiaenumtablesreset"></a>IDiaEnumTables::Reset
列挙シーケンスを先頭にリセットします。

## <a name="syntax"></a>構文

```C++
HRESULT Reset ();
```

## <a name="return-value"></a>戻り値
 `S_OK`を返します。
