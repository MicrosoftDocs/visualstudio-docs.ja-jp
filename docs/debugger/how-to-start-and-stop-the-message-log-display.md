# How to: Start and Stop the Message Log Display
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

[メッセージ ビュー](../debugger/messages-view.md) ウィンドウがアクティブなとき、Spy\+\+ の \[メッセージ\] メニューに \[ログ開始\] と \[ログ終了\] が表示され、対応する \[ログ開始\] ツール バーのボタンと \[ログ終了\] ツール バーのボタンがアクティブになります。  
  
### メッセージ ログの表示を開始するには  
  
1.  \[メッセージ\] メニューの \[ログ開始\] をクリックします。  
  
### メッセージ ログの表示を終了するには  
  
1.  \[メッセージ\] メニューの \[ログ終了\] をクリックします。
