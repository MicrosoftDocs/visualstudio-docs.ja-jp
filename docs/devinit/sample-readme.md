# <a name="sample-list"></a>サンプル リスト

> [!IMPORTANT]
> 2021 年 4 月 12 日以後は、Visual Studio 2019 から GitHub Codespaces への接続はサポートされなくなります。このプライベート プレビューは終了しています。 Microsoft では、クラウドを利用した内部ループと、さまざまな Visual Studio ワークロードに合わせて最適化された VDI ソリューションの進化し続けるエクスペリエンスに焦点を合わせています。 その一環として、`devinit` と関連ツールは利用できなくなります。 今後のプレビューとロードマップの情報のために、Visual Studio の開発者コミュニティ フォーラムに参加することをお勧めします。

次の表に、GitHub Codespaces で devinit を使用するために現在使用できる例の一覧を示します。

|---------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------|
| [**すべてのツール**](sample-all-tool.md)               | すべてのツールを使用する例。                                                                                              |
| [**eShopOnWeb**](sample-eshoponweb.md)            | [dotnet-architecture/eshoponweb](https://github.com/dotnet-architecture/eShopOnWeb) アプリをカスタマイズする例。          |
| [**プライベート ベータ版**](sample-private-preview.md)     | GitHub Codespaces Visual Studio プライベート ベータ版で使用されているカスタマイズの例。                                      |
| [**OpenCV**](sample-opencv.md)                   | OpenCV プロジェクトに必要なカスタマイズの例。                                                                |
| [**.NET Core ランタイム**](sample-dotnet-runtime.md) | .NET Core ランタイム [dotnet/runtime](https://github.com/dotnet/runtime) プロジェクトに必要なカスタマイズの例。 |
| [ **.NET Core アプリ**](sample-dotnet-core.md)        | devinit を使用して必要な .NET Core SDK をインストールするリポジトリの例。                                             |
| [**Node.js アプリ**](sample-nodejs.md)               | devinit を使用して Node.js Express プロジェクトの npm パッケージをインストールするリポジトリの例。                             |
