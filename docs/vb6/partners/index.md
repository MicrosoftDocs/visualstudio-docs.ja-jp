# <a name="visual-basic-60-offers-from-partners"></a>パートナーからの Visual Basic 6.0 プラン

このトピックでは、お客様が Visual Basic 6.0 から Visual Basic .NET への移行を成功させるために役立つ、パートナーの無償の商用ツールとソリューションについて情報を提供します。

Mobilize.Net の Visual Basic Upgrade Companion (VBUC) は、VB6 から .NET に移行するための業界最高レベルのツールです。 数百万行の実証された追跡レコードを C# と VB.NET に移行する VBUC は、時代遅れになったもののビジネスに不可欠なアプリケーションを、サードパーティへの依存にしばられることなく移行するのに役立ちます。 このツールのバージョンは無料で使用できます。</td>
