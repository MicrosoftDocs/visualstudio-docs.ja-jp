1. Visual Studio の [スタート] ウィンドウから **[新しいプロジェクトの作成]** を選択します。
1. **[ASP.NET Core Web アプリ]** を選択してから、 **[次へ]** を選択します。
1. 新しいアプリケーションの名前を入力 (または既定の名前を使用) し、ディスク上の場所を指定した後、 **[次へ]** を選択します。
1. 対象にする .NET バージョンを選択します。 わからない場合は、LTS (長期サポート) リリースを選択してください。

   ![Web プロジェクトの作成 - [追加情報] 画面](../../media/create-aspnet5-app/create-web-project3.png)

1. **[HTTPS 用の構成]** チェックボックスを選択するかオフにして、SSL サポートを使用するかどうかを選択します。
1. **[Docker サポートを有効にする]** チェック ボックスを選択します。
1. コンテナーの種類 (Windows または Linux) を選択し、**[作成]** を選択します。
