1. Visual Studio の [スタート] ウィンドウから **[新しいプロジェクトの作成]** を選択します。
1. **[ASP.NET Core Web アプリケーション]** を選択し、 **[次へ]** を選択します。
1. 新しいアプリケーションの名前を入力し (または既定の名前を使用)、 **[追加]** を選択します。
1. **[Web アプリケーション]** を選択します。
1. **[HTTPS 用の構成]** チェックボックスを選択するかオフにして、SSL サポートを使用するかどうかを選択します。
1. **[Docker サポートを有効にする]** チェック ボックスを選択します。
1. コンテナーの種類 (Windows または Linux) を選択し、 **[作成]** を選択します。
