# Visual Studio アーキテクチャ ツーリング ガイダンス
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

このアーキテクチャ ツーリング ガイダンスは、 Visual Studio 2010 のモデリング ツールの使用方法に関する実用的なガイダンスのダウンロード可能パッケージです。  アーキテクチャ ツーリング ガイダンスでは、すべてのモデリング ツールを統合的に使用する方法の概要を説明し、ツールが使用される最も一般的なシナリオについて説明します。
