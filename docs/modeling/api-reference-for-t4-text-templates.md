# T4 テキスト テンプレートの API リファレンス
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

テキスト テンプレート API を使用すると、[テキスト テンプレート](../modeling/code-generation-and-t4-text-templates.md)変換を呼び出し、カスタマイズできます。  
  
## 名前空間  
  
|名前空間|目的|  
|----------|--------|  
|<xref:Microsoft.VisualStudio.TextTemplating>|テキスト テンプレート変換機能のためのクラスが用意されています。  テキスト テンプレート変換エンジンは Visual Studio に統合されており、テキスト テンプレート ファイルを生成されるテキスト出力ファイルに変換します。|  
|<xref:Microsoft.VisualStudio.TextTemplating.Modeling>|UML モデルとドメイン固有言語に関係するテキスト変換機能を提供します \([!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] ModelBus へのアクセスなど\)。|  
|<xref:Microsoft.VisualStudio.TextTemplating.VSHost>|[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] のテキスト テンプレート サービスにアクセスできます。|
