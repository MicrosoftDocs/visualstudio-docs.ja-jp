# テキスト テンプレートでのエスケープ シーケンスの使用
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

テキスト テンプレートでは、テキスト テンプレート タグを生成したり、C\# コードの制御文字や引用符をエスケープしたりするために、エスケープ シーケンスを使用できます。  
  
 標準コード ブロックの開始タグと終了タグを出力ファイルに出力するには、タグを次のようにエスケープします。  
  
```  
\<# ... \#>  
```  
  
 他のテキスト テンプレート ディレクティブとコード ブロック タグでも、同じ方法を使用できます。  
  
 テキスト テンプレート タグをエスケープするための文字列をテキスト ブロックに含める場合は、次のエスケープ シーケンスを使用します。  
  
-   テキスト テンプレート タグの前に偶数個のエスケープ \(\\\) 文字が出現した場合は、その半分がテンプレート パーサーによって出力され、シーケンスはテキスト テンプレート タグとして処理されます。  たとえば、4 つのエスケープ文字がテキスト テンプレートに存在した場合、生成後のファイルには、2 つの "\\" 文字が存在することになります。  
  
-   テキスト テンプレート タグの前に奇数個のエスケープ \(\\\) 文字が出現した場合は、その半分の "\\" 文字とタグ自体 \(\<\# または \#\>\) がテンプレート パーサーによって出力されます。  タグはテキスト テンプレート タグとは見なされません。  
  
-   制御文字または引用符 \(C\# のみ\) をエスケープする以外の場所にエスケープ文字 \(\\\) が出現した場合、その文字は直接出力されます。
