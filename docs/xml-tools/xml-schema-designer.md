# <a name="xml-schema-designer-overview"></a>XML スキーマ デザイナーの概要

XML スキーマ デザイナー (XSD デザイナー) は、スキーマ セットをさまざまな抽象レベルで視覚的に表現するためのグラフィカル ツールです。 XSD デザイナーの主要コンポーネントを次に示します。

- [XML スキーマ エクスプローラー](../xml-tools/xml-schema-explorer.md)では、XML スキーマ ツリーを参照および移動して、検索を実行できます。

- [スタート ビュー](../xml-tools/start-view.md)は、XSD デザイナーの起点です。 スタート ビューからは、XSD デザイナーの他のビューに移動したり、スキーマ セットの詳細を表示したりすることができます。

- [グラフ ビュー](../xml-tools/graph-view.md)では、スキーマ セットの概要とスキーマ ノード間のリレーションシップを表示できます。

- [コンテンツ モデル ビュー](../xml-tools/content-model-view.md)には、単純型、複合型、要素、グループ、属性、属性グループなど、ローカル スキーマ ノードとグローバル スキーマ ノードの詳細がグラフィック表示されます。
