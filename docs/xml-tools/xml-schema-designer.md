# <a name="xml-schema-designer-overview"></a>XML スキーマデザイナーの概要

XML スキーマ デザイナー (XSD デザイナー) は、スキーマ セットをさまざまな抽象レベルで視覚的に表現するためのグラフィカル ツールです。 XSD デザイナーの主要なコンポーネントは次のとおりです。

- Xml スキーマ[エクスプローラー](../xml-tools/xml-schema-explorer.md)。 xml スキーマツリーを参照して移動したり、検索を実行したりすることができます。

- [スタートビュー](../xml-tools/start-view.md)。 XSD デザイナーの起動ポイントです。 [スタート] ビューでは、XSD デザイナーの他のビューに移動して、スキーマセットの詳細を確認できます。

- [グラフビュー](../xml-tools/graph-view.md)。スキーマセットの概要とスキーマノード間のリレーションシップを表示できます。

- [コンテンツモデルビュー](../xml-tools/content-model-view.md)。単純型、複合型、要素、グループ、属性、属性グループなど、ローカルスキーマノードとグローバルスキーマノードの詳細がグラフィカルに表示されます。
