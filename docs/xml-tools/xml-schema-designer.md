# <a name="xml-schema-designer-overview"></a>XML スキーマ デザイナーの概要

XML スキーマ デザイナー (XSD デザイナー) は、スキーマ セットをさまざまな抽象レベルで視覚的に表現するためのグラフィカル ツールです。 XSD デザイナーの主要なコンポーネントは次のとおりです。

- [XML スキーマ エクスプ ローラー](../xml-tools/xml-schema-explorer.md)を参照し、XML スキーマのツリーを移動し、検索を実行できます。

- [スタート ビュー](../xml-tools/start-view.md)、XSD デザイナーの起点であります。 スタート ビューからは、XSD デザイナーの他のビューに移動し、スキーマ セットの詳細を参照してください。

- [グラフ ビュー](../xml-tools/graph-view.md)、スキーマ セットとスキーマ ノード間のリレーションシップの概要を確認できます。

- [コンテンツ モデル ビュー](../xml-tools/content-model-view.md)、単純および複雑な型、要素、グループ、属性、および属性グループを含む、ローカルとグローバル スキーマ ノードの詳細のグラフィカル表現を提供します。
