# <a name="xml-schema-designer"></a>XML スキーマ デザイナー

XML スキーマ デザイナー (XSD デザイナー) は、スキーマ セットをさまざまな抽象レベルで視覚的に表現するためのグラフィカル ツールです。 XSD デザイナーには、次の主要コンポーネントがあります。

-   [XML スキーマ エクスプ ローラー](../xml-tools/xml-schema-explorer.md)参照し、XML スキーマのツリーを移動し、検索を実行することができます。

-   [スタート ビュー](../xml-tools/start-view.md)起動ポイントは、XSD デザイナーの以外の場合は、スタート ビューから、XSD デザイナーの他のビューに移動して、スキーマ セットの詳細を参照してください。

-   [グラフ ビュー](../xml-tools/graph-view.md)スキーマ セットとスキーマ ノード間のリレーションシップの概要を表示することができます。

-   [コンテンツ モデル ビュー](../xml-tools/content-model-view.md)単純型と複合型、要素、グループ、属性、および属性グループを含む、ローカルとグローバル スキーマ ノードの詳細のグラフィカル表現を提供します。
