# How to: Add Comments to Class Diagrams (Class Designer)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

コメントの図形を使用して、クラス ダイアグラムに注釈を付けることができます。  コメントの図形には、**\[テキスト\]** という 1 つのプロパティがあり、このプロパティにテキストを入力できます。  コメントの図形は、ダイアグラム領域だけに存在し、コードには存在しません。  
  
 コメントは、**クラス デザイナー**のクラス ダイアグラム ビューに存在します。同じプロジェクトに 2 番目のクラス ダイアグラムを開いた場合、最初のビューで作成したコメントは表示されません。  ダイアグラムを削除した場合は、そのダイアグラムに含まれるすべてのコメントも削除されます。  
  
 コメントの図形のサイズは変更できますが、背景色、フォント、フォント サイズなど、その他の外観は変更できません。  
  
### コメントを追加するには  
  
1.  **クラス デザイナーのツールボックス**からクラス ダイアグラムにコメントをドラッグします。  
  
2.  ダイアグラムで新しいコメントの図形をクリックし、目的のテキストを入力します。
