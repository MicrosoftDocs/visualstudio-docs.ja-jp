# <a name="use-pattern-matching"></a>パターン マッチングの使用

このリファクタリングは以下に適用されます。

- C#

**概要:** パターン マッチングを使用します。

**条件:** 新しい [C# パターン マッチング](https://docs.microsoft.com/dotnet/csharp/whats-new/csharp-9#pattern-matching-enhancements)機能を使用する必要があります。

**理由:** このリファクタリングにより、コードがより明確かつ簡潔になります。

## <a name="how-to"></a>方法

1. ステートメント内にカレットを置きます。

2. 行の任意の場所で **Ctrl**+ **.** キーを押すと、 **[クイック アクションとリファクタリング]** メニューをトリガーします。

3. **[パターン マッチングを使用します]** を選択します。

    ![クラスを抽象にする](media/use-pattern-matching-not-syntax.png)
