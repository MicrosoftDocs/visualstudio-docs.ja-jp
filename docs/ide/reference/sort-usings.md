# <a name="sort-usings"></a>using を並べ替える

このリファクタリングは以下に適用されます。

- C#

**概要:** using を並べ替えます。

**条件:** ファイルの先頭にある `using` ディレクティブを並べ替えて、アルファベット順にします。 

**理由:** using ディレクティブが見つけやすくなります。

## <a name="how-to"></a>方法

1. メニュー バーで **[編集]** を選択します。
2. **[Intellisense]**  >  **[using を並べ替えます]** を選択します。

   ![using を並べ替える](media/sort-usings.png)
