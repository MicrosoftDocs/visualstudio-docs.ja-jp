# <a name="evaluate-statement-command"></a>ステートメントの評価コマンド

指定したステートメントを評価し、表示します。

## <a name="syntax"></a>構文

```cmd
>Debug.EvaluateStatement text
```

## <a name="arguments"></a>引数

`text`

必須。 評価するステートメント。

## <a name="example"></a>例

```cmd
>Debug.EvaluateStatement args.Length
```
