# <a name="options-dialog-box-environment--preview-features"></a>[オプション] ダイアログ ボックス:[環境] \> [プレビュー機能]

**[オプション]** ダイアログ ボックスのこのページを使用して、Visual Studio のプレビュー機能を有効または無効にします。 **[プレビュー機能]** オプション ページは、Visual Studio でさまざまなプレビュー機能が利用できるようになるたびに変更されます。

![Visual Studio 2019 の [プレビュー機能] オプション ページ](media/environment-preview-features-page.png)

このダイアログ ボックスにアクセスするには、**[ツール]** メニューの **[オプション]** をクリックし、**[環境]** > **[プレビュー機能]** の順に選択します。 または、Visual Studio の **[検索]** ボックスに「**プレビュー機能**」と入力します (**Ctrl**+**Q** を押すと、**[検索]** ボックスにフォーカスが移動します)。
