# <a name="convert-dialog-box"></a>[変換] ダイアログ ボックス
**[変換]** ダイアログ ボックスでは、古いプロジェクトから新しいプロジェクトを作成するための変換ツールを選択できます。たとえば、Visual Basic 6 プロジェクトから [!INCLUDE[vbprvblong](../../ide/reference/includes/vbprvblong_md.md)] プロジェクトを作成するような場合です。 また、[Visual Studio プロジェクト コンバーター](https://github.com/ssvaidya/VSProjectConverter)を使用して、前のバージョンの Visual Studio で作成したプロジェクトをアップグレードすることもできます。   
  
## <a name="opening-the-convert-dialog-box"></a>[変換] ダイアログ ボックスを開く  
  
#### <a name="to-access-the-convert-dialog-box"></a>[変換] ダイアログ ボックスにアクセするには  
  
1.  メニュー バーから **[ファイル]**、 **[開く]**、 **[変換]**の順に選択します。  
  
     現在使用できるコード変換ツールの一覧が表示されます。  
  
2.  アプリケーションに適したツールを選択します。
