# <a name="-devenvexe"></a>/? (devenv.exe)
すべての `devenv` スイッチをそれぞれの簡単な説明とともに一覧表示するメッセージ ボックスを表示します。  
  
## <a name="syntax"></a>構文  
  
```  
devenv /?  
```
