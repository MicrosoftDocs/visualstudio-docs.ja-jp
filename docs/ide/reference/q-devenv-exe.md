# <a name="-devenvexe"></a>/? (devenv.exe)

すべての `devenv` スイッチを各スイッチの簡単な説明と共に一覧表示するメッセージ ボックスを表示します。

## <a name="syntax"></a>構文

```shell
devenv /?
```
