# <a name="refactoring-code-generation-and-quick-actions-in-visual-studio"></a>Visual Studio でのリファクタリング、コード生成、クイック アクション

Visual Studio には、コードを記述するときに役立つさまざまなツールが用意されています。  これらのツールの機能は言語の構文に密接に結び付けられているため、Visual Studio の各言語サービスには、それぞれ独自の機能があります。

## <a name="refactoring"></a>リファクタリング

リファクタリングとは、コードの動作を変更することなく、簡単にコードを維持、理解、拡張するためのプロセスです。  詳細については、次のページを参照してください。

* [リファクタリング (C#)](../csharp-ide/refactoring-csharp.md)

* [リファクタリング (Visual Basic)](../vb-ide/refactoring-vb.md)

* [コードの作成とリファクタリング (C++)](/cpp/ide/writing-and-refactoring-code-cpp)

* [Visual F# Power Tools](https://marketplace.visualstudio.com/items?itemName=FSharpSoftwareFoundation.VisualFPowerTools)&mdash;F# のリファクタリングのサポートは、このサード パーティ製の Visual Studio 拡張機能で提供されます。

## <a name="code-generation"></a>コード生成

Visual Studio では自動的に一部のソース コードが生成されるため、実際の作業に集中できます。 詳細については、次のページを参照してください。

* [コード生成 (C#)](../csharp-ide/code-generation-csharp.md)

* [コード生成 (Visual Basic)](../vb-ide/code-generation-vb.md)

## <a name="quick-actions"></a>クイック アクション

クイック アクションを使うと、コードのリファクタリング、生成、その他の変更を、1 つの操作で簡単に行うことができます。 これらのアクションは、電球アイコン ![小さい電球アイコン](media/vs2015_lightbulbsmall.png "VS2017_LightBulbSmall") を使うか、または適切なコード行にカーソルを置いて **Ctrl** + **.** キーを押すと 適用できます。

* [クイック アクション](quick-actions.md)

* [コード スタイルとクイック アクション](code-styles-and-quick-actions.md)
