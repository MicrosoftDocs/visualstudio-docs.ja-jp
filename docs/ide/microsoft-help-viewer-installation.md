# <a name="microsoft-help-viewer-installation"></a>Microsoft ヘルプ ビューアーのインストール
Visual Studio や SQL Server などのいくつかの製品では、Microsoft ヘルプ ビューアーでヘルプ コンテンツを表示できます。

ヘルプ ビューアーは、Visual Studio のオプションのインストール コンポーネントです。 Visual Studio インストーラーを使用してインストールするには、次の手順を実行します。

1. [スタート] メニューから **Visual Studio インストーラー**を開くか、Visual Studio が開いている場合は、**[ツール]** の **[ツールと機能を取得...]** を選ぶと、Visual Studio インストーラーが開きます。

2. **[個々のコンポーネント]** タブを選択し、**[コード ツール]** セクションの下で **[ヘルプ ビューアー]** を選択します。

   ![VS インストーラー ヘルプ ビューアー コンポーネント](media/help_viewer_vs_installer.png)

3. **[変更]** ボタンを選択して、Microsoft ヘルプ ビューアーのインストールを開始します。

Microsoft ヘルプ ビューアーを簡単にインストールするもう 1 つの方法は、**[クイック起動]** ボックスを使用することです。

1. Visual Studio タイトル バーにある **[クイック起動]** ボックスに「**ヘルプ ビューアー**」と入力します。

   ![[クイック起動] ボックス](media/help_viewer_quick_launch.png)

2. **ヘルプ ビューアー (個々のコンポーネント)** と呼ばれるのインストール結果を選択します。

3. 開いたダイアログ ボックスで、**[インストール]** ボタンを選択します。

   ![[インストール] ボタン](media/help_viewer_install.png)
