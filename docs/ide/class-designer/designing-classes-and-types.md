# <a name="designing-classes-and-types-class-designer"></a>クラスおよび型のデザイン (クラス デザイナー)

> [!NOTE]
>  クラス デザイナーを使うときは、生成されたコードを実行する前に調べて、コードがセキュリティ コンテキストに適していることを確認する必要があります。
