# ビルド プラットフォームについて
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

ソリューションの複数のバージョンを格納し、別のターゲット プラットフォームに適用されるプロパティを射影できます。  たとえば、x86 プラットフォームおよび x64 プラットフォームを対象とするデバッグ構成を対象とするデバッグ構成を作成できます。  簡単に複数の構成を作成できるようにすぐにアクティブなプラットフォームを変更できます。
