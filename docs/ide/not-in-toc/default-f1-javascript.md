# <a name="javascript-f1-help-for-visual-studio"></a>Visual Studio の JavaScript の F1 ヘルプ

JavaScript は、Web ページや他のアプリケーションに埋め込むことができるスクリプト言語です。 Microsoft Edge 従来版と Internet Explorer の場合、JavaScript の Microsoft による実装は、ECMAScript 言語仕様第 5 版に準拠しています。 これは、ECMA 規格に含まれていない別の機能も提供します。

言語キーワードとコード要素のリファレンス ヘルプについては、次を参照してください。

- JavaScript のヘルプについては、[MDN Web Docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference)。 検索ボックスに、探しているキーワードの名前を入力します。
- TypeScript のヘルプについては、[TypeScript ドキュメント](https://www.typescriptlang.org/docs)。
- Microsoft の公式ドキュメントでヘルプを探している場合は、docs.microsoft.com 上の[検索ページ](/search)。

> [!IMPORTANT]
> Microsoft は、[MDN Web Docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference) を 1 か所にまとめた Web の主要な開発リファレンスにするためのコミュニティ全体の取り組みに参加しています。 Microsoft による JavaScript API リファレンスの 500 以上のすべてのページが、[docs.microsoft.com](https://docs.microsoft.com) から対応する MDN に移植されました。 従来の JS 拡張機能のリファレンスも、履歴とサービスのために移植されています。
