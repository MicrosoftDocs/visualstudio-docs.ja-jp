# <a name="oops-no-f1-help-match-was-found"></a>問題が発生しました。 一致する `F1` のヘルプが見つかりませんでした

:::image type="content" source="../media/search-error-page-sm.png" alt-text="申し訳ございません。一致する項目が見つかりませんでした" border="false":::

## <a name="now-what"></a>次は何でしょうか。

申し訳ございません。 検索対象が見つからなかったようです。 

**ウィンドウ、キーワード、または API 名の `F1` ヘルプをお探しの場合:**
- あなたのお気に入りの検索エンジンで試す **[推奨]**
- [docs.microsoft.com](/) のツール バーを使用して検索する 
- [.NET API ブラウザー](/dotnet/api/)を検索するか、[テキスト コード エディターの概要](../../ide/writing-code-in-the-code-and-text-editor.md)をお読みください
- ナレッジ ベースで [Visual Studio のトラブルシューティング](/troubleshoot/visualstudio/welcome-visual-studio/)を検索してください


**クリックするつもりはなかった場合、** [`F1` ヘルプ キーを再マッピングするか無効にします](../not-in-toc/change-f1-help-key.md)
