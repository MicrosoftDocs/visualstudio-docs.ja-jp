# <a name="typescript-f1-help-for-visual-studio"></a>Visual Studio の TypeScript の F1 ヘルプ

TypeScript は、JavaScript の型指定されたスーパーセットとなる言語です。したがって、JavaScript の構文は TypeScript でも有効です。 TypeScript によって、さまざまな種類の値の使用方法に関する規則が追加されます。

言語キーワードとコード要素のリファレンス ヘルプについては、次を参照してください。

- TypeScript のヘルプについては、[TypeScript ドキュメント](https://www.typescriptlang.org/docs)。
- JavaScript のヘルプについては、[MDN Web Docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference)。 検索ボックスに、探しているキーワードの名前を入力します。
- Microsoft の公式ドキュメントでヘルプを探している場合は、docs.microsoft.com 上の[検索ページ](/search)。
