# <a name="globalize-applications"></a>アプリケーションのグローバル化

グローバリゼーションは、複数のロケールで機能するソフトウェア製品をデザインおよび開発するプロセスです。 次の記事は、Windows フォームと Web フォームの両方のページに当てはまります。

 [グローバルな Windows フォームおよび Web フォームにおけるカルチャ固有のクラス](../ide/culture-specific-classes-for-global-windows-forms-and-web-forms.md) カルチャの設定に従って日付、時刻、数値、通貨、その他の情報の書式を設定するクラスについて説明します。
