# <a name="internet-resources-used-by-visual-studio"></a>Visual Studio で使用されるインターネット リソース

Visual Studio は接続されている環境です。

- Visual Studio に[サインイン](../ide/signing-in-to-visual-studio.md)すると、ユーザー アカウントだけでなく、そのアカウントにリンクされているすべてのアカウントに関連付けられているすべてのサービスにアクセスできるようになります。 Visual Studio を使用するどの場所にでも、アカウントの設定はユーザーと共に移動します。  
- SDK などのインストールのコンポーネントに使用可能な更新プログラムがあった場合、Visual Studio は[ユーザーに通知](../ide/visual-studio-notifications.md)します。  
- [NuGet](/nuget/) パッケージをダウンロードして、ライブラリを取得することができます。  
- Visual Studio [拡張機能](../ide/finding-and-using-visual-studio-extensions.md)をダウンロードすることができます。  
- Azure サービスに接続することができます。  
- スタート ページは、さまざまな Web サイトからの最新の開発者向け情報で更新されます。  
- その他  

このセクションのトピックは、Visual Studio でインターネット リソースを使用するさまざまな方法に関するものです。
