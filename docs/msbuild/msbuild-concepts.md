# <a name="msbuild-concepts"></a>MSBuild の概念

MSBuild には、ビルド プラットフォームでソフトウェアを構築する方法を制御するために使用できる、基本的な XML スキーマが用意されています。 ビルド内のコンポーネントとその構築方法を指定するには、MSBuild の「プロパティ、アイテム、タスク、ターゲット」を使用します。
