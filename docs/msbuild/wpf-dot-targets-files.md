# WPF .Targets Files
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

[!INCLUDE[TLA#tla_winclient](../misc/includes/tlasharptla_winclient_md.md)] は、特別な .targets ファイル \(**Microsoft.WinFX.targets**\) にまとめられた [!INCLUDE[TLA2#tla_wpf](../msbuild/includes/tla2sharptla_wpf_md.md)] 固有の一連のタスクを追加することによって、[!INCLUDE[TLA#tla_msbuild](../msbuild/includes/tlasharptla_msbuild_md.md)] を拡張します。  このファイルは、[!INCLUDE[TLA#tla_winclient](../misc/includes/tlasharptla_winclient_md.md)] で [!INCLUDE[TLA2#tla_msbuild](../msbuild/includes/tla2sharptla_msbuild_md.md)] プロジェクトをビルドするために必要な [!INCLUDE[TLA2#tla_msbuild](../msbuild/includes/tla2sharptla_msbuild_md.md)] タスクのセットをまとめたものです。
