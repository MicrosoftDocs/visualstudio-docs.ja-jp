 アドイン: プライマリアプリケーションに追加されたユーティリティアプリケーション、ドライバー、またはその他のソフトウェア。 Visual Studio 統合開発環境 (IDE: integrated development environment) では、アドインは IDE の機能を拡張するオートメーションベースのアプリケーションです。

 オートメーションモデルオートメーションモデルは、以前のバージョンの Visual Studio では、機能拡張モデルとして知られています。これは、IDE を駆動する基になるルーチンへのアクセスを可能にするプログラミングインターフェイスです。 アドイン、ウィザード、およびマクロは、オートメーションモデルのオブジェクトを使用して、IDE の機能を制御または拡張します。

 コマンド UI コンテキストの GUID と、UI コマンドまたは要素 (ツールバーなど) の表示を関連付けます。 コマンド UI コンテキストは、ウィンドウにアタッチされていないので、選択コンテキストとは異なります。

- マクロの記録を有効にする GUID を割り当てます。

- デバッグモードをアクティブ化したり、エディターでデザインモードと実行モードを切り替えるために、GUID を割り当てます。
