# <a name="visual-language-dictionary-for-visual-studio"></a>Visual Studio のビジュアル言語辞書
Visual Studio では、そのデザインとしても使用し、映像などのイメージの構成要素の概念を示すために共通のビジュアル言語を使用します。 新しいアイコンをデザインするときは、独自のイメージで伝達する概念を識別するためにこの辞書を使用します。  
  
 **一般的なガイドライン**  
  
-   辞書の定義では、これらの視覚的概念を常に使用する必要があります。 それ以外の場合、一貫性のあるような意味を共通の要素をユーザーが混乱する可能性があります。  
  
-   画像が鮮明なと簡単に区別できるようにのアイコンでのみ使用する最も関連する、または重要な概念を選択します。  
  
-   各イメージの&1; つまたは&2; つの概念を使用します。  
  
-   さらに多くの方法の設計のアイコンでは、次を参照してください。[アイコンの設計](../../extensibility/ux-guidelines/images-and-icons-for-visual-studio.md#BKMK_IconDesign)します。  
  
 **セクションへジャンプします。**  
  
1.  [A ~ Z の概念](../../extensibility/ux-guidelines/visual-language-dictionary-for-visual-studio.md#BKMK_VLDConcepts)  
  
2.  [製品とプラットフォーム](../../extensibility/ux-guidelines/visual-language-dictionary-for-visual-studio.md#BKMK_VLDProducts)  
  
3.  [矢印を使用して一般的な概念](../../extensibility/ux-guidelines/visual-language-dictionary-for-visual-studio.md#BKMK_VLDArrows)  
  
4.  [状態インジケーター](../../extensibility/ux-guidelines/visual-language-dictionary-for-visual-studio.md#BKMK_VLDStatus)
