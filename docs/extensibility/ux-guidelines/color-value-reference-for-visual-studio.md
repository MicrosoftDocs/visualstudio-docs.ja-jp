# <a name="color-value-reference-for-visual-studio"></a>Visual Studio のカラー値の参照
このトピックでは、新しい色のトークンを作成するための参照です。 新しい UI を作成するときは、各主要なテーマの色とハイ コントラスト テーマ用には、その要素のシステムの色の名前を選択するのに既存の UI からのような色を使用できます。 

新しい色のトークンの作成の実装については、次を参照してください。[色のテーマ ツール](../../extensibility/internals/color-theming-tools.md)です。

既存の UI で使用する色のトークンを参照してください。 [for Visual Studio の色を共有](../../extensibility/ux-guidelines/shared-colors-for-visual-studio.md)です。

このリファレンス ガイドを使用する場合に注意してください。

-   **フォア グラウンドとバック グラウンドの両方の色合いを一緒に常にコピーします。** 色合いが好きという理由だけ、1 つの UI 要素別の背景から、前景を選択しないでください。  

-   **これらは、16 進コード、色のトークンではありません。** トークンを今後変更される可能性がどのように制御できないので、これらの UI 要素の実際の色のトークン名を使用することはできません。 代わりに、独自のカテゴリ内のトークンを作成し、表に示す色合いの値を使用します。 色のトークンの既存の UI で[for Visual Studio の色を共有](../../extensibility/ux-guidelines/shared-colors-for-visual-studio.md)です。

- **ハイ コントラスト見本の比較のみ表示されます。** ハイ コントラストの UI でのシステム カラー名を使用し、これら見本でスポイト ツールまたは色の選択を使用しないでください。 このトピックでハイ コントラスト見本は、Windows のハイコントラスト #1 テーマを反映します。 色の外観は、他のハイ コントラスト テーマで異なります。 

-   **最良の結果をペア カラー設定 UI を使用した例のようにします。**
