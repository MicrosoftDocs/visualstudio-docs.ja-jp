# <a name="color-value-reference-for-visual-studio"></a>Visual Studio の色の値のリファレンス

このトピックは、新しいカラートークンを作成するための参考資料です。 新しい UI を作成するときに、既存の UI と同様の色を使用して、各メインテーマの色と、その要素のシステムカラー名をハイコントラストテーマに選択できます。

新しいカラートークンの作成に関する実装情報については、「 [カラーテーマツール](../../extensibility/internals/color-theming-tools.md)」を参照してください。

既存の UI で使用するカラートークンについては、「 [Visual Studio の共有色](../../extensibility/ux-guidelines/shared-colors-for-visual-studio.md)」を参照してください。

このリファレンスガイドを使用する場合は、次の点に注意してください。

- **常に、前景色と背景色の両方を一緒にコピーします。** 1つの UI 要素から前景を選択しないでください。また、色合いが必要なので、別の UI 要素から背景を選択しないでください。

- **これらは、色トークンではなく、16進数のコードです。** これらの UI 要素の実際の色のトークン名を使用することはできません。これは、将来、トークンがどのように変更されるかを制御できないためです。 代わりに、独自のカテゴリ内にトークンを作成し、テーブルに表示される色合いの値を使用します。 既存の UI の色トークンは、「 [Visual Studio の共有色](../../extensibility/ux-guidelines/shared-colors-for-visual-studio.md)」にあります。

- **ハイコントラストの見本は比較のためだけに表示されます。** ハイコントラスト UI のシステムカラー名を使用します。これらの見本では、スポイトやカラーピッカーを使用しないでください。 このトピックのハイコントラストの見本は、Windows のハイコントラスト #1 テーマを反映しています。 色は、他のハイコントラストテーマでは外観が異なります。

- **最良の結果を得るには、例と同様に、カラーセットと UI をペアにします。**
