# <a name="add-commands-to-toolbars"></a>ツールバーにコマンドを追加する
ツールバーは、コマンドにバインドされたボタンを含む、水平または垂直のストリップです。 Visual Studio の多くの異なるウィンドウ (Visual Studio のメインウィンドウ、ソリューションエクスプローラー、独自のカスタムツールウィンドウ) に、独自のコマンドをツールバーに追加できます。

 このセクションでは、ツールバーのコマンドを使用して実行できるさまざまな操作について説明します。

## <a name="in-this-section"></a>このセクションの内容
- [ツールバーの追加](../extensibility/adding-a-toolbar.md) メインの Visual Studio ウィンドウのツールバーのセットにツールバーを追加する方法について説明します。

- [ツールウィンドウにツールバーを追加する](../extensibility/adding-a-toolbar-to-a-tool-window.md) ツールウィンドウにツールバーを追加する方法について説明します。

- [ソリューションエクスプローラーツールバーにコマンドを追加する](../extensibility/adding-a-command-to-the-solution-explorer-toolbar.md) ソリューションエクスプローラーのツールバーにコマンドを追加する方法について説明します。

- [ツールバーにメニューコントローラーを追加する](../extensibility/adding-a-menu-controller-to-a-toolbar.md) ツールバーにメニューコントローラーを追加する方法について説明します。

- [メニュー項目を動的に追加](../extensibility/dynamically-adding-menu-items.md) するさまざまなコンテキストでメニュー項目を動的に追加する方法について説明します。
