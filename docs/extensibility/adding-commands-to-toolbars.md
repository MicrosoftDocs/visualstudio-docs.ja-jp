# <a name="add-commands-to-toolbars"></a>ツールバーにコマンドを追加する
ツールバーは、コマンドにバインドされたボタンを含む水平または垂直のストリップです。 Visual Studio の多くのウィンドウ (メインの Visual Studio ウィンドウ、ソリューション エクスプローラー、および独自のカスタム ツール ウィンドウ) のツール バーに独自のコマンドを追加できます。

 このセクションでは、ツールバーのコマンドで実行できる操作について説明します。

## <a name="in-this-section"></a>このセクションの内容
- [ツールバーを追加する](../extensibility/adding-a-toolbar.md)Visual Studio のメイン ウィンドウのツール バー セットにツール バーを追加する方法について説明します。

- [ツール ウィンドウにツール バーを追加する](../extensibility/adding-a-toolbar-to-a-tool-window.md)ツール ウィンドウにツール バーを追加する方法について説明します。

- [ソリューション エクスプローラーのツール バーにコマンドを追加する](../extensibility/adding-a-command-to-the-solution-explorer-toolbar.md)ソリューション エクスプローラーのツール バーにコマンドを追加する方法について説明します。

- [メニュー コントローラをツールバーに追加する](../extensibility/adding-a-menu-controller-to-a-toolbar.md)メニュー コントローラをツール バーに追加する方法について説明します。

- [メニュー項目を動的に追加する](../extensibility/dynamically-adding-menu-items.md)さまざまなコンテキストでメニュー項目を動的に追加する方法について説明します。
