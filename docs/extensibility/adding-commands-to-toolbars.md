# <a name="add-commands-to-toolbars"></a>ツール バーにコマンドを追加する
ツール バーとは、コマンドにバインドされたボタンを含む、水平または垂直のストリップです。 Visual Studio のさまざまなウィンドウ (Visual Studio のメイン ウィンドウ、ソリューション エクスプローラー、独自のカスタム ツール ウィンドウ) のツール バーに独自のコマンドを追加できます。

 このセクションでは、ツール バーのコマンドに関して実行できるいくつかの操作を示します。

## <a name="in-this-section"></a>このセクションの内容
- [ツール バーを追加する](../extensibility/adding-a-toolbar.md) Visual Studio のメイン ウィンドウの一連のツール バーにツール バーを追加する方法について説明します。

- [ツール ウィンドウにツール バーを追加する](../extensibility/adding-a-toolbar-to-a-tool-window.md) ツール ウィンドウにツール バーを追加する方法について説明します。

- [ソリューション エクスプローラーのツール バーにコマンドを追加する](../extensibility/adding-a-command-to-the-solution-explorer-toolbar.md) ソリューション エクスプローラーのツール バーにコマンドを追加する方法について説明します。

- [ツール バーにメニュー コントローラーを追加する](../extensibility/adding-a-menu-controller-to-a-toolbar.md) ツール バーにメニュー コントローラーを追加する方法について説明します。

- [メニュー項目を動的に追加する](../extensibility/dynamically-adding-menu-items.md) さまざまなコンテキストでメニュー項目を動的に追加する方法について説明します。
