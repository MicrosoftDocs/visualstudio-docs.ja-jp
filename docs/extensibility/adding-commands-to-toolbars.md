# ツールバーにコマンドを追加します。
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

ツールバーは、水平または垂直方向のストリップで、コマンドにバインドされているボタンが含まれています。 Visual Studio での多くの異なるウィンドウのツールバーに独自のコマンドを追加することができます。 Visual Studio のメイン ウィンドウ、ソリューション エクスプ ローラーで、および独自のカスタム ツール ウィンドウです。  
  
 ここでは、さまざまな処理のツールバーのコマンドで行えることを示します。
