# <a name="extend-user-settings-and-options"></a>ユーザー設定とオプションを拡張する
Visual Studio で、ユーザー設定を保守するには、主に 2 つの方法があります。 **[ツール]**  >  **[オプション]** ページでは、全般と言語固有の両方のさまざまな種類の設定に対して、ユーザーが独自の値を設定できます。 ユーザー設定により、ユーザーは Visual Studio の全般的な構成プロパティを指定できます。 オプション ページの詳細については、「[オプションとオプション ページ](../extensibility/internals/options-and-options-pages.md)」を参照してください。 ユーザー設定の詳細については、「[ユーザー設定のサポート](../extensibility/internals/support-for-user-settings.md)」を参照してください。

- 「[オプション ページを作成する](../extensibility/creating-an-options-page.md)」オプション ページの作成方法について説明します。

- 「[設定カテゴリの作成](../extensibility/creating-a-settings-category.md)」設定カテゴリを作成する方法について説明します。

- 「[設定ストアの使用](../extensibility/using-the-settings-store.md)」設定ストアの使用方法について説明します。

- 「[設定ストアからサービス情報を取得する](../extensibility/getting-service-information-from-the-settings-store.md)」設定ストアから利用可能なサービスを取得する方法について説明します。

- 「[ユーザー設定ストアへの書き込み](../extensibility/writing-to-the-user-settings-store.md)」ユーザー設定ストアに書き込む方法について説明します。
