# LocationFieldMRUPrefix 要素 (Visual Studio テンプレート)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

**\[新しいプロジェクト\]** ダイアログ ボックスおよび **\[新しい項目の追加\]** ダイアログ ボックスで最近使用した \(MRU: Most Recently Used\) パスを指定します。  
  
## 構文  
  
```  
<LocationFieldMRUPrefix> ... </LocationFieldMRUPrefix>  
```  
  
## 属性および要素  
 以降のセクションでは、属性、子要素、および親要素について説明します。  
  
### 属性  
 なし。  
  
### 子要素  
 なし。  
  
### 親要素  
  
|要素|説明|  
|--------|--------|  
|[TemplateData](../extensibility/templatedata-element-visual-studio-templates.md)|テンプレートをカテゴリに分類し、**\[新しいプロジェクト\]** ダイアログ ボックス、または **\[新しい項目の追加\]** ダイアログ ボックスでどのように表示させるかを定義します。|  
  
## 解説  
 この要素は、[!INCLUDE[vsipprvsip](../extensibility/includes/vsipprvsip_md.md)] を介して生成されるテンプレートでのみ使用できます。
