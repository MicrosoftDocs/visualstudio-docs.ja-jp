# 接続済みサービス SDK
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Visual Studio 接続済みサービス SDK には、サービス プロバイダーのアプリの開発者で、Visual Studio IDE を離れることがなく、プロジェクトに追加できる簡単に Visual Studio 拡張機能を作成することができます。 このセクションには、Microsoft.VisualStudio.ConnectedServices API に関するリファレンス情報が含まれています。
