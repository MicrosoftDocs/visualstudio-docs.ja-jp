# <a name="process-debug-manager"></a>プロセス デバッグ マネージャー
プロセス デバッグ マネージャー (PDM) は、プログラムとプロセスを管理する Visual Studio のコンポーネントであり、セッション デバッグ マネージャーとデバッグ エンジンで使用できるようにします。

 PDM は、デバッグ可能なすべてのプロセスを管理します。 デバッグするには、プログラムを PDM に登録する必要があります。 この登録は、プログラムが起動された時点で、ポートまたはデバッグ エンジンによって行われます。
