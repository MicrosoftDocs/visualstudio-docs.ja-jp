# プロセスのデバッグ マネージャー
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

プロセス デバッグ マネージャーは \(PDM\)デバッグ セッションの経営者やデバッグ エンジンで使用できるようにするプログラムとプロセスを管理する Visual Studio のコンポーネントです。  
  
 PDM はデバッグできるすべてのプロセスを管理します。  デバッグするにはプログラムは PDM に登録する必要があります。  これはプログラムの実行またはデバッグ エンジンによって起動時されます。
