# <a name="process-debug-manager"></a>プロセスのデバッグ マネージャー
プロセスのデバッグ マネージャー (PDM) プログラムを管理している Visual Studio のコンポーネントであるし、マネージャーとデバッグ エンジンをデバッグするプロセス、セッションに使用できるようにします。  
  
 PDM は、デバッグ可能なすべてのプロセスを管理します。 デバッグするには、PDM にプログラムを登録する必要があります。 これは起動時に、プログラムは、ポートまたはデバッグ エンジンのいずれかです。
