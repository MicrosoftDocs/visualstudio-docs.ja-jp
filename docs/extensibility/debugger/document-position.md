# <a name="document-position"></a>ドキュメントの位置
[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] 、デバッグ、**ドキュメント配置**:  
  
-   IDE に認識されているソース ファイル内の位置の抽象化を提供します。 ほとんどの言語で今日では、ドキュメントの位置はようなもののソース ファイル内の位置。  
  
-   デバッグ エンジンにソース ドキュメント内の位置をについて説明します。  
  
-   によって実装される、 [IDebugDocumentPosition2](../../extensibility/debugger/reference/idebugdocumentposition2.md)インターフェイスです。
