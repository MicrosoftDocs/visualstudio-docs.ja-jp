# ドキュメントの位置
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

デバッグ [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] で  **ドキュメントの場所** :  
  
-   IDE で認識されるようにソース ファイルの場所の抽象化を提供します。  今日ほとんどの言語ではドキュメントの場所はソース ファイルの場所と考えることができます。  
  
-   デバッグ エンジンがソース ドキュメントの場所を記述します。  
  
-   [IDebugDocumentPosition2](../../extensibility/debugger/reference/idebugdocumentposition2.md) のインターフェイスが実装されます。
