# <a name="idebugaliasdispose"></a>IDebugAlias::Dispose
この別名を削除の対象としてマークします。

## <a name="syntax"></a>構文

```cpp
HRESULT Dispose();
```

```csharp
int Dispose();
```

## <a name="parameters"></a>パラメーター
 [なし] :

## <a name="return-value"></a>戻り値
 成功した場合は、S_OK を返します。それ以外の場合はエラー コードを返します。

## <a name="remarks"></a>解説
 このメソッドが呼び出されると、その別名は使用できなくなります。
