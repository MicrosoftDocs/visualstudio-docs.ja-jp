# <a name="name_match"></a>NAME_MATCH
名前を一致させる大文字と小文字のオプションを選択します。

## <a name="syntax"></a>構文

```cpp
typedef enum { 
   nmNone            = 0,
   nmCaseSensitive   = 1,
   nmCaseInsensitive = 2
} NAME_MATCH;
```

```csharp
public enum NameMatchOptions { 
   nmNone            = 0,
   nmCaseSensitive   = 1,
   nmCaseInsensitive = 2
}
```

## <a name="fields"></a>フィールド
 `nmNone`\
 オプションは指定されていません。

 `nmCaseSensitive`\
 一致する名前で大文字と小文字が区別されることを示します。

 `nmCaseInsensitive`\
 一致する名前で大文字と小文字が区別されないことを示します。
