# <a name="namematch"></a>NAME_MATCH
一致する名前の大文字のオプションを選択します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
typedef enum {   
   nmNone            = 0,  
   nmCaseSensitive   = 1,  
   nmCaseInsensitive = 2  
} NAME_MATCH;  
```  
  
```csharp  
public enum NameMatchOptions {   
   nmNone            = 0,  
   nmCaseSensitive   = 1,  
   nmCaseInsensitive = 2  
}  
```  
  
## <a name="members"></a>メンバー  
 nmNone  
 オプションは指定されていません。  
  
 nmCaseSensitive  
 照合する名前を大文字にすることを示します。  
  
 nmCaseInsensitive  
 照合する名前が大文字小文字を区別できないことを示します。
