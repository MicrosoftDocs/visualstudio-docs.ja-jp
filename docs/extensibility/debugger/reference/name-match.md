# <a name="name_match"></a>NAME_MATCH
一致する名前の case オプションを選択します。

## <a name="syntax"></a>構文

```cpp
typedef enum { 
   nmNone            = 0,
   nmCaseSensitive   = 1,
   nmCaseInsensitive = 2
} NAME_MATCH;
```

```csharp
public enum NameMatchOptions { 
   nmNone            = 0,
   nmCaseSensitive   = 1,
   nmCaseInsensitive = 2
}
```

## <a name="fields"></a>フィールド
 `nmNone`\
 オプションは指定されていません。

 `nmCaseSensitive`\
 照合する名前が大文字と小文字を区別することを示します。

 `nmCaseInsensitive`\
 照合する名前が大文字と小文字を区別しないことを示します。
