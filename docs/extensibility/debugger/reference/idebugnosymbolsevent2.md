# <a name="idebugnosymbolsevent2"></a>IDebugNoSymbolsEvent2
起動された[!INCLUDE[vsprvs](../../../code-quality/includes/vsprvs_md.md)]実行可能ファイルのシンボルが見つからないことをユーザーに警告するようにデバッガー UI に通知します。

## <a name="syntax"></a>構文

```
IDebugNoSymbolsEvent2 : IUnknown
```

## <a name="notes-for-implementers"></a>実装者向けの注意事項
 デバッグ エンジンによって実装され、デバッガー [!INCLUDE[vsprvs](../../../code-quality/includes/vsprvs_md.md)] UI によって使用されます。

## <a name="requirements"></a>必要条件
 ヘッダー: Msdbg.h
