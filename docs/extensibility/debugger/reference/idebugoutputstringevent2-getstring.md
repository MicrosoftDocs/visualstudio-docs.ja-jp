# <a name="idebugoutputstringevent2getstring"></a>IDebugOutputStringEvent2::GetString
表示可能なメッセージを取得します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT GetString(   
   BSTR* pbstrString  
);  
```  
  
```csharp  
int GetString(   
   out string pbstrString  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `pbstrString`  
 [out]表示可能なメッセージを返します。  
  
## <a name="return-value"></a>戻り値  
 成功した場合を返します`S_OK`、それ以外のエラー コードを返します。
