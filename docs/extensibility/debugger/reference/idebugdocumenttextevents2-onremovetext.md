# IDebugDocumentTextEvents2::onRemoveText
[!INCLUDE[vs2017banner](../../../code-quality/includes/vs2017banner.md)]

テキストが文書から削除されたデバッグのパッケージに通知します。  
  
## 構文  
  
```cpp#  
HRESULT onRemoveText(   
   TEXT_POSITION pos,  
   DWORD         dwNumToRemove  
);  
```  
  
```c#  
int onRemoveText(   
   enum_TEXT_POSITION pos,  
   uint               dwNumToRemove  
);  
```  
  
#### パラメーター  
 `pos`  
 \[入力\] テキストが削除されたかを示す [TEXT\_POSITION](../../../extensibility/debugger/reference/text-position.md) の構造体。  
  
 `dwNumToRemove`  
 \[入力\] 削除されたテキストの文字数を指定します。  
  
## 戻り値  
 正常に終了した場合戻り `S_OK`; それ以外の場合はエラー コード。
