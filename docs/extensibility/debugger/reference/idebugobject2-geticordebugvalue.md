# <a name="idebugobject2geticordebugvalue"></a>IDebugObject2::GetICorDebugValue
このオブジェクトに関連付けられている値を表すマネージド コード オブジェクトを取得します。

## <a name="syntax"></a>構文

```cpp
HRESULT GetICorDebugValue(
   IUnknown** ppUnk
);
```

```csharp
int GetICorDebugValue(
   out object ppUnk
);
```

## <a name="parameters"></a>パラメーター
`ppUnk`\
[出力] このエイリアスを表す `IUnknown` インターフェイス。 このインターフェイスは、`ICorDebugValue` インターフェイスに対してクエリを実行できます。

## <a name="return-value"></a>戻り値
 成功した場合は、S_OK を返します。それ以外の場合はエラー コードを返します。

## <a name="remarks"></a>解説
 `ICorDebugValue` オブジェクトは、値を表す共通言語ランタイム インターフェイスです。
