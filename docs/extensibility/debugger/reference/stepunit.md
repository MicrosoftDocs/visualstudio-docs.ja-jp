# <a name="stepunit"></a>STEPUNIT
ステップ実行するための増加単位を指定します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
enum enum_STEPUNIT {   
   STEP_STATEMENT   = 0,  
   STEP_LINE        = 1,  
   STEP_INSTRUCTION = 2  
};  
typedef DWORD STEPUNIT;  
```  
  
```csharp  
enum enum_STEPUNIT {   
   STEP_STATEMENT   = 0,  
   STEP_LINE        = 1,  
   STEP_INSTRUCTION = 2  
};  
```  
  
## <a name="members"></a>メンバー  
 STEP_STATEMENT  
 ステートメントで手順を実行します。  
  
 STEP_LINE  
 行をステップ実行します。  
  
 STEP_INSTRUCTION  
 命令で手順を実行します。  
  
## <a name="remarks"></a>Remarks  
 引数として渡される、[手順](../../../extensibility/debugger/reference/idebugprocess3-step.md)メソッド。
