# <a name="stepunit"></a>STEPUNIT
ステップ実行のステップ単位を指定します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
enum enum_STEPUNIT {   
   STEP_STATEMENT   = 0,  
   STEP_LINE        = 1,  
   STEP_INSTRUCTION = 2  
};  
typedef DWORD STEPUNIT;  
```  
  
```csharp  
enum enum_STEPUNIT {   
   STEP_STATEMENT   = 0,  
   STEP_LINE        = 1,  
   STEP_INSTRUCTION = 2  
};  
```  
  
## <a name="members"></a>メンバー  
 STEP_STATEMENT  
 ステートメントで手順を説明します。  
  
 STEP_LINE  
 行で手順を説明します。  
  
 STEP_INSTRUCTION  
 命令で手順を説明します。  
  
## <a name="remarks"></a>コメント  
 引数として渡される、[ステップ](../../../extensibility/debugger/reference/idebugprocess3-step.md)メソッドです。
