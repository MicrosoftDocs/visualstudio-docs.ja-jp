# <a name="idebugreference2getsize"></a>IDebugReference2::GetSize
参照の値のバイト単位のサイズを取得します。 将来使用するために予約されています。

## <a name="syntax"></a>構文

```cpp
HRESULT GetSize ( 
   DWORD* pdwSize
);
```

```csharp
int GetSize ( 
   out uint pdwSize
);
```

## <a name="parameters"></a>パラメーター
`pdwSize`\
[out]参照の値のバイト単位のサイズを返します。

## <a name="return-value"></a>戻り値
 常に `E_NOTIMPL` を返します。
