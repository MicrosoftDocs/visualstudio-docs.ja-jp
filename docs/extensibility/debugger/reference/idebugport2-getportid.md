# <a name="idebugport2getportid"></a>IDebugPort2::GetPortId
ポート id を取得します。

## <a name="syntax"></a>構文

```cpp
HRESULT GetPortId( 
   GUID* pguidPort
);
```

```csharp
int GetPortId( 
   out Guid pguidPort
);
```

## <a name="parameters"></a>パラメーター
`pguidPort`\
入出力ポートを識別する GUID を返します。

## <a name="return-value"></a>戻り値
 成功した場合はを返し `S_OK` ます。それ以外の場合はエラーコードを返します。
