# <a name="idebugport2getportid"></a>IDebugPort2::GetPortId
ポート識別子を取得します。

## <a name="syntax"></a>構文

```cpp
HRESULT GetPortId( 
   GUID* pguidPort
);
```

```csharp
int GetPortId( 
   out Guid pguidPort
);
```

## <a name="parameters"></a>パラメーター
`pguidPort`\
[out] ポートを識別する GUID を返します。

## <a name="return-value"></a>戻り値
 正常に終了した場合は、`S_OK` を返します。それ以外の場合は、エラー コードを返します。
