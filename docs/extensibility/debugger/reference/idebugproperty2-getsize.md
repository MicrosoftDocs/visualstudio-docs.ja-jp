# <a name="idebugproperty2getsize"></a>IDebugProperty2::GetSize
プロパティの値のサイズ (バイト単位) を取得します。

## <a name="syntax"></a>構文

```cpp
HRESULT GetSize ( 
   DWORD* pdwSize
);
```

```csharp
int GetSize ( 
   out uint pdwSize
);
```

## <a name="parameters"></a>パラメーター
`pdwSize`\
[出力] プロパティの値のサイズ (バイト単位) で返します。

## <a name="return-value"></a>戻り値
 成功した場合は、`S_OK` を返します。それ以外の場合は、エラー コードを返します。 プロパティのサイズが指定されていない場合は、`S_GETSIZE_NO_SIZE` を返します。
