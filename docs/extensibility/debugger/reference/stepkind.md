# <a name="stepkind"></a>STEPKIND
ステップ実行するためのステップの種類を指定します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
enum enum_STEPKIND {   
   STEP_INTO      = 0,  
   STEP_OVER      = 1,  
   STEP_OUT       = 2,  
   STEP_BACKWARDS = 3  
};  
typedef DWORD STEPKIND;  
```  
  
```csharp  
public enum enum_STEPKIND {   
   STEP_INTO      = 0,  
   STEP_OVER      = 1,  
   STEP_OUT       = 2,  
   STEP_BACKWARDS = 3  
};  
```  
  
## <a name="members"></a>メンバー  
 STEP_INTO  
 関数にステップ インします。  
  
 STEP_OVER  
 関数をステップ実行します。  
  
 STEP_OUT  
 関数からステップを実行します。  
  
 STEP_BACKWARDS  
 関数には旧バージョンと手順です。  
  
## <a name="remarks"></a>Remarks  
 引数として渡される、[手順](../../../extensibility/debugger/reference/idebugprocess3-step.md)メソッド。
