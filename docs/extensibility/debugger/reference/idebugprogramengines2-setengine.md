# <a name="idebugprogramengines2setengine"></a>IDebugProgramEngines2::SetEngine
プログラムまたはプログラムノードに、このプログラムのデバッグに使用するデバッグ エンジン (DE) を指定します。

## <a name="syntax"></a>構文

```cpp
HRESULT SetEngine( 
   REFGUID guidEngine
);
```

```csharp
int SetEngine( 
   ref Guid guidEngine
);
```

## <a name="parameters"></a>パラメーター
`guidEngine`\
[in]DE の GUID。

## <a name="return-value"></a>戻り値
 成功した場合は`S_OK`、 を返します。それ以外の場合は、エラー コードを返します。
