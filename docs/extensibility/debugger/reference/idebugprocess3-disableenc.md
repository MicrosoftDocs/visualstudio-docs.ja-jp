# <a name="idebugprocess3disableenc"></a>IDebugProcess3::DisableENC

## <a name="syntax"></a>構文  
  
```cpp  
HRESULT DisableENC(  
   EncUnavailableReason reason  
);  
```  
  
```csharp  
   EncUnavailableReason reason

```  
  
#### <a name="parameters"></a>パラメーター  
 `reason`
