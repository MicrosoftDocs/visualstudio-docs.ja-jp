# <a name="reference-visual-studio-debugging-apis"></a>参照 (Visual Studio のデバッグ API)

Visual Studio Debugging SDK リファレンスには、デバッガーとそのコンポーネントのランタイム アーキテクチャと機能の基本モデルの概要が説明されています。 このドキュメントでは、インターフェイスを介して定義する動作とは別に、説明するコンポーネントとインターフェイスに関する前提を可能な限り抑えています。

リファレンス セクションの目標は、可能な限り多様な言語、ランタイム、およびオペレーティング システムのセットに対するサポートを提供することです。
