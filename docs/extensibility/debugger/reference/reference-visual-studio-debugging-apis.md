# <a name="reference-visual-studio-debugging-apis"></a>参照 (Visual Studio のデバッグ API)

「Visual Studio デバッグ SDK リファレンス」では、デバッガーとそのコンポーネントの実行時のアーキテクチャと機能の基本的なモデルを概説します。 このドキュメントでは、インターフェイスによって定義される動作とは別に、記述されているコンポーネントとインターフェイスに関して、できるだけ少ない仮定を行います。

リファレンスセクションの目的は、さまざまな言語、実行時間、およびオペレーティングシステムを可能な限り多様にサポートすることです。
