# <a name="ienumdebugthreads2reset"></a>IEnumDebugThreads2::Reset
最初の要素を列挙型をリセットします。  
  
## <a name="syntax"></a>構文  
  
```cpp#  
HRESULT Reset(  
   void  
);  
```  
  
```c#  
int Reset();  
```  
  
## <a name="return-value"></a>戻り値  
 成功した場合、返す`S_OK`。 そうしないと、エラー コードを返します。  
  
## <a name="remarks"></a>コメント  
 このメソッドが呼び出された後、次の呼び出し、[次](../../../extensibility/debugger/reference/ienumdebugthreads2-next.md)メソッド列挙体の最初の要素を返します。
