# <a name="idebugprocess2terminate"></a>IDebugProcess2::Terminate
プロセスを終了します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT Terminate(   
   void   
);  
```  
  
```csharp  
int Terminate();  
```  
  
## <a name="return-value"></a>戻り値  
 成功した場合を返します`S_OK`、それ以外のエラー コードを返します。  
  
## <a name="remarks"></a>コメント  
 プロセスが終了すると、プロセス内のすべてのプログラムは終了です。[なし] は、任意コードの実行を許可します。
