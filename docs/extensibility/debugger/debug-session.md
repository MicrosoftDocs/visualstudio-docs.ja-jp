# デバッグ セッション
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

デバッガーのアーキテクチャという点では **セッション** :  
  
-   IDE の一つのインスタンスから起動またはアタッチされたすべてのプロセスの設定は関連するまたは無関係です。  
  
-   デバッグ セッションのマネージャー \(SDM\) によって実装されます。  詳細については、「[セッションのデバッグ マネージャー](../../extensibility/debugger/session-debug-manager.md)」を参照してください。
