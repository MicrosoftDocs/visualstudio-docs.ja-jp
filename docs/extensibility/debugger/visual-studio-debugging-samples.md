# Visual Studio のデバッグのサンプル
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

2 番目の例では [!INCLUDE[vsipsdk](../../extensibility/includes/vsipsdk_md.md)] を使用して Visual Studio デバッガーを拡張する方法を示します。  
  
-   デバッガーの式エバリュエーターの例ではマネージ言語で [!INCLUDE[vsipsdk](../../extensibility/includes/vsipsdk_md.md)] に含まれます。  [!INCLUDE[vsipsdk](../../extensibility/includes/vsipsdk_md.md)] のペインで\[ENT0ENT\] タブをクリックし言語の一覧に **\*\*\* C\# \*\*\*** \[入力\] ENT1ENT をクリックするか\[ENT4ENT\] をクリックします。  
  
-   Visual Studio のデバッグ エンジンの例ではネイティブ言語についてはMSDN コード ギャラリーで使用できます。  詳細についてはMicrosoft Web サイトのページを参照 : [Visual Studio Debug Engine Sample](http://go.microsoft.com/fwlink/?LinkId=150236)。
