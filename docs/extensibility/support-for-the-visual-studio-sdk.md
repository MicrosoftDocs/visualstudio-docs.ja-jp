# <a name="support-for-the-visual-studio-sdk"></a>Visual Studio SDK のサポート
Visual Studio 統合開発環境 (IDE) の拡張方法について質問があるお客様には、いくつかのサポート オプションを使用できます。

## <a name="free-support"></a>無料サポート

### <a name="microsoft-newsgroups"></a>マイクロソフト ニュースグループ
 Visual Studio 開発チームのメンバーや、Visual Studio を拡張する他の開発者は、多くの場合、microsoft.public.vstudio.拡張性ニュースグループに参加します。 このニュースグループは管理ニュースグループです。 MSDN ユニバーサル サブスクライバーの場合は、マイクロソフトが応答できるように、スパムのないエイリアスを登録していることを確認してください。

### <a name="msdn-forums"></a>MSDN フォーラム
 [MSDN フォーラムは、他の](https://social.msdn.microsoft.com/Forums/en-US/home)ユーザーや専門家とチャットする別の方法を提供します。 フォーラムのグラフィカル ユーザー インターフェイス (UI) は、ニュースグループよりも使いやすくするように設計されています。 [Visual Studio 機能拡張フォーラム](/azure/devops/integrate/index?view=azure-devops&viewFallbackFrom=vsts)は、機能拡張に関する質問に対する回答を検索する場合に適しています。 他の人がすでに同じ質問をしている可能性があるため、質問する前に必ず検索してください。

### <a name="visual-studio-gallery"></a>Visual Studio ギャラリー
 [Visual Studio マーケットプレース](https://marketplace.visualstudio.com/)は、拡張機能とサンプル コードの場所です。 Visual Studio の生産性とユーザー エクスペリエンスを向上させるツール、VSPackage の使用方法とマネージ機能拡張フレームワーク (MEF) を示すサンプル アプリケーション、およびその他の種類のサンプルを見つけることができます。

### <a name="visual-studio-sdk-samples"></a>ビジュアル スタジオ SDK のサンプル

VS SDK 拡張機能のオープン ソースの例については、GitHub[で参照してください](https://github.com/Microsoft/VSSDK-Extensibility-Samples)。 この GitHub リポジトリには、Visual Studio のさまざまな拡張可能な機能を示すサンプルが含まれています。

## <a name="included-support"></a>含まれるサポート

### <a name="visual-studio-product-support"></a>ビジュアル スタジオ製品サポート
 製品の購入に関してサポートが[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)]一部含まれています。 詳細については、「 [Visual Studio のサポート](https://msdn.microsoft.com/vstudio/cc136615.aspx)」を参照してください。 Microsoft は、拡張性に関[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)]する質問に回答するための専用サポート チームを維持しています。

### <a name="msdn-subscription-support"></a>MSDN サブスクリプションサポート
 MSDN サブスクリプション パッケージには、サポートが含まれている場合もあります。 詳細については、 [MSDN サブスクライバー サポート](https://msdn.microsoft.com/subscriptions/aa718661.aspx)を参照してください。

## <a name="paid-support"></a>有料サポート

### <a name="developer-tools-support"></a>開発者ツールのサポート

サポートも購入できます。 Microsoft[ヘルプとサポート センター](https://support.microsoft.com/supportforbusiness/productselection?fltadd=sps-business-1&sapId=4fd4947b-15ea-ce01-080f-97f2ca3c76e8)の開発者ツール カテゴリには、さまざまなサポート サービスがあります。

を使用して複雑なパッケージ[!INCLUDE[vsipsdk](../extensibility/includes/vsipsdk_md.md)]を開発したいお客様は、マイクロソフトとのプレミア サポート契約の購入を検討できます。 詳細については、[マイクロソフト プレミア サポート](https://support.microsoft.com/premier)を参照してください。 小規模企業の場合、[パートナー向けのアドバンスト サポート](https://partner.microsoft.com/support/advanced-cloud-support)は、従来のプレミア サポート製品に代わる魅力的な選択肢になる場合があります。

> 相談やアドバイスの取得には、標準サポートを使用できません。
