# <a name="support-for-the-visual-studio-sdk"></a>Visual Studio SDK のサポート
Visual Studio 統合開発環境 (IDE) を拡張する方法に疑問を抱かれているお客様向けに、いくつかのサポート オプションが用意されています。

## <a name="free-support"></a>無料サポート

### <a name="microsoft-newsgroups"></a>Microsoft ニュースグループ
 Visual Studio 開発チームのメンバー、および Visual Studio を拡張するその他の開発者は、microsoft.public.vstudio.extensibility ニュースグループに参加することがよくあります。 このニュースグループは、マネージド ニュースグループです。 MSDN ユニバーサル サブスクライバーの場合は、Microsoft が応答できるように、必ずスパムなしの別名を登録するようにしてください。

### <a name="msdn-forums"></a>MSDN フォーラム
 [MSDN フォーラム](https://social.msdn.microsoft.com/Forums/en-US/home)では、仲間や専門家とチャットする方法も用意されています。 このフォーラムのグラフィカル ユーザー インターフェイス (UI) は、ニュースグループよりも使いやすくなるように設計されています。 [Visual Studio 機能拡張フォーラム](/azure/devops/integrate/index?view=azure-devops&viewFallbackFrom=vsts&preserve-view=true)は、機能拡張に関する質問への回答を検索し始めるのに最適な場所です。 他のユーザーが既に同じ質問をしている可能性があるため、質問をする前に必ず検索してください。

### <a name="visual-studio-gallery"></a>Visual Studio ギャラリー
 [Visual Studio Marketplace](https://marketplace.visualstudio.com/) では、拡張機能とサンプル コードを提供しています。 Visual Studio で生産性とユーザー エクスペリエンスを向上させるツール、VSPackage と Managed Extensibility Framework (MEF) の使用方法を示すサンプル アプリケーション、その他の種類のサンプルなどを検索できます。

### <a name="visual-studio-sdk-samples"></a>Visual Studio SDK のサンプル

GitHub 上の [Visual Studio のサンプル](https://github.com/Microsoft/VSSDK-Extensibility-Samples) ページで、VS SDK 拡張機能のオープン ソースの例を検索できます。 この GitHub リポジトリには、Visual Studio の拡張可能な各種機能を示すサンプルが含まれています。

## <a name="included-support"></a>含まれるサポート

### <a name="visual-studio-product-support"></a>Visual Studio 製品サポート
 [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] 製品の購入には、何らかのサポートが含まれています。 詳細については、「[Visual Studio サポート](https://msdn.microsoft.com/vstudio/cc136615.aspx)」をご覧ください。 Microsoft では、[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] の機能拡張に関する質問に回答する専任のサポート チームを確保しています。

### <a name="msdn-subscription-support"></a>MSDN サブスクリプション サポート
 MSDN サブスクリプション パッケージにも、何らかのサポートが含まれている場合があります。 詳細については、[MSDN サブスクライバー サポート](https://msdn.microsoft.com/subscriptions/aa718661.aspx)に関するページをご覧ください。

## <a name="paid-support"></a>有料サポート

### <a name="developer-tools-support"></a>開発者ツールのサポート

サポートを購入することもできます。 [Microsoft ヘルプとサポート センター](https://support.microsoft.com/supportforbusiness/productselection?fltadd=sps-business-1&sapId=4fd4947b-15ea-ce01-080f-97f2ca3c76e8)の [開発者ツール] カテゴリから、さまざまなサポート オファリングを入手できます。

[!INCLUDE[vsipsdk](../extensibility/includes/vsipsdk_md.md)] を使用して複雑なパッケージを開発するお客様は、Microsoft との Premier サポート契約の購入をご検討いただけます。 詳細については、[Microsoft Premier サポート](https://support.microsoft.com/premier)に関するページをご覧ください。 中小企業のお客様は、従来の Premier サポート オファリングに代わる魅力的な手段として、[Advanced Support for Partners](https://partner.microsoft.com/support/advanced-cloud-support) をお選びいただけます。

> [!NOTE]
> Standard サポートでは、ご相談やアドバイスを受けることはできません。

VSIP プログラムと併用した場合の Premier サポート オファリングに関するご質問は、[vsipsvs@microsoft.com](mailto:vsipsvs@microsoft.com) までお寄せください。 新しい Partner Advantage プログラムに関するご質問は、[advantg@microsoft.com](mailto:advantg@microsoft.com) までお寄せください。

個々のサポート要求のご利用方法については、[個々のサポート要求](https://support.microsoft.com/supportforbusiness/productselection)に関するページをご覧ください。
