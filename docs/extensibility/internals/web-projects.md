# <a name="web-projects"></a>Web プロジェクト
Web プロジェクト システムは、Web プロジェクトを作成するためにプロジェクト システムです。 Web プロジェクトは、Web アプリケーションを作成します。 などの従来のプログラミング言語を使用して Web ページをプログラムする[!INCLUDE[vbprvb](../../code-quality/includes/vbprvb_md.md)]または[!INCLUDE[csprcs](../../data-tools/includes/csprcs_md.md)]を収集、ユーザーから情報を処理し、データベースに格納、および具合です。 関連付けられているコードを含む web ページは、スマート Web ページと呼ばれます。
