# <a name="test-area-6-delete"></a>テスト領域 6: 削除
このソース管理プラグインのテストの領域では、削除の各操作について説明します。  
  
 ソース コントロール内のアクションを削除するが応答する**ソリューション エクスプ ローラー**です。  
  
 削除できるアイテムの一覧を次に示します。  
  
-   ファイル  
  
-   フォルダー  
  
-   プロジェクト  
  
 によっては、プロジェクトの種類にオプションがあります**削除**(ディスク上のファイルのまま)、プロジェクトまたは**削除**プロジェクト (ディスク上のファイルを削除します)。 どちらの操作は、プロジェクトまたはから項目を削除します。**ソリューション エクスプ ローラー**です。  
  
## <a name="expected-behavior"></a>想定される動作  
 Delete テスト区分のテスト_ケースの通常の動作は次のとおりです。  
  
-   削除された項目が内で見えなくなる**ソリューション エクスプ ローラー**です。  
  
-   (おそらくプロンプトです。) が必要に応じて、削除されたプロジェクトまたは項目の親をチェック アウト  
  
-   チェック アウトを削除または項目を追加すると、それがない、**保留中のチェックイン**ウィンドウです。  
  
-   項目がまだソース管理ストア内で、削除した後でも存在し、手動で消去する必要があります。
