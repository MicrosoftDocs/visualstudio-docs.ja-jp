# <a name="outlining-and-hidden-text-in-a-legacy-language-service"></a>従来の言語サービスでのアウトライン表示と非表示のテキスト
アウトライン表示できるようになりますを単一行にテキスト行のシーケンスを折りたたみます。 たとえば、C++ では、すべてのメソッドをメソッド署名のみを示す&1; つの行に折りたたむできます。 非表示のテキストは、表示するか、非表示になっているテキスト行のシーケンスです。
