# <a name="source-control"></a>ソース管理
このセクションでは、ソース管理プラグインまたはソース管理 VSPackage のどちらかを使用して、[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] の統合された機能としてソース管理を実装するためのオプションについて説明します。

## <a name="in-this-section"></a>このセクションの内容
- [Essentials](../../extensibility/internals/source-control-integration-essentials.md)

 ソース管理に関する作業を開始するために必要になる重要な情報を提供します。

- [概要](../../extensibility/internals/source-control-integration-overview.md)

 ソース管理を実装するために使用できる 2 つのオプションの概要について説明します。

- [ソース管理プラグインの作成](../../extensibility/internals/creating-a-source-control-plug-in.md)

 [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] ソース管理ユーザー インターフェイス (UI) を使用して、ソース管理機能を提供するソース管理プラグインを作成する方法について説明します。

- [ソース管理 VSPackage の作成](../../extensibility/internals/creating-a-source-control-vspackage.md)

 ソース管理機能を提供するだけでなく、[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] ソース管理 UI をカスタマイズするためにも使用できるソース管理 VSPackage を作成する方法について説明します。
