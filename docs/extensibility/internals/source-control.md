# <a name="source-control"></a>ソース管理
このセクションでは、ソース管理プラグインまたはソース管理 VSPackage[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]を使用して、ソース管理を統合機能として実装するためのオプションについて説明します。

## <a name="in-this-section"></a>このセクションの内容
- [Essentials](../../extensibility/internals/source-control-integration-essentials.md)

 ソース管理の操作を開始するために必要な重要な情報を示します。

- [概要](../../extensibility/internals/source-control-integration-overview.md)

 ソース管理を実装するための 2 つのオプションの概要。

- [ソース管理プラグインの作成](../../extensibility/internals/creating-a-source-control-plug-in.md)

 ソース管理ユーザー インターフェイス (UI) を通じてソース管理機能を[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]提供するソース管理プラグインを作成する方法について説明します。

- [ソース管理 VSPackage の作成](../../extensibility/internals/creating-a-source-control-vspackage.md)

 ソース管理機能を提供するだけでなく、ソース管理 UI のカスタマイズに使用できるソース管理 VSPackage[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]を作成する方法について説明します。

 ソース管理プラグイン API のリファレンス。
