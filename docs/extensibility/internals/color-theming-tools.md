# <a name="color-theme-tools"></a>配色テーマツール
Visual Studio SDK には、色のテーマを作成して管理するためのツールがいくつか用意されています。

- [VSIX カラーエディター](../../extensibility/internals/vsix-color-editor.md)ツールでは、Visual Studio のカスタム色を作成および編集できます。

- [VSIX カラーコンパイラ](../../extensibility/internals/vsix-color-compiler.md)ツールは、既存の Visual Studio のテーマ *.xml* ファイルを、拡張機能の *pkgdef* ファイルに変換します。
