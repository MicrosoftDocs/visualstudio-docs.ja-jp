# <a name="color-theme-tools"></a>カラーテーマツール
Visual Studio SDK には、カラー テーマの作成と管理に役立ついくつかのツールが用意されています。

- [VSIX カラー エディター](../../extensibility/internals/vsix-color-editor.md)ツールは、Visual Studio のカスタム色を作成および編集できます。

- [VSIX カラー コンパイラ](../../extensibility/internals/vsix-color-compiler.md)ツールは、既存の Visual Studio テーマ *.xml*ファイルを、拡張機能の *.pkgdef*ファイルに変換します。
