# 色のテーマ ツール
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

VS SDK には、作成、および色のテーマを管理するためにいくつかのツールが用意されています。  
  
-   [VSIX の色エディター](../../extensibility/internals/vsix-color-editor.md) ツールは作成して、Visual Studio のカスタム カラーを編集します。  
  
-   [VSIX 色コンパイラ](../../extensibility/internals/vsix-color-compiler.md) ツールは、.pkgdef、拡張機能用に既存の Visual Studio のテーマの .xml ファイルを変換します。
