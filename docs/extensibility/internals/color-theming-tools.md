# <a name="color-theme-tools"></a>配色テーマ ツール
Visual Studio SDK には、配色テーマを作成して管理するためのツールがいくつか用意されています。

- [VSIX カラー エディター](../../extensibility/internals/vsix-color-editor.md) ツールを使用すると、Visual Studio のカスタム色を作成および編集できます。

- [VSIX カラー コンパイラ](../../extensibility/internals/vsix-color-compiler.md) ツールを使用すると、既存の Visual Studio のテーマである *.xml* ファイルを拡張機能用の *.pkgdef* ファイルに変換できます。
