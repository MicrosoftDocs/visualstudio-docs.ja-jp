# <a name="image-service-tools"></a>イメージ サービスのツール
VS SDK には、Visual Studio のイメージ サービスを使用して拡張機能を構築するときに役立つツールがいくつか用意されています。

- [Manifest from Resources](../../extensibility/internals/manifest-from-resources.md) ツールは、イメージ リソース (.png または .xaml ファイル) の一覧を受け取って、イメージ マニフェスト ファイルを生成します。

- [Manifest to Code](../../extensibility/internals/manifest-to-code.md) ツールは、イメージ マニフェスト ファイルを受け取って、C++、C#、VB、または .vsct ファイルで使用するラッパー ファイルを生成します。

- [Image Library Viewer](../../extensibility/internals/image-library-viewer.md) は、イメージ マニフェストの読み込み、編集、検索を行うことができます。
