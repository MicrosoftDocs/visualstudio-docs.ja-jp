# <a name="image-service-tools"></a>イメージ サービス ツール
VS SDK には、Visual Studio イメージのサービスを使用して拡張ビルダーのヘルプをいくつかのツールが含まれています。  
  
-   [リソースからマニフェスト](../../extensibility/internals/manifest-from-resources.md)ツールは、イメージ リソース (.png または .xaml ファイル) の一覧を取得し、イメージ マニフェスト ファイルを生成します。  
  
-   [コードにマニフェスト](../../extensibility/internals/manifest-to-code.md)ツールがイメージ マニフェスト ファイルを取得し、C++、c#、VB、または .vsct ファイルで使用されるラッパー ファイルを生成します。  
  
-   [イメージ ライブラリ ビューアー](../../extensibility/internals/image-library-viewer.md)読み込み、編集、および画像マニフェストを検索できます。
