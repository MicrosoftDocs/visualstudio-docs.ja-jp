# プロジェクトの拡張
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

プロジェクトおよびソリューションは、Visual Studio コードおよびリソース ファイルをコンパイルおよび展開の単位に整理する方法です。 プロジェクトの詳細についてを検索する [Projects \(Visual Studio SDK\)](../extensibility/extending-projects.md)です。  
  
 独自のプロジェクトの種類を作成するには、Visual Studio SDK、および管理されているパッケージ フレームワークとプロジェクトの場合にダウンロードできます [のプロジェクトのパッケージの管理フレームワーク](http://mpfproj12.codeplex.com/)します。 理解する独自のプロジェクトは、実装方法を参照してください。 [新しいプロジェクトの生成: 実際のところ、第 1 部します。](../extensibility/internals/new-project-generation-under-the-hood-part-one.md) と [新しいプロジェクトの生成: 実際のところ、第 2 部します。](../Topic/New%20Project%20Generation:%20Under%20the%20Hood,%20Part%20Two.md)です。  
  
 このセクションのトピックでは、独自のプロジェクトを作成する方法とさまざまな種類の Visual Studio ソリューションを管理する方法について説明します。
