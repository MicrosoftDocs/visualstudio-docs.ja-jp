# <a name="extend-properties-and-the-property-window"></a>プロパティとプロパティ ウィンドウを拡張する
**[プロパティ]** ウィンドウは、選択したオブジェクトのデザイン時のプロパティおよびイベントを表示および編集するために使用します。

 [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] でオブジェクトを選択すると、そのオブジェクトのパブリック プロパティが **[プロパティ]** ウィンドウに表示されます。 詳細については、「[プロパティ ウィンドウの概要](../extensibility/internals/properties-window-overview.md)」を参照してください。

## <a name="in-this-section"></a>このセクションの内容

- 「[プロジェクトのプロパティを取得する](../extensibility/getting-project-properties.md)」プロジェクトのプロパティを取得する方法について説明します。
