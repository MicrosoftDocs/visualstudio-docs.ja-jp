# <a name="objects"></a>オブジェクト
次の参照情報では、コア テキスト エディターのインスタンスを作成するために使用されるオブジェクトについて説明します。

## <a name="in-this-section"></a>このセクションの内容
- [VSCodeWindow オブジェクト](../extensibility/vscodewindow-object.md) 1 つ以上のテキスト ビューを囲むことができる特殊なドキュメント ウィンドウであるコード ウィンドウ オブジェクトについて説明します。

- [VSCodeWindowManager オブジェクト](../extensibility/vscodewindowmanager-object.md) 言語サービスによって実装され、表示要素を管理する役割を果たすコード ウィンドウ マネージャー オブジェクトについて説明します。

- [VSTextBuffer オブジェクト](../extensibility/vstextbuffer-object.md) Unicode テキストのストリーム (ファイルのデータ) を表すテキスト バッファー オブジェクトについて説明します。

- [VSTextView オブジェクト](../extensibility/vstextview-object.md) ユーザーがテキスト バッファーの Unicode テキストを表示したり、編集したりできるウィンドウであるテキスト ビューについて説明します。
