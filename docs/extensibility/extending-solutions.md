# <a name="extend-solutions"></a>ソリューションの拡張
ソリューションとは、アプリケーションを作成するために連携して動作する 1 つ以上のプロジェクトのグループです。

## <a name="in-this-section"></a>このセクションの内容
 次のチュートリアルでは、ソリューションを拡張する方法をいくつか示します。

1. [ソリューション エクスプローラーのフィルターを拡張](../extensibility/extending-the-solution-explorer-filter.md)する ソリューション エクスプローラーに表示される内容をフィルター処理する方法について説明します。

2. [ソリューションでのプロジェクトの読み込みを管理](../extensibility/managing-project-loading-in-a-solution.md)する方法では、ユーザーがソリューションを読み込むときのパフォーマンスを向上させるためにプロジェクトの読み込みを管理する方法について説明します。
