# <a name="extend-solutions"></a>ソリューションを拡張する
ソリューションは、アプリケーションを作成するために連携する 1 つ以上のプロジェクトのグループです。

## <a name="in-this-section"></a>このセクションの内容
 次のチュートリアルでは、ソリューションを拡張できる方法をいくつか紹介しています。

1. 「[ソリューション エクスプローラーのフィルターを拡張する](../extensibility/extending-the-solution-explorer-filter.md)」には、ソリューション エクスプローラーの表示内容をフィルター処理する方法が説明されています。

2. 「[ソリューションでのプロジェクトの読み込みを管理する](../extensibility/managing-project-loading-in-a-solution.md)」には、ユーザーがソリューションを読み込むときのパフォーマンスが向上するようにプロジェクトの読み込みを管理する方法が説明されています。
