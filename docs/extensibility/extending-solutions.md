# ソリューションを拡張します。
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

ソリューションは、連携してアプリケーションを作成する 1 つまたは複数のプロジェクトのグループです。  
  
## このセクションの内容  
 次のチュートリアルでは、ソリューションを拡張するいくつかの方法を説明します。  
  
1.  [ソリューション エクスプ ローラーのフィルターを拡張します。](../extensibility/extending-the-solution-explorer-filter.md) ソリューション エクスプ ローラーに表示される内容をフィルター処理する方法について説明します。  
  
2.  [ソリューションにプロジェクトの読み込みを管理します。](../extensibility/managing-project-loading-in-a-solution.md) ユーザーは、ソリューションを読み込むと、パフォーマンスを改善するプロジェクトの読み込みを管理する方法について説明します。
