# <a name="net-compiler-platform-package-version-reference"></a>.NET コンパイラ プラットフォーム パッケージのバージョン リファレンス

次の表は、Visual Studio のバージョンごとにサポートされている[.NET コンパイラ プラットフォーム (Roslyn) パッケージ](https://www.nuget.org/packages/Microsoft.Net.Compilers/)のバージョンを示しています。

たとえば、カスタム アナライザーが Visual Studio 2017 のすべてのバージョンで動作するようにするには、バージョン 2.0 の Microsoft.Net.Compiler を対象にする必要があります。

| ロスリンパッケージバージョン | サポートされる最小バージョンの Visual Studio |
| - | - |
| 3.x | Visual Studio 2019 |
| 2.10.0 | Visual Studio 2017 バージョン 15.9 |
| 2.9.0 | Visual Studio 2017 バージョン 15.8 |
| 2.8.2 | Visual Studio 2017 バージョン 15.7 |
| 2.7.0 | Visual Studio 2017 バージョン 15.6 |
| 2.6.1 | Visual Studio 2017 バージョン 15.5 |
| 2.4.0 | Visual Studio 2017 バージョン 15.4 |
| 2.3.2 | Visual Studio 2017 バージョン 15.3 |
| 2.2.0 | Visual Studio 2017 バージョン 15.2 |
| 2.1.0 | Visual Studio 2017 バージョン 15.1 |
| 2.0.0 | ビジュアル スタジオ 2017 RTM |
| 1.3.2 | ビジュアル スタジオ 2015 の更新プログラム 3 |
| 1.2.2 | ビジュアル スタジオ 2015 の更新プログラム 2 |
| 1.1.1 | ビジュアル スタジオ 2015 の更新プログラム 1 |
| 1.0.1 | ビジュアル スタジオ 2015 RTM |

> [!TIP]
> サポートされている最小バージョンの Visual Studio 2017 バージョンの Roslyn パッケージでは、Visual Studio 2019 のすべてのバージョンもサポートされています。
