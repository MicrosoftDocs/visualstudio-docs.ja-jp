# <a name="net-compiler-platform-package-version-reference"></a>.NET コンパイラ プラットフォーム パッケージのバージョンのリファレンス

次の表では[.NET コンパイラ プラットフォーム (Roslyn) パッケージ](https://www.nuget.org/packages/Microsoft.Net.Compilers/)バージョンが異なるバージョンの Visual Studio 2015 と Visual Studio 2017 でサポートされています。 たとえば、カスタム アナライザーが Visual Studio 2017 のすべてのバージョンで機能するためには、Microsoft.Net.Compilers のバージョン 2.0 を対象とする必要があります。

| Roslyn パッケージ バージョン | サポートされている Visual Studio のバージョン |
| - | - |
| 1.0.1 | Visual Studio 2015 RTM 以降 |
| 1.1.1 | Visual Studio 2015 update 1 以降 |
| 1.2.2 | Visual Studio 2015 update 2 以降 |
| 1.3.2 | Visual Studio 2015 update 3 以降 |
| 2.0.0 | Visual Studio 2017 RTM 以降 |
| 2.1.0 | Visual Studio 2017 バージョン 15.1 以降 |
| 2.2.0 | Visual Studio 2017 バージョン 15.2 以降 |
| 2.3.2 | Visual Studio 2017 バージョン 15.3 以降 |
| 2.4.0 | Visual Studio 2017 バージョン 15.4 以降 |
| 2.6.1 | Visual Studio 2017 バージョン 15.5 以降 |
| 2.7.0 | Visual Studio 2017 バージョン 15.6 以降 |
| 2.8.2 | Visual Studio 2017 バージョン 15.7 以降 |
| 2.9.0 | Visual Studio 2017 バージョン 15.8 以降 |
