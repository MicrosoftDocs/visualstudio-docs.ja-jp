# <a name="monitoring-gpu-utilization"></a>GPU 使用率を監視する

Linux リモート マシンの GPU 使用率を監視するには:

1. **サーバー エクスプローラー** で、 **[リモート マシン]** を展開します。
2. 監視するリモート マシンを **右クリック** します。

    ![サーバー エクスプローラーのスクリーンショット。AI Tools のフォルダー ツリーが表示されています。 Azure Demo が選択されています。Show Heat Map コマンドのコンテキスト メニューが開いています。](media/monitor-gpu/gpu-heatmap-0.png)

3. **[Show Heat Map]\(ヒート マップの表示\)** をクリックします。

    ![サーバー エクスプローラーの AI Tools Heat Map ウィンドウのスクリーンショット。概要、詳細、Azure DLVM 仮想マシンの GPU の Heat Map データが表示されています。](media/monitor-gpu/heatmap.png)
