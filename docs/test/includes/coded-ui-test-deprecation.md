> [!NOTE]
> UI 駆動型機能テストの自動化のためにコード化された UI テストは非推奨になりました。 Visual Studio 2019 は、コード化された UI テストが使用できる最後のバージョンです。 Web アプリのテストには [Selenium](https://docs.seleniumhq.org/) を使用し、デスクトップと UWP アプリのテストには [Appium と WinAppDriver を一緒に](https://github.com/Microsoft/WinAppDriver)使用することをお勧めします。 NUnit テスト フレームワークで iOS アプリや Android アプリをテストする場合は、[Xamarin.UITest](/appcenter/test-cloud/uitest/) を検討してください。
