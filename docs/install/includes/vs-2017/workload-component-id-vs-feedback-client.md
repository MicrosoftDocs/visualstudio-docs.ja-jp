## <a name="feedback-client"></a>Feedback Client

**ID:** Microsoft.VisualStudio.Workload.FeedbackClient

**説明:** Feedback Client では、関係者は Visual Studio Team Services または Team Foundation Server に関する豊富なフィードバックを送ることができます。

Microsoft.VisualStudio.Component.TestTools.FeedbackClient | Microsoft Feedback Client | 15.6.27406.0 | 必須
