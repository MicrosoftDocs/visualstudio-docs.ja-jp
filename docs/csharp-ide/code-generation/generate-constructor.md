# <a name="generate-a-constructor-in-c"></a>C# の場合、コンス トラクターを生成します。 #
**新機能:**すぐに、クラスの新しいコンス トラクターのコードを生成することができます。 

**正しく、**自動的に宣言する、新しいコンス トラクターを導入するか、既存のコンス トラクターを変更します。 

**理由:**が、この機能は、適切なパラメーターを自動的に生成を使用する前に、コンス トラクターを宣言することです。 さらに、既存のコンス トラクターを変更するには、この機能を使用して自動的に更新しない限り、callsite をすべての更新が必要です。

**方法:**はコンス トラクターを生成するいくつかの方法があります。
- [コンス トラクターを生成し、メンバーの選択](#pick)
- [選択したフィールドからのコンス トラクターを生成します。](#selection)
- [新しい使用法からコンス トラクターを生成します。](#usage)
- [既存のコンス トラクターにパラメーターを追加します。](#addparameter)
- [作成し、コンス トラクターのパラメーターからのフィールド/プロパティの初期化](#create)

## <a id = "pick"></a>コンス トラクターを生成し、メンバーの選択
1. クラス内の任意の空白行にカーソルを配置します。

   ![空の行にカーソル](media/constructor1_highlight.png)

1. 次に、次のいずれかの操作を行います。
   * **キーボード**
     * キーを押して**Ctrl + です。** トリガーを**クイック アクションとリファクタリング**メニュー**コンス トラクターの生成しています.**プレビュー ウィンドウのポップアップからです。
   * **マウス**
     * 右クリックし、選択、**クイック アクションとリファクタリング**メニュー**コンス トラクターの生成しています.**プレビュー ウィンドウのポップアップからです。
     * をクリックします ![電球](media/bulb.png) クラスの空の行にテキスト カーソルである場合、左余白に表示されるアイコン。

   ![コンス トラクターのプレビューを生成します。](media/constructor1_preview.png)

1. コンス トラクターのパラメーターとして使用するだけでなく (上矢印と下矢印) は、それらを注文するメンバーを選択できるダイアログが表示されます。 キーを押して**Ok**変更を確定します。
  
   ![選択メンバー ダイアログ](media/constructor1_dialog.png)

   >[!TIP] 
   >チェックすることができます、 **null チェックを追加**コンス トラクターのパラメーターを null チェックを自動的に生成する チェック ボックスです。

1. コンス トラクターは、指定した順序で選択したパラメーターを使用して作成されます。
   ![コンス トラクターの結果を生成します。](media/constructor1_result.png)
