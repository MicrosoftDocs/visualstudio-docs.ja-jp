# Value (XElement Dynamic Property)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

要素のコンテンツを取得または設定します。  
  
## 構文  
  
```  
elem.Value  
```  
  
## プロパティ値\/戻り値  
 要素の連結されたコンテンツを表す <xref:System.String>。  
  
## 解説  
 このプロパティは、<xref:System.Xml.Linq.XElement?displayProperty=fullName> クラスの <xref:System.Xml.Linq.XElement.Value%2A> プロパティに相当します。ただし、この動的プロパティは変更通知もサポートします。
