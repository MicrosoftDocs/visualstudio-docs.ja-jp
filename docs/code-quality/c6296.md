# <a name="c6296"></a>C6296
警告 C6296: for-loop の定義が無効です。ループ ボディは 1 度だけ実行されます

 この警告は、for ループが意図したとおりに機能しない可能性があることを示します。 インデックスが符号なしであり、ループがゼロからカウントされる場合、その本体は1回だけ実行されます。

## <a name="example"></a>例
 この警告が発生するコード例を次に示します。

```
void f( )
{
   unsigned int i;

   for (i = 0; i < 100; i--)
   {
      // code ...
   }
}
```

 この警告を解決するには、次のコードを使用します。

```
void f( )
{
   unsigned int i;

   for (i = 0; i < 100; i++)
   {
      // code ...
   }
}
```
