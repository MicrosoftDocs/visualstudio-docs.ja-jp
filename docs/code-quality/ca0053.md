# <a name="ca0053"></a>CA0053
規則のアセンブリを読み込めませんでした。

これは、規則のアセンブリをリフレクションで読み込めなかったことを示します。 ユーザーのカスタム規則が正しくビルドされなかったか、ファイル システム アクセスを拒否されたなど、他の理由によって分析ツールでアセンブリを読み込めなかった場合、これが発生することがあります。
