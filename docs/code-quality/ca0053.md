# <a name="ca0053"></a>CA0053
規則アセンブリをロードできませんでした。  
  
 これは、リフレクションによって、規則アセンブリを読み込めなかったことを示します。 原因としては、ユーザーがある正しく構築されていないカスタムのルール、または分析ツールは、ファイル システム アクセスが拒否されたなどに、他の理由のアセンブリを読み込むことができません。
