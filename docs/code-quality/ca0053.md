# <a name="ca0053"></a>CA0053
規則アセンブリを読み込むことができませんでした。

これは、リフレクションで規則アセンブリを読み込めなかったことを示します。 これは、ユーザーが正しくビルドされなかったカスタムルールを持っている場合、またはファイルシステムアクセスが拒否されたなどの理由で分析ツールがアセンブリを読み込むことができない場合に発生する可能性があります。
