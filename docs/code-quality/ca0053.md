# <a name="ca0053"></a>CA0053
規則アセンブリを読み込むことができませんでした。

 これは、規則アセンブリがリフレクションによって読み込まれていないことを示します。 これは、ユーザーがある正常にビルドされていないカスタム ルール場合に発生する可能性があります。 または分析ツールは、ファイル システムのアクセスが拒否されたなどに、上の理由から他のアセンブリを読み込むことができません。
