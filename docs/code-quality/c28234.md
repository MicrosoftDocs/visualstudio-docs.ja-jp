# C28234
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28234: \_At\_ 式は現在の関数に適用されません  
  
 この警告は `_At_` の式の値がユーザー補助オブジェクトを識別することを示します。
