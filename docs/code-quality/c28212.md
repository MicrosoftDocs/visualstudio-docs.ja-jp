# C28212
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28212: 注釈にはポインター式が必要です  
  
 この警告は、注釈 \(指定されていない関数\) への番号が付けられたパラメーターがポインター型または配列型であると想定されるが他の型が間違っています。  注釈は、修正する必要があります。
