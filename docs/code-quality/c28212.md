# <a name="c28212"></a>C28212
警告 C28212: 注釈にはポインター式が必要です

 この警告は、注釈の番号付きパラメーター (注釈が付けられている関数ではない) がポインターまたは配列型であると想定されていますが、他の型が検出されたことを示します。 注釈を修正する必要があります。
