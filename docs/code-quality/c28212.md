# <a name="c28212"></a>C28212
警告 C28212: 注釈にはポインター式が必要です  
  
 この警告は、注釈 (not 関数の注釈が付けられた) 番号付きのパラメーターがポインターまたは配列型になりますが必要ですが、その他のいくつかの種類が見つかりましたことを示します。 注釈は、修正する必要があります。
