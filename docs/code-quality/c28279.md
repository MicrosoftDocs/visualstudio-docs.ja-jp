# C28279
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28279: シンボルについて、'begin' が見つかりましたが対応する 'end' がありません  
  
 注釈の言語サポート開始および終了時 \(C\) 構造体の`{` と `}`、組み合わせが左右に取得。  これは、マクロが使用されている場合は回避できます。
