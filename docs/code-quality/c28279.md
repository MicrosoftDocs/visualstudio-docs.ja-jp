# <a name="c28279"></a>C28279
警告 C28279:シンボルについて、'begin' はありましたが、対応する 'end' がありません

 注釈言語では、begin と end (`{` および @no__t C) コンストラクトがサポートされていますが、ペアリングが不均衡になっています。 マクロが使用されている場合は、この状況を回避できます。
