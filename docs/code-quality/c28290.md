# C28290
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28290: 関数に対する注釈には、実際のパラメーター数よりも多くの外部参照が含まれます  
  
 コード分析ツールは、関数の注釈がパラメーターの実際の数よりも多くの外部エラーが含まれている場合は、この警告を報告します。
