# C28232
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28232: \_Pre、\_Post、または \_Deref は注釈に適用されません  
  
 この警告は、`_Pre_``_Post_`、または `_Deref_` の演算子とそれに続く注釈機能なしで注釈の式であることを示す; 修飾子は無視されましたが、これは正しくコード化されたコメントを示します。
