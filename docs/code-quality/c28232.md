# <a name="c28232"></a>C28232
警告 C28232: _Pre\_、_Post\_、または _Deref\_注釈には適用されません

 この警告では、ことを示します、 `_Pre_`、 `_Post_`、または`_Deref_`演算子が後続の関数型の注釈なし注釈式以外の場合は、修飾子は無視されましたが不適切に記述された注釈を示します。
