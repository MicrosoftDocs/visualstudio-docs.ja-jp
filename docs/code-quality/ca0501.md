# <a name="ca0501"></a>CA0501
レポート出力を読み取ることができません。

 これには、次の理由のいずれかのレポート出力を書き込むことができないときに発生します。

- ディスクがいっぱいです。

- ディレクトリに書き込むアクセス許可がありません。

- ディレクトリが存在しません
