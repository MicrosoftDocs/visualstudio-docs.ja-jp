# <a name="ca0501"></a>CA0501
出力レポートを読み取れません。  
  
 これは、エラーは、次の理由により、レポート出力を書き込むことはできないときに発生します。  
  
-   ディスクがいっぱいです。  
  
-   ディレクトリに書き込むアクセス許可なし  
  
-   ディレクトリは存在しません
