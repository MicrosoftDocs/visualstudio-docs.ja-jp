# <a name="ca0502"></a>CA0502
サポートされていない言語

ユーザーが、1 つ以上のページまたは C# または Visual Basic 以外の言語で記述されたクラスを含む ASP.NET プロジェクトを分析しようとしています。 このエラーが発生します。 このエラーを解決するには、プロジェクトからファイルを削除するか、コード分析を無効にします。
