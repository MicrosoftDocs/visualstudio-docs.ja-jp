# <a name="ca0502"></a>CA0502
サポートされていない言語

このエラーは、または Visual Basic 以外C#の言語で記述された1つ以上のページまたはクラスを含む ASP.NET プロジェクトを、ユーザーが分析しようとしたときに発生します。 このエラーを修正するには、プロジェクトからファイルを削除するか、コード分析を無効にします。
