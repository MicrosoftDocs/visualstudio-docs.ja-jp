# <a name="ca0502"></a>CA0502
サポートされていない言語  
  
 このエラーは、ユーザーが 1 つ以上のページまたは c# または Visual Basic 以外の言語で記述されたクラスを含む ASP.NET プロジェクトを分析しようとするときに発生します。 このエラーを解決するには、プロジェクトからファイルを削除またはコード分析を無効にします。
