# C28301
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28301: \<関数\> の最初の宣言に対して注釈がありません。\<note1\>\<ファイル名\>\(\<行\>\) を参照してください。\<note2\>  
  
 この警告は、注釈は、関数の最初の宣言で見つからなかった場合に報告されます。
