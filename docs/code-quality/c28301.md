# <a name="c28301"></a>C28301
warning C28301: \<function > の最初の宣言に注釈がありません。\<note1 > 「\<filename > (\<line >)」を参照してください。 \<note2 >

 この警告は、指定された関数の最初の宣言で注釈が見つからなかった場合に報告されます。
