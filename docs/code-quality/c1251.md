# C1251
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C1251: モデルを読み込めません。  
  
 コード分析ツールは、モデル ファイルの内部エラーが発生した場合、分析するコードでは、この警告を、報告します。
