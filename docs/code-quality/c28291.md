# <a name="c28291"></a>C28291
C28291 の警告: Post で null/notnull deref レベル 0 は関数に対して意味のない\<x > param で\<番号 >  
  
 コード分析ツールは、null と null 以外のプロパティを持つ逆参照レベル 0 のパラメーターの事後条件が指定されている場合、この警告を報告します。 このエラーは、位置の値を逆参照レベル 0 は変更できません。
