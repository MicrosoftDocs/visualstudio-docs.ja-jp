# <a name="c6303"></a>C6303
警告 C6303: 書式文字列の不一致: パラメーターとして渡されたワイド文字の文字列が \< 関数の呼び出しに文字列が必要な場合は、\<number > >

 この警告は、書式文字列によって文字列が必要であることが指定されていることを示します。 ただし、ワイド文字列が渡されています。 この不具合は、何らかの形式のクラッシュまたは破損の原因となる可能性があります。

## <a name="example"></a>例
 この警告が発生するコード例を次に示します。

```
#include <stdio.h>

void f()
{
  wchar_t buff[5] = L"hi";

  printf("%s", buff);
}
```

 この警告を修正するには、次のサンプルコードに示すように `%ls` を使用します。

```
#include <stdio.h>

void f()
{
  wchar_t buff[5] = L"hi";

  printf("%ls", buff);
}
```

 次のサンプルコードでは、安全な文字列操作関数 `printf_s` を使用して、この警告を修正します。

```
#include <stdio.h>

void f()
{
  wchar_t buff[5] = L"hi";

  printf_s("%ls",buff);
}
```
