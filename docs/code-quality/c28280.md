# <a name="c28280"></a>C28280
警告 C28280: シンボルについて、'end' が見つかりましたが対応する 'begin' がありません

 Begin と end 注釈言語をサポートしています (`{`と`}`C で)、構造とペアリングされている不均衡です。 マクロを使用する場合、このような状況を回避できます。
