# <a name="c28280"></a>C28280
警告 C28280: シンボルについて、'end' が見つかりましたが対応する 'begin' がありません

 注釈言語は begin と end (`{` と `}` C) コンストラクトをサポートしていますが、ペアリングが不均衡になっています。 マクロが使用されている場合は、この状況を回避できます。
