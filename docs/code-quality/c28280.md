# <a name="c28280"></a>C28280
警告 C28280:シンボルについて、'end' はありましたが、対応する 'begin' がありません

 注釈言語では、begin と end (`{` および @no__t C) コンストラクトがサポートされており、ペアリングの不均衡が発生しています。 マクロが使用されている場合は、この状況を回避できます。
