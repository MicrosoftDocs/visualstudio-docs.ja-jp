# <a name="c6525"></a>C6525
警告 C6525: 無効なサイズ指定です: プロパティの値が有効でない可能性があります

> [!NOTE]
>  この警告は、ソース コード注釈言語 (SAL) の非推奨のバージョンを使用しているコードでのみ発生します。 SAL の最新バージョンを使用するようコードを移植することをお勧めします。 詳細については、次を参照してください。 [c/c++ コードの不具合の削減に SAL 注釈を使用して](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md)です。

 この警告は、サイズが正しくないを指定するプロパティの値が使用されることを示します。 これは、場合は、サイズ パラメーターは、有効期間を使用して注釈が = なし。

## <a name="example"></a>例
 次のコードでは、ValidElements プロパティとしてマークされた有効なサイズのパラメーターを使用しているために、この警告が生成されます。

```
// C
#include <CodeAnalysis\SourceAnnotations.h>
void f([SA_Pre(ValidElements="*count")] char * px, [SA_Pre(Valid=SA_No)]size_t *count);

// C++
#include <CodeAnalysis\SourceAnnotations.h>
using namespace vc_attributes;
void f([Pre(ValidElements="*count")] char * px, [Pre(Valid=No)]size_t *count);

```

 この警告を解決するには、次のコードに示すように、有効なサイズ パラメーターを指定します。

```
// C
#include <CodeAnalysis\SourceAnnotations.h>
void f([SA_Pre(ValidElements="*count")] char * px, [SA_Pre(Valid=SA_Yes)]size_t *count);

// C++
#include <CodeAnalysis\SourceAnnotations.h>
using namespace vc_attributes;
void f([Pre(ValidElements="*count")] char * px, [Pre(Valid=Yes)]size_t *count);
```
