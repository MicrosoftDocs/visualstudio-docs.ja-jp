# C28286
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28286: 関数について、末尾付近で構文エラーが発生しました  
  
 コード分析ツールは、公算誤差がモデル ファイルに含まれている場合は、この警告を報告します。  一部のソース ファイルのエラーによって、このようなエラーが発生する場合があります。
