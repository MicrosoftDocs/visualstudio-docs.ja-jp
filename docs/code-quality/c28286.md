# <a name="c28286"></a>C28286
警告 C28286:関数について、構文エラーが最後の近くにあります

 コード分析ツールは、モデルファイルで発生する可能性のあるエラーが発生したときにこの警告を報告します。 いくつかのソースファイルエラーによって、このようなエラーが発生することもあります。
