# <a name="c6334"></a>C6334
警告 C6334: 演算子を伴う式に適用される sizeof 演算子は、予期しない結果を生成する可能性があります

 この警告は、`sizeof` 演算子が誤用されたことを示します。 式に適用した場合の @no__t 0 演算子は、結果として得られる式の型のサイズを生成します。

 たとえば、次のコードを使用します。

```
char     a[10];
size_t   x;

x = sizeof (a - 1);
```

 `x` には、9ではなく値4が割り当てられます。これは、結果として得られる式が配列 `a` のポインターではなく、単にポインターであるためです。

## <a name="example"></a>例
 この警告が発生するコード例を次に示します。

```
void f( )
{
  size_t x;
  char a[10];

  x= sizeof (a - 4);
  // code...
}
```

 この警告を解決するには、次のコードを使用します。

```
void f( )
{
  size_t x;
  char a[10];

  x= sizeof (a) - 4;
  // code...
}
```
