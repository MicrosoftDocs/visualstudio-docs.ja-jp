# <a name="c28283"></a>C28283
警告 C28283: シンボルについて、指定したサイズはまだサポートされていません  
  
 警告は、サイズの仕様"sentinel"注釈がゼロ以外の値を受信したことを示します。 基本的には、呼び出し元は、文字列がバイナリのゼロ以外の文字で終了したことを言いますしようとしました。
