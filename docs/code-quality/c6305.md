# <a name="c6305"></a>C6305
警告 C6305: sizeof 数と countof 数の間で不一致が発生した可能性があります

 この警告は、@no__t 0 の結果を保持している変数が、ポインターまたは `countof` 式から加算または減算されていることを示します。 これにより、ポインター演算で予期しないスケーリングが発生します。

## <a name="example"></a>例
 この警告が発生するコード例を次に示します。

```
void f(int *p)
{
  int cb=sizeof(int);
  //code...
  p +=cb; // warning 6305
}
```

 この警告を解決するには、次のコードを使用します。

```
void f(int *p)
{
  // code...
  p += 1;
}
```
