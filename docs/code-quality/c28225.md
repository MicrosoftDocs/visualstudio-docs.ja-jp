# <a name="c28225"></a>C28225
C28225 を警告します。注釈に正しい数の必須パラメーターがありません

 この警告は、名前付きの注釈が誤った数のパラメーターと共に使用されることを示します。 注釈は標準のマクロが使用されている場合があります。この警告は、一般的な使用で目にする必要はありません。
