# C28225
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28225: 注釈に見つかった必須パラメーターの数が正しくありません  
  
 この警告は名前付きな注釈は、パラメーターの数が使用されていることを示します。  これは、標準マクロが注釈に使用される可能ではありません; この警告は、一般的な使用方法で参照されると想定されていません。
