# <a name="c28225"></a>C28225
警告 C28225: 注釈に見つかった必須パラメーターの数が正しくありません

 この警告は、名前付きの注釈が正しくない数のパラメーターで使用されていることを示します。 これはできません。 注釈の標準マクロを使用している場合この警告は、一般的な使用で目にする必要はありません。
