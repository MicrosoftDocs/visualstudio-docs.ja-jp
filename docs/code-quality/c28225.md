# <a name="c28225"></a>C28225
警告 C28225:注釈に正しい数の必須パラメーターがありません

 この警告は、名前付き注釈がパラメーターの数が正しくないことを示しています。 注釈に標準のマクロが使用されている場合は、これを行うことはできません。この警告は、通常の使用では表示されません。
