# <a name="c28351"></a>C28351
警告 C28351: 注釈\<注釈 > 条件に動的な値 (変数) を使用できませんについて説明します。

 C と C++ の型が検査される注釈を適用するときに、この警告が報告されます。

 内の式、`_When_`定数に評価される必要があります。 `_When_`は無視されます。
