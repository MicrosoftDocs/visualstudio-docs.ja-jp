# C28351
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28351: 注釈 \<注釈\> には、動的な値 \(変数\) が使用できない条件が記述されています  
  
 この警告は、C\/C\+\+ の型がチェックされているで注釈が適用された場合です。  
  
 `_When_` の式は定数として評価される必要があります。  `_When_` は無視されます。
