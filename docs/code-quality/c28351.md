# <a name="c28351"></a>C28351
warning C28351: 注釈 \<annotation > では、動的な値 (変数) が条件内で使用できない場所について説明しています。

 この警告は、C/C++型の検査対象となる注釈が適用されたときに報告されます。

 @No__t_0 内の式は、定数に評価される必要があります。 @No__t_0 は無視されます。
