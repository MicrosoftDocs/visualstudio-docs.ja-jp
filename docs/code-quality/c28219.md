# <a name="c28219"></a>C28219
警告 C28219: 注釈のパラメーターには、列挙型のメンバーが必要です

 注釈へのパラメーターは、指定された `enum` 型のメンバーである必要がありますが、その他のシンボルが見つかりました。`enum` 型のメンバーを使用します。 これは通常、誤ってコード化された注釈マクロであることを示します。
