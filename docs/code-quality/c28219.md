# <a name="c28219"></a>C28219
警告 C28219:注釈 (注釈のパラメーター) には列挙型のメンバーが必要です

 注釈のパラメーターは、指定された @no__t 0 型のメンバーである必要がありますが、その他のシンボルが見つかりました。`enum` の型のメンバーを使用します。 これは通常、誤ってコード化された注釈マクロであることを示します。
