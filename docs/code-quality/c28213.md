# <a name="c28213"></a>C28213
警告 C28213:`_Use_decl_annotations_`注釈は、先行する宣言を変更しない限り、参照に使用される必要があります。 \<なぜ >

 先行する宣言が参照されている場合、この警告が報告されることがなく、必要な`_Use_decl_annotations_`注釈。
