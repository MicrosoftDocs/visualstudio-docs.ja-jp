# C28213
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28213: `_Use_decl_annotations_` 注釈は、変更、先行する宣言なしで、参照に使用される必要があります。\<理由\>  
  
 この警告は、前の宣言が `_Use_decl_annotations_` の注釈を指定せずに参照される場合です。
