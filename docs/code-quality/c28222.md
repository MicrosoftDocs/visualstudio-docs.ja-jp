# <a name="c28222"></a>C28222
警告 28222: _y\_、いいえ\_、または _Maybe\_の注釈が必要です  
  
 この警告は、注釈のパラメーター シンボルのいずれかを指定する必要のあることを示します`_Yes_`、 `_No_`、または`_Maybe_`、および他の記号が発生しました。 これは通常、不適切に記述された注釈マクロを示します。
