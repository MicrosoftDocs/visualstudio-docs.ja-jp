# C28222
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 28222: \_Yes\_、注釈を想定した\_No\_、または\_Maybe\_  
  
 この警告は、注釈にパラメーターがシンボル `_Yes_`、`_No_`、または `_Maybe_`の 1 人であると想定される他のシンボルが見つかりましたことを示します。  これは通常、誤ってコード化された注釈のマクロを使用します。
