# <a name="c28222"></a>C28222
警告 28222:\_はい\_、\_いいえ\_、または\_かもしれません\_の注釈が必要です

 この警告は、注釈のパラメーターが、`_Yes_`、`_No_`、または `_Maybe_` のいずれかの記号である必要があり、その他の記号が検出されたことを示します。 これは通常、誤ってコード化された注釈マクロであることを示します。
