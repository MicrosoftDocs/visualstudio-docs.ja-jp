# <a name="c28222"></a>C28222
警告 28222: 注釈には \_Yes \_、\_No \_、または \_Maybe が必要です

 この警告は、注釈のパラメーターが、`_Yes_`、`_No_`、`_Maybe_` のいずれかのシンボルであると想定され、他のシンボルが検出されたことを示します。 これは通常、誤ってコード化された注釈マクロであることを示します。
