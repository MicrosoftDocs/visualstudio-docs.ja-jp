# C28204
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28204: \<関数\>: このオーバーロードのうち 1 つだけと、\<ファイル名\>\(\<行\>\) にあるオーバーロードに \<パラメーター名\> に対する注釈が付けられています: 片方にのみ注釈を付けることはできません  
  
 この警告は、注釈にエラーがある場合です。
