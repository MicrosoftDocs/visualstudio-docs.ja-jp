# <a name="c28204"></a>C28204
警告 C28204:\<関数 >: このオーバー ロードとに 1 つの 1 つだけ\<ファイル名 > (\<行 >) の注釈が付けられて\<paramname >: 両方またはどちらもの注釈を付ける必要があります。  
  
 注釈でエラーがある場合は、この警告が報告されます。
