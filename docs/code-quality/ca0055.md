# <a name="ca0055"></a>CA0055
ファイルを読み込むことができませんでした:\<パス >

 コード分析ツールは、分析用に指定されたターゲット ファイルを読み込めませんでした。 これは、ファイルが見つからないか、ファイルへのアクセスが拒否された場合に発生します。 通常、ユーザーが入力アセンブリのプロパティをオーバーライドする場合に発生[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)]、または fxcopcmd.exe を手動で呼び出されました。
