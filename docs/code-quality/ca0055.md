# <a name="ca0055"></a>CA0055
ファイルを読み込むことができません:\<パス >

 コード分析ツールは、分析用に指定されたターゲット ファイルを読み込めませんでした。 これは、ファイルが見つからないか、ファイルへのアクセスが拒否された場合に発生します。 通常、これは、ユーザーは、Visual Studio で、入力アセンブリのプロパティをオーバーライドすることを選択または、fxcopcmd.exe を手動で呼び出された場合に発生します。
