# <a name="ca0057"></a>CA0057
違反は、現在のターゲットとルールのセットにマップすることはできません。

 FxCopCmd.exe は手動で起動されたプロジェクト ファイルとすると、このエラーが発生または/現在のターゲットとルールのセットにマップできない違反のある XML レポートをインポートします。

 分析を実行した場合にのみ発生ツールのコマンド ライン バージョンを使用します。
