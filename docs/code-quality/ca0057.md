# <a name="ca0057"></a>CA0057
現在のターゲットとルールのセットに違反をマップすることはできません

このエラーは、プロジェクトファイルまたはインポートされた XML レポートで FxCopCmd が手動で呼び出され、現在のターゲットとルールのセットにマップできない違反がある場合に発生します。

ツールのコマンドラインバージョンを使用して分析が行われた場合にのみ発生します。
