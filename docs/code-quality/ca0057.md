# <a name="ca0057"></a>CA0057
違反をターゲットと規則の現在のセットにマップすることはできません。

 FxCopCmd.exe はプロジェクト ファイルを使用して手動で起動された場合、このエラーが発生しますまたは/をターゲットと規則の現在のセットにマップすることはできませんの違反を持つ XML レポートをインポートします。

 分析を実行した場合にのみ発生コマンド ライン ツールのバージョンを使用します。
