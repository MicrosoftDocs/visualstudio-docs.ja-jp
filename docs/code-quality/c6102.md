# <a name="c6102"></a>C6102
警告 C6102:@No__t-0variable > を使用すると、失敗した関数呼び出しの行 \<location > になります。

 この警告は、前の関数呼び出しが失敗したときに `_Out_` パラメーターとしてマークされているため、変数が設定されていない場合に、 [C6001](../code-quality/c6001.md)ではなく報告されます。

 問題は、前に呼び出された関数が完全には注釈を付けられていないことである場合があります。 通常は、`_Always_`、`_Outptr_result_nullonfailure_` (COM コードの場合は `_COM_Outptr_`)、または関連する注釈が必要です。
