# <a name="c28244"></a>C28244
警告 C28244: 関数に対する注釈には、解析不能なパラメーター/外部注釈が含まれています  
  
 これは現在できない注釈; の標準マクロを使用している場合この警告は、一般的な使用で目にする必要はありません。
