# C28244
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28244: 関数に対する注釈には、解析不能なパラメーター\/外部注釈が含まれています  
  
 これは、標準マクロが注釈に使用した場合、現在可能ではありません; この警告は、一般的な使用方法で参照されると想定されていません。
