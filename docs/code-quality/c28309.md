# <a name="c28309"></a>C28309
警告 28309: < parameter_name >:注釈のオペランドは整数/列挙/ポインター型でなければなりません。 void オペランドおよび C++ オーバーロードされた演算子はサポートされません。 浮動小数点は整数で近似されます。 Types: \<typelist >。

 注釈式で void または関数を使用しようとしましたが、コード分析で処理できません。  このエラーは、通常、関数として実装されている @no__t 0 が使用される場合に発生しますが、その他のケースも発生する可能性があります。 @No__t-0typelist > の型を調べて、問題点についての手掛かりを探します。
