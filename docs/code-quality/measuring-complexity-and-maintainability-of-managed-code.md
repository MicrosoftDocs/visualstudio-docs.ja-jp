# <a name="measuring-complexity-and-maintainability-of-managed-code"></a>マネージ コードの複雑さと保守性の測定
最新のソフトウェア アプリケーションの複雑さの増加には、信頼性が高く、保守が容易なコードも難しくも長くなります。 近年、多くのソフトウェアのメジャー、コード メトリックスと呼ばれる開発されている開発者が作業のやり直しや向上テスト、コードが必要なを理解することができます。  
  
 開発者が使用できる[!INCLUDE[vsprvsts](../code-quality/includes/vsprvsts_md.md)]複雑さや、マネージ コードの保守容易性を測定するコード メトリックス データを生成します。 ソリューション全体または 1 つのプロジェクトのコード メトリックス データを生成できます。
