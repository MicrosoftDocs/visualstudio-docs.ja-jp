# マネージ コードの複雑さと保守性の測定
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

最新のソフトウェア アプリケーションの複雑さが増すにつれて、コードの信頼性や保守性の確保も難しくなっています。  近年では、コード メトリックスと呼ばれるソフトウェアの基準が数多く開発され、開発者は、コードの修正作業が必要な箇所やテスト作業の増加について把握しやすくなりました。  
  
 開発者は、[!INCLUDE[vsprvsts](../code-quality/includes/vsprvsts_md.md)] を使用して、マネージ コードの複雑さや保守性を測定するコード メトリックスを生成できます。  コード メトリクッス データは、ソリューション全体や単一のプロジェクトについて生成することができます。  
  
 ここでは、[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] によるコード メトリックスの生成とその使用方法を説明します。
