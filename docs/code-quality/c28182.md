# <a name="c28182"></a>C28182
警告 C28182:Null ポインターの逆参照

 **追加情報**: *\<ポインター 1 >* には @no__t と同じ NULL 値が含まれています。- *4ポインター 2 >* *\<note >*

 コード分析ツールは、ポインターが NULL になる可能性があることを確認したときにこの警告を報告します。 トレースの前にエラーが発生する可能性のある未確認のインスタンスがある場合、コード分析ツールは、すべてのインスタンスに対応するようにコードを変更できるように、最初のインスタンスの行番号を警告メッセージに追加します。

 *\<ポインター 2 >* は NULL である可能性があることが確認されます。 *\<ポインター 1 >* には*pointer2*と同じ値が含まれており、逆参照されています。 これらのポインターはコード内の場所がまったく異なる場合があるため、コード分析ツールがこの警告を報告している理由を特定できるように、両方のポインターが報告されます。

 条件の未確認のインスタンスが存在する場合、 *\< note >* は次のテキストに置き換えられます。「このような場合に発生する可能性がある以前の場所については *、行 \<number >* を参照してください。」

## <a name="example"></a>例
 次の例では、コード分析ツールによってこの警告メッセージが生成される可能性のあるコードを示します。 この例では、コード分析ツールによって、`if` ステートメントで `pNodeFree` が NULL であることが確認され、`if` の本体のコードパスが取得されます。 ただし、`nBlockSize` はゼロになる可能性があるため、`for` ステートメントの本体は実行されず、`pNodeFree` が変更されずに残ります。 次に `pNodeFree` が `pNode` に割り当てられ、NULL の逆参照が発生したときに `pNode` が使用されます。

```
typedef struct xlist {
    struct xlist *pNext;
    struct xlist *pPrev;
} list;

list *pNodeFree;
list *masterList;
int nBlockSize;

void fun()
{
    if (pNodeFree == 0)
    {
        list *pNode = masterList;

        for (int i = nBlockSize-1; i >= 0; i--, pNode--)
        {
            pNode->pNext = pNodeFree;
            pNodeFree = pNode;
        }
    }

    list* pNode = pNodeFree;
    pNode->pPrev = 0;
}
```

 コード分析ツールでは、次の警告が報告されます。

```
:\sample\testfile.cpp(24) : warning C28182: Dereferencing NULL pointer. 'pNode' contains the same NULL value as 'pNodeFree' did.: Lines: 12, 14, 16, 23, 24
```

## <a name="example"></a>例
 前の例を修正する方法の1つとして、NULL の逆参照が averted されるように、0の場合は @no__t 0 にすることをお勧めします。 次のコードは、この修正を示しています。

```
typedef struct xlist {
    struct xlist *pNext;
    struct xlist *pPrev;
} list;

list *pNodeFree;
list *masterList;
int nBlockSize;

void fun()
{
    if (pNodeFree == 0)
    {
        list *pNode = masterList;

        for (int i = nBlockSize-1; i >= 0; i--, pNode--)
        {
            pNode->pNext = pNodeFree;
            pNodeFree = pNode;
        }
    }

    list* pNode = pNodeFree;
    if(pNode != 0)
      pNode->pPrev = 0;
}
```
