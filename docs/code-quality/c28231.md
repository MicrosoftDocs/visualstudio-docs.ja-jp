# C28231
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28231: 注釈は配列内でのみ有効です  
  
 この警告は、注釈配列への引数が必要な別の型が見つかりましたことを示します。
