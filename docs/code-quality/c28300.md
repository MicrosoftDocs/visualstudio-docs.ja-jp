# <a name="c28300"></a>C28300
警告 C28300: <パラメーター名>: 演算子 <演算子名> に対する互換性のない型の、式のオペランドです  
  
 オペランドの型をパラメーターでは、演算子と互換性がない場合は、この警告が報告されます。
