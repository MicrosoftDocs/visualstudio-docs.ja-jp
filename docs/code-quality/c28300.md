# C28300
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28300: \<パラメーター名\>: 演算子 \<演算子名\> に対する互換性のない型の、式のオペランドです  
  
 この警告は、パラメーターのオペランドの型が演算子と互換性がない場合です。
