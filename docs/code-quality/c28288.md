# C28288
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28288: 関数の場合、\_At\_\(\) の注釈 \(無効なパラメーターの名前\) の構文エラー  
  
 コード分析ツールは、この警告を `SAL_at` と \(`__drv_at`\) の注釈使用されているレポートとパラメーター式は、現在のコンテキストに解釈することはできません。  これは「Return」のスペルミス パラメーターまたはメンバー名、または誤った\) を使用して、「この」というキーワードが含まれる場合があります。
