# <a name="c28288"></a>C28288
警告 C28288: 関数の場合 (_a) に構文エラー\_() 注釈 (無効なパラメーター名)  
  
 コード分析ツールは場合、この警告を報告、 `SAL_at` (`__drv_at`) 注釈を使用し、現在のコンテキストでは、パラメーター式を解釈できません。 スペルの正しくないパラメーターまたはメンバー名、または"return"または"this"キーワードのスペルミスを使用してこのなどがあります。
