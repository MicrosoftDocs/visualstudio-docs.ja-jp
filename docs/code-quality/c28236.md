# C28236
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28236: 注釈は式内で使用できません  
  
 この警告は式のコンテキストの注釈で宣言された関数を使用してが行われたことを示します。  これは、標準マクロが注釈に使用される可能ではありません; この警告は、一般的な使用方法で参照されると想定されていません。
