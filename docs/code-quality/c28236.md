# <a name="c28236"></a>C28236
警告 C28236: 注釈は式内で使用できません  
  
 この警告は、式のコンテキストで注釈として宣言された関数を使用する試みが行われたことを示します。 これはできません。 注釈の標準マクロを使用している場合この警告は、一般的な使用で目にする必要はありません。
