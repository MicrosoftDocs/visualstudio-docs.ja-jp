# <a name="c28221"></a>C28221
警告 C28221:注釈のパラメーターには文字列式が必要です

 この警告は、注釈のパラメーターは、文字列であることが必要です。 その他のいくつかの種類が見つかりましたを示します。 これは、通常、正しくコード化された注釈マクロを示しますされ、通常の使用で確認する必要はありません。
