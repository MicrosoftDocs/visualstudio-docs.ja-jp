# C28221
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28221: 注釈のパラメーターには文字列式が必要です  
  
 この警告は、注釈へのパラメーターは文字列であると想定されるそのほかの型が見つかりましたことを示します。  これは通常、誤ってコード化された注釈マクロを示し、正常な使用に参照されると想定されていません。
