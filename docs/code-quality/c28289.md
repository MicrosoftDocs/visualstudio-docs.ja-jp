# C28289
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28289: 関数の場合: ReadableTo または WritableTo にパラメーターとして制限仕様はありませんでした。  
  
 コード分析ツールは、関数パラメーターと注釈で説明したように miscoded にこの警告を報告します。
