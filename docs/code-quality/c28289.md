# <a name="c28289"></a>C28289
警告 C28289: 関数の場合: ReadableTo または WritableTo にパラメーターとして limit-spec がいません。

 コード分析ツールは、関数/パラメーターの注釈が誤った前述したときに、この警告を報告します。
