# <a name="c6387"></a>C6387
警告 C6387: \<argument > は \< の値 > ます。これは、関数の仕様に準拠していません \<function name >:行: x、y

 この警告は、注釈付きの関数パラメーターに予期しない値が渡された場合に発生します。 たとえば、`_In_` 注釈でマークされたパラメーターに null 値を渡すと、この警告が生成されます。

## <a name="example"></a>例
 次のコードでは、null パラメーターが `f(char *)` に渡されるため、この警告が生成されます。

```cpp

#include <sal.h>

_Post_ _Null_ char * g();

void f(_In_ char *pch);

void main()
{
    char *pCh = g();
    f(pCh); // Warning C6387
}
```

 この警告を解決するには、次のコードを使用します。

```cpp

#include <sal.h>

_Post_ _Notnull_ char * g();

void f(_In_ char *pch);

void main()
{
    char *pCh = g();
    f(pCh);
}
```
