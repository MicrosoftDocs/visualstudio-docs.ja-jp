# C1250
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C1250: プラグインを読み込むことができません。  
  
 コード分析ツールは、プラグインに内部エラーが発生した場合、分析するコードでは、この警告を、報告します。
