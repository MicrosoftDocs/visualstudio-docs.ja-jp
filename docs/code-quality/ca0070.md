# <a name="ca0070"></a>CA0070
このエラーは、次の条件の少なくとも 1 つが true とコード分析は、アセンブリを分析し、アクセス許可属性を検出したときに発生します。  
  
-   コード分析は、属性を含むアセンブリを見つけることはできません。  
  
-   コード分析には、アセンブリを読み込む権限がありません。  
  
-   コード分析によって読み込まれたアセンブリに属性が含まれていません。  
  
 ほとんどの場合は、コード分析ツールの最新バージョンがあることを確認して問題を解決できます。
