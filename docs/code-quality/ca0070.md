# <a name="ca0070"></a>CA0070

このエラーは、コード分析でアセンブリを分析し、アクセス許可属性が見つかったとき、次の条件のうち、少なくとも 1 つが当てはまるときに発生します。

- コード分析で、属性が含まれるアセンブリが見つからない。

- コード分析に、アセンブリを読み込むアクセス許可がない。

- コード分析で読み込まれたアセンブリに属性が含まれていない。

ほとんどの場合、コード分析ツールの最新版を用意することで問題は解決できます。
