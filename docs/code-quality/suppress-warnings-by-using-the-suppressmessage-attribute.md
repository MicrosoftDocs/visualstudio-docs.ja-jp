# SuppressMessage 属性を使用した警告の抑制
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告は適用されない、と示すことは、チーム メンバーに対し、コードはレビュー済みであり、警告は抑制することになった、と知らせるのに役立ちます。  ソース内抑制 \(ISS\) を使用することにより、開発者は警告が生成された場所に近接する警告を抑制する属性を設定できます。  ISS 属性をソース ファイルに直接追加するか、[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] IDE のショートカット メニューを使用できます。
