# <a name="ca0056"></a>CA0056
プロジェクト ファイルは、分析ツールの正しくないバージョンです。

 この警告は、FxCopCmd.exe は不適切なコード分析ツールのバージョンを含むプロジェクト ファイルを手動で呼び出された場合に発生します。
