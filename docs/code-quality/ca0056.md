# <a name="ca0056"></a>CA0056
プロジェクト ファイルが、分析ツールのバージョンが正しくありません。  
  
 FxCopCmd.exe は不適切なコード分析ツールのバージョンを含むプロジェクト ファイルを手動で呼び出す場合、この警告が発生します。
