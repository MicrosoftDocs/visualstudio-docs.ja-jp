# <a name="ca0056"></a>CA0056
プロジェクト ファイルに含まれている分析ツールのバージョンが正しくありません。

FxCopCmd.exe が手動で呼び出されたとき、プロジェクト ファイルに含まれているコード分析ツールのバージョンが正しくない場合、この警告が表示されます。
