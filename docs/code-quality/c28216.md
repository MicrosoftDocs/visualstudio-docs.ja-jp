# <a name="c28216"></a>C28216
警告 C28216: 注釈\_\_返す \_チェックは、特定の関数パラメーターの事後条件にのみ適用されます。

 `_Check_return_` 注釈が `__post`以外のコンテキストで適用されています。`__post` (または `__drv_out`) 修飾子が必要であるか、正しくない可能性があります。
