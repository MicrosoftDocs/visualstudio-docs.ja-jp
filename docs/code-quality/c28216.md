# <a name="c28216"></a>C28216
警告 C28216: \_Check \_return \_ 注釈は、特定の関数パラメーターの事後条件にのみ適用されます。

 @No__t_0 注釈が `__post` 以外のコンテキストで適用されています。`__post` (または `__drv_out`) 修飾子が必要であるか、正しくない可能性があります。
