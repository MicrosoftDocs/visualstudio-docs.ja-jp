# <a name="c28216"></a>C28216
警告 C28216:@No__t-0Check @ no__t-1return @ no__t-2 注釈は、特定の関数パラメーターの事後条件にのみ適用されます。

 @No__t 0 の注釈が `__post` 以外のコンテキストで適用されています。`__post` (また @no__t は 3) 修飾子が必要な場合や、間違って配置されている可能性があります。
