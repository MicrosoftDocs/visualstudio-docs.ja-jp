# <a name="c28216"></a>C28216
警告 C28216: _Check_return\_注釈は、特定の関数のパラメーターの事後条件にのみ適用されます。  
  
 `_Check_return_`コンテキスト内で注釈が以外の場合に適用`__post`; 必要があります、 `__post` (または`__drv_out`) 修飾子であるか、または配置することもありません正しくです。
