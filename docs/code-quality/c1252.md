# <a name="c1252"></a>C1252
警告 C1252: プラグイン間の依存関係が循環または不足しています: GUID が必要です

 コード分析ツールは、この警告を報告します。この警告は、分析対象のコードではなく、プラグインの依存関係で内部エラーが発生した場合に報告されます。
