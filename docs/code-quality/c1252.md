# C1252
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C1252: プラグイン間の依存関係が循環または不足しています: GUID が必要です  
  
 コード分析ツールは、プラグインの依存関係の内部エラーが発生した場合、分析するコードでは、この警告を、報告します。
