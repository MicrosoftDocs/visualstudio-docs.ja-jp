# <a name="c1252"></a>C1252
警告 C1252: プラグイン間の依存関係が循環または不足しています: GUID が必要です

 コード分析ツールは、分析対象のコードではなく、プラグインの依存関係を持つ内部エラーがある場合に、この警告を報告します。
