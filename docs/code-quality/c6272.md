# <a name="c6272"></a>C6272
警告 C6272: 浮動小数点値が引数として渡されました。 \<function への呼び出しに float が必要な場合は、\<number > >

 この警告は、書式指定文字列で float が必要であることが指定されていることを示します。たとえば、`printf,` の `%f` または `%g` の指定ではなく、整数や文字列などの非 float 値が渡されます。 この欠陥は、出力が正しくない可能性があります。ただし、状況によっては、クラッシュが発生する可能性があります。

## <a name="example"></a>例
 この警告が発生するコード例を次に示します。

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf(buff,"%s %f","a",i);
}
```

 この警告を解決するには、次のコードに示すように、`%f` の指定ではなく `%i` を使用します。

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf(buff,"%s %i","a",i);
}
```

 次のコードでは、セーフ文字列操作関数である @no__t 0 を使用して、この警告を修正します。

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf_s(buff,5,"%s %i","a",i); // safe version
}
```
