# <a name="c6272"></a>C6272
warning C6272: \<function への呼び出しに float が必要な場合に、float 以外の引数を引数として渡す \<number > >

 この警告は、書式指定文字列によって、浮動小数点が必要であることを示しています。たとえば、`printf,` には `%f` または `%g` のように指定しますが、整数や文字列などの非 float 型が渡されることを示します。 この欠陥は、出力が正しくない可能性があります。ただし、状況によっては、クラッシュが発生する可能性があります。

## <a name="example"></a>例
 この警告が発生するコード例を次に示します。

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf(buff,"%s %f","a",i);
}
```

 この警告を解決するには、次のコードに示すように、`%f` の指定ではなく `%i` を使用します。

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf(buff,"%s %i","a",i);
}
```

 次のコードでは、安全な文字列操作関数 `sprintf_s` を使用して、この警告を修正します。

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf_s(buff,5,"%s %i","a",i); // safe version
}
```
