# C28230
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28230: パラメーターの型にはメンバーがありません  
  
 この警告は、注釈への引数が `struct`、`class`にアクセスしようとした場合、または `union` と名前付きなメンバーがないことを示します。
