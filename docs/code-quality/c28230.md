# <a name="c28230"></a>C28230
警告 C28230:パラメーターの型にはメンバーがありません。

 この警告は、注釈への引数がアクセスしようとしたことを示します、 `struct`、 `class`、または`union`で名前付きのメンバーが存在しません。
