# <a name="c28245"></a>C28245
警告 C28245: 関数に対する注釈は、非メンバー関数上で 'this' に注釈を付けます  
  
 PREfast モデル ファイルに内部エラーが発生しました。 この警告は、一般的な使用法では実行されません。
