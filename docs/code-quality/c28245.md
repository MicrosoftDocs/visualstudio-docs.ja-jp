# C28245
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28245: 関数に対する注釈は、非メンバー関数上で 'this' に注釈を付けます  
  
 内部エラーが PREfast モデル ファイルで発生しました。  この警告は、一般的な使用方法に発生する必要があります。
