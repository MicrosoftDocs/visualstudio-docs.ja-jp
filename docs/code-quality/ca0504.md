# <a name="ca0504"></a>CA0504
規則ディレクトリは存在しないため、無視されました

警告 CA504 は、プロジェクトまたはビルド構成ファイルの**CodeAnalysisRuleDirectories**プロパティで指定されたディレクトリが見つからない場合に生成されます。 この問題を解決するには、有効なディレクトリパスを入力します。
