# <a name="ca0504"></a>CA0504
存在しないために、ルールのディレクトリは無視されました  
  
 ディレクトリが指定した場合に警告 CA504 が生成された、 **CodeAnalysisRuleDirectories**プロジェクトまたはビルドの構成ファイル内のプロパティが見つかりませんでした。 この問題を解決するには、有効なディレクトリ パスを入力します。
