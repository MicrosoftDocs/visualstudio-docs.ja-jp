# <a name="c26497-use_constexpr_for_function"></a>C26497 USE_CONSTEXPR_FOR_FUNCTION

コンパイル時の評価が必要な場合は、この関数% function% を `constexpr` としてマークできます。 「 [ C++コアガイドライン f. 4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rf-constexpr)」を参照してください。
