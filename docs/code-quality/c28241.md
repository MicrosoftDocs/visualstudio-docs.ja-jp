# <a name="c28241"></a>C28241
警告 C28241: パラメーターの関数に対する注釈は認識されません

 認識されていない注釈名が使用されました。 注釈に標準のマクロが使用されている場合は、これを行うことはできません。この警告は、通常の使用では表示されません。
