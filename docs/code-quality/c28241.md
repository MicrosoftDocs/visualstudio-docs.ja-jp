# <a name="c28241"></a>C28241
警告 C28241: パラメーターの関数に対する注釈は認識されません

 認識されない注釈名が使用されました。 これはできません。 注釈の標準マクロを使用している場合この警告は、一般的な使用で目にする必要はありません。
