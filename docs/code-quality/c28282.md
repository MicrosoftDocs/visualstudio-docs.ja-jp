# <a name="c28282"></a>C28282
警告 C28282:書式指定文字列は、前提条件の中に存在する必要があります

 この警告は、`_Pre_` (`__drv_in`) の注釈に含まれていない @no__t 0 の注釈が見つかったことを示します (関数のパラメーターは既定で `_Pre_`)。 注釈が @no__t 0 (`__drv_out`) の注釈を持つ明示的なブロックで使用されているかどうかを確認します。 その場合は、外側のブロックから注釈を削除して、それを @no__t 0 のコンテキストに配置します。
