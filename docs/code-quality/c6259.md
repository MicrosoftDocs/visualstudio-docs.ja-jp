# <a name="c6259"></a>C6259
警告 C6259: ラベル付きのコードに到達できません: (\<expression > & \< 定数 >) では、switch-expr は \<case-label に評価されません >

 この警告は、スイッチ式でのビットごとの and (`&`) 比較の結果によって発生した到達できないコードを示します。 Switch 式の定数に一致する case ステートメントは、アクセスできるだけです。その他の case ステートメントには到達できません。

## <a name="example"></a>例
 次のサンプルコードでは、式 `switch``(rand() & 3)` が case ラベルに評価されないため、この警告が生成されます (`case` `4`)。

```
#include <stdlib.h>

void f()
{
     switch (rand () & 3) {
        case 3:
            /* Reachable */
            break;
        case 4:
            /* Not reachable */
            break;
        default:
            break;
    }
}
```

 この警告を修正するには、到達できないコードを削除するか、case ステートメントで使用されている定数が正しいことを確認します。 次のコードは、到達できない case ステートメントを削除します。

```
#include <stdlib.h>

void f()
{
     switch (rand () & 3) {
        case 3:
            /* Reachable */
            break;
        default:
            break;
    }
}
```
