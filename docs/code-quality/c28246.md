# <a name="c28246"></a>C28246
警告 C28246: 関数 ' \<name > ' パラメーター ' \<parameter > ' の注釈がパラメーターの型と一致しません

 注釈を作成するときに、__ deref 演算子が非ポインター型に適用されました。
