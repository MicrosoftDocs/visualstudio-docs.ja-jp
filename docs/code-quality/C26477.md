# <a name="c26477-usenullptrnotconstant"></a>C26477 USE_NULLPTR_NOT_CONSTANT

`nullptr` 特殊な型を持つ`nullptr_t`特別な処理の null 処理を持つオーバー ロードできるようにします。 使用して`0`または`NULL`の代わりに`nullptr`タイプ セーフと推論をバイパスする`nullptr`を提供します。
