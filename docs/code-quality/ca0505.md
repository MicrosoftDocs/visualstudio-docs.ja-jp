# <a name="ca0505"></a>CA0505

非推奨になっているプロパティは、それに取って代わるプロパティが定義されているため、無視されます。

プロジェクトまたはビルドの構成ファイルで、非推奨になっているプロパティとそれに取って代わるプロパティの両方に値が与えられているとき、警告 CA505 が生成されます。 この問題を解決するには、非推奨になっているプロパティを削除します。
