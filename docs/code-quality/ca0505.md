# <a name="ca0505"></a>CA0505

Superceding プロパティが定義されているため、非推奨のプロパティは無視されます。

警告 CA505 は、非推奨のコード分析プロパティと置き換えるコード分析プロパティの両方にプロジェクトまたはビルド構成ファイルの値がある場合に生成されます。 この問題を解決するには、非推奨のプロパティを削除します。
