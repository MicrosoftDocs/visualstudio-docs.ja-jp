# <a name="ca0505"></a>CA0505

優先するプロパティが定義されているために、非推奨のプロパティは無視されます。

警告 CA505 は、非推奨と優先するコード分析プロパティをプロジェクトに値があるまたは構成ファイルをビルドするときに生成されます。 この問題を解決するには、非推奨のプロパティを削除します。
