# <a name="ca0059"></a>CA0059
コマンド ライン スイッチのエラー。

必要なコマンド ライン スイッチが不足しているとき、このエラーが発生します。たとえば、

/out または /console スイッチが指定されていない場合です。 タイムアウト値が間違っているなど、スイッチが無効な場合にもこのエラーは発生します。
