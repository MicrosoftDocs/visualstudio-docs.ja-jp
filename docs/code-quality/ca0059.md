# <a name="ca0059"></a>CA0059
コマンドラインスイッチエラー。

このエラーは、次のように、必要なコマンドラインスイッチがない場合に発生します。

/out または/console スイッチが指定されていません。 このエラーは、タイムアウト値が正しくないなど、無効なスイッチがある場合にも発生します。
