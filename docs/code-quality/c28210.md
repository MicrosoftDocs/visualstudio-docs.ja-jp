# <a name="c28210"></a>C28210
警告 28210: `_On_failure_` コンテキストの注釈を明示的なプリ コンテキストに含めることはできません

 注釈 `_On_failure_` は、`__post` コンテキストで明示的または暗黙的に指定されている必要があります。つまり、関数がを返した後に適用されます。  これを確認するには、`_drv_out` を使用します。
