# <a name="c28210"></a>C28210
警告 28210:@No__t-0 コンテキストの注釈は、明示的な先行コンテキスト内に存在することはできません

 注釈 `_On_failure_` は、`__post` のコンテキストで明示的または暗黙的に指定されている必要があります。つまり、関数がを返した後に適用されます。  このことを確認するには `_drv_out` を使用します。
