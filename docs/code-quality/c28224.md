# C28224
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28224: 注釈にはパラメーターが必要です  
  
 この警告は名前付きな注釈はパラメーターなしで使用され、少なくとも 1 個のパラメーターが必須であることを示します。  これは、標準マクロが注釈に使用される可能ではありません; この警告は、通常の使用で参照されると想定されていません。
