# <a name="c28224"></a>C28224
警告 C28224: 注釈にはパラメーターが必要です

 この警告は、パラメーターを指定せず、名前付きの注釈を使用しには、少なくとも 1 つのパラメーターが必要ことを示します。 これはできません。 注釈の標準マクロを使用している場合この警告は、通常の使用で目にする必要はありません。
