# <a name="c28224"></a>C28224
C28224 を警告します。注釈にはパラメーターが必要です

 名前付きの注釈を使用してパラメーターを指定せず、少なくとも 1 つのパラメーターが必要なことをこの警告を示します。 注釈は標準のマクロが使用されている場合があります。この警告は、通常の使用で目にする必要はありません。
