# <a name="c28243"></a>C28243
警告 C28243: 注釈には、パラメーターの関数のされた逆参照注釈が付けられた実際の型が許容します。  
  
 数`__deref`演算子が注釈には、パラメーターの型によって定義されているポインターのレベル数よりも多い。 いずれかを変更することで、注釈、または参照パラメーターのポインターのレベルの数に対応するこの警告を解決します。
