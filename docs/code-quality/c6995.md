# <a name="c6995"></a>C6995
C6995 の警告: XML ログ ファイルを保存できませんでした  
  
 この警告は、コード分析ツールがコード分析の出力である defect ログを作成できないことを示します。  
  
 このエラーは、ディスク エラーを示している可能性があります。 または指定したディレクトリにファイルを作成する権限を持っていないことを示します。
