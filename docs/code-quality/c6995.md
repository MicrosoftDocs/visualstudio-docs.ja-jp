# <a name="c6995"></a>C6995
警告 C6995: XML ログファイルを保存できませんでした

 この警告は、コード分析ツールがコード分析の出力である欠陥ログを作成できないことを示します。

 このエラーは、ディスクエラーが発生したか、指定されたディレクトリにファイルを作成するためのアクセス許可がないことを示している可能性があります。
