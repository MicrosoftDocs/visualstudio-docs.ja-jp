# C6995
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C6995: XML ログ ファイルを保存できませんでした  
  
 この警告は、コード分析ツールでは、コード分析の出力である問題のログを作成できないことを示します。  
  
 このエラーは、ディスク エラーを示すか、指定したディレクトリにファイルを作成するアクセス許可がないことを示す場合があります。
