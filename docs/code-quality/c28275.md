# <a name="c28275"></a>C28275
警告 C28275: _Macro_value パラメーター\_が null です  
  
 この警告は、モデル ファイルではなく、分析するコードの内部エラーがあることを示します。 *MacroValue*関数がパラメーターを指定せずに呼び出されました。
