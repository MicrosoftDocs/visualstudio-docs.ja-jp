# C28275
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28275: \_Macro\_value\_ へのパラメーターが null です  
  
 この警告は、モデル ファイルに内部エラーがあることを、分析するコードで示します。  *macroValue* 関数はパラメーターなしで呼び出されました。
