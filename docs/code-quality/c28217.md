# <a name="c28217"></a>C28217
警告 C28217: 関数について、注釈へのパラメーターの数と一致しませんファイルで検出されました。  
  
 現在の行と、メッセージ内の行にコメントが一致しません。 これはできません。 注釈の標準マクロを使用している場合この警告は、通常の使用で目にする必要はありません。
