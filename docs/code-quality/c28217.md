# C28217
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28217: 関数では、注釈に渡すパラメーターの個数はファイルにある一致しません。  
  
 現在の行とメッセージの行の注釈が矛盾しています。  これは、標準マクロが注釈に使用される可能ではありません; この警告は、通常の使用で参照されると想定されていません。
