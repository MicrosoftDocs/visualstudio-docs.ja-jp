# <a name="c28312"></a>C28312
警告 28312: \<関数 > \<パラメーター > の繰り返し宣言の < annotation_name > 注釈は、obsolescent バージョンの SAL です。

 注釈は古いバージョンであるため、同等の[SAL2](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md)に更新する必要があります。 この警告は、それより前に一貫性のない注釈の警告が生成された場合は生成されず、宣言ごとに一度しか報告されません。 より古い SAL については、残りの宣言を調べてください。
