警告 C28196: 要件が満たされていません  (式は true に評価されません)  
  
 この警告は、分析する関数の関数のプロトタイプがあることを示します、 `__notnull`、`__null`または`__drv_valueIs`上、`_Out_`パラメーターまたは戻り値が、値が返されますその注釈と整合性がありません。
