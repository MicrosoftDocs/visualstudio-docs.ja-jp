# <a name="c28196"></a>C28196
警告 C28196:要件が満たされていません。 (式は true に評価されません)

 この警告は、分析される関数の関数プロトタイプの @no__t が0、`__null`、または `__drv_valueIs` が `_Out_` パラメーターまたは戻り値に含まれていますが、返される値がその注釈と一致していないことを示しています。
