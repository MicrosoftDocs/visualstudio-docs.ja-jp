# <a name="c28215"></a>C28215
警告 C28215: typefix は、既に typefix のあるパラメーターには適用できません  
  
 適用する、`__typefix`その注釈は既にパラメーターに注釈はエラーです。 `__typefix`注釈は、いくつかの特別な場合にのみ使用され、この警告は、通常の使用で目にする必要はありません。
