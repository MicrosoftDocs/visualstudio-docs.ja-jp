# <a name="c28215"></a>C28215
警告 C28215:typefix は、既に typefix のあるパラメーターには適用できません

 注釈が既に存在するパラメーターに @no__t 0 の注釈を適用すると、エラーになります。 @No__t 0 の注釈はいくつかの特殊なケースでのみ使用されます。この警告は通常の使用時には表示されません。
