# <a name="ca0058"></a>CA0058

参照されたアセンブリを読み込めません。

このエラーは、分析ツールが、分析対象のアセンブリによって参照されるアセンブリを読み込むことができない場合に発生します。 このエラーが発生すると、予期しない状態が発生する可能性があるため、このエラーによって他の場所で CA0001 が発生する可能性があります。 Visual Studio 内からマネージコード分析ツールを使用している場合は、次のような理由で CA0058 が発生する可能性があります。

- 入力アセンブリがオーバーライドされたので、Visual Studio プロジェクトファイル内の参照の一覧の外部にある他のアセンブリを参照するアセンブリを指しています。

- FxCopCmd が実行される前にカスタムビルドステップを呼び出すようにプロジェクトを変更し、カスタムビルドプロセスによって入力アセンブリへの参照を追加すると、エラー CA0058 が生成されます。

- でC++は、.Vcproj で ForcedUsing を使用してアセンブリを参照することができます。また、コマンドラインをオーバーライドしてを使用するか、> \<#using をソースにのみ追加します。

どのような場合でも、この問題の解決策は、不足している参照を Visual Studio プロジェクト自体に追加することです。
