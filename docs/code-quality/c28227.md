# C28227
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28227: 注釈は、PrimOp \(前の宣言を参照\) になることもできません  
  
 この警告は名前付きな注釈を通常の注釈として宣言されていること、または PrimOp でして宣言されました。  これは、標準マクロが注釈に使用される可能ではありません; この警告は、一般的な使用方法で参照されると想定されていません。
