# <a name="c28227"></a>C28227
警告 C28227: 注釈こともできません、PrimOp (前の宣言を参照してください)

 この警告は、名前付きの注釈が、通常注釈として宣言されるもは以前として宣言、primop になることを示します。 これはできません。 注釈の標準マクロを使用している場合この警告は、一般的な使用で目にする必要はありません。
