# <a name="c28235"></a>C28235
警告 C28235: 関数は注釈として独立できません

 この警告は、注釈コンテキストの注釈に宣言されていない関数を使用する試みが行われたことを示します。 これには、スタンドアロンのコンテキストでプリミティブ操作 (PrimOp) の使用が含まれます。 これができない注釈の標準マクロを使用している場合。 この警告は、一般的な使用で目にする必要はありません。
