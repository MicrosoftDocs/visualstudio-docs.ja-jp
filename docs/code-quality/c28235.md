# <a name="c28235"></a>C28235
警告 C28235:関数は注釈として独立できません

 この警告は、注釈コンテキストで注釈として宣言されていない関数を使用しようとしたことを示します。 これには、スタンドアロンコンテキストでのプリミティブ演算 (PrimOp) の使用が含まれます。 注釈に標準のマクロを使用している場合は、これを行うことはできません。 この警告は、通常の使用では表示されません。
