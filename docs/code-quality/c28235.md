# <a name="c28235"></a>C28235
C28235 を警告します。関数は注釈として独立できません

 この警告は、注釈コンテキストでは注釈に宣言されていない関数を使用しようとすることを示します。 これは、スタンドアロンのコンテキストでプリミティブの操作 (PrimOp) の使用が含まれます。 注釈の標準マクロを使用している場合に考えられるないはずです。 この警告は、一般的な使用で目にする必要はありません。
