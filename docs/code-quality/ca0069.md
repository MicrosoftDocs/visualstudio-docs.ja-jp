# <a name="ca0069"></a>CA0069

Alternateplatform をお持ちです。 *FrameworkVersion1* が見つかりませんでした。 代わりに *FrameworkVersion2* を使用してください。 最適な分析結果を得るには、正しいバージョンの framework がインストールされていることを確認してください。
