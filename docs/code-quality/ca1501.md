# <a name="ca1501-avoid-excessive-inheritance"></a>CA1501: 継承を使用しすぎないでください

|||
|-|-|
|TypeName|AvoidExcessiveInheritance|
|CheckId|CA1501|
|カテゴリ|マイクロソフト.メンテナンス性|
|互換性に影響する変更点|あり|

## <a name="cause"></a>原因

型が、その継承階層内の 5 つ以上深いレベルにあります。

既定では、この規則では名前空間からすべての型のみが`System`除外されますが、これは[構成可能です](#configurability)。

## <a name="rule-description"></a>規則の説明

深いレベルで入れ子にされた型の確認、理解、および保守は困難です。 このルールは、分析を同じモジュール内の階層に制限します。

## <a name="how-to-fix-violations"></a>違反の修正方法

この規則違反を修正するには、継承階層の深さの深さから型を派生させるか、中間の基本型の一部を削除します。

## <a name="when-to-suppress-warnings"></a>警告を抑制する場合

この規則からの警告を抑制しても安全です。 ただし、コードの保守が難しい場合があります。 基本型の可視性によっては、この規則の違反を解決すると、互換性に影響する変更が生じる可能性があることに注意してください。 たとえば、パブリックな基本型を削除することは、互換性に影響する変更です。

## <a name="configurability"></a>構成可能の有無

### <a name="inheritance-excluded-type-or-namespace-names"></a>継承除外された型または名前空間名

型または型の名前空間が継承階層ツリーにカウントされないような 、型または名前空間の名前 ('|' で区切られた) を構成できます。 既定では、`System.*`名前空間のすべての型が除外されます。 設定する値に関係なく、このデフォルト値が追加されます。

| オプション値 | まとめ |
| --- | --- |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = MyType` | 'MyType' という名前のすべての型、または含まれる名前空間に 'MyType' と 'System' 名前空間のすべての型が含まれているすべての型に一致します。 |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = MyType1\|MyType2` | 'MyType1' または 'MyType2' という名前の名前のすべての型、または含まれる名前空間に 'MyType1' または 'MyType2' と 'System' 名前空間のすべての型が含まれているすべての型に一致します。 |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = T:NS.MyType` | 名前空間 'NS' の特定の型 'MyType' と 'System' 名前空間のすべての型に一致します。 |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = T:NS1.MyType1\|T:NS2.MyType2` | 特定の型 'MyType1' と 'MyType2' を、それぞれの完全修飾名と 'System' 名前空間のすべての型と一致します。 |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = N:NS` | 'NS' 名前空間のすべての型と 'System' 名前空間のすべての型に一致します。 |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = My*` | 名前が 'My' で始まるすべての型、または含まれる名前空間部分が 'My' で始まり、'System' 名前空間のすべての型が一致します。 |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = T:NS.My*` | 名前空間 'NS' で名前が 'My' で始まるすべての型と、'System' 名前空間のすべての型に一致します。 |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = N:My*` | 含む名前空間が 'My' で始まるすべての型と、'System' 名前空間のすべての型に一致します。 |
