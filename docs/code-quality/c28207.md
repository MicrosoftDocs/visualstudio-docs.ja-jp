# C28207
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28207: \<式\>: 左側のオペランドは構造体であるため、. を使用します  
  
 この警告は、構造体の逆参照のドット \(.\) がある場合に報告されます。
