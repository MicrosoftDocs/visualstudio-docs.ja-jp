# <a name="c6282"></a>C6282
警告 C6282:不適切な演算子です: ブール値のコンテキストで定数が割り当てられています。 代わりに、'==' を使用してください

 この警告は、変数への定数の割り当てがテストコンテキストで検出されたことを示します。 テストコンテキストで変数に定数を割り当てるのは、ほとんどの場合、正しくありません。 @No__t-0 を `==` に置き換えます。または、この警告を解決するには、テストコンテキストから割り当てを削除します。

## <a name="example"></a>例
 この警告が発生するコード例を次に示します。

```
void f( int i )
{
   while (i = 5)
   {
   // code
   }
}
```

 この警告を解決するには、次のコードを使用します。

```
void f( int i )
{
   while (i == 5)
   {
   // code
   }
}
```
