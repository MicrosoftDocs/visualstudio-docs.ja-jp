# C28223
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28223: 注釈、パラメーターの予期されるトークンまたは識別子が見つかりませんでした  
  
 この警告は、注釈にパラメーターが識別子であると想定される他のシンボルが見つかりましたことを示します。  これは通常、誤ってコード化された注釈のマクロを使用します。  この位置の C または C\+\+ キーワードを使用すると、このエラーが発生します。
