# <a name="c28223"></a>C28223
警告 C28223: 注釈ので期待されるトークン/識別子が見つかりませんでしたパラメーター。  
  
 この警告は、注釈のパラメーターは、識別子があると想定し、他の記号が発生したことを示します。 これは通常、不適切に記述された注釈マクロを示します。 この位置に C または C++ のキーワードの使用には、このエラーが発生します。
