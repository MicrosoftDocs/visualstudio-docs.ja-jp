# <a name="c28226"></a>C28226
警告 C28226: 注釈こともできません、PrimOp (現在の宣言内)  
  
 この警告は、名前付きの注釈は、PrimOp として宣言されても、通常注釈に宣言されていたことを示します。 これはできません。 注釈の標準マクロを使用している場合この警告は、通常の使用で目にする必要はありません。
