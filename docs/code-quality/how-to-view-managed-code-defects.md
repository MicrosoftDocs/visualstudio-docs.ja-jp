# 方法 : マネージ コードの障害を表示する
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

コード分析では、エラー、警告、およびメッセージはコード分析のウィンドウに表示されます。  コード分析のウィンドウは、プロジェクトまたはソリューションのコード分析を実行したときに自動的に表示されます。  
  
 コード分析のウィンドウで、たとえば P:  
  
-   **警告のタイトルをクリックします。**  
  
     警告についての追加情報がコード分析のウィンドウに表示されます。  警告が発生したソース ファイル行はソース・コード エディターが表示され、強調表示されます。  
  
-   **警告の ID を選択します。**警告の ID は **CA** がプレフィックスとして付けられ、桁数が含まれます。  たとえば、CA1234。  
  
     警告については、MSDN ライブラリのヘルプ トピックが表示されます。  ヘルプ トピックは、警告問題が解決する発生し、または警告の原因に関する詳細情報が含まれ、例を示します。
