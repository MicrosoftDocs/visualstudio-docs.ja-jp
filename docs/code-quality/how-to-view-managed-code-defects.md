# <a name="how-to-view-managed-code-defects"></a>方法 : マネージ コードの障害を表示する
コード分析では、エラー、警告、およびメッセージは、コード分析 ウィンドウで報告されます。 コード分析 ウィンドウは、プロジェクトまたはソリューションでコード分析を実行するときに自動的に表示されます。  
  
 コード分析 ウィンドウで、次の操作を実行できます。  
  
-   **警告のタイトルを選択します。**  
  
     警告の詳細については、コード分析 ウィンドウに表示されます。 警告を発生させたソース ファイル内の行が表示され、ソース コード エディターで強調表示されます。  
  
-   **警告 id を選択します。**警告 id プレフィックスが付きます**CA** 4 桁の数字を含めるとします。 たとえば、CA1234 です。  
  
     警告の MSDN ライブラリのヘルプ トピックが表示されます。 ヘルプ トピックでは、警告の原因に関する詳細情報が含まれています、警告の問題が発生し、解決方法を示す例が含まれています。
