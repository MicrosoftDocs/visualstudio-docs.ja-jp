# <a name="c28021"></a>C28021
C28021 の警告: パラメーター \<param > で注釈が付けられた\<anno > へのポインターにする必要があります  
  
 注釈が付けられたオブジェクトはポインター型ではない場合は、この警告が報告されます。 この注釈は併用できません`void`または整数型。
