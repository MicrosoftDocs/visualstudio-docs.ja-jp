# <a name="c28021"></a>C28021
警告 C28021:パラメーター \<param > \<anno > で注釈が付けられている場合は、ポインターである必要があります

 この警告は、注釈が付けられているオブジェクトがポインター型ではない場合に報告されます。 この注釈は @no__t 0 または整数型では使用できません。
