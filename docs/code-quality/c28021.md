# C28021
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28021: パラメーター \<param\> \(\<anno\> で注釈が付けられている\) はポインターである必要があります  
  
 この警告は、注釈が付けられたオブジェクトがポインター型の場合です。  この注釈は `void` または整数型では使用できません。
