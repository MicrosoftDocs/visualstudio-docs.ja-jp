# <a name="c28021"></a>C28021
警告 C28021: \<anno > で注釈が付けられているパラメーター \<param > はポインターでなければなりません

 この警告は、注釈が付けられているオブジェクトがポインター型ではない場合に報告されます。 この注釈は、`void` 型または整数型では使用できません。
