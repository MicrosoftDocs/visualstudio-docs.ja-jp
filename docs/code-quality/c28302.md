# <a name="c28302"></a>C28302
警告 C28302:参照C++パラメーター < parameter_name > の場合、\<annotation > に追加の `_Deref_` 演算子が見つかりました。

 この警告は、パラメーターで `_Deref_` の追加レベルが使用されている場合に報告されます。

 SAL2 では、参照パラメーターを処理するときに @no__t の追加レベルを使用する必要はありません。 この特定の注釈は明確で、正しく解釈されますが、修正する必要があります。

 多くの場合、これを修正するには、古い @no__t 0 の注釈を削除し、SAL2 構文を使用します。 @No__t-0 を使用して、注釈を付ける特定のオブジェクトを参照することが必要になる場合があります。
