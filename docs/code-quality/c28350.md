# C28350
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28350: 注釈 \<注釈\> には、条件付きで適用できない状況が記述されています  
  
 通常、この警告は、C\/C\+\+ の型がチェックされているで注釈が適用された場合に生成されます。
