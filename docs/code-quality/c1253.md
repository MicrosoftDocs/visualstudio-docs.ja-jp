# C1253
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C1253: ロード モデル ファイルにない。  
  
 コード分析ツールは、モデル ファイルの内部エラーが発生した場合、分析するコードでは、この警告を、報告します。
