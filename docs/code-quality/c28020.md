# C28020
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28020: 式 \<expr\> はこの呼び出しで true ではありません  
  
 この警告は、前に示した\_Satisfies\_の式が true の場合に報告されます。  一般的には、不正なパラメーターを示します。  
  
 これが関数宣言に発生した場合、注釈は不可能な状態を示します。
