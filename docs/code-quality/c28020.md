# <a name="c28020"></a>C28020
警告 C28020: この呼び出しで > \<expr 式は true ではありません

 この警告は、表示されている \_Satisfies \_ 式が true でない場合に報告されます。 多くの場合、パラメーターが正しくないことを示します。

 これが関数宣言で発生した場合、注釈は不可能な状態を示します。
