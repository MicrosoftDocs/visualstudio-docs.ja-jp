# <a name="c28020"></a>C28020
警告 C28020:式 \<expr > がこの呼び出しで true ではありません

 この警告は、表示されている @no__t 00@ no__t-1 式が true ではない場合に報告されます。 多くの場合、パラメーターが正しくないことを示します。

 これが関数宣言で発生した場合、注釈は不可能な状態を示します。
