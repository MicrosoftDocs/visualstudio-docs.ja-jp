# <a name="c28020"></a>C28020
警告 C28020: 式\<expr > がこの呼び出しで true でないです。  
  
 この警告が報告されたときに、_Satisfies\_表示されている式が true ではありません。 頻繁に無効なパラメーターを示します。  
  
 関数の宣言でこの場合は、注釈は不可能な状態を示すです。
