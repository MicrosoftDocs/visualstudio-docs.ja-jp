# <a name="c28239"></a>C28239
警告 C28239:パラメーターの注釈には、両方の値、stringValue、または longValue、さらに paramn=xxx が含まれます。 paramn=xxx のみを使用してください

 PREfast モデルファイルで内部エラーが発生しました。 通常の使用では、この警告は発生しません。
