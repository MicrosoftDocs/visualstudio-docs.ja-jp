# <a name="c28239"></a>C28239
警告 C28239: パラメーターの注釈には、値、stringValue、または longValue の両方が含まれています。および paramn = xxx。 paramn=xxx のみを使用してください

 PREfast モデルファイルで内部エラーが発生しました。 通常の使用では、この警告は発生しません。
