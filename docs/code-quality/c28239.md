# C28239
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28239: パラメーターの注釈の値、stringValue、または longValue;があります。と paramn\=xxx。  paramn\=xxx のみを使用してください  
  
 内部エラーが PREfast モデル ファイルで発生しました。  この警告は、一般的な使用方法に発生する必要があります。
