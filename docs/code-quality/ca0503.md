# <a name="ca0503"></a>CA0503
プロパティは推奨されません。 置き換える方のプロパティを使用します

 非推奨のコード分析プロパティには、ビルドまたはプロジェクトの構成ファイルで値が含まれている場合は、ca 0503 の警告が生成されます。 この警告を解決するには、それを置き換える、指定したプロパティを使用します。
