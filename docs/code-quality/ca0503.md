# <a name="ca0503"></a>CA0503

プロパティは非推奨となっています。 それに取って代わるプロパティを使用してください。

警告 CA0503 は、ビルドまたはプロジェクトの構成ファイルで非推奨のコード分析プロパティに値が含まれているときに生成されます。 この警告を解決するには、それに取って代わる指定のプロパティを使用します。
