# <a name="ca0503"></a>CA0503
プロパティは非推奨とされます。 置き換えるプロパティを使用する

警告 CA0503 は、非推奨のコード分析プロパティにビルドまたはプロジェクト構成ファイルの値が含まれている場合に生成されます。 この警告を解決するには、それを置き換える指定されたプロパティを使用します。
