# <a name="c6329"></a>C6329
警告 C6329:@No__t-0function > の呼び出しの戻り値を \< の数値と比較して確認することはできません >

 プログラムは、`CreateFile` の呼び出しの戻り値と数値を比較しています。 @No__t-0 が成功すると、オブジェクトへの開いているハンドルが返されます。 失敗した場合は `INVALID_HANDLE_VALUE` が返されます。

## <a name="example"></a>例
 このコードによって警告が発生する可能性があります。

```cpp
if (CreateFile() == NULL)
    {
        return;
    }
```

## <a name="example"></a>例
 このコードは、次のエラーを修正します。

```cpp
if (CreateFile() == INVALID_HANDLE_VALUE)
    {
        return;
    }
```
