# C28229
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28229: 注釈はパラメーターをサポートしません  
  
 この警告は、注釈がパラメーターなしで宣言されているときに注釈がパラメーターに使用されたことを示します。  これは、標準マクロが注釈に使用される可能ではありません; この警告は、一般的な使用方法で参照されると想定されていません。
