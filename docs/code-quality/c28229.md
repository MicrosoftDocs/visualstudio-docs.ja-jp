# <a name="c28229"></a>C28229
警告 C28229: 注釈はパラメーターをサポートしません

 この警告は、注釈がパラメーターを指定せずに宣言されている場合に、注釈がパラメーターを持つ使用されたことを示します。 これはできません。 注釈の標準マクロを使用している場合この警告は、一般的な使用で目にする必要はありません。
