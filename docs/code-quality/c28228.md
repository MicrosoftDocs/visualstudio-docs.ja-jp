# C28228
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28228: 注釈パラメーター: 注釈内で型を使用することはできません  
  
 この警告は、パラメーター型がサポートされていないことを示します。  注釈は、パラメーターとしてのみ型のセットを使用できません。  これは、標準マクロが注釈に使用される可能ではありません; この警告は、一般的な使用方法で参照されると想定されていません。
