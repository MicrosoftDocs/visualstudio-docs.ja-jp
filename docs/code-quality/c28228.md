# <a name="c28228"></a>C28228
警告 C28228: 注釈パラメーター: 注釈内で型を使用することはできません

 この警告は、パラメーターがサポートされていない型であることを示します。 注釈は、パラメーターとしてのみ型の制限のセットを使用できます。 これはできません。 注釈の標準マクロを使用している場合この警告は、一般的な使用で目にする必要はありません。
