# <a name="c28220"></a>C28220
警告 C28220: 注釈 (注釈のパラメーター) には整数式が必要です

 この警告は、注釈へのパラメーターが整数式であることが想定されており、他の型が検出されたことを示します。 これは通常、誤ってコード化された注釈マクロであることを示します。
