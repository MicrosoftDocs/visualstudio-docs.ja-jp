# <a name="c28220"></a>C28220
警告 C28220: 注釈 (注釈のパラメーター) には整数式が必要です  
  
 この警告は、注釈のパラメーターは整数式があると想定し、その他のいくつかの種類が見つかりましたことを示します。 これは通常、不適切に記述された注釈マクロを示します。
