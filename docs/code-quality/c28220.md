# C28220
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28220: 注釈 \(注釈のパラメーター\) には整数式が必要です  
  
 この警告は、注釈へのパラメーターが整数式であると想定されるそのほかの型が見つかりましたことを示します。  これは通常、誤ってコード化された注釈のマクロを使用します。
