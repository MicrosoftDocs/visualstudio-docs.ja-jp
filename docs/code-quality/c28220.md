# <a name="c28220"></a>C28220
C28220 を警告します。注釈 (注釈のパラメーター) には整数式が必要です

 この警告は、注釈のパラメーターの整数式のことが必要ですし、その他のいくつかの種類が見つかりましたことを示します。 これは通常、正しくコード化された注釈マクロを示します。
