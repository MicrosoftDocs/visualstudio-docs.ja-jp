# <a name="c26101"></a>C26101
警告 C26101:変数 \<var > に対して、インタロックされた操作を正しく使用できません。

 Windows Api は、さまざまなインタロック操作を提供します。 注釈 `_Interlocked_` は、ロックされていない操作によってのみ変数にアクセスする必要があることを指定します。 警告 C26101 は、アクセスが @no__t 0 の注釈と一致しない場合に発行されます。

## <a name="example"></a>例
 次の例では、`_Interlocked_` コントラクトの違反があるため、警告 C26101 が生成されます。

```

CRITICAL_SECTION cs;
typedef struct _DATA
{
    _Interlocked_ LONG data;
} DATA;

void Safe(DATA* p)
{
    InterlockedIncrement(&p->data); // OK
}

void Unsafe(DATA* p)
{
    p->data += 1; // Warning C26101
    EnterCriticalSection(&cs);
    LeaveCriticalSection(&cs);
}
```
