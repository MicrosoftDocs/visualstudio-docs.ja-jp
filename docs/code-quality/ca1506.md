# <a name="ca1506-avoid-excessive-class-coupling"></a>CA1506: クラス結合度を大きくしすぎないでください

|||
|-|-|
|TypeName|AvoidExcessiveClassCoupling|
|CheckId|CA1506|
|カテゴリ|マイクロソフト.メンテナンス性|
|互換性に影響する変更点|あり|

## <a name="cause"></a>原因

型またはメソッドは、他の多くの型と結合されます。 コンパイラによって生成された型は、このメトリックから除外されます。

## <a name="rule-description"></a>規則の説明

この規則は、型またはメソッドに含まれる一意の型参照の数をカウントすることによって、クラス結合度を計測します。

クラス結合の度合いが高い型とメソッドは、維持するのが難しい場合があります。 結合が低く、凝集度が高い型とメソッドを使用することをお勧めします。

## <a name="how-to-fix-violations"></a>違反の修正方法

この違反を修正するには、型またはメソッドを再設計して、結合先の型の数を減らしてみてください。

## <a name="when-to-suppress-warnings"></a>警告を抑制する場合

他の型に対する依存関係が多いにもかかわらず、型またはメソッドが保守可能と見なされる場合は、この警告を除外します。
