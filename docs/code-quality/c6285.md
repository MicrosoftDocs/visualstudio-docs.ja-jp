# <a name="c6285"></a>C6285
警告 C6285: (@no__t 0 以外の定数 > &#124; &#124; \<non 以外の定数 >) は常に0以外の定数です。 ビットごとの and 演算子を使用するつもりでしたか?

 この警告は、2つ以上の定数値が、テストコンテキストで発生する論理 or 演算の引数として検出されたことを示します。 この式は常に TRUE です。

 定数値が1より大きい場合は、論理 or の引数をビットフィールドにすることをお勧めします。 この場合、ビットごとの演算子がより適切な演算子である可能性があるかどうかを検討します。

## <a name="example"></a>例
 この警告が発生するコード例を次に示します。

```cpp
#include <stdio.h>
#define     TESTED_VALUE    0x37
#define     MASK            0xaa

void f()
{
  if (TESTED_VALUE || MASK)
  {
    puts("(TESTED_VALUE || MASK) True");
    // code ...
  }
  else
  {
    puts("(TESTED_VALUE || MASK) False");
    // code ...
  }
}
```

 この警告を解決するには、次のコードを使用します。

```cpp
#include <stdio.h>
#define     TESTED_VALUE    0x37
#define     MASK            0xaa

void f(int flag)
{
  if ((TESTED_VALUE & MASK)== flag)
  {
    puts("true");
    // code ...
  }
  else
  {
    puts("false");
    // code ...
  }
}
```
