# C6705
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C6705: 注釈エラー: \<予期された数値\> 個の引数が注釈 \<パラメーター\> に必要ですが、\<実際の数値\> 個が見つかりました  
  
 この警告は、注釈にエラーがある場合です。
