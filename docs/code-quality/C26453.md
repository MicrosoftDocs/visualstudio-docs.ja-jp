# <a name="warning-c26453-arithmetic-overflow-left-shift-of-a-negative-signed-number-is-undefined-behavior"></a>警告 C26453:算術オーバーフローの場合:負の符号付き数値の左シフトは未定義の動作

この警告は、負符号付き整数の値が不適切であることし、実装定義の動作をトリガーのシフトが残っていますを示します。

## <a name="example"></a>例

```cpp
void leftshift(int shiftCount)
{
  const auto result = -1 << shiftCount;  // C26453 reported here

  // code
}
```

 この警告を解決するには、次のコードを使用します。

```cpp
void leftshift(int shiftCount)
{
  const auto result = ((unsigned)-1) << shiftCount; // OK

  // code
}
```
