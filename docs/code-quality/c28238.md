# C28238
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28238: パラメーターの注釈には、複数の値、stringValue、および longValue が含まれています。  paramn\=xxx を使用してください  
  
 内部エラーが PREfast モデル ファイルで発生しました。  この警告は、一般的な使用方法に発生する必要があります。
