# <a name="c6302"></a>C6302
警告 C6302: 書式文字列の不一致: \<function の呼び出しでワイド文字列が必要な場合に、パラメーター \<number > として渡された文字列 >

 この警告は、書式文字列でワイド文字列が必要であることが指定されていることを示します。 ただし、文字列が渡されています。 この不具合は、何らかの形式のクラッシュまたは破損の原因となる可能性があります。

## <a name="example"></a>例
 次のサンプルコードでは、文字列が `wprintf` 関数に渡されるため、この警告が生成されます。

```
#include<stdio.h>

void f()
{
  char buff[5] = "hi";

  wprintf(L"%s", buff);
}
```

 次のサンプルコードでは、`%hs` を使用して、`wprintf` 関数で1バイト文字列を指定しています。

```
#include<stdio.h>

void f()
{
  char buff[5] = "hi";

  wprintf(L"%hs", buff);
}
```

 次のサンプルコードでは、安全な文字列操作関数 `wprintf_s` を使用して、この警告を修正します。

```
#include<stdio.h>

void f()
{
  char buff[5] = "hi";

  wprintf_s(L"%hs", buff);
}
```
