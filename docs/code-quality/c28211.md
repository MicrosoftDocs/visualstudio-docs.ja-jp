# C28211
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28211: SAL\_context には静的コンテキスト名が必要です  
  
 この警告は `_Static_context_` の注釈にオペランド ツールが定義されたコンテキストの名前である必要があることを示します。  この警告は、通常の使用には発生しません。
