# <a name="c28211"></a>C28211
警告 C28211: SAL_context には静的コンテキスト名が必要です  
  
 この警告が示すオペランド、`_Static_context_`注釈は、ツールが定義のコンテキストの名前を指定する必要があります。 この警告は、通常の使用では発生しません。
