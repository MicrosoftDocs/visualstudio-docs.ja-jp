# <a name="c28211"></a>C28211
警告 C28211: SAL_context には静的コンテキスト名が必要です

 この警告は、`_Static_context_` 注釈のオペランドが、ツールで定義されたコンテキストの名前である必要があることを示します。 通常の使用では、この警告は発生しません。
