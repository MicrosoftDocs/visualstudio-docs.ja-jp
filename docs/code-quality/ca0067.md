# <a name="ca0067"></a>CA0067
ディレクトリが見つかりません。  
  
 値、 **/ディレクトリ**FxCopCmd コマンド ライン オプションが見つかりませんでした。  
  
 この警告は、使用している場合に発生することができます、 **CodeAnalysisDependentAssemblyPaths**プロパティの[!INCLUDE[vstecmsbuild](../extensibility/internals/includes/vstecmsbuild_md.md)]追加の参照アセンブリ検索パスとそれらのパスのいずれかを指定する存在しません。  
  
-   CA0067 の警告が表示された場合は[CA0058](ca0058.md)エラーまたは[CA0060](ca0060.md)警告、通常、その他のエラーを解決するには CA0067 が解決されます。  
  
-   他のエラーまたは警告なし CA0067 の警告が表示されている場合、ディレクトリが必要ないため、警告を無視できます通常します。
