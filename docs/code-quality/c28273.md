# <a name="c28273"></a>C28273
C28273 を警告します。関数について、手がかりには関数宣言との一貫性がありません。

 この警告は、注釈で、分析するコードではなくエラーを示します。 関数定義に表示される注釈は、宣言に表示されるものと一貫性がありません。 2 つの注釈は、一致するように解決する必要があります。
