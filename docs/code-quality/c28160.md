# C28160
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28160: 注釈エラーです  
  
 この警告は `__drv_error` の注釈が見つかったと報告されます。  この注釈が固定される使用され `__drv_when` の注釈とパラメーターの特定の組み合わせを示すために使用できます。コーディング手法にフラグを設定するために使用されます。
