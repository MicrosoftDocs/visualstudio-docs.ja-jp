# <a name="c28160"></a>C28160
警告 C28160:エラーの注釈

 この警告は @no__t 0 の注釈が検出された場合に報告されます。 この注釈は、修正が必要なコーディングの手法にフラグを設定するために使用されます。また、パラメーターの特定の組み合わせを示すために @no__t 0 の注釈と共に使用できます。
