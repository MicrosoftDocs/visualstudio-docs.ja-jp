# <a name="c28163"></a>C28163
警告 C28163:Try/except ブロック内から関数を呼び出すことはできません

 この警告は、関数が、`try/except` ブロックで囲まれてはならない型の場合に、`try/except` ブロックで見つかったときに報告されます。 コード分析ツールにより、関数が呼び出された1つ以上のパスが @no__t 0 ブロック内に見つかりました。
