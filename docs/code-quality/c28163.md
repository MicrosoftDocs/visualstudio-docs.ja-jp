# <a name="c28163"></a>C28163
警告 C28163: 関数を try/except ブロック内から呼び出さないでください  
  
 関数は、型の場合、この警告は報告にすることはありません囲む必要があります、`try/except`ブロックに存在、`try/except`ブロックします。 コード分析ツールが呼び出される関数が内に少なくとも 1 つのパスが見つかりません、`try/except`ブロックします。
