# <a name="c1254"></a>C1254
警告 C1254:プラグインバージョンの不一致: バージョンが PREfast ドライバーのバージョンと一致しません

 コード分析ツールは、この警告を報告します。この警告は、解析対象のコードではなく、プラグインのバージョンで内部エラーが発生した場合に報告されます。
