# C1254
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C1254: プラグイン バージョンの不一致: バージョンは PREfast ドライバーのバージョンと一致していません  
  
 コード分析ツールは、プラグイン バージョンの内部エラーが発生した場合、分析するコードでは、この警告を、報告します。
