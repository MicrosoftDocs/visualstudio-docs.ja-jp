# C28208
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28208: 関数 \<関数\> は別のパラメーター リストを伴って \<ファイル\>\(\<行\>\) で以前定義されました。  分析ツールによっては正しくない結果が生じる場合があります  
  
 この警告は、関数の既知の定義が別の発生に一致しない場合に報告されます。
