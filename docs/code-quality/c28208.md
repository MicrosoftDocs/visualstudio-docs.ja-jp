# <a name="c28208"></a>C28208
警告 C28208:関数 \<function > \<file > で別のパラメーターリストで既に定義されています (\<line >)。 分析ツールによっては正しくない結果が生じる場合があります

 この警告は、関数の既知の定義が別のオカレンスと一致しない場合に報告されます。
