# <a name="c28208"></a>C28208
警告 C28208: 関数 \<function > は \<file > (\<line >) で別のパラメーターリストを使用して定義されています。 分析ツールによっては正しくない結果が生じる場合があります

 この警告は、関数の既知の定義が別のオカレンスと一致しない場合に報告されます。
