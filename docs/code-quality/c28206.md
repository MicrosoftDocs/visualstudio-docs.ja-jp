# C28206
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28206: \<式\>: 左側のオペランドは構造体をポイントするため、`->` を使用します  
  
 この警告は、構造体のポインターの逆参照表記 `->` である場合に報告されます。
