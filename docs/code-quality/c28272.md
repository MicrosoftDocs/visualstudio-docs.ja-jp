# <a name="c28272"></a>C28272
警告 C28272: 関数、パラメーターに対する注釈は、検証中は関数の宣言と整合性がありません  
  
 この警告は、注釈ではなく、コード分析中にエラーを示します。 関数定義は、上に表示される注釈は、宣言で表示されるものと一致しません。 2 つの注釈は、一致するように解決する必要があります。
