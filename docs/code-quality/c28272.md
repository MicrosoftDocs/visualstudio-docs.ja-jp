# C28272
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28272: 関数、パラメーターに対する注釈は、検証中は関数の宣言と整合性がありません  
  
 この警告は、注釈は、分析対象のコードのエラーを示します。  関数定義における注釈は宣言に表示されるものと矛盾しています。  2 種類の注釈は、一致する解決する必要があります。
