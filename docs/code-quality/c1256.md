# C1256
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C1256: PCH が書き込まれ、読み込む必要があります。  
  
 コード分析ツールは、ツールに内部エラーが発生した場合、分析するコードでは、この警告を、報告します。
