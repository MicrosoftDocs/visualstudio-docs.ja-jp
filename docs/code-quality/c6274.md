# <a name="c6274"></a>C6274
C6274 の警告: パラメーターとして渡された文字でない\<番号 > の文字が必要になるへの呼び出しで\<関数 >

 この警告は、書式指定文字列の文字が必要であるが指定されることを示します (たとえば、`%c`または`%C`仕様)、float、string、または構造体などの整数でない値が渡されるが、します。 この問題は、不正な出力が発生する可能性があります。

## <a name="example"></a>例
 この警告が発生するコード例を次に示します。

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str);
}
```

 この警告を解決するには、次のコードを使用します。

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str[0]);
}
```

 次の例では、安全な文字列操作関数では、 `sprintf_s`、この警告を解決します。

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf_s(buff,5,"%c", str[0]);
}
```
