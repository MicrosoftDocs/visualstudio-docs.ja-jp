# <a name="c28022"></a>C28022
C28022 の警告: 関数クラス\<classlist1 > この関数と一致していません関数クラス\<classlist2 > の typedef がそれを定義するために使用します。  
  
 注釈でエラーがある場合は、この警告が報告されます。 Typedef と関数自体の両方がある`_Function_class_`の注釈が、それらが一致しません。 両方使用すると一致する必要があります。
