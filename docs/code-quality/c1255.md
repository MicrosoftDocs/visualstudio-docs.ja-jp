# C1255
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C1255: プラグインの PCH データに不正な期間があります。  
  
 コード分析ツールは、ツールに内部エラーが発生した場合、分析するコードでは、この警告を、報告します。
