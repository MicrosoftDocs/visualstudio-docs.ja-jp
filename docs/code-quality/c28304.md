# <a name="c28304"></a>C28304
警告 C28304: 参照C++パラメーター < parameter_name > に対して、正しく配置されていない `_Notref_` 演算子が \<token > に適用されました。

 @No__t_0 演算子は、参照パラメーターを含むC++特殊な状況でのみ使用し、システム指定のマクロでのみ使用する必要があります。 直後に `_Deref_` 演算子または機能的な注釈が続く必要があります。
