> [**新しいプロジェクトの作成**] ダイアログの上部にある [検索] テキストボックスに「 **SharePoint** 」と入力して、テンプレートを検索することもできます。 また、[プロジェクトの種類] ドロップダウンボックスで [Office] を選択して、Office および SharePoint のテンプレートのみを表示するようにテンプレートの一覧をフィルター選択することもできます。 詳細については、「 [Visual Studio での新しいプロジェクトの作成](../../ide/create-new-project.md)」を参照してください。
