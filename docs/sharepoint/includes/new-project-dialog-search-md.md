>[!NOTE]
> **[新しいプロジェクトの作成]** ダイアログの上部にある [検索] テキスト ボックスに「**SharePoint**」と入力して、テンプレートを検索することもできます。 また、[プロジェクトの種類] ドロップダウン ボックスで [Office] を選択して、Office および SharePoint のテンプレートのみを表示するようにテンプレートの一覧をフィルターすることもできます。 詳細については、「[Visual Studio で新しいプロジェクトを作成する](../../ide/create-new-project.md)」を参照してください。
