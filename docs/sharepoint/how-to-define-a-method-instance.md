# <a name="how-to-define-a-method-instance"></a>方法: メソッド インスタンスを定義する
  モデルでは、すべてのメソッドの少なくとも 1 つのメソッドのインスタンスを定義する必要があります。  
  
 使用してメソッド インスタンスの追加、 **BDC メソッドの詳細**ウィンドウです。 Visual Studio が追加メソッドのインスタンスを追加すると、`<MethodInstance>`要素をモデル プロジェクト内のファイル、XML にします。 属性の詳細については、`<MethodInstance>`要素を参照してください[MethodInstance](http://go.microsoft.com/fwlink/?LinkID=169282)です。  
  
### <a name="to-define-a-method-instance"></a>メソッド インスタンスを定義するには  
  
1.  **BDC メソッドの詳細**ウィンドウで、メソッドのノードを展開し、展開、**インスタンス**ノード。  
  
2.  **メソッド インスタンスの追加**一覧で、選択**Finder インスタンスの作成**です。  
  
     新しいメソッド インスタンスの下に表示されます、**インスタンス**ノード。  
  
3.  メニュー バーで、次のように選択します。**ビュー**、選択**プロパティ ウィンドウ**します。  
  
4.  **プロパティ**ウィンドウ、メソッドのインスタンスのプロパティを設定します。 各プロパティの詳細については、次を参照してください。 [MethodInstance](http://go.microsoft.com/fwlink/?LinkID=169282)です。
