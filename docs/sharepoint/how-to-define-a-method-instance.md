# 方法: メソッド インスタンスを定義する
  モデルのどのメソッドでも、1 つ以上のメソッド インスタンスを定義する必要があります。  
  
 **\[BDC メソッドの詳細\]** ウィンドウを使用して、メソッド インスタンスを追加します。  メソッド インスタンスを追加すると、`<MethodInstance>` 要素がプロジェクト内のモデル ファイルの XML に追加されます。  `<MethodInstance>` の要素の属性の詳細については、参照します [MethodInstance](http://go.microsoft.com/fwlink/?LinkID=169282)。  
  
### メソッド インスタンスを定義するには  
  
1.  **\[BDC メソッドの詳細\]** ウィンドウで、メソッドのノードを展開し、**\[インスタンス\]** ノードを展開します。  
  
2.  **\[メソッド インスタンスの追加\]** の一覧で、**\[Finder インスタンスの作成\]** をクリックします。  
  
     **\[インスタンス\]** ノードの下に新しいメソッド インスタンスが表示されます。  
  
3.  メニュー バーで、**\[表示\]** をクリックします、**\[プロパティ ウィンドウ\]** をクリックします。  
  
4.  **プロパティ** ウィンドウで、メソッド インスタンスのプロパティを設定します。  各プロパティについての詳細については、参照します [MethodInstance](http://go.microsoft.com/fwlink/?LinkID=169282)。
