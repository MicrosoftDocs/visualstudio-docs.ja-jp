# Programming Concepts and Features for SharePoint Tools Extensions
  Visual Studio の SharePoint ツールに関連する重要なプログラミングの概念と機能がいくつかあります。  これらの概念と機能は、ほとんどの SharePoint ツールの拡張機能において重要な役割を果たします。  
  
## このセクションの内容  
 [Calling into the SharePoint Object Models](../sharepoint/calling-into-the-sharepoint-object-models.md)  
 SharePoint ツール拡張機能から、SharePoint クライアントと SharePoint サーバー オブジェクト モデルがどのように使用されるかを比較し、*SharePoint コマンド*を作成して SharePoint サーバー オブジェクト モデルを呼び出す方法について説明します。  
  
 [Using the SharePoint Project Service](../sharepoint/using-the-sharepoint-project-service.md)  
 SharePoint プロジェクト サービスが SharePoint ツール拡張機能に提供する機能について説明し、任意の種類の拡張機能のサービスを取得する手順を示します。  
  
 [Associating Custom Data with SharePoint Tools Extensions](../sharepoint/associating-custom-data-with-sharepoint-tools-extensions.md)  
 カスタム データを SharePoint ツールの拡張機能に含まれる特定の種類のオブジェクトに関連付ける方法について説明します。
