# 方法: SharePoint ソリューションにフィーチャーを追加する
  SharePoint ソリューションを作成すると、フィーチャーとパッケージがそれぞれ 1 つずつソリューションに追加されます。  フィーチャーの項目をさらに追加して、複数の異なる種類の SharePoint ファイルをまとめることができます。  
  
### ソリューション エクスプローラーでフィーチャーを追加するには  
  
1.  **\[ソリューション エクスプローラー\]** で、**\[フィーチャー\]** のショートカット メニューを開き、**\[フィーチャーの追加\(F\)\]** をクリックします。
