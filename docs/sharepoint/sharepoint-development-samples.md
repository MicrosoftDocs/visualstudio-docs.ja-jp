# <a name="sharepoint-development-samples"></a>SharePoint 開発のサンプル
  Visual Studio を使用して [MSDN コード ギャラリー](https://code.msdn.microsoft.com/)から SharePoint アプリケーションのサンプルを、ダウンロードしてインストールできます。

 各サンプルを個別にダウンロードできます。 サンプル パックをダウンロードすることもできます。サンプル パックには、関連性があり、テクノロジまたは記事を同じくするサンプルが含まれています。 ダウンロードしたサンプルについてソース コードの変更が発行された場合は、そのことが通知されます。
