# <a name="sharepoint-development-samples"></a>SharePoint 開発のサンプル
  Visual Studio を使用して、 [MSDN コードギャラリー](https://code.msdn.microsoft.com/)から SharePoint アプリケーションのサンプルをダウンロードしてインストールすることができます。

 各サンプルを個別にダウンロードできます。 また、テクノロジや記事を共有する関連サンプルを含むサンプルパックをダウンロードすることもできます。 ダウンロードしたサンプルについてソース コードの変更が発行された場合は、そのことが通知されます。
