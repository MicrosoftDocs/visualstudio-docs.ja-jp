# <a name="how-to-configure-workflow-project-properties-legacy"></a>方法: ワークフロー プロジェクトのプロパティを構成する (レガシ)
ワークフロー プロジェクトに関連したプロパティを表示または変更するには、次の手順に従います。  
  
### <a name="to-view-or-modify-workflow-project-properties"></a>ワークフロー プロジェクトのプロパティを表示または変更するには  
  
1.  場合**ソリューション エクスプ ローラー**ウィンドウが表示されている、select**ソリューション エクスプ ローラー**上、**ビュー**メニュー。  
  
2.  **ソリューション エクスプローラー**でプロジェクトを選択します。  
  
3.  場合**プロパティ**ウィンドウが表示されている、select**プロパティ ウィンドウ**上、**ビュー**メニュー。  
  
4.  プロジェクトで**プロパティ** ウィンドウを表示および変更するプロパティを変更します。
