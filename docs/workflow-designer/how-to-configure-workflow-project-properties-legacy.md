# 方法: ワークフロー プロジェクトのプロパティを構成する (レガシ)
ワークフロー プロジェクトに関連したプロパティを表示または変更するには、次の手順に従います。  
  
### ワークフロー プロジェクトのプロパティを表示または変更するには  
  
1.  **ソリューション エクスプローラー** ウィンドウが表示されない場合は、**\[表示\]** メニューの **\[ソリューション エクスプローラー\]** をクリックします。  
  
2.  **ソリューション エクスプローラー**でプロジェクトを選択します。  
  
3.  **\[プロパティ\]** ウィンドウが表示されていない場合は、**\[表示\]** メニューの **\[プロパティ ウィンドウ\]** をクリックします。  
  
4.  プロジェクトの **\[プロパティ\]** ウィンドウで、いずれかのプロパティを表示して、必要に応じて変更します。
