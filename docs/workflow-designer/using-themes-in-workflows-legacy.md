# ワークフロー内でのテーマの使用 (レガシ)
このトピックでは、[!INCLUDE[netfx35_long](../workflow-designer/includes/netfx35_long_md.md)] または [!INCLUDE[vstecwinfx](../workflow-designer/includes/vstecwinfx_md.md)] を対象とする従来の [!INCLUDE[wfd1](../workflow-designer/includes/wfd1_md.md)]を使用する際に、[!INCLUDE[wf](../workflow-designer/includes/wf_md.md)] のテーマを使用する方法について説明します。  
  
 [!INCLUDE[wf2](../workflow-designer/includes/wf2_md.md)] のテーマを使用すると、従来のデザイナーでワークフローの外観をカスタマイズできます。テーマは、ワークフローの背景色、前景色、スタイル、アイコン、その他のビジュアル要素を定義します。テーマを保存し、他のワークフローで再利用できます。  
  
 テーマを作成および編集するには、[\[テーマの構成\] ダイアログ ボックス \(レガシ\)](../Topic/Theme%20Configuration%20Dialog%20Box%20\(Legacy\).md)を使用します。このダイアログ ボックスを開くには、**\[ワークフロー\]** メニューの **\[新しいテーマの作成\]** をクリックするか、ワークフロー デザイン サーフェイスを右クリックして **\[新しいテーマの作成\]** をクリックします。  
  
 [\[ヘルプ\] \(\[オプション\] ダイアログ ボックス \- ワークフロー デザイナー\) \(レガシ\)](../Topic/Themes,%20Workflow%20Designer,%20Options%20Dialog%20Box%20\(Legacy\).md)を使用すると、既定の保存されているカスタム テーマを選択できます。このダイアログ ボックスを開くには、**\[ワークフロー\]** メニューの **\[カスタム テーマの選択\]** をクリックした後 \(またはワークフロー デザイン サーフェイスを右クリックして **\[カスタム テーマの選択\]** をクリックした後\)、**\[ワークフロー デザイナー\]** オプションをクリックします。適切なテーマをクリックして、**\[OK\]** をクリックします。
