# <a name="debugging-workflows-with-the-workflow-designer"></a>ワークフロー デザイナーを使用したワークフローのデバッグ

Windows ワークフロー デザイナーには、ワークフローとカスタム アクティビティをデバッグする機能が用意されています。 プロセスと動作は、Visual Studio 2010 の既定のデバッガーに似ています。 このセクションの各トピックでは、従来のデバッグ手順に加えて、XAML および式のデバッグを行う方法について説明します。
