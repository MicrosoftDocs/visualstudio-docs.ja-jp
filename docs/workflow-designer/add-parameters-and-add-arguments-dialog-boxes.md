# [パラメーターの追加] ダイアログ ボックスと [引数の追加] ダイアログ ボックス
次の表では、<xref:System.Activities.Statements.InvokeMethod> デザイナーで使用される **\[パラメーターの追加\]** ダイアログ ボックスのユーザー インターフェイス \(UI\) 要素について説明します。  
  
|||  
|-|-|  
|**方向**|パラメーターがメソッドに入っていくデータ フロー、メソッドから出てくるデータ フロー、またはその両方のいずれを表すのかを指定します。|  
|**型**|新しいパラメーターの型の名前。|  
|**値**|新しいパラメーターに既定値を割り当てるために使用される VB の式。|  
  
 次の表では、<xref:System.Activities.Statements.InvokeDelegate> デザイナーで使用される **\[引数の追加\]** ダイアログ ボックスのユーザー インターフェイス \(UI\) 要素について説明します。  
  
|||  
|-|-|  
|**名前**|引数の名前。|  
|**方向**|引数がデリゲートに入っていくデータ フロー、デリゲートから出てくるデータ フロー、またはその両方のいずれを表すのかを指定します。|  
|**型**|新しい引数の型の名前。|  
|**値**|デリゲート引数のこのインスタンスに使用する値。|
