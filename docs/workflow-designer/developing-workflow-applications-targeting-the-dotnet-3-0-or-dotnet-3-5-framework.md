# <a name="developing-workflow-applications-targeting-the-net-30-or-net-35-framework"></a>.NET 3.0 または .NET 3.5 Framework を対象とするワークフロー アプリケーションの開発

Visual Studio 2010 は、.NET Framework 4 を対象とする Windows Workflow Foundation (WF) アプリケーションを作成するための Windows ワークフロー デザイナーを提供します。 デザイナーは、グラフィカルに作成してワークフロー アプリケーションのデバッグを使い慣れた Visual Studio ユーザー インターフェイスを使用します。

Visual Studio 2010 は、.NET Framework version 3.5、または、WinFX をターゲットに使用できる従来のワークフロー デザイナーにもアクセスを提供します。 ここでは、この従来のデザイナーの使用方法について説明します。
