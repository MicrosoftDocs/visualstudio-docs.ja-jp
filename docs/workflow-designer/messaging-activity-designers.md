# <a name="messaging-activity-designers"></a>メッセージング アクティビティ デザイナー

メッセージングアクティビティデザイナーは、Windows Workflow Foundation (WF) アプリケーション内から Windows Communication Foundation (WCF) メッセージを送受信するメッセージングアクティビティを作成および構成するために使用されます。 .NET Framework 4 で5つのメッセージングアクティビティが導入されました。 ワークフローデザイナーには、ワークフロー内のメッセージングを管理するための2つのテンプレートデザイナーが用意されています。

このセクションに含まれるトピックと、次の表に示すトピックでは、ワークフローデザイナーアクティビティデザイナーとテンプレートデザイナーの使用方法について説明します。

- <xref:System.Activities.Activity>

- <xref:System.ServiceModel.Activities.CorrelationScope>

- <xref:System.ServiceModel.Activities.Receive>

- <xref:System.ServiceModel.Activities.Send>

- <xref:System.ServiceModel.Activities.ReceiveReply>

- <xref:System.ServiceModel.Activities.SendReply>

- <xref:System.ServiceModel.Activities.TransactedReceiveScope>
