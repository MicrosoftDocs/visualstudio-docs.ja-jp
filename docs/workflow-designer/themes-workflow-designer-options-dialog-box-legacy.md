# [ヘルプ] ([オプション] ダイアログ ボックス - ワークフロー デザイナー) (レガシ)
このトピックでは、従来の [!INCLUDE[wfd1](../workflow-designer/includes/wfd1_md.md)]の **\[ヘルプ\]** \(\[オプション\] ダイアログ ボックス \- ワークフロー デザイナー\) の使用方法について説明します。[!INCLUDE[netfx35_long](../workflow-designer/includes/netfx35_long_md.md)] または [!INCLUDE[vstecwinfx](../workflow-designer/includes/vstecwinfx_md.md)] を対象とする必要がある場合は、従来の[!INCLUDE[wfd2](../workflow-designer/includes/wfd2_md.md)]を使用します。  
  
 テーマは、ワークフローの背景色、前景色、スタイル、アイコン、その他のビジュアル要素を定義します。テーマを保存し、他のワークフローで再利用できます。**\[ヘルプ\]** \(\[オプション\] ダイアログ ボックス \- ワークフロー デザイナ\) を使用すると、既定の保存されているカスタム テーマを選択できます。このダイアログ ボックスを表示するには、**\[ワークフロー\]** メニューの **\[カスタム テーマの選択\]** をクリックします。または、ワークフロー デザイン サーフェイスを右クリックして **\[カスタム テーマの選択\]** をクリックします。  
  
 次の表は、**\[ヘルプ\]** \(\[オプション\] ダイアログ ボックス \- ワークフロー デザイナ\) のユーザー インターフェイス \(UI\) 要素を示しています。  
  
|UI 要素|説明|  
|-----------|--------|  
|**新規作成**|クリックすると、[\[テーマの構成\] ダイアログ ボックス \(レガシ\)](../Topic/Theme%20Configuration%20Dialog%20Box%20\(Legacy\).md)が表示され、新しいテーマを作成できます。|  
|**変更**|クリックすると、[\[テーマの構成\] ダイアログ ボックス \(レガシ\)](../Topic/Theme%20Configuration%20Dialog%20Box%20\(Legacy\).md)が表示され、選択したテーマを変更できます。|  
|**参照**|ワークフロー テーマ ファイル \(ファイル名の拡張子 .wtm\) を参照するには、これをクリックします。|  
|**テーマの選択**|テーマ リストのテーマ名を示す列見出し。リストからテーマを選択し、**\[変更\]** をクリックして編集するか、**\[OK\]** をクリックして選択されたテーマをワークフローに使用します。|  
|**説明**|テーマ リストのテーマの説明を示す列見出し。|  
|**場所**|テーマ リストのテーマの場所を示す列見出し。|
