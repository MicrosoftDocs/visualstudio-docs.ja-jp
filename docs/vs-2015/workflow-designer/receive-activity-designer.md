# <a name="receive-activity-designer"></a>Receive アクティビティ デザイナー
**Receive**アクティビティ デザイナーは、アクティビティを作成および<xref:System.ServiceModel.Activities.Receive>構成するために使用します。 <xref:System.ServiceModel.Activities.Receive> アクティビティは、メッセージ (<xref:System.ServiceModel.Channels.Message>、<xref:System.IO.Stream>、<xref:System.Xml.Linq.XElement> などの組み込みの型、アプリケーション定義のデータ コントラクト、メッセージ コントラクト、またはシリアル化可能な XML クラス) を受信するアクティビティです。

## <a name="the-receive-activity"></a>Receive アクティビティ
 <xref:System.ServiceModel.Activities.Receive> アクティビティでは、使用されている受信コンテンツの型に応じて、単一または複数のアイテムを受信できます。 <xref:System.ServiceModel.Activities.SendReply> アクティビティは、サービスでの要求/応答メッセージ交換パターンの一部としてメッセージを受信する <xref:System.ServiceModel.Activities.Receive> アクティビティにバインドできます。

### <a name="using-the-receive-activity-designer"></a>Receive アクティビティ デザイナーの使用
 **受信**アクティビティ デザイナーは**View**、**ツールボックス**の**メッセージ**カテゴリにあります。 **Toolbox** [!INCLUDE[wfd2](../includes/wfd2-md.md)] **Toolbar**

 **受信**アクティビティ デザイナーは **、ツールボックス**からドラッグして、通常はアクティビティが[!INCLUDE[wfd2](../includes/wfd2-md.md)]配置されている場所にドロップできます。 この操作により、Receive という既定の <xref:System.ServiceModel.Activities.Receive> を持つ <xref:System.Activities.Activity.DisplayName%2A> アクティビティが作成されます。 <xref:System.Activities.Activity.DisplayName%2A>**は、受信**アクティビティ デザイナーのヘッダーまたはプロパティ グリッドの **[表示名**] ボックスで編集できます。

 アクティビティを<xref:System.ServiceModel.Activities.SendReply>作成して選択した<xref:System.ServiceModel.Activities.Receive>アクティビティにバインドするには、**受信**アクティビティ デザイナーを右クリックし、コンテキスト メニューの [SendReply の作成] をクリックすると **、Receive**デザイナの下に**SendReplyToReceive**デザイナーが表示されます。 **Create SendReply** <xref:System.ServiceModel.Activities.SendReply> アクティビティは、サービスでの要求/応答メッセージ交換パターンの一部として応答メッセージを送信するアクティビティであり、 これは、**送信返信受信**デザイナーで構成できます。

 または、**ツールボックス**の<xref:System.ServiceModel.Activities.Receive>**メッセージング**カテゴリの**ReceiveAndSendReply**テンプレート デザイナーを使用して、構成済みアクティビティと<xref:System.ServiceModel.Activities.SendReply>アクティビティのペアを作成できます。 [!INCLUDE[crabout](../includes/crabout-md.md)]**テンプレートの****使用**については、[受信と送信の返信のトピック](../workflow-designer/receiveandsendreply-template-designer.md)を参照してください。
