# <a name="themes-workflow-designer-options-dialog-box-legacy"></a>[ヘルプ] ([オプション] ダイアログ ボックス - ワークフロー デザイナー) (レガシ)
このトピックでは、従来のの [オプション] ダイアログボックスの [ **テーマ] ワークフローデザイナー** 使用する方法について説明し [!INCLUDE[wfd1](../includes/wfd1-md.md)] ます。 [!INCLUDE[wfd2](../includes/wfd2-md.md)] または [!INCLUDE[netfx35_long](../includes/netfx35-long-md.md)] を対象とする必要がある場合は、従来の[!INCLUDE[vstecwinfx](../includes/vstecwinfx-md.md)]を使用します。

 テーマは、ワークフローの背景色、前景色、スタイル、アイコン、その他のビジュアル要素を定義します。 テーマを保存し、他のワークフローで再利用できます。 [ **テーマ] ([オプション** ] ダイアログボックスワークフローデザイナー) を使用して、[既定] および [保存されたカスタムテーマ] を選択できます。 このダイアログボックスを開くには、[**ワークフロー** ] メニューの [**カスタムテーマの選択**] をクリックします。 または、ワークフローデザインサーフェイスを右クリックし、[ **カスタムテーマの選択**] をクリックします。

 次の表は、[テーマ] ([オプション] ダイアログボックス- **ワークフローデザイナー)** のユーザーインターフェイス (UI) 要素を示しています。

|UI 要素|説明|
|----------------|-----------------|
|**[新規作成]**|クリックすると、[ [テーマの構成] ダイアログボックス (レガシ)](../workflow-designer/theme-configuration-dialog-box-legacy.md) が開き、新しいテーマを作成できます。|
|**変更...**|クリックすると、[ [テーマの構成] ダイアログボックス (レガシ)](../workflow-designer/theme-configuration-dialog-box-legacy.md) が開き、選択したテーマを変更できます。|
|**[参照]**|ワークフロー テーマ ファイル (ファイル名の拡張子 .wtm) を参照するには、これをクリックします。|
|**テーマの選択**|テーマ リストのテーマ名を示す列見出し。 一覧からテーマを選択し、[ **変更** ] をクリックして編集するか、[ **OK** ] をクリックして選択したテーマをワークフローで使用します。|
|**説明**|テーマ リストのテーマの説明を示す列見出し。|
|**場所**|テーマ リストのテーマの場所を示す列見出し。|
