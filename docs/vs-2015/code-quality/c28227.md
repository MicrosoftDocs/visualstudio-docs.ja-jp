# <a name="c28227"></a>C28227
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

このトピックの最新バージョンをご覧[C28227](https://docs.microsoft.com/visualstudio/code-quality/c28227)します。  
  
警告 C28227: 注釈こともできません、PrimOp (前の宣言を参照してください)  
  
 この警告は、名前付きの注釈が、通常の注釈として宣言されていることも、PrimOp に宣言されていたことを示します。 注釈は標準のマクロが使用されている場合があります。この警告は、一般的な使用で目にする必要はありません。
