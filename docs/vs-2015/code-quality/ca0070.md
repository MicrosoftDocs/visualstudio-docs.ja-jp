# <a name="ca0070"></a>CA0070

このエラーは、次の条件の少なくとも 1 つが true とコード分析は、アセンブリを分析し、アクセス許可属性が発生したときに発生します。

-   コード分析は、属性を含むアセンブリを見つけることができません。

-   コード分析には、アセンブリを読み込むためのアクセス許可がありません。

-   コード分析によって読み込まれるアセンブリに属性が含まれていません。

ほとんどの場合、コード分析ツールの最新バージョンがあることを確認することにより、問題を解決できます。
