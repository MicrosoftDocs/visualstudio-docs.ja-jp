# <a name="c28020"></a>C28020
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28020: 式\<expr > この呼び出しで true でないです。  
  
 この警告が報告されたときに、 \_Satisfies\_表示されている式が true ではありません。 頻繁に無効なパラメーターを示します。  
  
 関数の宣言でこのような場合、注釈は、不可能な状態を示します。
