# <a name="c28020"></a>C28020
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28020: \<expr> この呼び出しでは式が true ではありません  
  
 この警告は、 \_ \_ 表示されている式が true でない場合に報告されます。 多くの場合、パラメーターが正しくないことを示します。  
  
 これが関数宣言で発生した場合、注釈は不可能な状態を示します。
