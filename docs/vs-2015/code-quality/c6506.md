# <a name="c6506"></a>C6506
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6506: 無効な注釈:\<名 > プロパティは、ポインターまたは配列型の値でのみ使用します。  
  
 この警告は、ポインターまたは配列型以外の型で、プロパティを使用することを示します。 アクセス、Tainted、および有効なプロパティは、すべてのデータ型で使用できます。 ValidBytesConst、ValidElementsConst、ElementSize、NullTerminted などの他のプロパティは、ポインター、メンバー、または配列型へのポインターをサポートします。 プロパティとサポートされているデータ型の完全な一覧を参照してください。[注釈プロパティ](http://msdn.microsoft.com/f77b4370-6bda-4294-bd2a-e7d0df182a3d)します。  
  
## <a name="example"></a>例  
 この警告が発生するコード例を次に示します。  
  
```cpp  
#include<sal.h>  
void f(_Out_ char c)  
{  
    c = 'd';  
}  
```  
  
 この警告を解決するには、次のサンプル コードに示すようにポインターまたは配列型を使用します。  
  
```cpp  
#include<sal.h>  
void f(_Out_ char *c)  
{  
    *c = 'd';  
}  
```
