[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

最新のソフトウェア アプリケーションの複雑さが増すには、信頼性が高く、保守しやすいコードを行う苦労も増加します。 近年、多くのソフトウェアのメジャー、コード メトリックスと呼ばれる開発され開発者の作業のやり直しや増加テスト、コードが必要なを理解することができます。
