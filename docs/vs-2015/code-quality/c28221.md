# <a name="c28221"></a>C28221
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28221: 注釈のパラメーターには文字列式が必要です  
  
 この警告は、注釈のパラメーターが文字列であることが予期されており、他の型が検出されたことを示します。 これは通常、正しくコーディングされていない注釈マクロを示しており、通常の使用時に表示されるとは限りません。
