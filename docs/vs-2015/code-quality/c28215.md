警告 C28215: typefix は、既に typefix のあるパラメーターには適用できません  
  
 適用、`__typefix`その注釈が既にあるパラメーターに注釈は、エラーです。 `__typefix`注釈は、いくつかの特殊なケースでのみ使用して、この警告は、通常の使用で確認する必要はありません。
