# <a name="c6995"></a>C6995
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6995: XML ログ ファイルを保存できませんでした  
  
 この警告は、コード分析ツールでコード分析の出力は、欠陥のログを作成できないことを示します。  
  
 このエラーは、ディスク エラーを示している可能性があります。 または指定したディレクトリにファイルを作成するアクセス許可を持っていないことを示します。
