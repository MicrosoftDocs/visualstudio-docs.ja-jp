# <a name="c28283"></a>C28283
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28283: シンボルについて、指定したサイズはまだサポートされていません  
  
 この警告は、サイズ指定 "sentinel" 注釈が0以外の値を受け取ったことを示します。 基本的に、呼び出し元は、文字列がバイナリ0以外の文字で終了したと言っていました。
