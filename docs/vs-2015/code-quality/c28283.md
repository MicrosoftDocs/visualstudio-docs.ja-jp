# <a name="c28283"></a>C28283
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28283: シンボルについて、指定したサイズはまだサポートされていません  
  
 警告は、サイズの仕様"sentinel"注釈に 0 以外の値が受信したことを示します。 基本的には、呼び出し元は、文字列はバイナリのゼロ以外の文字で終了されていることをしようとしました。
