# <a name="c6216"></a>C6216
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6216: 意味の異なる整数型の間でのコンパイラ挿入されたキャストです: ブール型から HRESULT  
  
 This warning indicates that a Boolean is being used as an `HRESULT` without being explicitly cast. ブール型は 0 以外の値で成功を示します。`S_OK` の成功値 (`HRESULT`) は値 0 によって示されます。 The typical failure value for functions that return a Boolean false is a success status when it is tested as an `HRESULT`. This is likely to lead to incorrect results.  
  
## <a name="example"></a>例  
 この警告が発生するコード例を次に示します。  
  
```  
#include <windows.h>  
BOOL IsEqual(REFGUID, REFGUID);  
  
HRESULT f( REFGUID riid1, REFGUID riid2 )  
{  
  // code ...  
  return IsEqual(riid1, riid2);    
}  
```  
  
 この警告を解決するには、次のコードを使用します。  
  
```  
#include <windows.h>  
BOOL IsEqual(REFGUID, REFGUID);  
  
HRESULT f( REFGUID riid1, REFGUID riid2 )  
{  
  if (IsEqual(riid1, riid2) == TRUE)  
  {  
    // code ...  
    return S_OK;  
  }  
  else  
  {  
    // code ...  
    return E_FAIL;  
  }  
}  
```  
  
 この警告では、`SCODE` 型は `HRESULT` と等価です。  
  
 For more information, see [SUCCEEDED Macro](https://go.microsoft.com/fwlink/?LinkId=92738) and [FAILED Macro](https://go.microsoft.com/fwlink/?LinkId=180875).
