# <a name="c28351"></a>C28351
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

warning C28351: 注釈 \<注釈 > は、動的な値 (変数) が条件内で使用できない場所を示します。  
  
 この警告は、C/C++型の検査対象となる注釈が適用されたときに報告されます。  
  
 `_When_` 内の式は、定数に評価される必要があります。 `_When_` は無視されます。
