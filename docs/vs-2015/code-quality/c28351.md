このトピックの最新バージョンをご覧[C28351](https://docs.microsoft.com/visualstudio/code-quality/c28351)します。  
  
警告 C28351: 注釈\<注釈 > 条件で動的な値 (変数) を使用できませんについて説明します。  
  
 C と C++ の型は検査されている注釈が適用されるときに、この警告が報告されます。  
  
 内の式、`_When_`定数に評価する必要があります。 `_When_`は無視されます。
