# <a name="c6993"></a>C6993
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 6993: コード分析は OpenMP コンストラクトを無視し、シングルスレッド コードを解析しています  
  
 この警告は、コード アナライザーが分析できない Open MP プラグマを検出したことを示します。
