# <a name="c6530"></a>C6530
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 6530: 認識されない書式文字列スタイル\<名 >  
  
> [!NOTE]
>  この警告は、ソース コード注釈言語 (SAL) の非推奨のバージョンを使用しているコードでのみ発生します。 最新バージョンの SAL を使用するコードを移植することをお勧めします。 詳細については、[C と C++ コードの欠陥を削減する SAL 注釈を使って](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md)を参照してください。  
  
 この警告は、FormatString プロパティが scanf または printf 以外の値を使用していることを示します。 この警告を解決するには、コードを確認し、スタイル プロパティの有効な値を使用します。  
  
## <a name="example"></a>例  
 次のコードでこの警告を生成する、`Style`プロパティ。  
  
```  
// C  
#include <CodeAnalysis\SourceAnnotations.h>  
void f([SA_FormatString(Style="printfd")] char *px);   
  
// C++  
#include <CodeAnalysis\SourceAnnotations.h>  
using namespace vc_attributes;  
void f([FormatString(Style="printfd")] char *px);  
  
```  
  
 この警告を解決するには、次のコードに示すように、有効なスタイルを使用します。  
  
```  
// C  
#include <CodeAnalysis\SourceAnnotations.h>  
void f([SA_FormatString(Style="printf")] char *px);   
  
// C++  
#include <CodeAnalysis\SourceAnnotations.h>  
using namespace vc_attributes;  
void f([FormatString(Style="printf")] char *px);   
```
