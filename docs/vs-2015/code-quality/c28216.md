# <a name="c28216"></a>C28216
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28216: \_Check_return\_注釈は、特定の関数のパラメーターの後の状態にのみ適用されます。  
  
 `_Check_return_`コンテキストで注釈が以外に適用`__post`; 必要とする可能性を`__post`(または`__drv_out`) 修飾子、またはその可能性があります配置正常。
