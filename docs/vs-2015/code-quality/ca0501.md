# <a name="ca0501"></a>CA0501

出力レポートを読み取れません。

これは、次のいずれかの理由により、出力レポートに書き込めない場合に発生します。

- ディスクがいっぱいです

- ディレクトリに書き込むためのアクセス許可がありません

- ディレクトリが存在しません
