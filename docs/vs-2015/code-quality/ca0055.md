# <a name="ca0055"></a>CA0055

ファイルを読み込めませんでした: \<path> 。

コード分析ツールは、分析用に指定されたターゲットファイルを読み込むことができませんでした。 このエラーは、ファイルが見つからなかった場合、またはファイルへのアクセスが拒否された場合に発生します。 通常、このエラーは、Visual Studio で入力アセンブリプロパティをオーバーライドすることをユーザーが選択した場合、または fxcopcmd.exe 手動で呼び出された場合に発生します。
