# <a name="c6997"></a>C6997
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 6997: この場所の注釈は意味がなく、無視されます。  
  
 注釈を `extern "C" {...}` に適用できません。 特定のオブジェクトに注釈を適用してください。
