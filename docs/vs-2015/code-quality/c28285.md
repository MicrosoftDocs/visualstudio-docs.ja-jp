# <a name="c28285"></a>C28285
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28285: 関数について、パラメーターに構文エラーがあります  
  
 コード分析ツールでは、モデル ファイルの考えられるエラーが発生した場合に、この警告を報告します。 このようなエラーもいくつかのソース ファイルのエラーがあります。
