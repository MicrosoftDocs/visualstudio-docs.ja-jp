# <a name="ca0503"></a>CA0503

プロパティが非推奨とされます。 置き換える方のプロパティを使用します。

非推奨のコード分析プロパティには、ビルドまたはプロジェクトの構成ファイル内の値が含まれている場合は、警告 CA0503 が生成されます。 この警告を解決するには、優先する指定したプロパティを使用します。
