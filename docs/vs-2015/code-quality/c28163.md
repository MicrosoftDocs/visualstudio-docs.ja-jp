警告 C28163: 関数を try/except ブロック内から呼び出さないでください  
  
 関数が型の場合に、この警告は報告されますですることはありません囲む必要があります、`try/except`ブロックがある、`try/except`ブロックします。 コード分析ツールが呼び出される関数が内に少なくとも 1 つのパスが見つかりません、`try/except`ブロックします。
