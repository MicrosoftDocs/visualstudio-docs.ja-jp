# <a name="c28222"></a>C28222
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 28222: \_ \_ \_ \_ 注釈には Yes、no、または \_ \_ 必要な可能性があります  
  
 この警告は、注釈のパラメーターが、、またはのいずれかのシンボルであることが想定されて `_Yes_` `_No_` おり、他のシンボルが検出されたことを示し `_Maybe_` ます。 これは通常、誤ってコード化された注釈マクロであることを示します。
