警告 28222: (_y)\_、(_n)\_、または _Maybe\_の注釈が必要です  
  
 この警告は、注釈のパラメーター シンボルのいずれかを指定する必要のあることを示します`_Yes_`、 `_No_`、または`_Maybe_`、および他の記号が発生しました。 これは通常、正しくコード化された注釈マクロを示します。
