# <a name="ca0056"></a>CA0056

プロジェクトファイルの分析ツールのバージョンが正しくありません。

この警告は、コード分析ツールのバージョンが正しくないプロジェクトファイルで FxCopCmd が手動で呼び出された場合に発生します。
