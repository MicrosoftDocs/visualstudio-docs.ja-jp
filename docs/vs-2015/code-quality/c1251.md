# <a name="c1251"></a>C1251
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

C1251 を警告します。モデルを読み込めません。  
  
 コード分析ツールは、分析対象のコードではなく、モデル ファイルの内部エラーがある場合に、この警告を報告します。
