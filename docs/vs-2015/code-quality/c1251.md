# <a name="c1251"></a>C1251
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C1251: モデルを読み込めません。  
  
 コード分析ツールは、分析されているコードではなく、モデルファイルに内部エラーが発生した場合に、この警告を報告します。
