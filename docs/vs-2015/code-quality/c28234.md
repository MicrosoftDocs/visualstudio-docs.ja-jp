# <a name="c28234"></a>C28234
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28234: \_ At \_ 式は現在の関数に適用されません  
  
 この警告は、式の値によって `_At_` ユーザー補助オブジェクトが識別されないことを示します。
