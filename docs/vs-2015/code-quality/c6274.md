# <a name="c6274"></a>C6274
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

このトピックの最新バージョンをご覧[C6274](https://docs.microsoft.com/visualstudio/code-quality/c6274)します。  
  
警告 C6274: 文字以外のパラメーターとして渡される\<数 > 文字が必要になるへの呼び出しで\<関数 >  
  
 この警告は、文字が必要である書式指定文字列を指定することを示します (たとえば、`%c`または`%C`仕様) など、float、string、または構造体の整数でない値が渡されるが、します。 この問題は、不正な出力が発生する可能性があります。  
  
## <a name="example"></a>例  
 この警告が発生するコード例を次に示します。  
  
```  
#include <stdio.h>  
#include <string.h>  
  
void f(char str[])  
{  
   char buff[5];  
  
   sprintf(buff,"%c",str);  
}  
```  
  
 この警告を解決するには、次のコードを使用します。  
  
```  
#include <stdio.h>  
#include <string.h>  
  
void f(char str[])  
{  
   char buff[5];  
  
   sprintf(buff,"%c",str[0]);  
}  
```  
  
 次の例では、安全な文字列操作関数は、 `sprintf_s`、この警告を解決します。  
  
```  
#include <stdio.h>  
#include <string.h>  
  
void f(char str[])  
{  
   char buff[5];  
  
   sprintf_s(buff,5,"%c", str[0]);  
}  
```
