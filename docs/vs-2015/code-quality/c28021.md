# <a name="c28021"></a>C28021

警告 C28021: パラメーター \<param > で注釈が付けられた\<anno > へのポインターにする必要があります  
  
 注釈が付けられたオブジェクトは、ポインター型ではない場合、この警告が報告されます。 この注釈は併用できません`void`または整数型。
