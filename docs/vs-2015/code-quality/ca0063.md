規則セット ファイルまたはその依存ルール セット ファイルのいずれかの読み込みに失敗しました。

指定したルール セットが見つかりませんでした。 または、ルール セットに含まれる規則セット ファイルの 1 つ見つかりませんでした。 すべての ruleset の規則セットに含まれているが、ディスク上に存在しを使用してプロジェクトに適切な規則セット ディレクトリが指定されていることを確認、 **CodeAnalysisRuleSetDirectories** MSBuild のプロパティ。

エラーをデバッグするには、テキスト エディターで、ルール セット ファイルを調べます。 ルール セット ファイルへのパスを検索するには、ソリューション エクスプ ローラーでプロジェクトを右クリックし、をクリックして**プロパティ**、 をクリックし、**コード分析**します。 ルールセット ファイルがで選択されていることを確認します**この規則セットを実行**します。 ルール セットへのパスは、[説明] フィールドに表示されます。

確認、**パス**属性の値はすべて、 **Include**要素。 含めるパスは、親/現在のルール セット ファイル、環境変数、および絶対パスへの相対パスを使用できます。 例えば:

```
<Include Path="%PUBLIC%\Documents\RuleSets\alldesign.ruleset" Action="Default" />
<Include Path="..\alldesign.ruleset" Action="Default" />
<Include Path="C:\Rulesets\alldesign.ruleset" Action="Default" />
```

これらのそれぞれのパスを含めるしがすべて有効であることを確認します。

場合によっては、ruleset の場所は、MSBuild プロパティに依存することができます。 MSBuild プロパティは、ルール セットから参照できません。 この問題を回避するには、追加の検索パスを指定、 **CodeAnalysisRuleSetDirectories** MSBuild のプロパティ。 このシナリオでの ruleset の名前のみを指定します、**パス**の属性、 **Include**要素。
