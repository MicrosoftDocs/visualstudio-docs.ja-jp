# <a name="c28280"></a>C28280
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28280: シンボルについて、'end' が見つかりましたが対応する 'begin' がありません  
  
 注釈の言語には、開始と終了がサポートされています (`{`と`}`C で)、構造とペアリングされている負荷が不均衡です。 マクロを使用する場合、このような状況を回避できます。
