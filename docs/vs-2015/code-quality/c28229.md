# <a name="c28229"></a>C28229
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28229: 注釈はパラメーターをサポートしません  
  
 この警告は、注釈がパラメーターを指定せずに宣言されている場合に、注釈がパラメーターを持つ使用されたことを示します。 注釈は標準のマクロが使用されている場合があります。この警告は、一般的な使用で目にする必要はありません。
