このトピックの最新バージョンをご覧[C28220](https://docs.microsoft.com/visualstudio/code-quality/c28220)します。  
  
警告 C28220: 注釈 (注釈のパラメーター) には整数式が必要です  
  
 この警告は、注釈のパラメーターの整数式のことが必要ですし、その他のいくつかの種類が見つかりましたことを示します。 これは通常、正しくコード化された注釈マクロを示します。
