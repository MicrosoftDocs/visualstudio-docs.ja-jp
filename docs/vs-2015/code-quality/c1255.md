# <a name="c1255"></a>C1255
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C1255: プラグインの PCH データが不適切な長さ。  
  
 コード分析ツールは、ツールで、分析対象のコードではなく、内部エラーがある場合に、この警告を報告します。
