# <a name="c1255"></a>C1255
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C1255: プラグインの PCH データの長さが正しくありません。  
  
 コード分析ツールは、分析されているコードではなく、ツールに内部エラーが発生したときにこの警告を報告します。
