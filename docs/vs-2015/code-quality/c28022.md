# <a name="c28022"></a>C28022
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28022: この関数の関数クラスは、 \<classlist1> \<classlist2> 定義に使用された typedef の関数クラスと一致しません。  
  
 この警告は、注釈にエラーがある場合に報告されます。 Typedef と関数自体の両方に注釈があり `_Function_class_` ますが、これらは一致しません。 両方が使用されている場合は、一致する必要があります。
