[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

このトピックの最新バージョンをご覧[C28225](https://docs.microsoft.com/visualstudio/code-quality/c28225)します。  
  
警告 C28225: 注釈に見つかった必須パラメーターの数が正しくありません  
  
 この警告は、名前付きの注釈が誤った数のパラメーターと共に使用されることを示します。 注釈は標準のマクロが使用されている場合があります。この警告は、一般的な使用で目にする必要はありません。
