# <a name="c28196"></a>C28196
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28196: 要件が満たされていません  (式は true に評価されません)  
  
 この警告は、分析される関数の関数プロトタイプにまたは `__notnull` `__null` `__drv_valueIs` パラメーターまたは戻り値があるに `_Out_` もかかわらず、返された値がその注釈と一致しないことを示します。
