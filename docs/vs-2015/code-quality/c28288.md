# <a name="c28288"></a>C28288
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

このトピックの最新バージョンをご覧[C28288](https://docs.microsoft.com/visualstudio/code-quality/c28288)します。  
  
警告 C28288: 関数について、_at 構文エラー\_() 注釈 (無効なパラメーター名)  
  
 コード分析ツールは、この警告を報告、 `SAL_at` (`__drv_at`) 注釈を使用し、現在のコンテキストでは、パラメーター式を解釈できません。 これには、スペルの正しくないパラメーターまたはメンバー名、または"return"または"this"キーワードのスペルミスを使用してが含まれます。
