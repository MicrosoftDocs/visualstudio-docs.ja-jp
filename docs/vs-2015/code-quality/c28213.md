# <a name="c28213"></a>C28213
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

warning C28213: `_Use_decl_annotations_` 注釈は、変更せずに前の宣言を参照するために使用する必要があります。 \<理由 >  
  
 この警告は、前の宣言が、必要な `_Use_decl_annotations_` 注釈を付けずに参照された場合に報告されます。
