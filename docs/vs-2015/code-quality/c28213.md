# <a name="c28213"></a>C28213
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28213:`_Use_decl_annotations_`先行する宣言を変更しない限り、参照に注釈を使用する必要があります。 \<理由 >  
  
 先行する宣言が参照されている場合、この警告は報告せず、必要な`_Use_decl_annotations_`注釈。
