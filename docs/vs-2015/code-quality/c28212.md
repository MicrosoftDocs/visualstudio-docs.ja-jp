[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

このトピックの最新バージョンをご覧[C28212](https://docs.microsoft.com/visualstudio/code-quality/c28212)します。  
  
警告 C28212: 注釈にはポインター式が必要です  
  
 この警告は、注釈 (not 関数の注釈が付けられた) を番号付きのパラメーターがポインターまたは配列型にする必要がありますが、その他の種類が検出されたことを示します。 注釈は、修正する必要があります。
