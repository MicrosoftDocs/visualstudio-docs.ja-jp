# <a name="c1257"></a>C1257
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 1257: プラグインの初期化に失敗します。  
  
 コード分析ツールは、この警告を報告します。この警告は、解析対象のコードではなく、プラグインに内部エラーが発生した場合に報告されます。
