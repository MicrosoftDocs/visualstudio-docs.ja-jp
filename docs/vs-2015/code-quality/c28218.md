# <a name="c28218"></a>C28218
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28218: 関数パラメーターについて、注釈のパラメーター数がファイルで検出されたものと一致しません  
  
 現在の行と、メッセージ内の行のコメントは、一貫性がありません。 注釈は標準のマクロが使用されている場合があります。この警告は、通常の使用で目にする必要はありません。
