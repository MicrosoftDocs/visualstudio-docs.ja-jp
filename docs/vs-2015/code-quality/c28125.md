# <a name="c28125"></a>C28125
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28125: 関数は try/except ブロック内から呼び出される必要があります  
  
 ドライバーは、 [ProbeForRead](https://msdn.microsoft.com/library/windows/hardware/ff559876.aspx)、 [ProbeForWrite](https://msdn.microsoft.com/library/windows/hardware/ff559879.aspx)、 [MmProbeAndLockPages](https://msdn.microsoft.com/library/windows/hardware/ff554664.aspx)などの try/except ブロック内から呼び出す必要がある関数を呼び出しています。  
  
## <a name="example"></a>例  
 次のコード例では、この警告が生成されます。  
  
```  
ProbeForRead(addr, len, 4);  
```  
  
 次のコード例では、この警告は生成されません。  
  
```  
  
__try  
{  
    ProbeForRead(addr, len, 4);  
}  
__except(EXCEPTION_EXECUTE_HANDLER)  
{  
    Status = GetExceptionCode();  
    ... report error status  
}  
```
