# <a name="c28289"></a>C28289
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28289: 関数: ReadableTo または WritableTo に、パラメーターとして制限仕様がありませんでした  
  
 コード分析ツールでは、関数またはパラメーターの注釈が誤って示されている場合に、この警告を報告します。
