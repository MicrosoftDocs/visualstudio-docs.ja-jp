# <a name="c28160"></a>C28160
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28160: 注釈エラーです  
  
 この警告は `__drv_error` 注釈が検出された場合に報告されます。 この注釈は、修正が必要なコーディングの手法にフラグを設定するために使用されます。また、パラメーターの特定の組み合わせを示すために `__drv_when` 注釈と共に使用することもできます。
