# <a name="c28223"></a>C28223
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28223: 注釈、パラメーターに必要なトークン/識別子が見つかりませんでした  
  
 この警告は、注釈のパラメーターが識別子であり、他のシンボルが検出されたことを示します。 これは通常、誤ってコード化された注釈マクロであることを示します。 この位置で C またC++はキーワードを使用すると、このエラーが発生します。
