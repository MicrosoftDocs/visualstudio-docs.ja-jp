# <a name="c28223"></a>C28223
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

このトピックの最新バージョンをご覧[C28223](https://docs.microsoft.com/visualstudio/code-quality/c28223)します。  
  
警告 C28223: 注釈の必要なトークン/識別子を見つかりませんでしたパラメーター。  
  
 この警告は、注釈のパラメーターの識別子のことが必要ですし、他の記号が見つかりましたことを示します。 これは通常、正しくコード化された注釈マクロを示します。 この位置での C または C++ のキーワードの使用には、このエラーが発生します。
