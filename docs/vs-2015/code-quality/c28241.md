# <a name="c28241"></a>C28241
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28241:パラメーターの関数の注釈は認識されません  
  
 認識されない注釈名が使用されました。 注釈は標準のマクロが使用されている場合があります。この警告は、一般的な使用で目にする必要はありません。
