# <a name="c28275"></a>C28275
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28275:\_Macro_value\_ のパラメーターは null です  
  
 この警告は、分析対象のコードではなく、モデル ファイルの内部エラーがあることを示します。 *MacroValue*パラメーターを指定せず関数を呼び出しました。
