# <a name="ca0057"></a>CA0057

現在のターゲットとルールのセットに違反をマップすることはできません。

このエラーが発生するのは、プロジェクトファイルまたはインポートされた XML レポートで、現在のターゲットとルールのセットにマップできない違反を含む FxCopCmd.exe が手動で呼び出された場合です。

ツールのコマンドラインバージョンを使用して分析が行われた場合にのみ発生します。
