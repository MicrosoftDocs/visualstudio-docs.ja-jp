# <a name="c1250"></a>C1250
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C1250: プラグインを読み込めませんでした。  
  
 コード分析ツールは、プラグインで、分析対象のコードではなく、内部エラーがある場合に、この警告を報告します。
