# <a name="c1250"></a>C1250
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C1250: プラグインを読み込めません。  
  
 コード分析ツールは、この警告を報告します。この警告は、解析対象のコードではなく、プラグインに内部エラーが発生した場合に報告されます。
