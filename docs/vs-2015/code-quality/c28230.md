# <a name="c28230"></a>C28230
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28230: パラメーターの型にはメンバーがありません  
  
 この警告は、注釈の引数が `struct`、`class`、または `union` にアクセスしようとしたときに、名前付きのメンバーが存在しないことを示します。
