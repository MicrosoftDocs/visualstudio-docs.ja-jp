# <a name="c1253"></a>C1253
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C1253: モデル ファイルを読み込めませんでした。  
  
 コード分析ツールは、分析対象のコードではなく、モデル ファイルの内部エラーがある場合に、この警告を報告します。
