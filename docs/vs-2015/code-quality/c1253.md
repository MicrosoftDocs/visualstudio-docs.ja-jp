# <a name="c1253"></a>C1253
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C1253: モデルファイルを読み込むことができません。  
  
 コード分析ツールは、分析されているコードではなく、モデルファイルに内部エラーが発生した場合に、この警告を報告します。
