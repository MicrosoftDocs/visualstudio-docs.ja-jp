# <a name="c28211"></a>C28211
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28211: SAL_context には静的コンテキスト名が必要です  
  
 この警告は、注釈のオペランドが `_Static_context_` ツールで定義されたコンテキストの名前である必要があることを示します。 通常の使用では、この警告は発生しません。
