# <a name="c28226"></a>C28226
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28226: 注釈こともできません、PrimOp (現在の宣言内)  
  
 この警告は、名前付きの注釈は、primop になるとして宣言されても、通常の注釈に宣言されていたことを示します。 注釈は標準のマクロが使用されている場合があります。この警告は、通常の使用で目にする必要はありません。
