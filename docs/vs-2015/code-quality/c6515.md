# <a name="c6515"></a>C6515
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6515: 無効な注釈:\<名 > プロパティは、ポインター型の値でのみ使用します。  
  
 この警告は、非ポインター型へのポインターに使用するためのプロパティが適用されたことを示します。 注釈プロパティの一覧は、次を参照してください。[注釈プロパティ](http://msdn.microsoft.com/f77b4370-6bda-4294-bd2a-e7d0df182a3d)します。  
  
## <a name="example"></a>例  
 この警告が発生するコード例を次に示します。  
  
```cpp  
#include <sal.h>  
  
void f(_Readable_bytes_(c) char pc,  size_t c)  
{  
    // code ...  
}  
```  
  
 この警告を解決するには、次のコードを使用します。  
  
```  
#include <sal.h>  
  
void f(_Readable_bytes_(c) char * pc,  size_t c)  
{  
    // code ...  
}  
```
