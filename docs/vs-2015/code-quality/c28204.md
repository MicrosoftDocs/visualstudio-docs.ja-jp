# <a name="c28204"></a>C28204
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28204:\<関数 >: このオーバー ロードとに 1 つの 1 つだけ\<ファイル名 > (\<行 >) の注釈が付けられた\<paramname >: 両方またはどちらもの注釈を付ける必要があります。  
  
 この警告は、注釈内でエラーがある場合に報告されます。
