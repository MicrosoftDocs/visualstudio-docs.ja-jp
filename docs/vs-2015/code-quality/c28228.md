# <a name="c28228"></a>C28228
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

このトピックの最新バージョンをご覧[C28228](https://docs.microsoft.com/visualstudio/code-quality/c28228)します。  
  
警告 C28228: 注釈パラメーター: 注釈内で型を使用することはできません  
  
 この警告は、パラメーターがサポートされていない型であることを示します。 注釈は、パラメーターとしてのみ型の制限のセットを使用できます。 注釈は標準のマクロが使用されている場合があります。この警告は、一般的な使用で目にする必要はありません。
