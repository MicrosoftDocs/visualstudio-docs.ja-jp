# <a name="c28300"></a>C28300
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28300: < parameter_name >: 演算子 < operator_name に対する互換性のない型の式のオペランド >  
  
 この警告は、パラメーターのオペランドの型が演算子と互換性がない場合に報告されます。
