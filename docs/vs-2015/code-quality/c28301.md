# <a name="c28301"></a>C28301
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28301: \<関数 > の最初の宣言に対する注釈がありません。\<note1 > 「\<filename > (\<行 >)」を参照してください。 \<note2 以降 >  
  
 この警告は、指定された関数の最初の宣言で注釈が見つからなかった場合に報告されます。
