# <a name="c28232"></a>C28232
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28232: _Pre\_、_Post\_、または _Deref\_注釈には適用されません  
  
 この警告では、ことを示します、 `_Pre_`、 `_Post_`、または`_Deref_`後続の関数型の注釈なし注釈式に演算子が表示されます。 修飾子は無視されましたが正しくコード化された注釈を示します。
