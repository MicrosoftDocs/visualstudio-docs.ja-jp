# <a name="ca0059"></a>CA0059

コマンド ライン スイッチのエラー。

必要なコマンド ライン スイッチが欠落している場合、たとえば、このエラーが発生します

/out または"/console"スイッチが指定されていません。 このエラーは、不適切なタイムアウト値など、無効なスイッチがある場合にも発生します。
