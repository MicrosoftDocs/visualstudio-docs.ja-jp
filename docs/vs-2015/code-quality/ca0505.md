# <a name="ca0505"></a>CA0505

優先するプロパティが定義されているために、非推奨のプロパティは無視されます。

指定した非推奨とされます両方と優先されるコード分析プロパティをプロジェクト内の値を含めることがまたは構成ファイルをビルド警告 CA505 が生成されます。 この問題を解決するには、非推奨のプロパティを削除します。
