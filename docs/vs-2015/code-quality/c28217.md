# <a name="c28217"></a>C28217
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

C28217 を警告します。関数について、注釈へのパラメーター数がファイルで検出されたものと一致しません  
  
 現在の行と、メッセージ内の行のコメントは、一貫性がありません。 注釈は標準のマクロが使用されている場合があります。この警告は、通常の使用で目にする必要はありません。
