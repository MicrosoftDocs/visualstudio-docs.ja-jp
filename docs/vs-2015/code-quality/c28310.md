# <a name="c28310"></a>C28310
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

このトピックの最新バージョンをご覧[C28310](https://docs.microsoft.com/visualstudio/code-quality/c28310)します。  
  
警告 28310: < annotation_name > 注釈\<関数 >\<パラメーター > SAL バージョンがありません。  
  
 ソース コードで使用されているすべての SAL 注釈は、SAL_name を使用して適用された注釈バージョンである必要があります。 これは、マクロ定義で修正する必要があります。  
  
 この警告は、宣言ごとに一度だけ報告されます。 より古い SAL については、残りの宣言を調べてください。
