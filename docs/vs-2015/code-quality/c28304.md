# <a name="c28304"></a>C28304
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

このトピックの最新バージョンをご覧[C28304](https://docs.microsoft.com/visualstudio/code-quality/c28304)します。  
  
警告 C28304: C++ 参照パラメーター < パラメーター名 >、不適切に`_Notref_`に適用される演算子が見つかりました\<トークン >。  
  
 `_Notref_`演算子は、システム指定のマクロのみと C++ 参照パラメーターを使用する特殊な状況でのみ使用する必要があります。 直後にする必要があります、`_Deref_`演算子または関数型の注釈。
