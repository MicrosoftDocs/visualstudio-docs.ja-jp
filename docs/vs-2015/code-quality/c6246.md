# <a name="c6246"></a>C6246
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6246: のローカル宣言\<変数 > 外側のスコープで同じ名前の宣言を隠します。 追加情報: は、以前の宣言を参照してください。\<場所 >。  
  
 この警告は、2 つの宣言がローカル スコープで同じ名前を指定することを示します。 外側のスコープの名前は、内側のスコープで宣言によって隠されています。 任意の外側のスコープの宣言の使用を目的とすると、ローカル宣言の使用になります。  
  
## <a name="example"></a>例  
 この警告が発生するコード例を次に示します。  
  
```  
#include <stdlib.h>  
#define UPPER_LIMIT 256  
int DoSomething( );  
  
int f( )  
{  
  int i = DoSomething( );  
  if (i > UPPER_LIMIT)  
  {  
    int i;  
    i = rand( );  
  }  
  return i;  
}  
```  
  
 この警告を解決するには、次のコードに示すように、別の変数名を使用します。  
  
```  
#include <stdlib.h>  
#define UPPER_LIMIT 256  
int DoSomething( );  
  
int f ( )  
{  
  int i = DoSomething( );  
  if (i > UPPER_LIMIT)  
  {  
    int j = rand( );  
    return j;  
  }  
  else  
  {  
    return i;  
  }  
}  
```  
  
 のみ、この警告は、スコープの重複を識別します。
