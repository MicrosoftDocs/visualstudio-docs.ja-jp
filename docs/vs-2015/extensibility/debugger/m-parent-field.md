# <a name="mparent-field"></a>m_parent フィールド
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

このトピックの最新バージョンをご覧[m_parent フィールド](https://docs.microsoft.com/visualstudio/extensibility/debugger/m-parent-field)します。  
  
バッキング フィールドは、<xref:System.Threading.Tasks.Task?displayProperty=fullName>プロパティの親。  
  
 **Namespace:** <xref:System.Threading.Tasks?displayProperty=fullName>  
  
 **アセンブリ:** mscorlib (mscorlib.dll 内)  
  
 .NET Framework からこの内部メンバーにアクセスできないため、次の構文には共通中間言語 (CIL) が提供されます。  
  
## <a name="syntax"></a>構文  
  
```  
.field private initonly class System.Threading.Tasks.Task m_parent  
```
