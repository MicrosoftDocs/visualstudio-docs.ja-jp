# <a name="visual-studio-debugging-samples"></a>Visual Studio のデバッグのサンプル
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

2つのサンプルは、を使用して Visual Studio デバッガーを拡張する方法を示して [!INCLUDE[vsipsdk](../../includes/vsipsdk-md.md)] います。  
  
- マネージ言語のデバッガー式エバリュエーターのサンプルは、に含まれてい [!INCLUDE[vsipsdk](../../includes/vsipsdk-md.md)] ます。 ウィンドウで、[ [!INCLUDE[vsipsdk](../../includes/vsipsdk-md.md)] **サンプル** ] タブをクリックし、言語の一覧で [ **すべて** ] または [ **C#** ] をクリックして、[ **デバッガー**] をクリックします。  
  
- ネイティブ言語の Visual Studio デバッグエンジンサンプルは、MSDN コードギャラリーから入手できます。 詳細については、GitHub のサンプルを参照してください。 [Visual Studio デバッグエンジンのサンプル](https://github.com/microsoftarchive/msdn-code-gallery-microsoft/tree/master/Visual%20Studio%20Product%20Team/Visual%20Studio%20Debug%20Engine%20Sample)です。
