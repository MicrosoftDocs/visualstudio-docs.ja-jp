# <a name="idebugnosymbolsevent2"></a>IDebugNoSymbolsEvent2
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

信号、[!INCLUDE[vsprvs](../../../includes/vsprvs-md.md)]デバッガーの UI をユーザーに警告シンボルが起動した実行可能ファイルを見つけることできませんでした。  
  
## <a name="syntax"></a>構文  
  
```  
IDebugNoSymbolsEvent2 : IUnknown  
```  
  
## <a name="notes-for-implementers"></a>実装についてのメモ  
 デバッグ エンジンによって実装されで使用される、[!INCLUDE[vsprvs](../../../includes/vsprvs-md.md)]デバッガーの UI。

 ヘッダー:Msdbg.h
