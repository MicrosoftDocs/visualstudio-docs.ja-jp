# <a name="idebugdocumenttextevents2ondestroy"></a>IDebugDocumentTextEvents2::onDestroy

このトピックの最新バージョンをご覧[IDebugDocumentTextEvents2::onDestroy](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugdocumenttextevents2-ondestroy)します。

## <a name="syntax"></a>構文  
  
```cpp#  
HRESULT onDestroy(   
   void
```  

```csharp

```
