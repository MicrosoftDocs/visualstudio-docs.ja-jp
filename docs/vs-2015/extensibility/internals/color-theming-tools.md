# <a name="color-theming-tools"></a>カラー テーマのツール
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

このトピックの最新バージョンをご覧[カラー テーマのツール](https://docs.microsoft.com/visualstudio/extensibility/internals/color-theming-tools)します。  
  
VS SDK には、作成し、配色テーマを管理するのに役立ついくつかのツールが用意されています。  
  
-   [VSIX カラー エディター](../../extensibility/internals/vsix-color-editor.md)ツールの作成し、Visual Studio のカスタム色を編集できます。  
  
-   [VSIX カラー コンパイラ](../../extensibility/internals/vsix-color-compiler.md)ツールでは、.pkgdef、拡張機能を既存の Visual Studio のテーマの .xml ファイルを変換します。
