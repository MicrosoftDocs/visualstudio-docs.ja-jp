# <a name="test-area-5-change-source-control"></a>テスト領域 5: ソース管理の変更
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

このソース管理プラグインのテスト領域では、[ **ソース管理の変更** ] コマンドを使用したソース管理の変更について説明します。  
  
 [**ソース管理の変更**] コマンドでは、ユーザーに4つの基本機能が用意されています。  
  
- **束縛**  
  
   ソリューション/プロジェクトとバージョンストア間のソース管理リンクを確立または再確立することをユーザーに許可します。  
  
- **Unbind**  
  
   接続ごとにソース管理からプロジェクトまたはソリューションを削除します。  
  
- **接続/切断:**  
  
  管理されたソリューションの接続状態またはオフライン状態を切り替えます。これについては、「区分3」で説明しています。 詳細については、「 [テスト領域 3: チェックアウト/](../../extensibility/internals/test-area-3-check-out-undo-checkout.md)チェックアウトの取り消し」を参照してください。  
  
## <a name="command-menu-access"></a>コマンドメニューへのアクセス  
 [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)]テストケースでは、次の統合開発環境メニューパスが使用されます。  
  
 ソース管理の変更:**ファイル**、 **ソース管理**、 **ソース管理の変更**。
