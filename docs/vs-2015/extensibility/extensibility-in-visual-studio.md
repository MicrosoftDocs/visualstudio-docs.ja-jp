# <a name="extensibility-in-visual-studio"></a>Visual Studio の機能拡張
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Visual Studio では、多くの機能拡張オプションを提供しています。 独自の Sdk を作成し、Windows、Windows Phone、および Azure Sdk (Visual Studio の一部としてインストールされます) を使用して、Visual Studio 自体を拡張することができます。

## <a name="extend-visual-studio"></a>Visual Studio を拡張する
 Visual Studio SDK を使用すると、コマンド、メニュー、ウィンドウ、エディター、プロジェクトなど、Visual Studio のすべての部分を簡単に拡張できます。 詳細については、「 [Visual STUDIO SDK](../extensibility/visual-studio-sdk.md)」を参照してください。

## <a name="create-your-own-sdks"></a>独自の Sdk を作成する
 独自のプラットフォームおよび拡張 Sdk の作成、パッケージ化、および配置の方法については、「 [ソフトウェア開発キットの作成](../extensibility/creating-a-software-development-kit.md)」を参照してください。

## <a name="roslyn-extensibility"></a>Roslyn の拡張性
 拡張に Roslyn を使用する方法については、 [.NET Compiler Platform ("Roslyn") の拡張機能](../extensibility/dotnet-compiler-platform-roslyn-extensibility.md)に関する説明を参照してください。

## <a name="azure-sdk"></a>Azure SDK
 Azure sdk を有効にする方法については、「azure [sdk の有効化](../extensibility/enabling-the-azure-sdk.md)」を参照してください。
