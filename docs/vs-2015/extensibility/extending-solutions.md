# <a name="extending-solutions"></a>ソリューションの拡張
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

ソリューションは、連携してアプリケーションを作成する 1 つまたは複数のプロジェクトのグループです。  
  
## <a name="in-this-section"></a>このセクションの内容  
 次のチュートリアルでは、ソリューションを拡張するいくつかの方法を説明します。  
  
1.  [ソリューション エクスプ ローラーのフィルターを拡張する](../extensibility/extending-the-solution-explorer-filter.md)ソリューション エクスプ ローラーの表示をフィルター処理する方法について説明します。  
  
2.  [プロジェクトがソリューションの読み込みを管理する](../extensibility/managing-project-loading-in-a-solution.md)ユーザー ソリューションの読み込み時に、パフォーマンスを向上させるためにプロジェクトの読み込みを管理する方法について説明します。
