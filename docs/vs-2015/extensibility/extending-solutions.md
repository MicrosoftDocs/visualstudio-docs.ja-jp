# <a name="extending-solutions"></a>ソリューションの拡張
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

ソリューションは、アプリケーションを作成するために連携して動作する1つ以上のプロジェクトをグループ化したものです。  
  
## <a name="in-this-section"></a>このセクションの内容  
 次のチュートリアルでは、ソリューションを拡張するいくつかの方法について説明します。  
  
1. [ソリューションエクスプローラーフィルターを拡張](../extensibility/extending-the-solution-explorer-filter.md) すると、ソリューションエクスプローラーに表示される内容をフィルター処理する方法が説明されます。  
  
2. [ソリューションでプロジェクトの読み込みを管理](../extensibility/managing-project-loading-in-a-solution.md) するプロジェクトの読み込みを管理して、ユーザーがソリューションを読み込むときのパフォーマンスを向上させる方法について説明します。
