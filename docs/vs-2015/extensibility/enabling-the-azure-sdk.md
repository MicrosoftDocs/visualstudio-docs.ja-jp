# <a name="enabling-the-azure-sdk"></a>Azure SDK の有効化
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Azure SDK for .NET は、Azure でスケーラブルな Web アプリケーションとサービスを作成、構成、ビルド、デバッグ、実行、およびパッケージ化するのに役立ちます。  
  
 Azure SDK は、[ **新しいプロジェクト** ] ダイアログボックスからダウンロードできます。  
  
 [!INCLUDE[note_settings_general](../includes/note-settings-general-md.md)]  
  
### <a name="to-enable-the-azure-sdk"></a>Azure SDK を有効にするには  
  
1. [ **ファイル** ] メニューで、新しいプロジェクトを作成します。 ([ **新規** ] をクリックし、[ **プロジェクト**] をクリックします)。  
  
2. **Visual Basic**または**C#** のノードを展開し、[**クラウド**] をクリックします。  
  
3. 中央のウィンドウで、[ **Get AZURE SDK for .net**] を選択し、[ **OK**] をクリックします。  
  
     インストーラーに表示される指示に従います。  
  
    > [!NOTE]
    > Azure Tools をインストールするには、インターネットインフォメーションサービス7.0 と ASP.NET アプリケーション開発コンポーネントがインストールされている必要があります。 これらのコンポーネントがインストールされていない場合、Azure Tools セットアップでインストールするように求められます。
