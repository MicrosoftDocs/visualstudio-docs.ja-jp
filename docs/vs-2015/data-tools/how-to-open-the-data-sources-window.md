# <a name="how-to-open-the-data-sources-window"></a>方法: [データ ソース] ウィンドウを開く
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

**データソース**ウィンドウには、プロジェクトのデータ ソースが表示されます。 使用する、**データソース** ウィンドウから、プロジェクト内のフォームに項目をドラッグしてデータ バインド コントロールのユーザー インターフェイスを作成するウィンドウ。 参照してください[Visual Studio でのデータにコントロールを Windows フォームのバインド](../data-tools/bind-windows-forms-controls-to-data-in-visual-studio.md)します。  
  
 設定することができます、**データソース**ウィンドウを使用して、**データ ソースの構成**ウィザード。 詳細については、次を参照してください。[データ ソース構成ウィザード](http://msdn.microsoft.com/library/c4df7de5-5da0-4064-940c-761dd6d9e28f)します。  
  
### <a name="to-open-the-data-sources-window"></a>[データ ソース] ウィンドウを開くには  
  
-   メニュー バーで、**ビュー**、**その他の Windows**、**データソース**(または、Shift + Alt + D キーを押します)。  
  
    > [!IMPORTANT]
    >  データ ソースの作成と操作がサポートされている場合にのみ、プロジェクトにデータ ソースを追加できます。 たとえば、開くことはできません、**データソース**Windows ストア アプリ用のプロジェクトでのウィンドウ。
