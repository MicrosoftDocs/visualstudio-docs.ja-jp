# <a name="how-to-edit-tableadapter-queries"></a>方法 : TableAdapter クエリを編集する
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

TableAdapter クエリの編集、 [TableAdapters の編集](../data-tools/editing-tableadapters.md)で、**データセット デザイナー**します。 不要になったアプリケーションのニーズに適している場合、TableAdapter クエリを変更する必要があります。 (または、クエリを作成できます追加 TableAdapter 上。 新しいクエリを追加する方法の詳細については、次を参照してください[方法: TableAdapter クエリの作成](../data-tools/how-to-create-tableadapter-queries.md)。)。  
  
> [!NOTE]
>  場合、 [TableAdapter 構成ウィザード](http://msdn.microsoft.com/library/3a373dd9-7b34-4d3c-a48b-69414512bac8)の代わりに開きます、 **TableAdapter クエリ構成ウィザード**、TableAdapter のメインを選択した`Fill`クエリとのない 1 つ、TableAdapter の追加のクエリ。 TableAdapter を編集する方法についてのメインの`Fill`クエリを参照してください[方法: Tableadapter の編集](http://msdn.microsoft.com/library/ca178745-e35a-45f1-a395-23cddfd8f855)します。  
  
 ![複数のクエリによる TableAdapter](../data-tools/media/tableadapter.gif "TableAdapter")  
  
### <a name="to-edit-a-tableadapter-query"></a>TableAdapter クエリを編集するには  
  
1.  データセットを開き、**データセット デザイナー**します。 詳細については、次を参照してください。[方法: データセット デザイナーでデータセットを開く](http://msdn.microsoft.com/library/36fc266f-365b-42cb-aebb-c993dc2c47c3)します。  
  
2.  編集する TableAdapter クエリを選択します。  
  
3.  TableAdapter クエリを右クリックして**構成**します。  
  
     **TableAdapter クエリ構成ウィザード**が開き、クエリまたはクエリを実行するためにストアド プロシージャを変更できるようになります。  
  
4.  完了、 **TableAdapter クエリ構成ウィザード**を変更します。 詳細については、次を参照してください。 [TableAdapters の編集](../data-tools/editing-tableadapters.md)します。
