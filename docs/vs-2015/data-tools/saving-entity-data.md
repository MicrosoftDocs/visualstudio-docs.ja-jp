# <a name="saving-entity-data"></a>エンティティ データの保存
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

WCF Data Services からデータベースに接続するエンティティを作成、更新、または削除した場合は、変更内容をデータベースに明示的に保存する必要があります。 以下のトピックでは、WCF Data Services アプリケーションでエンティティ データの変更を保存する方法について説明します。  
  
|トピック|説明|  
|-----------|-----------------|  
|[方法: エンティティを追加、変更、および削除する](http://msdn.microsoft.com/library/a00f8933-b232-4445-95ba-adc634f055d8)|データ サービスを使用してエンティティ データの変更を保存する方法について説明します。|  
|[方法: 非同期データ サービス クエリを実行する](http://msdn.microsoft.com/library/902a2dc1-d0e9-4b00-90a8-becc4cb1f6a7)|データ サービスを使用して変更を非同期に保存する方法について説明します。|
