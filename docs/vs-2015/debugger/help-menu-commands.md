# <a name="help-menu-commands"></a>[ヘルプ] メニューのコマンド
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

**[ヘルプ]** メニューには、ヘルプ ファイルを開くためのコマンドが用意されています。  
  
 **Index**  
 Spy + + ヘルプの [コンテンツ] 画面を開きます。  
  
 **ヘルプの使い方**  
 ヘルプの使用方法を説明するヘルプファイルを開きます。  
  
 **Spy++ のバージョン情報**  
 Spy + + のソフトウェアバージョンと著作権情報を表示します。
