2.  **ビュー** ] メニューの [選択**プロパティ**します。  
  
 プロパティのダイアログ ボックスはモーダルではないため、表示ウィンドウで他のアイテムをクリックして、ダイアログ ボックスの選択項目に関する情報が表示されます。
