# <a name="window-properties-dialog-box"></a>[ウィンドウ プロパティ] ダイアログ ボックス
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

このダイアログ ボックスを使用して、特定のウィンドウの詳細を確認します。 このダイアログ ボックスを表示するには、[[ウィンドウ ビュー]](../debugger/windows-view.md) ウィンドウにフォーカスを移動します。 ツリーで任意のウィンドウ ノードを選択し、 **[ビュー]** メニューから **[プロパティ]** を選択します。  
  
## <a name="in-this-section"></a>このセクションの内容  
 [[全般] タブ ([ウィンドウのプロパティ] ダイアログボックス)](../debugger/general-tab-window-properties-dialog-box.md)  
 選択したウィンドウに関する情報を表示します。  
  
 [[スタイル] タブ ([ウィンドウプロパティ] ダイアログボックス)](../debugger/styles-tab-window-properties-dialog-box.md)  
 選択したウィンドウに適用されているスタイルを表示します。  
  
 [Windows の [タブ] ([ウィンドウのプロパティ] ダイアログボックス)](../debugger/windows-tab-window-properties-dialog-box.md)  
 選択したウィンドウに関連するウィンドウの情報を表示します。  
  
 [[クラス] タブ ([ウィンドウプロパティ] ダイアログボックス)](../debugger/class-tab-window-properties-dialog-box.md)  
 選択したウィンドウのクラスに関する情報を表示します。  
  
 [[プロセス] タブ ([ウィンドウのプロパティ] ダイアログボックス)](../debugger/process-tab-window-properties-dialog-box.md)  
 選択されたウィンドウのプロセスを識別します。
