# <a name="idiaenumlinenumbersreset"></a>IDiaEnumLineNumbers::Reset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

先頭に、列挙体シーケンスをリセットします。  
  
## <a name="syntax"></a>構文  
  
```cpp#  
HRESULT Reset();  
```  
  
## <a name="return-value"></a>戻り値  
 `S_OK` を返します。
