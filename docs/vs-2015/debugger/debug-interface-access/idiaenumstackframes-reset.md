# <a name="idiaenumstackframesreset"></a>IDiaEnumStackFrames::Reset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

列挙のシーケンスを最初にリセットします。  
  
## <a name="syntax"></a>構文  
  
```cpp#  
HRESULT Reset();  
```  
  
## <a name="return-value"></a>戻り値  
 `S_OK` を返します。
