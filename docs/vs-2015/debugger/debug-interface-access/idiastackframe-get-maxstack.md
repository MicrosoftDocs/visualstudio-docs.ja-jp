# <a name="idiastackframegetmaxstack"></a>IDiaStackFrame::get_maxStack
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

このトピックの最新バージョンをご覧[idiastackframe::get_maxstack](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiastackframe-get-maxstack)します。  
  
フレームのスタックにプッシュされるバイトの最大数を取得します。  
  
## <a name="syntax"></a>構文  
  
```cpp#  
HRESULT get_maxStack (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `pRetVal`  
 [out]スタックにプッシュするバイトの最大数を返します。  
  
## <a name="return-value"></a>戻り値  
 成功した場合、返します`S_OK`します。 返します`S_FALSE`プロパティがサポートされていない場合。 それ以外の場合はエラー コードを返します。
