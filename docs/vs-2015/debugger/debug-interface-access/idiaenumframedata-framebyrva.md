# <a name="idiaenumframedataframebyrva"></a>IDiaEnumFrameData::frameByRVA
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

相対仮想アドレス (RVA) でフレームを返します。  
  
## <a name="syntax"></a>構文  
  
```cpp#  
HRESULT frameByRVA(   
   DWORD           relativeVirtualAddress,  
   IDiaFrameData** frame  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 relativeVirtualAddress  
 から目的のフレームの RVA。  
  
 frame  
 入出力指定されたアドレスを含むフレームを表す [IDiaFrameData](../../debugger/debug-interface-access/idiaframedata.md) オブジェクトを返します。  
  
## <a name="return-value"></a>戻り値  
 正常に終了した場合は、`S_OK` を返します。 `S_FALSE`指定されたアドレスと一致するフレームデータがない場合は、を返します。 それ以外の場合はエラー コードを返します。
