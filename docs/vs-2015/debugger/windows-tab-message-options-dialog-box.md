# <a name="windows-tab-message-options-dialog-box"></a>[ウィンドウ] タブ ([メッセージ オプション] ダイアログ ボックス)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

使用して、 **Windows**メッセージの一覧のスコープを設定するタブ[メッセージ ビュー](../debugger/messages-view.md)します。 表示する、[メッセージ オプション ダイアログ ボックス](../debugger/message-options-dialog-box.md)、選択**ログ メッセージ**から、**スパイ**メニュー。  
  
 次の設定は [使用可能な**Windows** ] タブ。  
  
 **ファインダー ツール**  
 ウィンドウを選択するために使用するドラッグ アンド ドロップ ツールです。  
  
 **Spy++ を非表示に**  
 Spy++ のメイン ウィンドウが、**メッセージ**ダイアログ ボックスで、他のアプリケーションの上に表示します。  
  
 **親ウィンドウ**  
 選択したウィンドウとその直接の親ウィンドウのメッセージが表示されます。  
  
 **子ウィンドウ**  
 選択したウィンドウとウィンドウの入れ子になった子など、そのすべての子ウィンドウのメッセージを表示します。  
  
 **同じスレッドの Windows**  
 選択したウィンドウと同じスレッドによって所有されているその他のすべての windows メッセージを表示します。  
  
 **同じプロセスの Windows**  
 選択したウィンドウと同じプロセスによって所有されているその他のすべての windows メッセージを表示します。  
  
 **システム内のすべての Windows**  
 すべてのウィンドウのメッセージが表示されます。  
  
 **既定値として設定を保存します。**  
 新しいメッセージのストリーム ウィンドウに対する上記の設定を保存します。 これらの設定は、spy++ を終了するときにも保存されます。
