# <a name="error-mixed-mode-debugging-for-ia64-processes-is-unsupported"></a>エラー: IA64 プロセスの混合モードのデバッグはサポートされていません
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

このトピックの最新バージョンをご覧[エラー: IA64 プロセスの混合モードのデバッグはサポートされていません](https://docs.microsoft.com/visualstudio/debugger/error-mixed-mode-debugging-for-ia64-processes-is-unsupported)します。  
  
[!INCLUDE[vsprvs](../includes/vsprvs-md.md)] デバッガーは、Itanium ベースのプロセスでのネイティブ コードとマネージド コードの混合のデバッグをサポートしません。  
  
### <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   デバッグのため、32 ビット バージョンのアプリケーションをビルドします。
