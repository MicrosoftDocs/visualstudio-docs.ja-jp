# <a name="spy-views"></a>Spy++ ビュー
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

このトピックの最新バージョンをご覧[spy++ ビュー](https://docs.microsoft.com/visualstudio/debugger/spy-increment-views)します。  
  
Spy++ では、コンピューターで windows、メッセージ、プロセス、および利用可能なスレッドのツリー ビューが表示されます。  
  
## <a name="in-this-section"></a>このセクションの内容  
 [ウィンドウ ビュー](../debugger/windows-view.md)  
 使用可能なウィンドウとコントロールのツリー ビューが表示されます。  
  
 [メッセージ ビュー](../debugger/messages-view.md)  
 ウィンドウ、プロセス、またはスレッドに関連付けられているメッセージのストリームを表示します。  
  
 [プロセス ビュー](../debugger/processes-view.md)  
 アクティブなプロセスのツリー ビューが表示されます。  
  
 [スレッド ビュー](../debugger/threads-view.md)  
 アクティブなスレッドのツリー ビューが表示されます。
