# <a name="font-dialog-box-microsoft-spy-help"></a>[フォント] ダイアログ ボックス (Microsoft Spy++ ヘルプ)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

このダイアログ ボックスを使用すると、[Spy++ ビュー](../debugger/spy-increment-views.md) ウィンドウで使用される文字フォントを構成できます。 このダイアログ ボックスを表示するには、 **[ビュー]** メニューを展開し、 **[フォント]** を選択します。  
  
 次の設定を使用できます。  
  
 **フォント**  
 フォーカスがある Spy + + ビューウィンドウの文字フォント。  
  
 **[スタイル]**  
 フォーカスがある [Spy + + ビュー] ウィンドウのフォントスタイル。  
  
 **[サイズ]**  
 フォーカスがある [Spy + + ビュー] ウィンドウのフォントサイズ。  
  
 **フォントを既定値として保存**  
 Spy + + ビューを開くときに、現在のフォント設定を設定として保存します。
