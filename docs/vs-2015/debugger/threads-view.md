# <a name="threads-view"></a>スレッド ビュー
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

スレッド ビューは、システム上のウィンドウに関連付けられているすべてのスレッドの単純な一覧です。 各スレッドのスレッド ID とモジュール名が表示されます。 プロセスは含まれませんが、選択したスレッドを所有するプロセスを簡単に見つけることができます。  
  
## <a name="procedures"></a>プロシージャ  
  
#### <a name="to-open-the-threads-view"></a>スレッド ビューを開くには  
  
1. **[Spy]** メニューから **[スレッド]** を選択します。  
  
   ![Spy&#43;&#43; スレッドビュー](../debugger/media/spy-threads.png "Spy + + _Threads")  
   Spy++ スレッド ビュー  
  
   上の図は、スレッド ビューを示しています。
