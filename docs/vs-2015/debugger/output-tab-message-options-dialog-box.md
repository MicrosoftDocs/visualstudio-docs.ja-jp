# <a name="output-tab-message-options-dialog-box"></a>[出力] タブ ([メッセージ オプション] ダイアログ ボックス)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

**[出力]** タブを使用し、各メッセージからどのデータを [[メッセージ ビュー]](../debugger/messages-view.md) に一覧表示するか指定します。 [[メッセージ オプション]](../debugger/message-options-dialog-box.md) ダイアログ ボックスを表示するには、 **[Spy]** メニューから **[ログ メッセージ]** を選択します。  
  
 **[出力]** タブでは、次の設定を使用できます。  
  
 **行番号**  
 行番号を表示します。  
  
 **メッセージの入れ子レベル**  
 入れ子になったメッセージを1つのレベルにつき1つの期間でプレフィックスします。  
  
 **生のメッセージ パラメーター**  
 16進数の **wParam** と **lParam** の値を表示します。  
  
 **デコードされたメッセージ パラメーター**  
 **WParam**と**lParam**の値のメッセージ固有のデコードの結果を表示します。  
  
 **生の戻り値**  
 16進数の **lResult** 戻り値を表示します。  
  
 **デコードされた戻り値**  
 **LResult**戻り値のメッセージ固有のデコードの結果を表示します。  
  
 **メッセージの配信元時間**  
 Windows システムが起動してからの経過時間 (ポストされたメッセージの場合のみ)。  
  
 **メッセージのマウス位置**  
 メッセージがポストされたときのマウスの画面座標 (ポストされたメッセージの場合のみ)。  
  
 **最大行数**  
 現在選択されているメッセージビューで保持される行数を制限します。  
  
 **また、メッセージをファイルに記録します**  
 メッセージログの出力ファイルを指定します。 この出力ファイルは、メッセージ ログ ウィンドウと同期で書き込まれます。  
  
 **設定を既定値として保存**  
 新しいメッセージストリームウィンドウの前の設定を保存します。 これらの設定は Spy++ を終了したときに保存されます。
