 **次回からこのダイアログ ボックスを表示しない**  
 このチェック ボックスをオンにすると、以後**エディット コンティニュ**では、アクセス許可を求めることなく、コード変更を適用します。 この警告を再度オンにするには、**[オプション]** ダイアログ ボックスで、**[デバッグ]** フォルダーを開き、**[エディット コンティニュ]** ページをクリックし、**[デバッグ コマンドで呼び出す]** と **[常に確認メッセージを表示する]** をオンにします。
