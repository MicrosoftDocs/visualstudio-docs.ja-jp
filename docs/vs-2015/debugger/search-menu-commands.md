# <a name="search-menu-commands"></a>[検索] メニューのコマンド
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

**[検索]** メニューには、特定のウィンドウを検索するためのコマンドが用意されています。  
  
 **ウィンドウ/プロセス/スレッド/メッセージの検索**  
 [**ウィンドウ** / **プロセス** / **スレッド**の / **messagesearch** ] ダイアログボックスを開きます。このダイアログボックスでは、 **findertool**を使用したり、検索条件を指定したりできます。  
  
 **[次を検索]**  
 現在のビューで、次に一致するウィンドウ、プロセス、スレッド、またはメッセージを検索します。 このメニュー項目 (および関連するツールバー ボタン) は、一意ではない有効な検索結果がある場合にのみ使用できます。  
  
 **[前を検索]**  
 現在のビューで、前の一致するウィンドウ、プロセス、スレッド、またはメッセージを検索します。 このメニュー項目 (および関連するツールバー ボタン) は、一意ではない有効な検索結果がある場合にのみ使用できます。
