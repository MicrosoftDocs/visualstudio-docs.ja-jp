# <a name="releasing-a-visual-studio-integration-product"></a>Visual Studio 統合製品のリリース
このセクションのトピックでは、VSPackage をビルドしてインストールし、Visual Studio 統合テスト (VSIT) を使用して、Visual Studio 統合製品をリリースする方法を説明します。

## <a name="in-this-section"></a>このセクションの内容
 [Vspackage のインストール](../misc/installing-vspackages.md) Vspackage をビルドしてインストールする方法と、複数のバージョンの Visual Studio を同時に実行しているユーザーをサポートする方法について説明します。

 [Regpkg ユーティリティ](../extensibility/internals/regpkg-utility.md) VSPackage を登録し、配置用に準備するために使用される RegPkg ユーティリティについて説明します。

 [プレミアパートナーエディションのセットアップおよびユーザーガイド](https://msdn.microsoft.com/8ee4dad7-95d3-4f2d-a8d4-3ba9a80ecae2) には、独自の開発ツールとコンピューター言語を追加できるように、VSIP パートナーに Visual Studio 統合開発環境 (IDE) の再頒布可能バージョンが用意されています。
