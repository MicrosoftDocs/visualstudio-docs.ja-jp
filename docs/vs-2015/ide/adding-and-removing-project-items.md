# <a name="adding-and-removing-project-items"></a>プロジェクト項目の追加と削除
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

**ソリューション エクスプローラー**でプロジェクトまたはソリューション ノードを右クリックし、**[追加]** または **[削除]** を選択して、プロジェクトを追加または削除できます。 ソリューションからプロジェクトを削除した場合、またはプロジェクトから項目を削除した場合、それらの項目がディスクまたはソース管理から削除されることはありません (ただし、完全に削除するオプションを選択した場合を除く)。
