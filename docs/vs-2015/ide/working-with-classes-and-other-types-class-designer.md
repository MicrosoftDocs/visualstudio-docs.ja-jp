[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

このトピックの最新バージョンをご覧[クラスとその他の種類 (クラス デザイナー) の使用](https://docs.microsoft.com/visualstudio/ide/working-with-classes-and-other-types-class-designer)します。
