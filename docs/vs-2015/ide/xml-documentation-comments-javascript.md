# <a name="xml-documentation-comments-javascript"></a>XML ドキュメント コメント (JavaScript)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

このトピックの最新バージョンをご覧[Visual Studio 2017 ドキュメント](/visualstudio/)します。  
  
JavaScript では、ソース コードの特別なコメント フィールドに XML タグを含めることで、コードのドキュメントを作成できます。 これらの XML タグは、JavaScript オブジェクトに関する情報を提供するために IntelliSense によって使用されます。  
  
 コード コメントの位置は、特定のタグによって異なります。  
  
 次のタグがサポートされます。  
  
-   [\<非推奨とされます >](../ide/deprecated-javascript.md)  
  
-   [\<フィールド >](../ide/field-javascript.md)  
  
-   [\<loc >](../ide/loc-javascript.md)  
  
-   [\<param >](../ide/param-javascript.md)  
  
-   [\<returns>](../ide/returns-javascript.md)  
  
-   [\<署名 >](../ide/signature-javascript.md)  
  
-   [\<summary>](../ide/summary-javascript.md)  
  
-   [\<value>](../ide/value-javascript.md)  
  
-   [\<var >](../ide/var-javascript.md)
