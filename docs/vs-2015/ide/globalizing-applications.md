# <a name="globalizing-applications"></a>アプリケーションのグローバル化
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

グローバリゼーションは、複数のロケールで機能するソフトウェア製品をデザインおよび開発するプロセスです。 このセクションの内容は、Windows フォームと Web フォームの両方のページに当てはまります。

## <a name="in-this-section"></a>このセクションの内容
 [グローバル Windows フォームと Web フォームのカルチャ固有のクラス](../ide/culture-specific-classes-for-global-windows-forms-and-web-forms.md) カルチャの設定に従って、日付、時刻、数値、通貨、およびその他の情報の書式を設定するクラスについて説明します。
