# <a name="how-to-print-class-diagrams-class-designer"></a>方法: クラス ダイアグラムを印刷する (クラス デザイナー)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

このトピックの最新バージョンをご覧[方法: クラス ダイアグラムを印刷する (クラス デザイナー)](https://docs.microsoft.com/visualstudio/ide/how-to-print-class-diagrams-class-designer)します。  
  
Visual Studio の印刷機能を利用し、クラス ダイアグラムを印刷できます。  
  
### <a name="to-print-a-class-diagram"></a>クラス ダイアグラムを印刷するには  
  
1.  クラス ダイアグラムを表示します。 (必要に応じて、[クラス ダイアグラム] タブを選択して表示します。)  
  
2.  **[ファイル]** メニューの **[印刷]** をクリックします。  
  
     クラス ダイアグラム全体が印刷されます。 場合によっては、適切なサイズで印刷するために、[ページ設定] ダイアログ ボックスで設定を調整する必要があります。
