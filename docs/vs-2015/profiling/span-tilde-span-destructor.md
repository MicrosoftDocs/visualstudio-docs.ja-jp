# <a name="spanspan-destructor"></a>span::~span デストラクター
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

`span` オブジェクトを破棄し、そのリソースを解放します。  
  
## <a name="syntax"></a>構文  
  
```  
~span();  
```  
  
## <a name="requirements"></a>必要条件  
 **ヘッダー:** cvmarkersobj.h  
  
 **名前空間:** Concurrency::diagnostic
