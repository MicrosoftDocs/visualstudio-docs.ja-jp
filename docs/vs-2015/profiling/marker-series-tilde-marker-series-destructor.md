# <a name="markerseriesmarkerseries-destructor"></a>marker_series::~marker_series デストラクター
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

このトピックの最新バージョンをご覧[marker_series:: ~ marker_series デストラクター](https://docs.microsoft.com/visualstudio/profiling/marker-series-tilde-marker-series-destructor)します。  
  
marker_series オブジェクトを破棄し、割り当てられているすべてのリソースを解放します。  
  
## <a name="syntax"></a>構文  
  
```  
~marker_series();  
```

 **ヘッダー:** cvmarkersobj.h  
  
 **名前空間:** Concurrency::diagnostic
