# <a name="xml-schema-designer"></a>XML スキーマ デザイナー
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

XML スキーマ デザイナー (XSD デザイナー) は、スキーマ セットをさまざまな抽象レベルで視覚的に表現するためのグラフィカル ツールです。 XSD デザイナーには、次の主要コンポーネントがあります。

- [Xml スキーマエクスプローラー](../xml-tools/xml-schema-explorer.md)を使用すると、xml スキーマツリーを参照して移動し、検索を実行できます。

- [スタートビュー](../xml-tools/start-view.md)は、XSD デザイナーの開始点です。[スタート] ビューでは、XSD デザイナーの他のビューに移動して、スキーマセットの詳細を確認できます。

- [グラフビュー](../xml-tools/graph-view.md)では、スキーマセットの概要とスキーマノード間のリレーションシップを確認できます。

- [コンテンツモデルビュー](../xml-tools/content-model-view.md)には、単純型、複合型、要素、グループ、属性、属性グループなど、ローカルスキーマノードとグローバルスキーマノードの詳細がグラフィカルに表示されます。
