# <a name="xslt-profiler"></a>XSLT プロファイラー
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

XSLT プロファイラーは、XSLT ドキュメントの開発とデバッグに役立つパフォーマンス分析プロファイラー ツールです。 XSLT プロファイラーを使用して詳細な XSLT パフォーマンス レポートを作成することにより、XSLT コード内のパフォーマンス関連の問題を計測および評価できるようになります。 XSLT プロファイラーには、XSL および XSLT のスタイル シートの最適化に役立つヒントが含まれており、最高のパフォーマンスが必要になる XSLT ベースのアプリケーションには不可欠です。  
  
 XSLT Profiler は Visual Studio 2010 の一部であるし、Visual Studio から使用**XML**メニュー。  
  
 ![XSLT Profiler](../xml-tools/media/xsltprofilermenu.gif "XSLTProfilerMenu")  
  
 XSLT プロファイラーは最初に Visual Studio 2008 のアドインとして導入されました。 詳細については、の関連リソース」セクションを参照してください。、 [XSLT Profiler for Visual Studio Feb 08 Community Technology Preview](http://go.microsoft.com/fwlink/?LinkId=142987)ページ。  
  
> [!NOTE]
>  Visual Studio 2010 の一部の画面は、これらのドキュメントの説明と異なる場合があります。これは、一部のビジュアル要素が Visual Studio 2008 から変更されたためです。
