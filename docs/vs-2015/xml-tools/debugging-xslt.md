# <a name="debug-xslt"></a>XSLT のデバッグ
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Visual Studio デバッガーを使用して XSLT をデバッグすることができます。 このデバッガーは、ブレークポイントの設定や、XSLT 実行状態の表示などをサポートします。 XSLT デバッグは Visual Studio Professional エディションで使用できます。

 [XSLT の既定のテンプレート](../xml-tools/xslt-default-templates.md)XSLT デバッガーが既定のテンプレートを処理する方法について説明します。
