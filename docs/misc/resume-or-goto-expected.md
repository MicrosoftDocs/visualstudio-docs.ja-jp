# &#39;Resume&#39; または &#39;GoTo&#39; が必要です
`On Error` ステートメントに、`Resume` 句も `GoTo` 句も含まれていません。 どちらか 1 つの句を使用する必要があります。  
  
 **エラー ID:** BC32019  
  
### このエラーを解決するには  
  
-   `Resume` 句か `GoTo` 句を `On Error` ステートメントに追加します。
