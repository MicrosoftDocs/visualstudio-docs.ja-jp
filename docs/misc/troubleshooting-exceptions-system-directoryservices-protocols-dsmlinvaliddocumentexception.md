# 例外のトラブルシューティング : System.DirectoryServices.Protocols.DsmlInvalidDocumentException
この例外は、DSML \(Directory Services Markup Language\) の要求または応答のドキュメントが、整形済みの XML ではない場合、または DSMLV2 スキーマで検証できない場合にスローされます。
