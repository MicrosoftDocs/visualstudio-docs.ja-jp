# 既定のアセンブリ参照 &#39;&lt;reference&gt;&#39; は既に自動的に追加されています。無視されます。
既定のアセンブリ参照を追加しようとしましたが、アセンブリが既に追加されているため、無視されます。  
  
 **エラー ID:** BC2024  
  
### このエラーを解決するには  
  
-   余分なコードを削除します。
