# ラベル &#39;&lt;labelname&gt;&#39; が定義されていません
行ラベルまたは行番号が参照されていますが、参照のスコープ内で定義されていません。 ラベルは、参照を含むプロシージャ内にある必要があります。  
  
 **エラー ID:** BC30132  
  
### このエラーを解決するには  
  
-   行ラベルと参照が同じプロシージャ内に含まれるようにコードを再構築します。
