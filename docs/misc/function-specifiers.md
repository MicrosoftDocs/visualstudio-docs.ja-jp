# 関数指定子
[inline](../misc/inline-inline-forceinline.md) キーワードと [virtual](/visual-cpp/cpp/virtual-specifier) キーワードは、関数宣言で指定子として使用できます。 この **virtual** の使い方は、クラス定義の base\-class 指定子の使い方とは異なります。
