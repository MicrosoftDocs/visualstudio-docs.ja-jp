# TargetFilePath は既存のフォルダーを指定します
パラメーター `TargetFilePath` は、既存のディレクトリを指定していますが、新しいディレクトリの名前を指定する必要があります。  
  
### このエラーを解決するには  
  
-   `TargetFilePath` を変更して、既存ではなく、新しいディレクトリを指定します。
