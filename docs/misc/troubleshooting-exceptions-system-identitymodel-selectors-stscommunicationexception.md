# 例外のトラブルシューティング : System.IdentityModel.Selectors.StsCommunicationException
<xref:System.IdentityModel.Selectors.StsCommunicationException> 例外は、セキュリティ トークン サービスとの通信に問題がある場合にスローされます。  
  
 この問題は、次の条件が原因で発生することがあります。  
  
-   ネットワーク エラーが発生した  
  
-   MEX エンドポイントに到達できない  
  
-   セキュリティ トークン サービスのエンドポイント ID が有効でない
