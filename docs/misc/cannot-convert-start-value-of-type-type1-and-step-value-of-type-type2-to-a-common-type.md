# 型 &#39;&lt;type1&gt;&#39; の初期値と型 &#39;&lt;type2&gt;&#39; のステップ値を共通の型に変換できません。
指定された `For...Next` コントロールの値は、型が異なり、共通の型に変換できないため、無効です。  
  
### このエラーを解決するには  
  
1.  すべての値を共通の型に変換できるように、1 つ以上の値の型を変更します。
