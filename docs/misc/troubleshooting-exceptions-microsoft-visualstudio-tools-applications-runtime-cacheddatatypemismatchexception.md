# 例外のトラブルシューティング : Microsoft.VisualStudio.Tools.Applications.Runtime.CachedDataTypeMismatchException
この例外は、キャッシュ内のオブジェクトの XML シリアル化状態が、キャッシュされたデータ マニフェストの型情報と一致しない場合にスローされます。
