# MSBuild エラー MSB3168
**MSB3168: 重複した項目 '\<パッケージ\>' は無視されます。**  
  
 この警告は、ブートストラップで 2 つの同じブートストラップ パッケージをインストールするように指定した場合に発生します。  この場合、ブートストラップは 1 つのインスタンスだけインストールします。  
  
### このエラーを解決するには  
  
-   MSBuild エラーの原因と解決方法の詳細については、「[追加のリソース](../msbuild/additional-msbuild-resources.md)」を参照してください。
