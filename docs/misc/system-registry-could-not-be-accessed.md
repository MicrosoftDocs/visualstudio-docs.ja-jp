# System registry could not be accessed.
このエラーが発生するのは一般的に、オペレーティング システムがレジストリ ファイルを見つけられない場合、ファイルが破損している場合、またはファイルの読み取りまたは書き込みに失敗した場合です。  
  
### このエラーを解決するには  
  
1.  Visual Studio を再インストールします。
