# 例外のトラブルシューティング : System.Printing.PrintingCanceledException
<xref:System.Printing.PrintingCanceledException> 例外は、キャンセルされた印刷ジョブにコードでアクセスしようとするときに発生します。  
  
## コメント  
 印刷機能を備え、印刷ジョブをキャンセルできる WPF \(Windows Presentation Foundation\) アプリケーションを作成している場合は、この例外を正しく処理してください。 この種の例外が未処理の場合、Windows Presentation Foundation アプリケーションで応答が停止する可能性があります。
