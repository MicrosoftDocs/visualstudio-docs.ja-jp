# この操作は、ファイルが閉じられているときにのみ実行することができます
ファイルが開いている場合は、ファイルが閉じられているときにのみ実行できる操作を実行しようとします。  
  
### このエラーを解決するには  
  
-   ファイルを閉じて、操作をやり直します。
