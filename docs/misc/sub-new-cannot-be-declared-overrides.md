# &#39;Sub New&#39; を &#39;Overrides&#39; として宣言することはできません
コンストラクターは、継承されたコンストラクターをオーバーライドすることを示します。 コンストラクターはオーバーライドできません。  
  
 **エラー ID:** BC30283  
  
### このエラーを解決するには  
  
-   `Sub` 宣言から `Overrides` キーワードを削除します。
