# This project cannot be added to the solution because it already exists in the solution in a hidden Solution Folder.
あるプロジェクトを 1 つのソリューションの複数の場所に共存させることはできません。  
  
### このエラーを解決するには  
  
1.  非表示のソリューション フォルダーを表示するには、非表示のソリューション フォルダーが存在するソリューションまたはソリューション フォルダーを選択し、**\[プロジェクト\]** メニューの **\[フォルダーの再表示\]** を選択します。
