# プロジェクト レベルの条件付きコンパイル定数の形式が有効ではありません: &lt;error&gt;
無効な条件付きコンパイル オプションが、プロジェクトに指定されています。  
  
 **エラー ID:** BC31031  
  
### このエラーを解決するには  
  
1.  このプロジェクトで使用されている条件付きコンパイル定数を確認します。
