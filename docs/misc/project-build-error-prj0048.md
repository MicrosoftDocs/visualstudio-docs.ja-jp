# プロジェクト ビルド エラー PRJ0048
ビルドは既に進行中であるため、ビルドを開始できませんでした。  
  
 別のビルドが実行中である場合、ビルドを開始することはできません。 現在のビルドが完了するまで待つか、またはビルドを取り消します。
