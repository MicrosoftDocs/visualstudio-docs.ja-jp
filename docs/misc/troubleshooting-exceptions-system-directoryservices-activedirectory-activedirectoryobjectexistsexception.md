# 例外のトラブルシューティング : System.DirectoryServices.ActiveDirectory.ActiveDirectoryObjectExistsException
この例外は、Active Directory オブジェクトが作成され、基になるディレクトリ ストアにそのオブジェクトが既に存在する場合にスローされます。
