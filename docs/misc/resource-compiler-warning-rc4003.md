# リソース コンパイラの警告 RC4003
マクロ 'identifier' に指定された実引数の数が少なすぎます  
  
 指定されている識別子に指定された実引数の数が、識別子のマクロ定義で指定された仮引数の数を下回っています。  
  
 定義で仮パラメーターが参照されており、それに対応する実パラメーターを指定しなかった場合、マクロ展開で空のテキストに置き換えられます。
