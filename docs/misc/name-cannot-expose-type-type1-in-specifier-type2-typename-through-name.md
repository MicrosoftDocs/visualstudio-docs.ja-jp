# &#39;&lt;name&gt;&#39; が、&lt;name&gt; によって &lt;specifier&gt; &lt;type2&gt; &#39;&lt;typename&gt;&#39; で型 &lt;type1&gt;を公開できません。
パブリック クラス `MyClass` の外部でプライベート型を公開しようとしました。  
  
 **エラー ID:** BC30508  
  
### このエラーを解決するには  
  
-   型 `Public` を宣言します。
