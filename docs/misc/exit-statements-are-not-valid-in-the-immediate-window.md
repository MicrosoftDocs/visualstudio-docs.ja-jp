# &#39;Exit&#39; ステートメントは、イミディエイト ウィンドウでは有効ではありません
`Exit` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30713  
  
### このエラーを解決するには  
  
-   デバッグ コードから `Exit` ステートメントを削除します。
