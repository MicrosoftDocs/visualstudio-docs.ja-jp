# CA0502
サポートされていない言語です。  
  
 このエラーは、C\# や Visual Basic 以外の言語で作成されたページやクラスが含まれる ASP.NET プロジェクトを分析しようとしたときに発生します。  このエラーを修正するには、そのファイルをプロジェクトから削除するか、コード分析を無効にします。
