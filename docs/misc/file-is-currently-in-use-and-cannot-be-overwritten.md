# File is currently in use and cannot be overwritten.
現在のファイルは保存できません。  このエラーが発生するのは一般的に、同時に開いているファイルと同じ名前を使用して、あるファイルを保存しようとした場合です。  
  
### このエラーを解決するには  
  
1.  同じ名前のファイルを閉じてやり直します。  
  
     または  
  
     ファイルの名前を変更して保存します。
