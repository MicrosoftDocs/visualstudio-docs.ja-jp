# Some Unicode characters in this file will not be saved.
通常、このエラーが発生するのは、Unicode コード ページを適用する以外に保存できない文字がファイルに含まれる場合です。  
  
### Unicode 文字が含まれるファイルを保存するには  
  
1.  **\[キャンセル\]** をクリックします。  
  
2.  **\[ファイル\]** メニューの **\[保存オプションの詳細設定\]** をクリックします。  
  
3.  **\[保存オプションの詳細設定\]** ダイアログ ボックスで、ファイルに合ったエンコーディングを選択します。
