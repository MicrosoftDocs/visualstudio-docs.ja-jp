# &#39;Error&#39; ステートメントは、イミディエイト ウィンドウでは有効ではありません
`Error` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30724  
  
### このエラーを解決するには  
  
-   デバッグ コードから `Error` ステートメントを削除します。
