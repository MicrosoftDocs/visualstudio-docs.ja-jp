# &#39;&lt;assemblyname&gt;&#39; はアセンブリであるため、モジュールとして参照できません
アセンブリをモジュールとして使用しようとしました。  
  
 **エラー ID:** BC31077  
  
### このエラーを解決するには  
  
-   モジュールへの参照を変更します。
