# Unable to retrieve resource information
.resx ファイルの階層構造をスキャンするのに失敗しました。  
  
 `Preparing resources` のビルド手順の過程で \(**出力**ウィンドウに表示される\)、拡張子 .resx の全ファイルのプロジェクト階層構造がプロジェクト システムによってスキャンされます。  これにより、.resources バイナリ ファイルに変換する必要のある .resx という XML ファイルの一覧が作成され、プロジェクト出力にリソースとしてインクルードできるようになります。  
  
 **このエラーを解決するには**  
  
-   [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] を再起動します。  それでもエラーが発生する場合は、この問題をマイクロソフト プロダクト サポート サービスまでご連絡ください。  
  
     このエラーによりビルド エラーが発生します。
