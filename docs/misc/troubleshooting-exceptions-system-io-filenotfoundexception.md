# 例外のトラブルシューティング : System.IO.FileNotFoundException
<xref:System.IO.FileNotFoundException> 例外は、ディスク上に存在しないファイルまたはディレクトリにアクセスしようとするとスローされます。  
  
## 関連するヒント  
 **指定された場所にファイルが存在することを確認します。**  
 ファイルが存在しない場合は、この例外がスローされます。 詳細については、「<xref:Microsoft.VisualBasic.FileIO.FileSystem.FileExists%2A>」を参照してください。  
  
 **相対パスを使用するとき、現在のディレクトリが正しいことを確認します。**  
 現在のディレクトリが正しくない場合は、相対パスも正しくありません。
