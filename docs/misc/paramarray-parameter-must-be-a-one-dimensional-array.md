# ParamArray パラメーターは 1 次元配列でなければなりません
`ParamArray` 引数は、1 次元配列型として宣言する必要があります。 この宣言では、`ParamArray` 引数に複数の次元を指定しています。  
  
 **エラー ID:** BC30051  
  
### このエラーを解決するには  
  
-   `ParamArray` 引数名に続くかっこ内のすべてのコンマを削除します。
