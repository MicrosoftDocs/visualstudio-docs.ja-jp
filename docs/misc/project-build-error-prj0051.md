# プロジェクト ビルド エラー PRJ0051
'file' をターゲット ディレクトリにコピーできませんでした。  対象のファイルが使用中でないことを確認してください。  
  
 ビルド システムがファイルをコピーできませんでした。  
  
 ファイルがコンピューターで開いていないこと、または別のプログラムで使用中ではないことをご確認ください。
