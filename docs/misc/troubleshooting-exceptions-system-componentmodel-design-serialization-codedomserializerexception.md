# 例外のトラブルシューティング : System.ComponentModel.Design.Serialization.CodeDomSerializerException
この例外は、CodeDOM シリアライザーでエラーが検出された場合にスローされます。  
  
## コメント  
 この例外によって、シリアライザーからエラーの位置や種類に関する情報が返されます。
