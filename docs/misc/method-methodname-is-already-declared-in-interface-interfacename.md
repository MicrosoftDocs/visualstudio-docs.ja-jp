# メソッド &#39;&lt;methodname&gt;&#39; は、インターフェイス &#39;&lt;interfacename&gt;&#39; で既に宣言されています
重複してインターフェイス メソッドの 2 回目の宣言を行いました。  
  
 **エラー ID:** BC30377  
  
### このエラーを解決するには  
  
-   重複している宣言を削除します。
