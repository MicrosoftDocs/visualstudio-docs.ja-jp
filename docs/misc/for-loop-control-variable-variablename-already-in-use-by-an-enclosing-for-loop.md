# For Loop コントロール変数 &#39;&lt;variablename&gt;&#39; は、それを囲む For loop によって既に使用されています
入れ子になったすべての `For` ループが一意のコントロール変数を使用する必要があります。  
  
 **エラー ID:** BC30069  
  
### このエラーを解決するには  
  
-   コントロールの変数名のいずれかを変更します。
