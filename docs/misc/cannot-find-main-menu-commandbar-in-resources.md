# Cannot find main menu commandbar in resources.
このエラーは、統合開発環境 \(IDE: Integrated Development Environment\) の起動時に、MenuBar コマンド バーが見つからない場合に発生します。  .dll ファイルが削除されたか、移動されたか、または破損しています。  
  
### このエラーを解決するには  
  
-   **\[プログラムの追加と削除\]** を使用して、本製品を修復または再インストールします。
