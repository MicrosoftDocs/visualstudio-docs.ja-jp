# クラス &#39;&lt;classname&gt;&#39; には、オーバーライドされていない &#39;MustOverride&#39; メンバーが含まれているため、&#39;New&#39; は使用できません。
オーバーライドされていない `MustOverride` メンバーを含むクラスで `New` の使用を試みました。  
  
 **エラー ID:** BC30376  
  
### このエラーを解決するには  
  
-   `New` ステートメントを削除します。
