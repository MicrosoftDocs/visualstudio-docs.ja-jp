# パスの長さが 259 文字を超えるため、アセンブリ &#39;&lt;assemblyname&gt;&#39; を作成できません
このアセンブリのパスが 259 文字の最大長を超えています。  
  
 **エラー ID:** BC31518  
  
### このエラーを解決するには  
  
-   短い名前を使用するか浅いファイル階層を選択して、短いパスを作成します。
