# &#39;GoTo&#39; ステートメントは、イミディエイト ウィンドウでは有効ではありません。
`GoTo` ステートメントは分岐を行います。これはデバッグ コンテキストで使用できません。  
  
 **エラー ID:** BC30133  
  
### このエラーを解決するには  
  
-   **\[イミディエイト\]** ウィンドウで `GoTo` ステートメントを実行しないようにします。
