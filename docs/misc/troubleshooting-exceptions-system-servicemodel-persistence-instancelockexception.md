# 例外のトラブルシューティング : System.ServiceModel.Persistence.InstanceLockException
<xref:System.ServiceModel.Persistence.InstanceLockException> 例外は、インスタンスがロック状態であるため操作を実行できない場合に、<xref:System.ServiceModel.Persistence.LockingPersistenceProvider> を継承するクラスによって使用されます。
