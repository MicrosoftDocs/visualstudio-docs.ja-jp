# 例外のトラブルシューティング : System.IO.PathTooLongException
<xref:System.IO.PathTooLongException> 例外は、パス名またはファイル名がシステム定義の最大長を超えている場合にスローされます。  
  
## 関連するヒント  
 **パスの長さがシステムで定義された最大値を超えないことを確認します。**  
 Windows ベースのプラットフォームでは、パスは 248 文字未満、ファイル名は 260 文字未満にする必要があります。
