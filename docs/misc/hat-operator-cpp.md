# ^ 演算子 (C++)
`^` 演算子は、Visual C\+\+ ではコンテキストに応じて 2 つの意味を持ちます。  詳細については、次のトピックを参照してください。  
  
-   「[ビット処理排他的 OR 演算子: ^](../Topic/Bitwise%20Exclusive%20OR%20Operator:%20%5E.md)」。あらゆる種類の C\+\+ で使用されます。  
  
-   「[オブジェクト演算子 \(^\) へのハンドル](/visual-cpp/windows/handle-to-object-operator-hat-cpp-component-extensions)」。[!INCLUDE[cppwrt_short](../misc/includes/cppwrt_short_md.md)] および [!INCLUDE[cppcli](../misc/includes/cppcli_md.md)] で使用されます。
