# 構文エラー (Visual Basic エラー)
ソース コード行は、不特定の構文エラーを生成します。  
  
 **エラー ID:** BC30035  
  
### このエラーを解決するには  
  
1.  ソース コード行で使用される各キーワードのドキュメントを確認します。 ドキュメントのコード例と、このエラーを生成するソース行とを比較します。  
  
2.  エラーの原因を特定できない場合は、状況に関する情報を収集し、マイクロソフト プロダクト サポート サービスにお知らせください。
