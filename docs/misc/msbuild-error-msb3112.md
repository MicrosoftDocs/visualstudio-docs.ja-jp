# MSBuild エラー MSB3112
**MSB3112: 同じ ID '\<アセンブリ\>' が指定されたアセンブリが 2 つ以上あります。**  
  
 この警告は、同じ ID を持つ参照アセンブリが複数ある場合に生成されます。  
  
### このエラーを解決するには  
  
-   参照アセンブリのどれかをプロジェクトから削除します。
