# プロパティを &#39;&lt;modifiername&gt;&#39; として宣言することはできません
`Property` 宣言で使用されている構文にエラーがあります。 通常、このエラーは、`WithEvents` などの指定されたキーワードを `Property` 宣言で使用できないことを意味します。  
  
 **エラー ID:** BC30639  
  
### このエラーを解決するには  
  
-   `Property` 宣言の構文を確認します。
