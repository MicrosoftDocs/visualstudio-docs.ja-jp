# Add-in &lt;name&gt; caused an access violation. Remove it from the Add-in Manager?
このアドインがクラッシュしたため、統合開発環境でほかの問題が発生する可能性があります。  
  
### このエラーを解決するには  
  
1.  \[はい\] をクリックして、使用できるアドインの一覧からこのアドインを削除します。  
  
     または  
  
     \[いいえ\] をクリックします。  
  
    > [!NOTE]
    >  アドインの作成者に連絡し、この問題を報告してください。
