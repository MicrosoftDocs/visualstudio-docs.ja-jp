# 初期化エラー
プログラムの初期化中にエラーが発生しました。 インストール中にインストールを途中でキャンセルしたことが原因になっている可能性があります。  
  
 **エラー ID:** BC2000  
  
### このエラーを解決するには  
  
-   インストールをやり直します。
