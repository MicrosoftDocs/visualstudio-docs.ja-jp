# Object with program ID &lt;number&gt; cannot be created.
使用しようとしているクラス文字列が無効です。  このメッセージは、オペレーティング システムのエラーです。  
  
### このエラーを解決するには  
  
1.  Visual Studio または影響を受けているアプリケーションを再インストールしてからやり直します。
