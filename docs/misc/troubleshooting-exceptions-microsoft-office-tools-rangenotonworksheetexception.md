# 例外のトラブルシューティング : Microsoft.Office.Tools.RangeNotOnWorksheetException
<xref:Microsoft.Office.Tools.RangeNotOnWorksheetException> 例外は、ビュー コントロールが、その Controls コレクションに対応するワークシートではない範囲で、プログラムによって作成されたときにスローされます。  
  
## 関連するヒント  
 指定する範囲が、正しいワークシートに対応するようにします。
