# 例外のトラブルシューティング : System.MulticastNotSupportedException
この例外は、組み合わせることができないデリゲートの種類のインスタンスを、組み合わせようとした場合にスローされます。 オペランドの 1 つが null 参照 \(Visual Basic では `Nothing`\) であると、例外はスローされません。
