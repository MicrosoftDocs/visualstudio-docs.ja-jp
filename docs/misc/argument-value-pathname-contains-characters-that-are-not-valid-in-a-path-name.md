# 引数値 &#39;&lt;pathname&gt;&#39; は、パス名に有効でない文字を含んでいます。
パス名に無効な文字が 1 つ以上含まれています。  
  
### このエラーを解決するには  
  
-   パス名から無効な文字を削除します。
