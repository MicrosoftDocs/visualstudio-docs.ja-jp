# 例外のトラブルシューティング : System.IdentityModel.Selectors.IdentityValidationException
<xref:System.IdentityModel.Selectors.IdentityValidationException> 例外は、受信者の証明書が有効でない場合にスローされます。 たとえば、証明書をルート証明書まで正常にチェーンできない場合などに発生します。
