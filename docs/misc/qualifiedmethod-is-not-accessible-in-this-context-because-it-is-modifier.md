# &#39;&lt;qualifiedmethod&gt;&#39; は &#39;&lt;modifier&gt;&#39; であるため、このコンテキストではアクセスできません
修飾子が適用されているためにこのコンテキストでアクセスできないメソッドにアクセスしようとしました。  
  
 **エラー ID:** BC30390  
  
### このエラーを解決するには  
  
-   修飾子を削除します。
