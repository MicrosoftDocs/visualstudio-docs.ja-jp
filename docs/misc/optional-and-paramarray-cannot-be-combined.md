# &#39;Optional&#39; と &#39;ParamArray&#39; を組み合わせて使用することはできません
`Optional` 修飾子と `ParamArray` 修飾子は同時に指定できません。  
  
 **エラー ID:** BC30642  
  
### このエラーを解決するには  
  
-   `ParamArray` または `Optional` を指定し、両方は指定しないでください。
