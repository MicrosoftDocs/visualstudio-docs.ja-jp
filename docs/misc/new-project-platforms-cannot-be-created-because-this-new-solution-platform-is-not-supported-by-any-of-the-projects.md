# New project platforms cannot be created because this new solution platform is not supported by any of the projects.
このエラーは、**\[新しいプロジェクト プラットフォームを作成する\]** が選択されていて、どのプロジェクトでも新しいソリューション プラットフォームがサポートされていない場合に発生します。  
  
### このエラーを解決するには  
  
-   **\[新しいプロジェクト プラットフォームを作成する\]** の選択を解除し、**\[OK\]** をクリックします。  
  
     または  
  
     **\[新しいプロジェクト プラットフォームを作成する\]** の選択を解除し、ドロップダウン リストで別のプラットフォームを選択してから、**\[OK\]** をクリックします。
