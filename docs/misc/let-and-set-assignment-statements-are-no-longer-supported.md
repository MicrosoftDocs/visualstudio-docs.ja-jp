# &#39;Let&#39; および &#39;Set&#39; 代入ステートメントは、現在サポートされていません
`Let` および `Set` は代入で使用できません。  
  
 **エラー ID:** BC30807  
  
### このエラーを解決するには  
  
-   代入ステートメントから `Let` および `Set` を削除します。次に例を示します。  
  
     `X = 2`
