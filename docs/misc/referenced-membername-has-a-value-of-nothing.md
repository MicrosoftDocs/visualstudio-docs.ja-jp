# 参照された &#39;&lt;membername&gt;&#39; には値 &#39;Nothing&#39; が指定されています
参照されている項目に値が割り当てられていないか、値 `Nothing` が割り当てられました。  
  
 **エラー ID:** BC30759  
  
### このエラーを解決するには  
  
-   項目名のスペルが正しいこと、および正しく設定されていることを確認します。
