# 部分メソッド &#39;&lt;methodname&gt;&#39; では &#39;Implements&#39; キーワードを使用できません
部分メソッド定義では、インターフェイスまたはインターフェイス メンバーを実装できません。  
  
 **エラー ID:** BC31439  
  
### このエラーを解決するには  
  
1.  部分メソッド定義から `Implements` ステートメントを削除します。
