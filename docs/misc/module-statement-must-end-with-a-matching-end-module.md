# &#39;Module&#39; ステートメントの終わりには、対応する &#39;End Module&#39; を指定しなければなりません
モジュール ブロックは、`End``Module` ステートメントで終了する必要があります。  
  
 **エラー ID:** BC30625  
  
### このエラーを解決するには  
  
-   `End Module` ステートメントを `Module` ブロックの最後に追加します。
