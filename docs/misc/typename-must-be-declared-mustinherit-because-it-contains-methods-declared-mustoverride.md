# &#39;&lt;typename&gt;&#39; には、&#39;MustOverride&#39; として宣言されているメソッドが含まれているため、&#39;MustInherit&#39; として宣言する必要があります
`MustOverride` メンバーを含む型は、`MustInherit` として宣言する必要があります。  
  
 **エラー ID:** BC31411  
  
### このエラーを解決するには  
  
-   型に `MustInherit` 修飾子を追加します。
