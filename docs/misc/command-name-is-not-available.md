# Command &lt;name&gt; is not available.
このエラーが発生するのは一般的に、**\[コマンド\]** ウィンドウまたは **\[検索\]** ボックスに、Visual Studio の現在の状態に対して有効でないコマンドを入力した場合です。  たとえば、Visual Studio でソリューションが開かれていない場合、`File.CloseSolution`  コマンドは有効ではありません。  
  
### このエラーを解決するには  
  
1.  別のコマンドを選択します。  
  
     または  
  
     使用するコマンドの選択を変更します。
