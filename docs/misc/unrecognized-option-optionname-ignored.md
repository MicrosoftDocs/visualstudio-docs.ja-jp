# オプション &lt;optionname&gt; を認識できません。無視されました
コマンド ラインから無効なオプションが指定されています。  
  
 **エラー ID:** BC2007  
  
### このエラーを解決するには  
  
-   無効なオプションがないか、指定したオプションを確認します。
