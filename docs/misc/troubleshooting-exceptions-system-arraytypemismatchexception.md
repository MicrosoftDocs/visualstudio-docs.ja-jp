# 例外のトラブルシューティング : System.ArrayTypeMismatchException
配列で宣言された型に要素を変換できないと、<xref:System.ArrayTypeMismatchException> 例外がスローされます。  
  
## 関連するヒント  
 **オブジェクトの型が配列型に変換できることを確認します。**  
 2 つの型の間で変換できる必要があります。  
  
 詳細については、「[配列](/dotnet/visual-basic/programming-guide/language-features/arrays/index)」を参照してください \(Visual Basic を使用している場合\)。
