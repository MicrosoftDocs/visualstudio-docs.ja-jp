# 例外のトラブルシューティング : System.Windows.Media.Animation.AnimationException
<xref:System.Windows.Media.Animation.AnimationException> 例外は、プロパティをアニメーション化するときにエラーが発生した場合にスローされます。  
  
## コメント  
 このクラスは、負の <xref:System.Windows.Media.Animation.Timeline.Duration%2A> を指定するか、<xref:System.Windows.Media.Animation.Timeline.AccelerationRatio%2A> や <xref:System.Windows.Media.Animation.Timeline.DecelerationRatio%2A> に無効な値を設定してアニメーションを作成した場合など、タイミングに関連するエラーには使用されません。
