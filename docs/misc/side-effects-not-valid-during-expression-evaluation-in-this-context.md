# このコンテキストでは、式の評価中に副作用は使用できません
このコンテキストでは、オブジェクトの状態を変更するステートメントは使用できません。  
  
 **エラー ID:** BC30700  
  
### このエラーを解決するには  
  
-   その他の項目の状態を変更せずにステートメントを評価できるように、コードを変更します。
