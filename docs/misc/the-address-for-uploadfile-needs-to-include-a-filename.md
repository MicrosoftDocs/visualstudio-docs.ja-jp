# UploadFile のアドレスにはファイル名が含まれていなければなりません
`UploadFile` メソッドでファイルをアップロードする場合、アップロード先のパスにファイル名を指定する必要があります。  
  
### このエラーを解決するには  
  
-   ファイル名を指定します。
