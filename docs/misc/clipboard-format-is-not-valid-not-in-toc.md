# Clipboard format is not valid.
クリップボードは、特定の形式のデータだけを受け付けます。  このエラーが発生するのは一般的に、クリップボードに置こうとしたコンポーネントからのデータが、クリップボードと互換性のない形式である場合です。  
  
### このエラーを解決するには  
  
1.  コンポーネントのドキュメントを参照して、コンポーネントからデータを転送するときにクリップボードを使用できるかどうかを確認します。  
  
     または  
  
2.  他の項目を選択してやり直してください。
