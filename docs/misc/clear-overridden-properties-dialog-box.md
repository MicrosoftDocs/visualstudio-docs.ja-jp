# [オーバーライドされたプロパティのクリア] ダイアログ ボックス
> [!NOTE]
>  このダイアログ ボックスは [!INCLUDE[vs_dev10_long](../code-quality/includes/vs_dev10_long_md.md)] ではサポートされていません。  
  
 ユーザー定義プロパティ シート \(.props ファイル\) をプロジェクト構成に追加すると、既定では、そのプロパティ シートで定義されたすべてのプロパティがプロジェクト構成に継承されます。  このダイアログ ボックスは、プロジェクト構成に継承されるのが望ましくないプロパティを削除するために使用します。
