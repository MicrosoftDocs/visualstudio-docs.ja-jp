# &#39;&lt;labelname&gt;&#39; は、このステートメントを含まない &#39;SyncLock&#39; ステートメントの内側にあるため、&#39;GoTo &lt;labelname&gt;&#39; は有効ではありません
`SyncLock` ブロックに分岐することはできません。  
  
 **エラー ID:** BC30755  
  
### このエラーを解決するには  
  
-   `SyncLock` ブロックの前にラベルが配置されるように、コードを再構成します。
