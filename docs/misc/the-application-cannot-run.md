# The application cannot run.
Visual Studio の実行に必要な十分なメモリがありません。  
  
### このエラーを解決するには  
  
1.  Visual Studio の実行に必要な最小要件をコンピューターが満たしていることを確認します。  
  
2.  コンピューターで実行中のほかのプログラムを閉じて、メモリを解放します。  
  
     または  
  
     Visual Studio を再インストールします。
