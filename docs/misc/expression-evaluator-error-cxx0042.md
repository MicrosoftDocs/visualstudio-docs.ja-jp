# 式エバリュエーター エラー CXX0042
行番号が正しくありません。  
  
 このエラーが発生することは許されません。  
  
 エラーの状況を確認して、Microsoft Corporation に通知してください。[http:\/\/support.microsoft.com\/support\/visualc\/report\/](http://support.microsoft.com/support/visualc/report/) ではオンラインでエラーを報告することができます。また、マイクロソフト製品サポート サービスに電話をかけることができます。 詳細については、Visual C\+\+ の **\[ヘルプ\]** メニューで、**\[テクニカル サポート\]** をクリックしてください。  
  
 このエラーは CAN0042 と同じです。
