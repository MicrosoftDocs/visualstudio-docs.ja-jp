# 内部コンパイラ エラー: コード ジェネレーターが正しくない入力を取得しました
[!INCLUDE[vbprvb](../code-quality/includes/vbprvb_md.md)] コンパイラが内部で記述したファイルの 1 つを正しく解釈できませんでした。  
  
 **エラー ID:** BC30011  
  
### このエラーを解決するには  
  
1.  プログラムをもう一度コンパイルし、エラーがまだ発生するかどうか確認します。  
  
2.  エラーがまだ発生する場合は、[!INCLUDE[vbprvb](../code-quality/includes/vbprvb_md.md)] コンパイラを再インストールします。  
  
3.  再インストールした後もエラーが再発する場合は、状況に関する情報を収集し、マイクロソフト プロダクト サポート サービスに通知してください。
