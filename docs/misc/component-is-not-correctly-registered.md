# Component is not correctly registered.
このエラーが発生するのは一般的に、レジストリが破損したためにシステムからレジストリにアクセスできない場合か、項目が正しい形式でレジストリに書き込まれていない場合です。  
  
### このエラーを解決するには  
  
1.  コンポーネントを再インストールします。
