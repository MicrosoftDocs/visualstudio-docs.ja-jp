# 例外のトラブルシューティング : Microsoft.VisualStudio.Tools.Applications.Runtime.UpdateFailedException
この例外は、ドキュメント内のアプリケーション マニフェストとは異なるバージョンのアプリケーション マニフェストを配置マニフェストが参照しても、ドキュメント内のアプリケーション マニフェストが更新されない場合にスローされます。
