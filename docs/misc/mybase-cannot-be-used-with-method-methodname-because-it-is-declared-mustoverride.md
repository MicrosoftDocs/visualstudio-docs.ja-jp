# メソッド &#39;&lt;methodname&gt;&#39; は &#39;MustOverride&#39; として宣言されているため、&#39;MyBase&#39; と共に使用することはできません。
`MustOverride` が宣言されているメソッドと共に `MyBase` を使用しようとしました。  
  
 **エラー ID:** BC30399  
  
### このエラーを解決するには  
  
-   `MustOverride` 宣言を削除します。
