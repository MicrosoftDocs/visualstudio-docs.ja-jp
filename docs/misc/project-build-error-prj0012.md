# プロジェクト ビルド エラー PRJ0012
システム リソースが非常に少ない可能性があります。 ビルドの起動に必要なスレッドを作成できません。  
  
 システム リソースが低い場合:  
  
-   実行中のアプリケーションのいくつかを終了します。  
  
-   Visual Studio を再起動してリソースを解放します。  
  
-   再起動します。
