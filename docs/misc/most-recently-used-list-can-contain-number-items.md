# Most recently used list can contain &lt;number&gt; items.
MRU リスト （最後に使用した一覧） には、最大 24 項目を含むことができます。  このエラーが発生するのは一般的に、\[オプション\] ダイアログ ボックスの \[環境\] の全般ペインにある \[最後に使用した一覧に表示する項目 \<number\> 項目\] ボックスに、1 より小さい数値または 24 より大きい数値を入力した場合です。  
  
### このエラーを解決するには  
  
1.  ボックスに 1 ～ 24 の間の数値を入力します。
