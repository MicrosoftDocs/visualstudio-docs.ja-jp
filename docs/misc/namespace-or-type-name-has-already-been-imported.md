# 名前空間または型 &#39;&lt;name&gt;&#39; は既にインポートされています
名前空間を複数回インポートしようとしました。  
  
 **エラー ID:** BC31051  
  
### このエラーを解決するには  
  
-   不要な `Imports` ステートメントを削除します。
