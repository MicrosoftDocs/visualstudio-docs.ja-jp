# 例外のトラブルシューティング : System.ObjectDisposedException
閉じられたストリームやレジストリ キーなど、破棄されたオブジェクトを操作しようとすると <xref:System.ObjectDisposedException> 例外がスローされます。  
  
## 関連するヒント  
 **リソースを使用する前にリソースを解放していないことを確認します。**  
 たとえば、ストリームを操作する場合、そのストリームが閉じられていないことを確認します。
