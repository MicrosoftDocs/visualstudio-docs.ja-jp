# 例外のトラブルシューティング : System.IdentityModel.Tokens.SecurityTokenValidationException
<xref:System.IdentityModel.Tokens.SecurityTokenValidationException> 例外は、受け取ったセキュリティ トークンが無効である場合にスローされます。 通常、この例外は、<xref:System.IdentityModel.Selectors.SecurityTokenAuthenticator.ValidateToken%2A> メソッドによってスローされます。
