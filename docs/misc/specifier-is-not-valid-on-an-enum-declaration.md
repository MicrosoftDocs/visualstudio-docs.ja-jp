# &#39;&lt;specifier&gt;&#39; は、Enum 宣言では有効ではありません
`Enum` 宣言で無効な指定子が使用されました。  
  
 **エラー ID:** BC30396  
  
### このエラーを解決するには  
  
1.  指定子を削除します。
