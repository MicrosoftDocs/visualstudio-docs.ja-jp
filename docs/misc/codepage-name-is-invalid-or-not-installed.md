# コードページ &#39;&lt;name&gt;&#39; は無効か、インストールされていません
指定されたコードページは無効か、見つかりません。  
  
 **エラー ID:** BC2016  
  
### このエラーを解決するには  
  
-   コードをチェックして、特定の状況に適したソリューションを判断します
