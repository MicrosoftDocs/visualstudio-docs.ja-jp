# MSBuild エラー MSB8020
*\<プラットフォーム\>* \(プラットフォーム ツールセット \= '*\<バージョン\>*'\) のビルド ツールが見つかりません。*\<バージョン\>* ビルド ツールを使用してビルドするには、*\<プラットフォーム\>* ビルド ツールをインストールしてください。  または、\[プロジェクト\] メニューを選択するかソリューションを右クリックし \[ソリューションの再ターゲット\] を選択して、現在の Visual Studio ツールにアップグレードすることもできます。
