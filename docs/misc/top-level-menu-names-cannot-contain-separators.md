# Top-level menu names cannot contain separators.
このエラーが発生するのは一般的に、ダッシュ \(\-\) などの区切り記号を含むテキストをメニュー名にドラッグしようとした場合です。  最上位のメニューには、メニュー項目のセットを区切るために予約されている文字は使用できません。  
  
### このエラーを解決するには  
  
1.  区切り記号を使用しないテキストを入力します。
