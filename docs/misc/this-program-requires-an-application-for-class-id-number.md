# This program requires an application for class ID &lt;number&gt;.
要求されたアプリケーションが見つかりません。  このメッセージは、オペレーティング システムのエラーです。  
  
### このエラーを解決するには  
  
1.  該当するアプリケーションを再インストールしてやり直します。  
  
     または  
  
     Visual Studio を再インストールしてやり直します。
