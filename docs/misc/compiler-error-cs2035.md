# コンパイラ エラー CS2035
コマンド ライン構文エラー: 'compiler\_option' オプションには ':\<number\>' が必要です  
  
 一部のコンパイラ オプションでは、値が必要です。  
  
## 使用例  
 次の例では CS2035 が生成されます。  
  
```  
// CS2035.cs // compile with: /baseaddress // CS2035 expected  
```
