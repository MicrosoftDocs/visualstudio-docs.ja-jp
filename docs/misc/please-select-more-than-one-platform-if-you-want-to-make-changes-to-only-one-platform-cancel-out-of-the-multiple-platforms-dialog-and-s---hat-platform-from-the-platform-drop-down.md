# 2 つ以上のプラットフォームを選択してください。 1 つのプラットフォームのみを変更する場合は、[複数のプラットフォーム] ダイアログ ボックスで [キャンセル] をクリックして終了し、[プラットフォーム] ボックスの一覧からプラットフォームを選択してください。
このエラーが発生するのは、一般的に、**\[複数のプラットフォーム\]** ダイアログ ボックスで 1 つのプラットフォームが選択された場合です。  
  
### このエラーを解決するには  
  
1.  追加のプラットフォームを選択します。  
  
     または  
  
2.  1 つのプラットフォームを更新する場合は、\[キャンセル\] を選択して、\[プラットフォーム\] ボックスの一覧で正しいプラットフォームを選択します。
