# コンパイラ エラー CS0648
'type' はこの言語ではサポートされていない型です  
  
 別の言語 \(C\+\+ など\) から生成されたメタデータに、マネージ型としてマークされていない型が含まれていました。 この型に関する情報はメタデータに含まれていますが、C\# で作成するプログラムにこの型を使用することはできません。
