# リソース コンパイラの致命的なエラー RC1107
使用法が無効です。RC \/? を使用します \(ヘルプ用\)  
  
 無効なリソース コンパイラ オプションが指定されました。 有効なオプションは、\/r、\/v、\/d、\/fo、\/l、\/i、\/x、および\/c です。
