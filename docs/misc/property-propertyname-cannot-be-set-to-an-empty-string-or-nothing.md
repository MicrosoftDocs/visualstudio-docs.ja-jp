# プロパティ &lt;propertyname&gt; は、空の文字列または Nothing に設定することはできません
プロパティには、空の文字列 \(""\) または `Nothing` 以外の値が必要です。 値の計算に誤りがある可能性があります。  
  
### このエラーを解決するには  
  
-   有効な値を指定します。
