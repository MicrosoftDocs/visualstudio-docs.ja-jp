# 引数 &#39;Access&#39; は有効ではありません Output モードの有効な値は、&#39;OpenAccess.Write&#39; および &#39;OpenAccess.Default&#39; です
Output モードでアクセスの値が正しくありません。  
  
### このエラーを解決するには  
  
1.  値を `OpenAccess.Write` または `OpenAccess.Default` のどちらかに変更します。
