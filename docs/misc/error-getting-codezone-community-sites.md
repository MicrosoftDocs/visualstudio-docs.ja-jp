# Error getting Codezone Community sites.
このエラーは、オンライン Web サービスへの接続が見つからない場合に発生します。  この Web サービスにより、**\[オプション\]** ダイアログ ボックスの **\[ヘルプ\]** の **\[環境\]** 下にある **\[オンライン\]** ページの情報が作成されます。  
  
### このエラーを解決するには  
  
1.  アクティブなインターネット接続が存在することを確認します。  
  
     または  
  
     プロキシ サーバー経由でインターネットに接続している場合は、プロキシ サーバーの設定を確認します。  
  
2.  もう一度オンライン ページにアクセスしてみます。
