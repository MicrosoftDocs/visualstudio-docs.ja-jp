# コンパイラ エラー CS1555
Main メソッドに指定された 'class' が見つかりませんでした  
  
 [\/main](/dotnet/csharp/language-reference/compiler-options/main-compiler-option) コンパイラ オプションにクラスが指定されましたが、クラス名がソース コードに見つかりませんでした。
