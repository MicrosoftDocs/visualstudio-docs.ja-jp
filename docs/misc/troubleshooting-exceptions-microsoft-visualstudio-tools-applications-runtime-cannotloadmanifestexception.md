# 例外のトラブルシューティング : Microsoft.VisualStudio.Tools.Applications.Runtime.CannotLoadManifestException
この例外は、マニフェストがドキュメントに存在しない場合、破損している場合、または無効な場合に、マニフェストを読み込むことができないとスローされます。
