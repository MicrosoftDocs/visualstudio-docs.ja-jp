# 例外のトラブルシューティング : System.Net.NetworkInformation.PingException
<xref:System.Net.NetworkInformation.Ping.Send%2A> メソッドまたは <xref:System.Net.NetworkInformation.Ping.SendAsync%2A> メソッドが、例外をスローするメソッドを呼び出したときにスローされる例外。 たとえば、サーバーで ping をサポートしていない場合や、ファイアウォールで操作がブロックされる場合などです。
