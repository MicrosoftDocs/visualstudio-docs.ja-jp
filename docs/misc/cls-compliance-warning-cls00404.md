# CLS 準拠の警告 CLS00404
メンバー識別子は、Unicode Standard 3.0 の Technical Report の Annex 7 に従う必要があります  
  
 メンバー識別子が、Unicode Standard 3.0 の Technical Report の Annex 7 に従うことを確認します。
