# 引数 &#39;Access&#39; は使用できません。 Input モードに使用できる値は、&#39;OpenAccess.Read&#39; および &#39;OpenAccess.Default&#39; です
`Input` モードでは Access 値は使用できません。  
  
### このエラーを解決するには  
  
-   値を `OpenAccess.Read` または `OpenAccess.Default` のどちらかに変更します。
