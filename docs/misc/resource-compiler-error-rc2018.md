# リソース コンパイラ エラー RC2018
文字 'hexnumber' は認識できません  
  
 指定された 16 進数に対応する ASCII 文字がソース ファイルにありますが、無効な文字です。  
  
 このエラーは、ソース ファイルの破損が原因で発生する場合があります。
