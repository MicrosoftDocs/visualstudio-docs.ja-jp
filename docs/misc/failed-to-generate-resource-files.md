# Failed to generate resource files
.NET リソース マネージャーによる判別の可能なバイナリ リソース ファイルをプロジェクト システムが作成する過程で、.resx ファイルを読み取るときに一般エラーが発生しました。  
  
 一般に、このエラーは、無効な .resx ファイルが原因で発生します。  
  
 この警告によりビルド エラーが発生することはありません。  ただし、このエラーが表示された .resx ファイルが、アセンブリ マニフェスト リソースとしてプロジェクトにインクルードされることはありません。
