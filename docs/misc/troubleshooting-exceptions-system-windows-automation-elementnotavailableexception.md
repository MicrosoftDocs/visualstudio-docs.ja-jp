# 例外のトラブルシューティング : System.Windows.Automation.ElementNotAvailableException
<xref:System.Windows.Automation.ElementNotAvailableException> 例外は、使用できなくなったユーザー インターフェイスの一部分に対応する UI <xref:System.Windows.Automation.AutomationElement> にアクセスしようとしたときに発生する例外に関する情報を格納します。  
  
## コメント  
 この例外は、閉じられたダイアログ ボックス内、閉じられたアプリケーション内、または終了されたアプリケーション内に要素が存在している場合に発生する可能性があります。
