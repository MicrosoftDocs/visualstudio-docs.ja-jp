# プログラミング言語の選択、Visual Studio パッケージ ウィザード
この手順を使用して次の処理を行うことができます。  
  
-   VSPackage を作成するときに使用するプログラミング言語を選びます。  
  
-   キー ファイルを生成するか既存のキー ファイルを使用して、VSPackage アセンブリに署名します。  
  
 **Visual C\#**  
 C\# で VSPackage を作成します。  
  
 **Visual Basic**  
 Visual Basic で VSPackage を作成します。  
  
 **Visual C\+\+**  
 Visual C\+\+ で VSPackage を作成します。  
  
## アセンブリの署名のためのキーを取得  
 **アセンブリに署名するために新しいキー ファイルを生成**  
 ウィザードによって、キー ファイルが生成されます。 生成されるキー ファイルの名前は、key.snk です。  
  
 **アセンブリに署名するには、次のキー ファイルをご使用ください**  
 既存のキー ファイルへのパスを入力するか、**\[参照\]** をクリックしてファイルを取得します。 ウィザードによってファイルのコピーが作成され、key.snk という名前になります。  
  
 **キャンセル**  
 VSPackage の作成を取り消します。  
  
 **前へ**  
 ウィザードの前のページに移動します。  
  
 **次へ**  
 ウィザードの次のページに移動します。  
  
 **完了**  
 VSPackage を作成します。
