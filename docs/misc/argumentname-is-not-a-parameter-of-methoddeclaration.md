# &#39;&lt;argumentname&gt;&#39; は &#39;&lt;methoddeclaration&gt;&#39; のパラメーターではありません
プロシージャ呼び出しで、プロシージャによって定義されていない追加の引数が指定されています。  
  
 **エラー ID:** BC30272  
  
### このエラーを解決するには  
  
-   引数リストから余分な引数を削除します。
