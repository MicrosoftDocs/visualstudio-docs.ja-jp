# 例外のトラブルシューティング : System.Data.ReadOnlyException
読み取り専用のデータ列の値を変更しようとすると、<xref:System.Data.ReadOnlyException> 例外がスローされます。  
  
## 関連するヒント  
 **列を読み取り\/書き込みに変更します。**  
 この例外は、読み取り専用のデータ列の値を変更しようとするとスローされます。 詳細については、「<xref:System.Data.DataColumn>」を参照してください。  
  
 **データを変更しようとする列が正しいことを確認します。**  
 この例外は、読み取り専用のデータ列の値を変更しようとするとスローされます。 詳細については、「<xref:System.Data.DataColumn>」を参照してください。
