# Invalid assembly strong name or name not specified
プロジェクト システムのオブジェクト モデルで、厳密な名前を持つアセンブリ \(署名されたアセンブリ\) にビルドされるプロジェクトに対してキー名またはキー コンテナーを指定するためのプロジェクト プロパティが、空白になっています。  このメッセージが表示された場合、ビルドは失敗します。  
  
### このエラーを解決するには  
  
1.  キー コンテナー名またはキー ファイルのいずれかがコードに指定されていることを確認します。
