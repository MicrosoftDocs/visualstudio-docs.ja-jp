# &#39;&lt;name&gt;&#39; で宣言された &lt;type1&gt; &#39;&lt;typename&gt;&#39; と &lt;type2&gt; &#39;&lt;typename&gt;&#39; は、&lt;namespace&gt; &#39;&lt;namespacename&gt;&#39; で競合しています
クラスやモジュールなど、宣言された 2 つの要素には、同じ名前空間の同じ名前があります。  
  
 **エラー ID:** BC30175  
  
### このエラーを解決するには  
  
-   いずれかの要素の宣言を別の名前空間に移動します。  
  
-   いずれかの要素の名前を変更します。
