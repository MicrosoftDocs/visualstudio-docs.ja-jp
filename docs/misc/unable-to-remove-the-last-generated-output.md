# Unable to remove the last generated output
最後に生成された既知の出力が、ディスクまたはプロジェクトから削除されていません。  
  
 カスタム ツールによってエラーが返された場合、最近生成された出力 \(存在する場合\) がディスクおよびプロジェクトから削除されます。  
  
 このエラーは、生成された出力がディスク上でロックされている場合に発生します。  
  
 **このエラーを解決するには**  
  
-   [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] を閉じて再起動します。
