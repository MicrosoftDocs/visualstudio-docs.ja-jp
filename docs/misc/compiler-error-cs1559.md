# コンパイラ エラー CS1559
'object' はインポートされているため、Main メソッドに対して使うことはできません。  
  
 [\/main](/dotnet/csharp/language-reference/compiler-options/main-compiler-option) コンパイラ オプションに無効なクラスが指定されています。このクラスは [Main](/dotnet/csharp/programming-guide/main-and-command-args/main-and-command-line-arguments) メソッドの場所として使用することはできません。
