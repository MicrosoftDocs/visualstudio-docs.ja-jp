# リソース コンパイラの致命的なエラー RC1206
コマンドラインで指定されたコード ページがレジストリに存在しません  
  
 コード ページの指定 \(\/c\) オプションの後ろに指定されているコード ページの位置が、レジストリに指定されていません。 SETUP を実行して、指定されたコード ページをインストールします。
