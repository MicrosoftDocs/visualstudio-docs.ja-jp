# リソース コンパイラ エラー RC2146
LANGUAGE ステートメントにコンマがありません  
  
 第 1 言語と第 2 言語を区切るコンマがありません。  
  
 **LANGUAGE** ステートメントでは次の構文を使用しなければなりません。  
  
 **LANGUAGE** *primary\_language\_ID*,*secondary\_language\_ID*
