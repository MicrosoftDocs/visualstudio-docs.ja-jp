# パラメーター リストの末尾が必要です
プロシージャの宣言で、`ParamArray` を指定した後に、他の 1 つ以上の引数を指定しています。  
  
 **エラー ID:** BC30192  
  
### このエラーを解決するには  
  
-   `ParamArray` を引数リストの末尾に移動します。
