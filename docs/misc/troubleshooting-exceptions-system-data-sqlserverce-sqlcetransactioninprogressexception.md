# 例外のトラブルシューティング : System.Data.SqlServerCe.SqlCeTransactionInProgressException
`System.Data.SqlServerCe.SqlCeTransactionInProgressException` 例外は、別のトランザクションの進行中に、アプリケーションがデータベースを変更しようとした場合に発生します。 詳細については、「[SQL Server Compact 3.5 クラス ライブラリ](http://go.microsoft.com/fwlink/?LinkID=102595)」をご覧ください。
