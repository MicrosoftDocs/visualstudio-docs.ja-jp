# MSBuild エラー MSB1039
**MSB1039: ツールセットのバージョンを指定してください。**  
  
 **\/ToolsVersion** スイッチが使用されましたが、値が指定されていません。  
  
### このエラーを解決するには  
  
-   **\/ToolsVersion** スイッチを使用する場合は、空でない有効な値を `ToolsVersion` として指定してください。
