# モジュール属性 &#39;&lt;attributename&gt;&#39; は、有効ではありません: &lt;error&gt;
共通言語ランタイムは、この属性をモジュールで受け入れることができません。  
  
 **エラー ID:** BC30130  
  
### このエラーを解決するには  
  
1.  宣言から属性を削除するか、コンテキストにとって正しい属性で置き換えます。  
  
2.  引用されているエラー メッセージを確認し、適切な処理を行います。
