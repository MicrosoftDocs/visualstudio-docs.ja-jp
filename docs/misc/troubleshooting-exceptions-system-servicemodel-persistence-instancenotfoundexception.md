# 例外のトラブルシューティング : System.ServiceModel.Persistence.InstanceNotFoundException
<xref:System.ServiceModel.Persistence.InstanceNotFoundException> 例外は、次の場合にスローされます。  
  
-   完了としてマークされた永続的なサービス インスタンスに対する操作が実行されている。  
  
-   <xref:System.ServiceModel.Persistence.SqlPersistenceProviderFactory> によって作成された永続化プロバイダーが、データベースで見つからない状態データをロック、ロック解除、または処理しようとしている。
