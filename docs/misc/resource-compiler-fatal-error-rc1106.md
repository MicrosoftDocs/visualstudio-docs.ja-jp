# リソース コンパイラの致命的なエラー RC1106
無効なスイッチ: オプション  
  
 オプションが有効なリソース コンパイラ オプションではありません。  
  
 有効なオプションは次のとおりです。  
  
 \/r  
  
 .res ファイルを生成します。  
  
 \/v  
  
 詳細 \(進行状況メッセージの出力\)  
  
 \/d  
  
 シンボルを定義します。  
  
 \/fo  
  
 .res ファイルの名前を変更します。  
  
 \/l  
  
 16 進の既定の言語 ID です。  
  
 \/I  
  
 **INCLUDE** 検索にパスを追加します。  
  
 \/x  
  
 **INCLUDE** 環境変数を無視します。  
  
 \/c  
  
 NLS 変換で使用するコード ページを定義します。
