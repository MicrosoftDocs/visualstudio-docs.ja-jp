# バイナリ ライセンス ファイルを生成できませんでした。 出力ファイル &#39;file&#39; を開けませんでした。 &lt;理由&gt;
プロジェクト システムによって、古いライセンス ファイルのコピーを削除できませんでした。 プロジェクト システムは、ライセンスを準備する過程で、テキストの .licx ファイルを .NET ライセンス システムが解釈できるバイナリのライセンス ファイルに変換します。  
  
 **このエラーを解決するには**  
  
-   [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] を再起動します。  
  
     このエラーが発生した場合、ビルド プロセスは失敗します。
