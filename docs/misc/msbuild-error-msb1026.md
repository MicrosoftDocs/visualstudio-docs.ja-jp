# MSBuild エラー MSB1026
**スキーマ ファイルが存在しません。**  
  
 指定された場所にスキーマ ファイルが存在しません。  
  
### このエラーを解決するには  
  
1.  スキーマ ファイルの絶対位置または相対位置を指定した場合は、指定したディレクトリにスキーマ ファイルが存在するかどうかを確認してください。
