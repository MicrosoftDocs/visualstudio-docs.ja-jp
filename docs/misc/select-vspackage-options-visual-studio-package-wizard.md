# VSPackage のオプションの選択、Visual Studio パッケージ ウィザード
この手順を使用して、VSPackage のオプション コンポーネントを選択できます。  
  
 **メニュー コマンド**  
 VSPackage のコマンドを作成できます。  
  
 新しいコマンドは **\[ツール\]** メニューに追加され、既定で有効になります。  
  
 このオプションを選択すると、別のウィザード ページが表示され、コマンドの設定を定義できます。  
  
 **ツール ウィンドウ**  
 VSPackage のツール ウィンドウを作成できます。  
  
 ツール ウィンドウのコマンドは、**\[表示\]** メニューの **\[その他のウィンドウ\]** サブメニューに追加されます。  
  
 このオプションを選択すると、ツール ウィンドウの設定を定義できるように別のウィザード ページが表示されます。  
  
 **カスタム エディター**  
 VSPackage のカスタム エディターを作成できます。  
  
 生成されたエディター コードは基本的なものであり、独自のエディター コードに置き換えることができます。  
  
 このオプションを選択すると、エディターの設定を定義できるように別のウィザード ページが表示されます。  
  
 **キャンセル**  
 VSPackage の作成をキャンセルします。  
  
 **前へ**  
 ウィザードの前のページに移動します。  
  
 **次へ**  
 ウィザードの次のページに移動します。  
  
 **完了**  
 VSPackage を作成します。
