# メンバー &#39;&lt;membername&gt;&#39; がクラス &#39;&lt;classname&gt;&#39; に見つかりません
メンバー '\<membername\>' がクラス '\<classname\>' に見つかりません。 このエラーは通常、'Microsoft.VisualBasic.dll' が適切でない場合に発生します。  
  
 定義済みのメンバーが見つかりませんでした。  
  
 **エラー ID:** BC31097  
  
### このエラーを解決するには  
  
1.  プログラムをもう一度コンパイルし、エラーがまだ発生するかどうか確認します。  
  
2.  エラーがまだ発生する場合は、作業内容を保存し、[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] を再起動します。  
  
3.  エラーが続く場合は、[!INCLUDE[vbprvb](../code-quality/includes/vbprvb_md.md)] を再インストールします。  
  
4.  再インストールした後にエラーが続く場合は、マイクロソフト プロダクト サポート サービスに通知してください。
