# 例外のトラブルシューティング : System.Threading.SemaphoreFullException
この例外は、<xref:System.Threading.Semaphore.Release%2A> メソッドのオーバーロードが、カウントが最大値に達しているセマフォで呼び出された場合にスローされます。
