# Clipboard cannot be emptied.
このエラーが発生するのは一般的に、クリップボードに削除できない情報が含まれているために、項目をコピーしてもクリップボードに置けない場合です。  
  
### このエラーを解決するには  
  
1.  クリップボードを使用しているほかのプログラムを終了して、やり直します。
