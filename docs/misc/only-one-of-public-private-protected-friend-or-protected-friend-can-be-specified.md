# &#39;Public&#39;、&#39;Private&#39;、&#39;Protected&#39;、&#39;Friend&#39;、または &#39;Protected Friend&#39; のいずれかのみを指定できます。
1 つの要素の宣言内に、競合するアクセシビリティがあります。 アクセシビリティを 1 つだけ指定できます。  
  
 **エラー ID:** BC30176  
  
### このエラーを解決するには  
  
-   1 つのアクセシビリティを選び、残りのキーワードを宣言から削除します。
