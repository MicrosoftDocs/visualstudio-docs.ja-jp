# 例外のトラブルシューティング : System.Printing.PrintCommitAttributesException
<xref:System.Printing.PrintCommitAttributesException> 例外は、エラー状態が原因で、<xref:System.Printing.PrintSystemObject> によって、オブジェクトが表す実際のコンピューター、プリンター、またはデバイスに属性がコミットされない場合にスローされます。
