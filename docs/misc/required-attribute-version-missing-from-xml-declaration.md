# 必要な属性 &#39;version&#39; が XML 宣言にありません
XML ドキュメント リテラルの `version` 属性がありません。  
  
 **エラー ID:** BC31153  
  
### このエラーを解決するには  
  
-   `<?xml version="1.0"?>` のように、XML ドキュメントの宣言に `version` 属性を追加します
