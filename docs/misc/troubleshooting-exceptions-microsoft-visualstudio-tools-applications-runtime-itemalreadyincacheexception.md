# 例外のトラブルシューティング : Microsoft.VisualStudio.Tools.Applications.Runtime.ItemAlreadyInCacheException
この例外は、ドキュメント内のキャッシュに項目を追加しようとした場合に、その項目が既にキャッシュ内に存在しているとスローされます。
