# コンパイラ エラー CS0014
必要なファイル 'file' が見つかりませんでした  
  
 コンパイラで必要なファイルがシステムにありません。 パスが正しいことを確認してください。 ファイルが [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] システム ファイルの場合は、インストール環境を修復したり、[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] を完全に削除して再インストールしたりする必要がある場合があります。
