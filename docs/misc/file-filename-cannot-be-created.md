# File &lt;filename&gt; cannot be created.
通常、このエラーが発生するのは、コード コメント Web レポートの作成を試みたときに、現在のユーザーに書き込みアクセス権がなかった場合です。  
  
### このエラーを解決するには  
  
1.  書き込みアクセス権を持っているかどうかを確認します。  
  
2.  ページを保存できるだけの十分な領域があるかどうかを確認します。
