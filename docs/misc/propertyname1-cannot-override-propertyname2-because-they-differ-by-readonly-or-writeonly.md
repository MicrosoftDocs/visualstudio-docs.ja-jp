# &#39;&lt;propertyname1&gt;&#39; と &#39;&lt;propertyname2&gt;&#39; は、&#39;ReadOnly&#39; であるか、または &#39;WriteOnly&#39; であるかのみが異なるため、オーバーライドすることはできません
1 番目のプロパティを、`ReadOnly` か `WriteOnly` かの違いしかない 2 番目のプロパティでオーバーライドしようとしました。  
  
 **エラー ID:** BC30362  
  
### このエラーを解決するには  
  
-   プロパティどうしの間に `ReadOnly` と `WriteOnly` 以外の違いがあるようにします。
