# XML エンティティの終わりに &#39;;&#39; が必要です
XML エンティティ参照 \(例: `>` など\) で最後のセミコロン \(;\) が見つかりません。  
  
 **エラー ID:** BC31178  
  
### このエラーを解決するには  
  
-   XML エンティティ参照の末尾にセミコロンを含めます。
