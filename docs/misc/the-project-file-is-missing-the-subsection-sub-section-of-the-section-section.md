# The project file is missing the &#39;subsection&#39; sub-section of the &#39;section&#39; section
SubSection \(subsection\) がインクルードで、セクション \(section\) がファイルです。  
  
 このエラーの原因として最も可能性が高いのは、プロジェクト ファイルを手動で編集したことです。  
  
 **このエラーを解決するには**  
  
-   すべてのファイルを表示モードで作業して \(ソリューション エクスプローラーのツール バーにある **\[すべてのファイルを表示\]** ボタンをクリック\)、関連するファイルをすべてインクルードします。  
  
     このエラーが表示された場合、プロジェクトは開きますが、どのファイルもプロジェクトに追加されません。
