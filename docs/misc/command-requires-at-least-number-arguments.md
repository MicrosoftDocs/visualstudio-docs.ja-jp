# Command requires at least &lt;number&gt; arguments.
このエラーが発生するのは一般的に、入力されたコマンドの引数が少なすぎる場合です。  
  
### このエラーを解決するには  
  
1.  ドキュメントでコマンドの正しい構文を確認して、やり直します。
