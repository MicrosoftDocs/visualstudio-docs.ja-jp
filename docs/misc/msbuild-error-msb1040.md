# MSBuild エラー MSB1040
**MSB1040: ToolsVersion が無効です。**  
  
 **\/ToolsVersion** スイッチで指定した `ToolsVersion` の値が無効です。  
  
### このエラーを解決するには  
  
-   コマンド ラインで **\/ToolsVersion** スイッチを使用してビルドを実行する場合は、有効な `ToolsVersion` の値を指定してください。
