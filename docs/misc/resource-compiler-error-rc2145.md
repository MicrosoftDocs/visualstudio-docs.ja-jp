# リソース コンパイラ エラー RC2145
主言語 ID が大きすぎます  
  
 主言語 ID は、16 進数の有効な言語 ID にする必要があります。 有効な言語 ID の一覧については、「[ランタイム ライブラリ リファレンス](/visual-cpp/c-runtime-library/locale-names-languages-and-country-region-strings)」の「*言語および国\/地域識別文字列*」を参照してください。
