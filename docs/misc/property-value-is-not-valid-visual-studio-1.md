# Property value is not valid. (Visual Studio) (1)
このエラーが発生するのは一般的に、プロパティに対して無効な値が入力された場合です。  たとえば、プロパティがブール値のプロパティで、有効な値が TRUE と FALSE である場合、MAYBE という値はそのプロパティに対して無効になります。  
  
### このエラーを解決するには  
  
1.  プロパティに対して有効な値を入力します。  
  
    > [!TIP]
    >  オンライン ドキュメントでプロパティ名を検索して、有効な値を確認してください。
