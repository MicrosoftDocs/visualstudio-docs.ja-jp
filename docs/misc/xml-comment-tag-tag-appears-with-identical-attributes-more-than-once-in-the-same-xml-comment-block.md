# 同一の属性を伴う XML コメント タグ &#39;&lt;tag&gt;&#39; が同じ XML コメント ブロック内に複数回記述されています
同一の属性を伴う XML コメント タグ \<tag\> が同じ XML コメント ブロック内に複数回記述されています。 XML コメントは無視されます。  
  
 XML コメントに、同じ属性を持つ複数のタグが含まれています。  
  
 **エラー ID:** BC42305  
  
### このエラーを解決するには  
  
-   重複しているタグを削除します。
