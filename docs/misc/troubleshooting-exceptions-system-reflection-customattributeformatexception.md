# 例外のトラブルシューティング : System.Reflection.CustomAttributeFormatException
カスタム属性のバイナリ形式が無効な場合にスローされる例外。  
  
## コメント  
 エラーをスローするデータ型の作成に使用する、ツールのバグに起因することがあります。
