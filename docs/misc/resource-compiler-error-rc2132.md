# リソース コンパイラ エラー RC2132
VALUE、BLOCK、または END のキーワードが必要です  
  
 **VERSION** リソースのブロックが正しく終了していないか、新しいブロックが正しく開始されていません。 このため、**VERSION** ステートメントが有効なブロックではなくなります。
