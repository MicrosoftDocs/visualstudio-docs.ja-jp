# リソース コンパイラ エラー RC2138
'\/\*' が終了コメントに対応しません  
  
 コメント開始シーケンス \(`/*`\) がコメント終了シーケンス \(`*/`\) と一致しませんでした。このエラーは、コメントを入れ子にすることで発生する場合があります。
