# 2 つ以上の構成を選択してください。 1 つの設定のみを変更したい場合は、[複数の構成] ダイアログ ボックスで [キャンセル] をクリックして終了し、[構成] のドロップダウン一覧から構成を選択してください。
このエラーが発生するのは、一般的に、**\[複数の構成\]** ダイアログ ボックスで 1 つのソリューション構成が選択された場合です。  
  
### このエラーを解決するには  
  
1.  追加のソリューション構成を選択します。  
  
     または  
  
2.  1 つの構成を更新する場合は、\[キャンセル\] を選択し、**\[ソリューションのプロパティ\]** ダイアログ ボックスの \[構成\] ボックスの一覧から適切なソリューション構成を選択します。
