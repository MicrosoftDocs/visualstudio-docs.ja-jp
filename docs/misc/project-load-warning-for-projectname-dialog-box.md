# [&lt;プロジェクト名&gt; プロジェクトの読み込みに関連する警告] ダイアログ ボックス
IDE で開かれたプロジェクトの設定が正しく読み込まれない場合に、このダイアログ ボックスが表示されます。  
  
 **\[指定されたエラーを修正し、プロジェクトを再度読み込んでください。\]**  
 無効なパスやファイル名など、問題に関する情報が示されます。  エラーを解決するには、プロジェクトのプロパティ ページから、すべての無効な参照を削除または置換します。  プロジェクトを保存し、IDE で再読み込みします。  
  
 **\[今後このセッションでこのダイアログを表示しない\]**  
 このチェック ボックスをオンにすると、\[\<Projectname\> プロジェクトの読み込みに関連する警告\] ダイアログ ボックスは、次回 IDE を開くまで表示されなくなります。
