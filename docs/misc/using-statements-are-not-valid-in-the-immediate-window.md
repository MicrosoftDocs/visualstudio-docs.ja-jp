# &#39;Using&#39; ステートメントは、イミディエイト ウィンドウでは有効ではありません。
`Using` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30186  
  
### このエラーを解決するには  
  
-   **\[イミディエイト\]** ウィンドウから、`Using` ステートメントを削除します。
