# コンパイラ エラー CS1565
競合するオプションが指定されました: Win32 リソース ファイル、Win32 アイコン  
  
 [\/win32res](/dotnet/csharp/language-reference/compiler-options/win32res-compiler-option) コンパイラ オプションと [\/win32ico](/dotnet/csharp/language-reference/compiler-options/win32icon-compiler-option) コンパイラ オプションを同じコンパイルで両方とも指定することは正しくありません。
