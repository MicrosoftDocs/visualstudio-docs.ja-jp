# 構造体 &#39;&lt;structurename&gt;&#39; には既定のプロパティがないため、インデックス処理を実行できません
既定のプロパティがない構造体で、既定のプロパティの構文を使用しようとしています。  
  
 **エラー ID:** BC30690  
  
### このエラーを解決するには  
  
-   標準プロパティの構文を使用して、構造体のプロパティにアクセスします。
