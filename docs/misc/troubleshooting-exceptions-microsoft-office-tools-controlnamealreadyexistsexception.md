# 例外のトラブルシューティング : Microsoft.Office.Tools.ControlNameAlreadyExistsException
<xref:Microsoft.Office.Tools.ControlNameAlreadyExistsException> 例外は、プロジェクトにあるコントロールと同名のコントロールをプログラムで作成しようとしたときにスローされます。  
  
## 関連するヒント  
 **コントロールの 1 つには別の名前を使用してください。**  
 コントロール名は一意である必要があります。
