# MSBuild エラー MSB1012
**応答ファイルを指定します。**  
  
 @ スイッチには応答ファイルを指定する必要があります。  
  
### このエラーを解決するには  
  
-   応答ファイルを指定します。  @\<file name\> の構文で、`@BuildHW.txt` のように指定します。  
  
-   コマンド ラインで @ スイッチを使用することはできません。
