# 引数 &#39;&lt;argumentname&gt;&#39; は、型 &#39;Date&#39; には変換できません
変数を型を `Date` に変更する変換が試みられましたが、値が有効な日付として認識されません。  
  
### このエラーを解決するには  
  
-   有効な変換を検索します。
