# 例外のトラブルシューティング : Microsoft.VisualStudio.Tools.Applications.Runtime.NonCacheableTypeException
この例外は、型をキャッシュしようとしたときに、型が <xref:System.Data.DataSet> または <xref:System.Data.DataTable> ではないか、「[Caching Data](http://msdn.microsoft.com/ja-jp/e60dc731-2cb0-4de6-a861-0f0d131d780a)」に指定されている要件を満たす型ではない場合にスローされます。
