# 例外のトラブルシューティング : Microsoft.VisualStudio.Tools.Applications.Runtime.ControlNotFoundException
コードで、ドキュメントにないコントロールにアクセスしようとすると、<xref:Microsoft.VisualStudio.Tools.Applications.Runtime.ControlNotFoundException> の例外がスローされます。 このコントロールは、エンド ユーザーまたは他の何らかのコードによって削除された可能性があります。 コントロールにアクセスする前にコントロールをチェックする方法はありません。コントロールが削除されると、ソリューションは機能しません。
