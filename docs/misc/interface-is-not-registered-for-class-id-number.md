# Interface is not registered for class ID &lt;number&gt;.
インターフェイスが登録されていません。  このメッセージは、オペレーティング システムのエラーです。  
  
### このエラーを解決するには  
  
1.  Visual Studio または影響を受けているアプリケーションを再インストールしてからやり直します。
