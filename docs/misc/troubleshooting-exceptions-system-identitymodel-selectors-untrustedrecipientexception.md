# 例外のトラブルシューティング : System.IdentityModel.Selectors.UntrustedRecipientException
<xref:System.IdentityModel.Selectors.UntrustedRecipientException> 例外は、トークンを要求しているエンティティの情報をユーザーが調べて、そのエンティティを信頼しないことにした場合にスローされます。 ユーザーは、CardSpace ユーザー インターフェイスで **\[Windows CardSpace の無効化\]** をクリックします。
