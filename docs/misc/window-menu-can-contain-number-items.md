# Window menu can contain &lt;number&gt; items.
\[ウィンドウ\] メニューの \[ウィンドウ一覧\] では、最大 24 のウィンドウ名しか表示できません。  このエラーが発生するのは一般的に、\[オプション\] ダイアログ ボックスの \[環境\] の全般ペインにある **\[\<数\> 項目がウィンドウ メニューに表示されています\]** ボックスに、1 より小さい数値または 24 より大きい数値を入力した場合です。  
  
### このエラーを解決するには  
  
1.  ボックスに 1 ～ 24 の間の数値を入力します。
