# &#39;Get&#39; は既に宣言されています
コード ブロックで `Get` ステートメントが複数回出現します。`Get` ステートメントは、プロパティの値を取得するのに使用する `Get` プロパティ プロシージャを宣言します。  
  
 **エラー ID:** BC30443  
  
### このエラーを解決するには  
  
-   不要な `Get` ステートメントを削除します。
