# エディター オプション、Visual Studio パッケージ ウィザード
このステップでは、エディターに関連付けられているファイル名拡張子、エディターを使用して作成されたドキュメントの既定名として使用されているファイル名、およびエディターおよびファイルの種類に関連付けられているアイコンを定義します。  
  
 **エディター名**  
 新しいエディターの名前を入力します。 この名前は、**\[プログラムから開く\]** ダイアログ ボックスのエディターに使用されます。  
  
 **ファイル拡張子**  
 エディターに関連付けられているファイル名拡張子を定義します。 複数の拡張子を指定するには、コンマ区切りのリストを入力します。 エディターは、定義したファイル名拡張子を持つファイル用の既定の [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] エディターです。  
  
 **既定のファイル名**  
 エディターで作成するドキュメントの既定名のベースとして使用される名前を入力します。  
  
 **アイコンの変更**  
 クリックして、**\[アイコンの変更\]** ダイアログ ボックスを開きます。 このダイアログ ボックスで、エディターのアイコンを選択することができます、また、別のファイルの場所を参照することができます。 このファイルは .ico ファイルである必要があります。 選択したアイコンは、エディター ウィンドウに表示され、エディターによって生成されるファイルに関連付けられます。  
  
 **キャンセル**  
 ウィザードをキャンセルおよび停止します。  
  
 **戻る**  
 ウィザードの前のページに移動します。  
  
 **完了**  
 VSPackage を作成します。
