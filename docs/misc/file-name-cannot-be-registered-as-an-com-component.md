# File &lt;name&gt; cannot be registered as an COM component.
ファイルをプロジェクトの参照に追加できません。  このエラーが発生するのは、ファイルが破損している場合、ファイルにタイプ ライブラリが含まれない場合、使用しているファイルの種類に互換性がない場合、または操作の完了に必要な十分なリソースがシステムにない場合です。  
  
### このエラーを解決するには  
  
1.  ファイルに有効なタイプ ライブラリが含まれることを確認して、やり直します。  
  
     または  
  
     ファイルをインストールした製品を再インストールします。  
  
    > [!NOTE]
    >  ファイルをインストールした製品が Visual Studio でない可能性があります。
