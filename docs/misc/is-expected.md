# &#39;Is&#39; が必要です
`TypeOf` 演算子に `Is` 句がありません。  
  
 **エラー ID:** BC30224  
  
### このエラーを解決するには  
  
-   `TypeOf` 演算子の後に `Is` 句を追加します。たとえば、`TypeOf MyControl Is ComboBox` のように記述します。
