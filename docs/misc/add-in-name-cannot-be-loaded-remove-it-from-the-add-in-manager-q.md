# Add-in &lt;name&gt; cannot be loaded. Remove it from the Add-in Manager?
選択されたアドインが見つかりません。  このエラーは通常、選択したアドインに .dll やその他の必要なコンポーネントが見つからないときに発生します。  
  
### このエラーを解決するには  
  
1.  アドインを再度インストールします。
