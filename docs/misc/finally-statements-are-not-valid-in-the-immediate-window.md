# &#39;Finally&#39; ステートメントは、イミディエイト ウィンドウでは有効ではありません
`Finally` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30716  
  
### このエラーを解決するには  
  
-   デバッグ コードから `Finally` ステートメントを削除します。
