# 匿名型の初期化子は、デバッグ ウィンドウでは使用できません
デバッグ セッション中に、匿名型の初期化子が **\[ウォッチ\]** ウィンドウまたは **\[イミディエイト\]** ウィンドウに入力されました。 匿名型の初期化子は、デバッグ ウィンドウでは使用できません。  
  
 **エラー ID:** BC36577  
  
### このエラーを解決するには  
  
-   コード エディターで匿名型の初期化子をテストします。
