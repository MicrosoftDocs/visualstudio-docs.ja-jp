# Package &lt;name&gt; failed to load.
パッケージを開けませんでした。  このエラーが発生するのは一般的に、パッケージが正しく登録されていない場合、パッケージにダイナミック リンク ライブラリ \(DLL: Dynamic Link Library\) が見つからない場合、DLL が破損している場合、またはパッケージに必要な別のコンポーネントの DLL が見つからない場合です。  
  
### このエラーを解決するには  
  
1.  パッケージが付属する製品を再インストールします。
