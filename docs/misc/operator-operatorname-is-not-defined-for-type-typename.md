# 演算子 &#39;&lt;operatorname&gt;&#39; は型 &#39;&lt;typename&gt;&#39; に対して定義されていません
`String` などの型に対して単項演算子 \(`+`、`-`、または `Not`\) を使用しようとしましたが、これは定義されていません。  
  
 **エラー ID:** BC30487  
  
### このエラーを解決するには  
  
-   単項演算子を式の適切な演算子に置き換えます。
