# 例外のトラブルシューティング : System.DirectoryServices.Protocols.LdapException
この例外は、ライトウェイト ディレクトリ アクセス プロトコル \(LDAP\) から、<xref:System.DirectoryServices.Protocols.ResultCode> に含まれないエラー コードが返された場合にスローされます。
