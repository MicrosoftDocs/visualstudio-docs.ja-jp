# 例外のトラブルシューティング : System.Transactions.TransactionInDoubtException
この例外は、不明なトランザクションで操作しようとした場合にスローされます。  
  
## コメント  
 登録されたリソース マネージャーに接続できないとき、トランザクションは不明になります。
