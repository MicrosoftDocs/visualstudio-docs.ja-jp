# &#39;&lt;statementname&gt;&#39; ステートメントには配列が必要です
`Erase` と `ReDim` ステートメントは、配列のデータ型を必要とします。 指定された型が配列ではありません。  
  
 **エラー ID:** BC30049  
  
### このエラーを解決するには  
  
-   データ型を配列型に変更します。
