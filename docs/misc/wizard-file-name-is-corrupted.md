# Wizard file &lt;name&gt; is corrupted.
ウィザードの .vsz ファイルの形式が正しくありません。  このエラーが発生するのは、.vsz ファイルが破損している場合、またはファイルが Visual Studio 以外の製品のファイルである場合です。  
  
### このエラーを解決するには  
  
1.  ウィザードが Visual Studio のウィザードであることを確認します。  
  
     または  
  
     Visual Studio を再インストールして、破損ファイルを置き換えます。
