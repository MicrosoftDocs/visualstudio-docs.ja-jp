# 例外のトラブルシューティング : System.Data.SqlServerCe.SqlCeException
`System.Data.SqlServerCe.SqlCeException` 例外は、基になるプロバイダーが SQL Server Compact データ ソースから警告またはエラーを返した場合にスローされます。 このエラーの原因について詳しくは、MSDN Web サイトの「[SQL Server Compact のエラー](http://msdn2.microsoft.com/library/ms171849.aspx)」をご覧ください。  
  
## 解説  
 `System.Data.SqlServerCe.SqlCeException` には、常に、`System.Data.SqlServerCe.SqlCeError` のインスタンスが少なくとも 1 つ含まれます。 このクラスは継承できません。  
  
 詳しくは、「[SQL Server Compact 3.5 クラス ライブラリ](http://go.microsoft.com/fwlink/?LinkID=102595)」をご覧ください。
