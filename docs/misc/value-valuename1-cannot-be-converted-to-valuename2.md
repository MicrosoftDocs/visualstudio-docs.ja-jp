# 値 &#39;&lt;valuename1&gt;&#39; を &#39;&lt;valuename2&gt;&#39; に変換できません
2 つの互換性のない型の間で型変換が試行されました。  
  
 **エラー ID:** BC30742  
  
### このエラーを解決するには  
  
-   キャストされている型を確認します。
