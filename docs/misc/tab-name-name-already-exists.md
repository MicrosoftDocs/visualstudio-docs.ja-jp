# Tab name &lt;name&gt; already exists.
\[ツールボックス\] の各タブは、一意の名前である必要があります。  このエラーが発生するのは一般的に、\[ツールボックス\] のタブに入力した名前が \[ツールボックス\] の既存のタブと同じ場合です。  
  
### このエラーを解決するには  
  
1.  タブに一意の名前を入力します。
