# Visual Basic コンパイラは、次のエラーから回復できません: &lt;エラー&gt;
[!INCLUDE[vbprvb](../code-quality/includes/vbprvb_md.md)] コンパイラは、次のエラーから回復できません: \<`error`\>。  
  
 一部の IDE 機能が無効になっています。  
  
 [!INCLUDE[vbprvb](../code-quality/includes/vbprvb_md.md)] コンパイラで内部エラーが発生しました。  
  
 **エラー ID:** BC31021  
  
### このエラーを解決するには  
  
1.  作業内容を保存し、[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] を再起動します。  
  
2.  エラーがまだ発生する場合は、[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] を再インストールします。  
  
3.  再インストールした後にエラーが続く場合は、マイクロソフト プロダクト サポート サービスに通知してください。
