# &#39;Variant&#39; 型は現在サポートされていません。&#39;Object&#39; 型を使用してください
`Object` データ型を `Variant` の代わりに使用します。  
  
 **エラー ID:** BC30804  
  
### このエラーを解決するには  
  
-   アイテムを `Object` として宣言します。
