# 式 &#39;&lt;name&gt;&#39; はプロシージャではありませんが、プロシージャ呼び出しのターゲットとして発生します
プロシージャではない式に対してプロシージャ呼び出しを行いました。  
  
### このエラーを解決するには  
  
1.  プロシージャの名前のスペルを間違えていないか確認します。  
  
2.  \<`name`\> によって呼び出されるプロシージャを指定します。
