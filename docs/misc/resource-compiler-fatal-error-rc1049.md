# リソース コンパイラの致命的なエラー RC1049
数値の引数 'string' が無効です  
  
 リソース コンパイラでは数値の引数が必要でしたが、指定された文字列を受け取りました。  
  
 このエラーは、必要な \\x プレフィックスのない 16 進数が指定されたか、浮動小数点数の形式が正しくない場合に、発生する可能性があります。
