# &#39;Set&#39; ステートメントの終わりには、対応する &#39;End Set&#39; を指定しなければなりません
`Set` プロパティ プロシージャは、必ず `End Set` ステートメントで終了する必要があります。  
  
 **エラー ID:** BC30633  
  
### このエラーを解決するには  
  
-   `Set` プロパティ プロシージャが `End Set` コンストラクトで終了していることを確認します。
