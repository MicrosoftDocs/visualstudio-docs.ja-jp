# 例外のトラブルシューティング : Microsoft.VisualStudio.Tools.Applications.Runtime.CannotGetCachedDataManifestException
この例外は、キャッシュされたすべてのデータ オブジェクトを一覧表示するマニフェストが破損しているか、紛失している場合にスローされます。
