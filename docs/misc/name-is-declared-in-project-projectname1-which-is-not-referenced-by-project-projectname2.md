# &#39;&lt;name&gt;&#39; は、プロジェクト &#39;&lt;projectname2&gt;&#39; が参照していないプロジェクト &#39;&lt;projectname1&gt;&#39; で宣言されています
プログラミング要素が、現在のプロジェクトに参照が含まれていないプロジェクトで定義されています。  
  
 **エラー ID:** BC32004  
  
### このエラーを解決するには  
  
-   現在のプロジェクトで、目的の要素が入っているプロジェクトへの参照を追加します。
