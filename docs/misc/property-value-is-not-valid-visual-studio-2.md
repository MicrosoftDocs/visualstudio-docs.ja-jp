# Property value is not valid. (Visual Studio) (2)
このエラーが発生するのは一般的に、プロパティに対して無効な値が入力された場合です。  たとえば、整数値を誤った形式で入力したり、short 値に対して入力する値が大きすぎる場合です。  
  
### このエラーを解決するには  
  
1.  プロパティに対して有効な値を入力します。  
  
    > [!TIP]
    >  オンライン ドキュメントを検索して、プロパティに有効な値を確認してください。
