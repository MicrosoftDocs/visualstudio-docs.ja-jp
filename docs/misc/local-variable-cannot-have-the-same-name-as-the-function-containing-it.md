# ローカル変数に、それを含む関数と同じ名前を指定することはできません
`Dim` ステートメントで、その変数を含む `Function` プロシージャと同じ名前の変数を指定しています。  
  
 **エラー ID:** BC30290  
  
### このエラーを解決するには  
  
-   変数の宣言を削除するか、変数の名前を変更します。
