# &#39;Return&#39; ステートメントは、イミディエイト ウィンドウでは有効ではありません
`Return` ステートメントは分岐を行うため、デバッグ コンテキストでは使用できません。  
  
 **エラー ID:** BC30147  
  
### このエラーを解決するには  
  
-   **\[イミディエイト\]** ウィンドウで `Return` ステートメントを実行しないようにします。
