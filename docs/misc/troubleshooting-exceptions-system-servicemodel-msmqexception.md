# 例外のトラブルシューティング : System.ServiceModel.MsmqException
<xref:System.ServiceModel.MsmqException> 例外は、メッセージ キューが返すエラーをカプセル化します。 この例外は、メッセージ キュー トランスポートまたはメッセージ キュー統合チャネルによってスローされます。
