# アセンブリに署名できません: &lt;エラー メッセージ&gt;
[!INCLUDE[vbprvb](../code-quality/includes/vbprvb_md.md)] コンパイラはアセンブリ リンカー \(Al.exe、Alink とも呼ばれる\) を呼び出し、マニフェストを伴うアセンブリを生成します。 リンカーは、作成された後のアセンブリに署名する際にエラーを報告しました。 これは、無効なキーが原因である可能性があります。  
  
 **エラー ID:** BC30146  
  
### このエラーを解決するには  
  
1.  引用符で囲まれたエラー メッセージを調べ、「[Al.exe ツールのエラーと警告](http://msdn.microsoft.com/ja-jp/7f125d49-0a03-47a6-9ba9-d61a679a7d4b)」というトピックで、より詳細な説明とアドバイスを参照します。  
  
2.  エラーが続く場合は、状況に関する情報を収集し、マイクロソフト プロダクト サポート サービスに通知してください。
