# &#39;&lt;typename&gt;&#39; で定義された拡張メソッド &#39;&lt;methodname&gt;&#39; のパラメーター &#39;&lt;parametername&gt;&#39; には、一致する引数が既に存在します
1 つの引数が、同じプロシージャ呼び出しで複数回にわたって名前で渡されています。  
  
 **エラー ID:** BC36584  
  
### このエラーを解決するには  
  
-   引数名のいずれかのスペルが誤っていないかどうかを確認します。  
  
-   引数リストから、重複している引数を削除します。
