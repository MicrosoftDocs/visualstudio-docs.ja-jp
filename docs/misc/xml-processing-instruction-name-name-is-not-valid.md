# XML 処理命令の名前 &#39;name&#39; が有効ではありません
XML 処理命令に指定した名前が正しくありません。  
  
 **エラー ID:** BC31174  
  
### このエラーを解決するには  
  
-   XML 処理命令の名前のエラーを修正するか、または削除します。
