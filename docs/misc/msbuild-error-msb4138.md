# MSBuild エラー MSB4138
**MSB4138: レジストリの場所 "'\<key\>'" に文字列以外のデータが指定されています。**  
  
 カスタム ツールセット構成に指定されたレジストリの場所が無効です。  
  
### このエラーを解決するには  
  
-   ツールセット定義に誤りがなく、指定されたレジストリ キーが存在することを確認してください。
