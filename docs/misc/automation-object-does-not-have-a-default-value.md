# オートメーション オブジェクトには既定値がありません
[!INCLUDE[vbprvb](../code-quality/includes/vbprvb_md.md)] が、指定したオブジェクトの既定のメンバーを判断できません。  
  
### このエラーを解決するには  
  
-   オブジェクトのドキュメントを確認し、プロパティまたはメソッドを明示的にご指定ください。
