# Unable to customize.
このエラーは、アプリケーションのデバッグ中にメニューまたはツール バーをカスタマイズしようとした場合に発生します。  
  
### このエラーを解決するには  
  
-   デバッグを終了してから、ツール バーまたはメニューをカスタマイズします。  
  
     または  
  
     デバッグを中止してから、ツール バーまたはメニューをカスタマイズします。
