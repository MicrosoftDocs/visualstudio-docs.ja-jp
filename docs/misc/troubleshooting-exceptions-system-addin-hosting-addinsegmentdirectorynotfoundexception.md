# 例外のトラブルシューティング : System.AddIn.Hosting.AddInSegmentDirectoryNotFoundException
<xref:System.AddIn.Hosting.AddInSegmentDirectoryNotFoundException> 例外は、パイプライン ディレクトリ構造にセグメント ディレクトリがない場合にスローされます。  
  
## コメント  
 アドインがアクティブになると、アドインのホストへのパイプラインが構築されます。 アドイン システムでは、「[パイプライン開発の必要条件](http://msdn.microsoft.com/ja-jp/ef9fa986-e80b-43e1-868b-247f4c1d9da5)」に説明されているように、指定したディレクトリ名の指定したディレクトリ構造内にパイプライン セグメントが配置される必要があります。
