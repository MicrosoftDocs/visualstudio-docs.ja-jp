# プロジェクト ビルドの警告 PRJ0045
ファイル 'file' に割り当てられているカスタム ビルド規則 'rule' の 'Outputs' プロパティが設定されていません。 規則の実行はスキップされます。  
  
 カスタム ビルド規則の Outputs プロパティが定義されていません。 カスタム ビルド規則では、ビルド アクションの出力が必要です。
