# ツールには有効な実行可能ファイルが必要です。 このツールを保存しない場合は、[外部ツール] ダイアログで [削除] をクリックしてください。
通常、このエラーが発生するのは、\[外部ツール\] ダイアログ ボックスにツールを追加するときに、実行可能ファイルを指定しなかった場合です。  
  
### このエラーを解決するには  
  
1.  ツールを保存する場合は、**\[OK\]** をクリックし、ツールの **\[コマンド\]** ボックスにファイル名を入力するか、省略記号 \(...\) ボタンをクリックしてファイルを参照します。  
  
2.  ツールを保存しない場合は、**\[OK\]** をクリックし、**\[外部ツール\]** ダイアログ ボックスの **\[削除\]** をクリックします。
