# &#39;!&#39; の左側に、型パラメーター、クラスまたはインターフェイス型を持つオペランドが必要ですが、このオペランドの型は &#39;&lt;type&gt;&#39; です
ピリオド \(`.`\) の代わりに感嘆符 \(`!`\) を使用するディクショナリ メンバー アクセスは、クラスまたはインターフェイスでのみ有効です。  
  
 **エラー ID:** BC30103  
  
### このエラーを解決するには  
  
-   感嘆符の左側の式を、定義済みのクラスまたはインターフェイス型に評価される式で置き換えます。
