# 例外のトラブルシューティング : Microsoft.VisualStudio.Tools.Applications.Runtime.CannotFindStartupObjectAssemblyException
この例外は、アプリケーション マニフェストに指定されたスタートアップ オブジェクトがメイン ソリューション アセンブリに含まれない場合にスローされます。
