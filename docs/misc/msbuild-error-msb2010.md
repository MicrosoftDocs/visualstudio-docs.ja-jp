# MSBuild エラー MSB2010
**要素 \<{1}\> の子要素 \<{0}\> は有効ではありません。**  
  
 指定された要素は、有効な子ではありません。  
  
### このエラーを解決するには  
  
-   指定した要素のスペルを確認します。  
  
-   プロジェクト ファイルに変更や破損がないことを確認します。  変更または破損がある場合は、プロジェクトを作成するのに使用したバージョンの [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] でプロジェクトをいったん開いて保存し直してから、再び変換してください。
