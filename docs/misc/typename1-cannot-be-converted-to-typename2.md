# &#39;&lt;typename1&gt;&#39; は &#39;&lt;typename2&gt;&#39; に変換できません
2 つの互換性のない型の間で型変換が試行されました。  
  
 **エラー ID:** BC30741  
  
### このエラーを解決するには  
  
-   キャストされている型を確認します。
