# No suitable target for find was specified or another find error occurred.
通常、このエラーが発生するのは、検索スコープを \[現在のドキュメント\] に設定して検索を試みたときに、開いているファイルがなかった場合です。  
  
### このエラーを解決するには  
  
1.  ファイルを開いて再検索するか、検索スコープを変更します。
