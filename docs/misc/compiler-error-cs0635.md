# コンパイラ エラー CS0635
'attribute' : System.Interop.UnmanagedType.CustomMarshaller には名前付き引数 ComType および Marshal が必要です  
  
 マーシャリング形式が **System.Interop.UnmanagedType.CustomMarshaller** の場合、**ComType** および **Marshal** 引数を指定する必要があります。
