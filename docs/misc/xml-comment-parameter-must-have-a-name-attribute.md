# XML コメント パラメーターには &#39;name&#39; 属性がなければなりません
XML コメント パラメーターには 'name' 属性を指定しなければなりません。 XML コメントは無視されます。  
  
 XML タグに必要な `name` 属性がありません。  
  
 **エラー ID:** BC42308  
  
### このエラーを解決するには  
  
-   タグに必要な `name` 属性を指定します。
