# 例外のトラブルシューティング : System.ServiceModel.ActionNotSupportedException
<xref:System.ServiceModel.ActionNotSupportedException> 例外は、通常、呼び出された操作に関連するアクションが、サーバー上の操作のどのアクションにも一致しない場合に、クライアントでスローされます。
