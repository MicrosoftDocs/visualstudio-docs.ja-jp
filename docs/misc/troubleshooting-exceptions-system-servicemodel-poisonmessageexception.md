# 例外のトラブルシューティング : System.ServiceModel.PoisonMessageException
<xref:System.ServiceModel.PoisonMessageException> 例外は、メッセージが "ポイズン" であると見なされる場合にスローされます。 メッセージを配信しようとする試みが繰り返し失敗すると、そのメッセージはポイズンであると見なされます。
