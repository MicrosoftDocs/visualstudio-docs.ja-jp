# &#39;&lt;type3&gt;&#39; が参照型でないため、&#39;&lt;type1&gt;&#39; 型の値を &#39;&lt;type2&gt;&#39; に変換できません。
ステートメントでは、対象となる配列の要素のデータ型が参照型ではない別の配列型に、配列型を変換しようとしています。  
  
 **エラー ID:** BC30333  
  
### このエラーを解決するには  
  
-   両方の配列の要素のデータ型を確認して、エラーの原因を特定します。
