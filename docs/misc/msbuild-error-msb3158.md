# MSBuild エラー MSB3158
**MSB3158: カルチャ '\<カルチャ\>' のリソースが見つかりませんでした。  カルチャ '\<カルチャ\>' を使用します。**  
  
 この警告は、指定した製品が、指定したカルチャを使用するパッケージ マニフェスト ファイル \(package.xml\) を見つけられない場合に生成されます。  
  
### このエラーを解決するには  
  
-   適切なパッケージ マニフェスト ファイルを提供します。
