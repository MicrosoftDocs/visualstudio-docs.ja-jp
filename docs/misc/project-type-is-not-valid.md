# Project type is not valid.
このエラーが発生するのは一般的に、**\[新しいプロジェクトの追加\]** コマンドの引数 *templatename* に入力した値が正しくない場合です。  
  
### このエラーを解決するには  
  
1.  ドキュメントで **\[新しいプロジェクトの追加\]** コマンドの正しい構文を確認し、入力し直します。
