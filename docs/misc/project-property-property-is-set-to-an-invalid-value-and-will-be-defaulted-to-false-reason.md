# Project property &#39;property&#39; is set to an invalid value and will be defaulted to FALSE. &lt;reason&gt;
永続化されたブール型 \(Boolean\) のプロパティ値が、次に示す値のいずれにも当てはまりません。  
  
-   1 または 0  
  
-   On または Off  
  
-   true または false  
  
 このエラーの原因として最も可能性が高いのは、プロジェクト ファイルを手動で編集したことです。  
  
 **このエラーを解決するには**  
  
-   プロジェクト ファイルを再編集し、適切な状態に戻す必要があります。  
  
     影響を受けたプロパティの値は、false に設定されます。
