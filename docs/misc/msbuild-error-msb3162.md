# MSBuild エラー MSB3162
**MSB3162: 項目 '\<パッケージ\>' では、依存する項目 '\<パッケージ\>' が見つかりませんでした。**  
  
 このエラーは、ブートストラップ パッケージが別のブートストラップ パッケージに依存しているにもかかわらず、依存関係にあるブートストラップの必要条件が見つからない場合に発生します。
