# 例外のトラブルシューティング : System.Drawing.Printing.InvalidPrinterException
無効なプリンター設定でプリンターにアクセスしようとすると、<xref:System.Drawing.Printing.InvalidPrinterException> 例外がスローされます。  
  
## 関連するヒント  
 **プリンターが存在することを確認します。**  
 無効なプリンター設定で最も多いのは、存在しないプリンターを参照しているというものです。 詳細については、「<xref:System.Drawing.Printing>」を参照してください。  
  
 **既定のプリンターがインストールされていることを確認します。**  
 プリンターが指定されていない場合は、既定のプリンターがインストールされていることを確認します。 詳細については、「<xref:System.Drawing.Printing.PrintDocument.PrinterSettings%2A>」を参照してください。
