# MSBuild エラー MSB3186
**MSB3186: 生成されたマニフェストのアセンブリ ID をタスク入力パラメーターから生成できません。**  
  
 このエラーは、ビルド処理でアプリケーション マニフェストまたは配置マニフェストのアセンブリ名を推定できない場合に生成されます。  アセンブリ名が明示的に指定されていません。べース マニフェストに ID がなく、エントリ ポイント ID も指定されていません。
