# File &lt;name&gt; size is incorrect.
ファイルに必要なバイト数がありません。このファイルは、切り捨てられたか破損している可能性があります。  
  
### このエラーを解決するには  
  
1.  Visual Studio を閉じてから再起動します。  
  
2.  エラーが継続する場合は、Visual Studio を再インストールしてファイルを修復します。
