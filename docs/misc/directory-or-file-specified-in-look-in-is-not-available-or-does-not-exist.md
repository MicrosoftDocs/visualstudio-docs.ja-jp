# Directory or file specified in Look In is not available or does not exist.
このエラーが発生するのは一般的に、\[参照\] の一覧で指定したパスまたはファイル名が見つからない場合です。  
  
### このエラーを解決するには  
  
1.  パスとファイル名を入力し直します。  
  
     または  
  
2.  \[参照\] の一覧および \[有効な項目\] ボックスを使用して、正しいファイルまたはフォルダーに移動します。
