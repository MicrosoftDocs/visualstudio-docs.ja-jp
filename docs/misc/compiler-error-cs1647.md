# コンパイラ エラー CS1647
式が長すぎるか複雑すぎるため、'code' 付近ではコンパイルできません  
  
 コードを処理するコンパイラでスタック オーバーフローが発生しました。 このエラーを解決するには、コードを簡略化します。 コードが正しい場合は、[製品サポート](http://msdn.microsoft.com/ja-jp/77e75b8b-817d-45bf-9c38-458930d873b4)にお問い合わせください。
