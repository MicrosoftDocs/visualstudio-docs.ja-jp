# &#39;Catch&#39; ステートメントは、[イミディエイト] ウィンドウでは有効ではありません。
`Catch` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30715  
  
### このエラーを解決するには  
  
-   デバッグ コードから `Catch` ステートメントを削除します。
