# リソース コンパイラ エラー RC2119
ダイアログ タイトルに引用符で囲まれた文字列が必要です。  
  
 **DIALOG** ステートメントの CAPTION オプションの `captiontext` フィールドは、二重引用符で囲まれた ASCII 文字列である必要があります。
