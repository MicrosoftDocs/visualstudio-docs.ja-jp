# 例外のトラブルシューティング : System.SystemException
`System` 名前空間内の定義済み例外の基底クラスを定義します。  
  
## コメント  
 このクラスには、システムによって定義される例外と、アプリケーションによって定義される例外とを区別する役割があります。
