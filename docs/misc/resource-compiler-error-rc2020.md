# リソース コンパイラ エラー RC2020
基数の桁数が無効です  
  
 指定した `digit` は、`radix` によって指定された基数に関して正しい桁数ではありません。 桁数または基数が正しくない可能性があります。  
  
 8 進数の場合は 0 ～ 7 の数字、16 進数の場合は 0 ～ 9 の数字または A ～ E までの文字でなければなりません。
