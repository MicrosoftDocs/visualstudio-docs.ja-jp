# 引数 &#39;&lt;argumentname&gt;&#39; は、-1 以上でなければなりません
引数の値が \-1 未満です。  
  
### このエラーを解決するには  
  
1.  引数の値が有効な範囲内にあることを確認します。  
  
2.  引数が計算によって指定される場合は、計算で処理される値を確認します。
