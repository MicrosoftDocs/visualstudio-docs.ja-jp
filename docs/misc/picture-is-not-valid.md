# ピクチャが有効ではありません
フォームまたはコントロールの `Picture` プロパティに、ビットマップ、アイコン、または Windows メタファイル以外のグラフィック形式を割り当てようとしました。  
  
### このエラーを解決するには  
  
1.  `Picture` プロパティに読み込もうとしているファイルが、[!INCLUDE[vbprvb](../code-quality/includes/vbprvb_md.md)] でサポートされている有効なグラフィックス ファイルであることを確認してください。
