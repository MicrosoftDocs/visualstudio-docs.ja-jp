# チュートリアル: VSPackage の配置
Visual Studio 2010 以降、VSPackage を配置するには VSIX テクノロジを使用する必要があります。 VSIX の詳細については、「[Visual Studio 拡張機能を配布](../extensibility/shipping-visual-studio-extensions.md)」を参照してください。
