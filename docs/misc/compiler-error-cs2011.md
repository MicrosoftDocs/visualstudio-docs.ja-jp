# コンパイラ エラー CS2011
応答ファイル 'file' を開けません  
  
 コンパイルで応答ファイルが指定されましたが、コンパイラではそのファイルを見つけて開くことができませんでした。  
  
 応答ファイルの詳細については、「[@ \(応答ファイルの指定\)](/dotnet/csharp/language-reference/compiler-options/response-file-compiler-option)」を参照してください。
