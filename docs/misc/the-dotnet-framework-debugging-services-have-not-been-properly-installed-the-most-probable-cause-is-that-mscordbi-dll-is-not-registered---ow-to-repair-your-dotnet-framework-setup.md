# .NET Framework デバッグ サービスが正しくインストールされていません。 mscordbi.dll が正しく登録されていないことが原因と考えられます。 .NET Framework セットアップの修正方法の詳細については、[ヘルプ] をクリックしてください。
このエラーが発生するのは一般的に、Visual Studio が mscordbi.dll ファイルを見つけられなかった場合です。mscordbi.dll ファイルは、共通言語ランタイムのデバッグ サービスと共にインストールされます。  
  
### このエラーを解決するには  
  
-   コンピューターにインストールされている .NET Framework を修復します。
