# メンバーにアクセスできません。
指定されたメンバーにアクセスできませんでした。  
  
 **エラー ID:** BC30749  
  
### このエラーを解決するには  
  
1.  対象メンバーが定義されていること、およびメンバー名を正しく入力したことを確認します。  
  
2.  モジュールで宣言されたいずれかのメンバーにアクセスしてみます。 宣言されているモジュールがまだ読み込まれていないために、デバッグ環境ではメンバーを特定できないという場合もあります。
