# MSBuild エラー MSB8012
*{0}\(\\{1\\}\)* が *\\{2\\}* の OutputFile プロパティ値 \(*\\{3\\}*\) と一致しません。 これでは、プロジェクトのビルドが正しく行われない可能性があります。 この問題を解決するには、$\(OutDir\)、$\(TargetName\)、および $\(TargetExt\) の各プロパティ値が、%\(\\{4\\}.OutputFile\) で指定されている値と一致することを確認してください。
