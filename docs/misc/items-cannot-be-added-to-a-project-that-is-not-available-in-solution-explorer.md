# ソリューション エクスプローラーで使用できないプロジェクトに項目を追加することはできません。
このエラーが発生するのは一般的に、指定されたプロジェクトがソリューション エクスプローラーで開かれていない場合です。 Visual Studio は、ソリューション エクスプローラーで現在使用できないプロジェクトには項目を追加できません。  
  
### このエラーを解決するには  
  
1.  指定したプロジェクトを現在のソリューションに追加するか、指定したプロジェクトを含むソリューションを開きます。
