# 例外のトラブルシューティング : System.ComponentModel.Win32Exception
この例外は、Win32 エラー コードの場合にスローされます。  
  
## コメント  
 Win32 エラー コードは、表示されるときに数値表記からシステム メッセージに変換されます。<xref:System.ComponentModel.Win32Exception.NativeErrorCode%2A> プロパティを使用すると、エラー コードの数値表記にアクセスできます。
