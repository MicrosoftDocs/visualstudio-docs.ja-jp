# Visual Studio フィードバック ポリシー
フィードバックをお送りいただき、ありがとうございます。  開発者ライティング チームでは、お寄せいただいたフィードバックを基にドキュメントの改善を図っております。  フィードバック内容について検討させていただく際に、電子メールをお送りして確認やご意見をお願いする場合があります。  それ以外の目的でユーザーの皆様の電子メール アドレスを使用することはありません。また、検討作業の完了後、電子メール アドレスは破棄いたします。  Microsoft のプライバシー ポリシーの詳細については、[http:\/\/privacy.microsoft.com\/default.aspx](http://privacy.microsoft.com/default.aspx) を参照してください。
