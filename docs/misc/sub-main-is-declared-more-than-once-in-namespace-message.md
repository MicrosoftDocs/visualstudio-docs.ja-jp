# &#39;Sub Main&#39; が &#39;&lt;namespace&gt;&#39;: &lt;message&gt; で 2 回以上宣言されています
`Sub Main` は、名前空間の内部で 1 回だけ宣言できます。  
  
 **エラー ID:** BC30738  
  
### このエラーを解決するには  
  
-   プロジェクト内に `Sub Main` プロシージャがただ 1 つのみ存在することを確認します。
