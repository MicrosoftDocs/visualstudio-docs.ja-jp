# 例外のトラブルシューティング : System.InvalidProgramException
プログラムに無効な MSIL \(Microsoft intermediate language\) またはメタデータが含まれている場合にスローされる例外。 通常、プログラムを生成したコンパイラにエラーがあることを示します。
