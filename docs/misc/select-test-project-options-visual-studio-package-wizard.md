# テスト プロジェクトのオプションの選択、Visual Studio パッケージ ウィザード
この手順を使用して次の処理を行うことができます。  
  
-   VSPackage の統合テスト プロジェクトを生成します。  
  
-   VSPackage の単体テスト プロジェクトを生成します。  
  
 **統合テスト プロジェクト**  
 統合テスト プロジェクトを作成します。 統合テストは、Visual Studio の実験用インスタンスで実行され、自動化モデルを使用して、Visual Studio でパッケージの統合をテストします。  
  
 **単体テスト プロジェクト**  
 単体テスト プロジェクトを作成します。 単体テストは、パッケージの個別のクラスとメンバーをテストするために使用できる、小規模の MSTest インスタンスです。  
  
 **キャンセル**  
 新しいプロジェクトのコマンドを取り消し、停止します。  
  
 **戻る**  
 前の手順に戻ります。  
  
 **完了**  
 VSPackage を作成します。
