# Command allows at most one argument.
このエラーが発生するのは一般的に、引数を取らないコマンドまたは引数を 1 つだけ取るコマンドに対して、入力した引数が多すぎる場合です。  
  
### このエラーを解決するには  
  
1.  ドキュメントでコマンドの正しい構文を確認して、やり直します。
