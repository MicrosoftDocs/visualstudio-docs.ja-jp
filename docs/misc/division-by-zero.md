# Division by zero.
0 による除算は、数学的に不可能です。  このエラーが発生するのは一般的に、除数として使用する式の値が 0 である場合です。  
  
### このエラーを解決するには  
  
1.  スペル、論理、または大文字小文字の誤りがないか、式を確認します。
