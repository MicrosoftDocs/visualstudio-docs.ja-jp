# Wizard file &lt;name&gt; format is not recognized.
ウィザードを開けません。  このエラーの原因は一般的に、ウィザード ファイルの破損、またはウィザードと Visual Studio のバージョン番号の不一致によります。  
  
### このエラーを解決するには  
  
1.  ウィザード ファイルのバージョン番号が、`VSWIZARD 7.0` のように VSWIZARD の値として示されている Visual Studio のバージョンと同じであることを確認します。  
  
     または  
  
     Visual Studio を再インストールして、破損ファイルを修復します。
