# An invalid cookie was passed to UnregisterExtenderProvider().
既に有効ではないクッキーを指定しました。  
  
### このエラーを解決するには  
  
1.  RegisterExtenderProvider\(\) を呼び出し、同様のクッキー値を取得します。
