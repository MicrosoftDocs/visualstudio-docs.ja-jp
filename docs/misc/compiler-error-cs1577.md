# コンパイラ エラー CS1577
アセンブリの生成に失敗しました \-\- 理由  
  
 C\# コンパイラがソース コードを正常にコンパイルした後、コンパイルのアセンブリ生成部分が失敗しました。 詳細については、ALink ユーティリティのエラーに関するドキュメント \(「[Al.exe ツールのエラーと警告](http://msdn.microsoft.com/ja-jp/7f125d49-0a03-47a6-9ba9-d61a679a7d4b)」\) を参照してください。
