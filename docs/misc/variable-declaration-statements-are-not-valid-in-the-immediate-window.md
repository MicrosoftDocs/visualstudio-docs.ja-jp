# 変数宣言ステートメントは、[イミディエイト] ウィンドウでは有効ではありません
変数宣言は、**\[イミディエイト\]** ウィンドウでは使用できません。  
  
 **エラー ID:** BC30709  
  
### このエラーを解決するには  
  
-   **\[イミディエイト\]** ウィンドウで変数を宣言しません。
