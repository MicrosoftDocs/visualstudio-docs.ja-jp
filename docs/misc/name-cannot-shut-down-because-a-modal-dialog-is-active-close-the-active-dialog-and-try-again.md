# &lt;name&gt; cannot shut down because a modal dialog is active. Close the active dialog and try again.
このエラーが発生するのは一般的に、オペレーティング システムがシャットダウンしようとするときにアプリケーションでモーダル ダイアログが開いているため、アプリケーションを閉じることができない場合です。  
  
### このエラーを解決するには  
  
1.  モーダル ダイアログ ボックスを閉じてから、アプリケーションを閉じるかオペレーティング システムをシャットダウンします。
