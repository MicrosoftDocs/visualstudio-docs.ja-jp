# Type library &lt;name&gt; was not found.
Visual Studio は、タイプ ライブラリの DTE.olb と vsext.tlb がないと実行できません。  このエラーが発生するのは一般的に、これら 2 つのファイルが削除されている場合です。  
  
### このエラーを解決するには  
  
1.  Visual Studio を再インストールします。
