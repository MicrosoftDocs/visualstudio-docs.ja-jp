# プロパティ &#39;&lt;propertyname&gt;&#39; は &#39;WriteOnly&#39; です
`WriteOnly` で宣言されているプロパティから値を取得しようとしました。  
  
 **エラー ID:** BC30524  
  
### このエラーを解決するには  
  
-   プロパティ宣言から `WriteOnly` 指定子を削除します。
