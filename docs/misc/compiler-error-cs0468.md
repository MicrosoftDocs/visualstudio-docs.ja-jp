# コンパイラ エラー CS0468
型 'type1' と型 'type2' の間にあいまいさがあります  
  
 このエラーは、コンパイルされるアセンブリに同じ完全修飾名を持つ 2 つの型がある場合に生成されます。 これは、両方とも追加されたモジュールにあるか、または 1 つが追加されたモジュールにあり、もう 1 つがソースにある場合に発生する可能性があります。
