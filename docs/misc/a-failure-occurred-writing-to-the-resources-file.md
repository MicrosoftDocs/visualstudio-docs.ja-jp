# A failure occurred writing to the resources file
プロジェクト システムによって .resources 出力ファイルを書き込むことができませんでした。  プロジェクト システムは、リソースを準備する過程で、.NET リソース マネージャー用に、拡張子が .resx の XML ファイルをバイナリの .resources ファイルに変換します。  
  
 オペレーティング システム固有の原因がある場合は、このメッセージの最後に追加されます。  .resx ファイルの名前も追加されます。  
  
 このエラーが発生すると、ビルド処理が失敗します。
