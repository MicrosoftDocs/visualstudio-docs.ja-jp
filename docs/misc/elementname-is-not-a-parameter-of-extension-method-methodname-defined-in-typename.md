# &#39;&lt;elementname&gt;&#39; は &#39;&lt;typename&gt;&#39; で定義されている拡張メソッド &#39;&lt;methodname&gt;&#39; のパラメーターではありません
プロシージャ呼び出しで、拡張メソッドによって定義されていない追加の引数が指定されています。  
  
 **エラー ID:** BC36585  
  
### このエラーを解決するには  
  
-   引数リストから余分な引数を削除します。
