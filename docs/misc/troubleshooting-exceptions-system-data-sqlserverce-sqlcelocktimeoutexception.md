# 例外のトラブルシューティング : System.Data.SqlServerCe.SqlCeLockTimeoutException
`System.Data.SqlServerCe.SqlCeLockTimeoutException` 例外は、ロックのタイムアウトに達した場合、およびロックの待機中に SQL Server Compact がタイムアウトになった場合にスローされます。 既定のロック タイムアウト時間は 2000 ミリ秒です。 詳細については、「[SQL Server Compact 3.5 Class Library](http://go.microsoft.com/fwlink/?LinkID=102595)」を参照してください。
