# 例外のトラブルシューティング : System.IdentityModel.Selectors.CardSpaceException
<xref:System.IdentityModel.Selectors.CardSpaceException> 例外は、CardSpace サービス レベルで 1 つ以上の例外が発生した場合にスローされます。 エラーの原因は、イベント ログに格納されます。  
  
## コメント  
 この例外が発生する条件には、ストアや帳簿の読み込み中にエラーが発生した場合などがあります。
