# &#39;&lt;propertyname&gt;&#39; は、同じ名前のメンバーと &lt;type&gt; &#39;&lt;typename&gt;&#39; で競合する &#39;&lt;membername&gt;&#39; を暗黙的に定義しています
型メンバーの名前が、プロパティに対して暗黙的に作成されたメンバーの名前と競合します。 プロパティは、定義されているメンバーと競合する可能性がある複数の暗黙的な変数を暗黙的に作成します。  
  
 **エラー ID:** BC31060  
  
### このエラーを解決するには  
  
-   明示的に宣言されたメンバーの名前を変更して、名前の競合を削除します。
