# Clipboard cannot be opened.
このエラーが発生するのは一般的に、項目の切り取り、コピー、または貼り付けを実行しようとしても、クリップボードにアクセスできない場合です。  
  
### このエラーを解決するには  
  
1.  クリップボードを使用しているほかのプログラムを終了して、やり直します。
