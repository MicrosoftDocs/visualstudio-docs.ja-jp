# 例外のトラブルシューティング : System.Workflow.Runtime.WorkflowOwnershipException
<xref:System.Workflow.Runtime.WorkflowOwnershipException> 例外は、ワークフロー ランタイム エンジンが、現在別のワークフロー ランタイム エンジンのインスタンスによって読み込まれているワークフロー インスタンスを読み込もうとした場合にスローされます。 また、この例外は、ワークフローが読み込みを行っている際に指定されていた所有権のタイムアウト時間が経過した後に、ワークフロー ランタイム エンジンがワークフローを保存しようとした場合にもスローされます。
