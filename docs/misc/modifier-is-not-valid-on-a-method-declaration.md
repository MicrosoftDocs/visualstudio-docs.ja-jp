# &#39;&lt;modifier&gt;&#39; は、メソッド宣言では有効ではありません
`Function` または `Sub` ステートメントに無効なキーワードが含まれています。  
  
 **エラー ID:** BC30242  
  
### このエラーを解決するには  
  
-   `Function` ステートメントまたは `Sub` ステートメントから無効なキーワードを削除します。
