# 例外のトラブルシューティング : System.IO.DirectoryNotFoundException
ファイル パスまたはディレクトリの一部が見つからない場合、<xref:System.IO.DirectoryNotFoundException> 例外がスローされます。  
  
## 関連するヒント  
 **指定した場所にディレクトリが表示されることを確認します。**  
 指定した場所にディレクトリが存在することを確認します。  
  
 **相対パスを使用するとき、現在のディレクトリが正しいことを確認します。**  
 現在のディレクトリが正しくないと、相対パスも正しくなくなります。
