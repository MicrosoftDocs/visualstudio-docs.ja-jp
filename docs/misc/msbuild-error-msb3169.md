# MSBuild エラー MSB3169
**MSB3169: ブートストラップの生成中にエラーが発生しました: '\<エラー\>'**  
  
 このメッセージには、ブートストラップ ビルドでキャッチされないエラーが表示されます。  
  
### このエラーを解決するには  
  
-   MSBuild エラーの原因と解決方法の詳細については、「[追加のリソース](../msbuild/additional-msbuild-resources.md)」を参照してください。
