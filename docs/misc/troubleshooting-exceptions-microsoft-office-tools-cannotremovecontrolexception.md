# 例外のトラブルシューティング : Microsoft.Office.Tools.CannotRemoveControlException
<xref:Microsoft.Office.Tools.CannotRemoveControlException> 例外は、プログラムで追加されたのではないコントロールを、プログラムで削除しようとしたときにスローされます。  
  
## 関連するヒント  
 プログラムで削除できるのは、プログラムで追加したコントロールのみです。  
 -   プログラムで削除できるのは、プログラムで追加したコントロールのみです。 実行時ではなく、設計時にドキュメントに追加したコントロールの場合、削除できません。
