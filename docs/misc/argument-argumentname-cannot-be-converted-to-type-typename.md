# 引数 &#39;&lt;argumentname&gt;&#39; は、型 &#39;&lt;typename&gt;&#39; には変換できません
変換で、引数を正しくない型に変更しようとしています。  
  
### このエラーを解決するには  
  
-   正しい型に変更します。
