# リソース コンパイラ エラー RC2174
ビットマップ ファイルのファイル名が 2.03 形式ではありません  
  
 ビットマップに、バージョン 2.03 より前の形式が使用されています。 ビットマップは、バージョン 3.00 以降の形式を使用して変換または再描画する必要があります。
