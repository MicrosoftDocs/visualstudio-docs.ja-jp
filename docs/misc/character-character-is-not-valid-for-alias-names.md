# Character &lt;character&gt; is not valid for alias names.
このエラーが発生するのは一般的に、エイリアス名の一部として入力された文字が正しくない場合です。  エイリアス名には、空白、\=、\[、\]、\/、\-、" は使用できません。  
  
### このエラーを解決するには  
  
1.  不正な文字を除いてエイリアス名を入力し直します。
