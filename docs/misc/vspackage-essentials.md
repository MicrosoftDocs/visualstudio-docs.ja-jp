# VSPackage の基本事項
VSPackage は、UI 要素、サービス、プロジェクト、エディター、およびデザイナーで Visual Studio を構成および拡張するソフトウェア モジュールです。 VSPackage は、[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] の中心のアーキテクチャ単位です。[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] 自体のほとんどが VSPackage のコレクションとして記述されています。  
  
 [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] の機能拡張アプリケーションを作成するには、まず VSPackage を作成します。 次に、メニュー コマンド、ツール ウィンドウ、ドキュメント ウィンドウ、またはその他のコードを追加して [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] IDE を拡張できます。  
  
 VSPackage を使用するには、Visual Studio SDK をインストールする必要があります。 このチュートリアルに従うには、Visual Studio SDK をインストールする必要があります。 詳細については、「[Visual Studio SDK](../extensibility/visual-studio-sdk.md)」を参照してください。
