# Operating system does not match requirements for object with class ID &lt;number&gt;.
このエラーが発生するのは一般的に、互換性のないオペレーティング システムまたは互換性のないバージョンのオペレーティング システムが検出された場合です。  
  
### このエラーを解決するには  
  
1.  アプリケーションに必要なシステム要件を確認します。  
  
2.  互換性のあるオペレーティング システムのいずれかをインストールしてやり直します。
