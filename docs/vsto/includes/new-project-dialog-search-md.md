> [!NOTE]
> **[新しいプロジェクトの作成]** ダイアログの上部にある [検索] テキスト ボックスに Office 製品の名前を入力して、テンプレートを検索することもできます。 たとえば、「**Excel**」と入力すると、Excel のすべてのプロジェクト テンプレートが表示されます。
また、[プロジェクトの種類] ドロップダウン ボックスで **[Office]** を選択して、Office のテンプレートのみを表示するようにテンプレートの一覧をフィルターすることもできます。  詳細については、「[Visual Studio で新しいプロジェクトを作成する](../../ide/create-new-project.md)」を参照してください。
