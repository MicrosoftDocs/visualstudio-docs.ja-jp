# <a name="understand-coauthoring-and-add-ins"></a>共同作成とアドインについて

共同作成は、クラウド (つまり、OneDrive、OneDrive for Business、または SharePoint Online) でホストされているブックを他のユーザーと同時に編集できるようにする、Excel for Windows デスクトップの機能です。 この機能を追加することで、ユーザーが共同作成している場合でも円滑に動作するようにするために、Excel ソリューションの調整が必要になる場合があります。 [Excel での共同作成](/office/vba/excel/concepts/about-coauthoring-in-excel)に関するページを参照してください。

注: Windows Desktop Excel の共同作成は 2017 年に導入され、現時点では Office365 サブスクライバーが使用できます。 Office 2016 以前の永続的ライセンスを購入したユーザーは、現在、共同作成機能にアクセスすることはできません。 (Windows ストアの Excel Online、Excel for Android、Excel for iOS、および Excel Mobile も、共同作成をサポートしています)。
