# <a name="understand-coauthoring-and-add-ins"></a>共同編集とアドインについて

共同編集は、クラウドでホストされているブック (OneDrive、OneDrive for Business、または SharePoint Online) を他のユーザーと同時に編集できるようにする Excel for Windows desktop の機能です。 この機能を追加することで、ユーザーが共同作業している場合でも、円滑に動作するように Excel ソリューションの調整が必要になる場合があります。 「 [Excel の共同編集につい](/office/vba/excel/concepts/about-coauthoring-in-excel)て」を参照してください。

注: Windows デスクトップ用共同編集 Excel は2017で導入され、現在は Office365 サブスクライバーで使用できます。 Office 2016 以前に永続的ライセンスを購入したユーザーは、現在、共同編集機能にアクセスできません。 (Excel Online、Excel for Android、excel for iOS、および Windows ストアの Excel Mobile は、共同編集もサポートしています)。
