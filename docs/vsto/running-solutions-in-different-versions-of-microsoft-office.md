 次の表は、Microsoft Office のバージョンが、Visual Studio の以前のバージョンを使用して作成されたソリューションを実行し、どのバージョンの[!INCLUDE[vsto_runtime](../vsto/includes/vsto-runtime-md.md)]と .NET Framework が各ソリューションに必要です。
